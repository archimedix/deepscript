# Lebanon Power Map

> Mappa del potere libanese: confessionalismo, dinastie, connessioni internazionali

## Executive Summary

Il Libano rappresenta un caso unico di **potere confessionale istituzionalizzato**: dal 1943 le cariche sono distribuite per religione (Presidente Maronita, PM Sunnita, Speaker Sciita). Questo sistema ha cristallizzato dinastie politiche che controllano il paese attraverso reti familiari, bancarie e connessioni con potenze regionali.

| Metrica | Valore |
|---------|--------|
| **Sistema** | Repubblica confessionale |
| **Confessioni riconosciute** | 18 |
| **Popolazione** | ~5.5M |
| **Diaspora** | ~14M |
| **Crisi bancaria** | $93B depositi bloccati (2019-) |
| **Debito/PIL** | >150% |

## Pattern di Potere

### 1. Confessionalismo come Lock-in

Il sistema garantisce potere perpetuo alle stesse famiglie:
- Nessun censimento dal 1932
- Quote fisse per confessione
- Impossibile riforma senza consenso unanime

### 2. Banche come Estrattori

```
Depositi cittadini → Banche → Prestiti a Stato → Default
                        ↓
                  Profitti privati
                        ↓
                  Perdite socializzate
```

### 3. Proxy delle Potenze Regionali

| Potenza | Proxy/Canale | Confessione target |
|---------|--------------|-------------------|
| Iran | Hezbollah | Sciiti |
| Saudi | Hariri, Future Mvt | Sunniti |
| Francia | Establishment | Maroniti |
| USA | Esercito, AUB | Trasversale |

### 4. Diaspora come Safety Valve

14M libanesi all'estero vs 5.5M in patria:
- Rimesse = % significativa PIL
- Elite emigra, sistema resta
- Billionaires diaspora non cambiano sistema
