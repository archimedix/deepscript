# Corea del Sud: Mappa del Potere

> Il caso studio perfetto di simbiosi stato-conglomerato: come una famiglia controlla il 23% del PIL con il 4% delle azioni

---

## Sintesi

La Corea del Sud rappresenta un **laboratorio unico** per studiare le dinamiche del potere post-1945:

- **Stato sviluppista** che ha creato i chaebol (1961-1979)
- **Democrazia formale** dal 1987, ma **oligarchia economica** de facto
- **Cross-shareholding** che permette controllo con proprieta' minima
- **Revolving door** sistematico governo-conglomerati
- **Ciclo corruzione-impeachment** ricorrente

## Tesi: Il Modello Coreano

### Caratteristiche Distintive

1. **Stato sviluppista**: Governo crea campioni nazionali (1960s)
2. **Cross-shareholding**: Controllo senza proprieta'
3. **Famiglia al centro**: Successione dinastica
4. **Revolving door**: Funzionari → board aziendali
5. **Corruzione sistemica**: Non eccezione ma regola
6. **Democrazia fragile**: 2 impeachment in 8 anni

### Il Paradosso Coreano

> Samsung = 23% del PIL coreano

La Corea e' una **democrazia formale** dove:
- 5 famiglie controllano 50% del mercato azionario
- Il fondo pensione statale (NPS) sostiene i controllanti
- I presidenti vengono impeached per corruzione chaebol
- Gli eredi condannati vengono graziati "per interesse nazionale"

### Lezione per DeepScript

Il caso coreano dimostra che:
- Il potere economico puo' **catturare** lo stato democratico
- Le strutture azionarie possono **mascherare** il controllo reale
- Il **revolving door** e' il meccanismo di trasmissione
- I **fondi pensione pubblici** possono essere strumentalizzati
- La **grazia presidenziale** e' lo strumento di impunita'

## Query Neo4j Utili

```cypher
-- Mappa completa Samsung
MATCH path = (f:Family {id: 'lee-samsung'})<-[:MEMBER_OF]-(p:Person)
OPTIONAL MATCH (p)-[r:AFFILIATED_WITH]->(o:Organization)
RETURN path, r, o

-- Cross-shareholding Samsung
MATCH (o1:Organization)-[r:STAKE_IN]->(o2:Organization)
WHERE o1.id CONTAINS 'samsung' OR o2.id CONTAINS 'samsung'
RETURN o1.id, o2.id, r.share

-- Revolving door governo-Samsung
MATCH (p:Person)-[:AFFILIATED_WITH]->(gov:Organization {id: 'governo-corea'})
MATCH (p)-[:AFFILIATED_WITH]->(sam:Organization)
WHERE sam.id CONTAINS 'samsung'
RETURN p.id, p.note
```
