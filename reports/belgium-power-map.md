# Mappa del Potere in Belgio

> Crocevia d'Europa: capitale UE/NATO, dinastia Bilderberg, birra e finanza

## Executive Summary

Il Belgio e' uno dei paesi piu' influenti al mondo rispetto alle dimensioni, grazie alla sua posizione di capitale de facto dell'Unione Europea e sede NATO. Il sistema di potere belga e' caratterizzato da:

| Caratteristica | Dettaglio |
|----------------|-----------|
| **Modello** | Monarchia federale, hub istituzionale europeo |
| **Casa Reale** | Saxe-Coburg-Gotha (dal 1831), legami pan-europei |
| **Multinazionali** | AB InBev (#1 birra mondiale), Solvay, UCB, Umicore |
| **Holding storiche** | Sofina, GBL, Societe Generale de Belgique (†) |
| **Forum chiave** | Bilderberg (Davignon chairman 13 anni), ERT (co-fondata a Bruxelles) |
| **Infrastruttura critica** | SWIFT (messaggistica bancaria globale), Euroclear |
| **Peculiarita'** | Sede EU/NATO, architetti integrazione europea (Spaak, Davignon, Van Rompuy) |

### Le Cifre del Potere

| Metrica | Valore |
|---------|--------|
| **PIL** | €600B (2024) - 24a economia mondiale |
| **PIL pro capite** | ~€52,000 |
| **Billionaires** | 8 (2024) |
| **Famiglie piu' ricche** | Van Damme/de Spoelberch/de Mevius (AB InBev) €40-50B combinati |
| **Popolazione** | 11.6 milioni |
| **Azienda piu' grande** | AB InBev €180B market cap |
| **Transazioni SWIFT/giorno** | 42 milioni messaggi |

## Pattern di Potere

### 1. Piccolo Paese, Influenza Globale

| Fattore | Meccanismo |
|---------|------------|
| **Posizione** | Crocevia geografico, capitale EU de facto |
| **Neutralita' storica** | Sede istituzioni "neutrali" |
| **Multilinguismo** | FR/NL/DE facilita mediazione |
| **Elite compatta** | Poche famiglie, network denso |

### 2. Revolving Door Belga

```
GOVERNO → COMMISSIONE EU → BUSINESS → FORUM

Davignon: Min.Esteri → Comm.EU → SG Belgique → Bilderberg Chair
Van Rompuy: PM → EU Council → Think Tanks → Trilateral
Maystadt: Min.Finanze → BEI → Academia
```
