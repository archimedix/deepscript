## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1977 |
| **Fondatori** | British Coal, Railway Industry, Barclays Bank pension funds |
| **Nome originale** | Coal Investment Nominees for Venture Capital |
| **Contesto** | Veicolo per investimenti delle pensioni pubbliche britanniche |

### Timeline Storica

| Anno | Evento |

| **2024** | Acquisizione Grant Thornton UK per 1.3 miliardi GBP |
| **2024** | Exit Arcaplanet a Fressnapf |
| **2025** | Rajagopalan e Leslie lasciano, Alexandra Hess President |
| **2025** | Exit STADA a CapVest (largest European LBO 2025) |
