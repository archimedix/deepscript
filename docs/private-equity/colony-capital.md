## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1991, Los Angeles |
| **Fondatore** | [Tom Barrack](../persons/tom-barrack.md) |
| **Investitori iniziali** | Bass Brothers, GE Capital |
| **Contesto** | Crisi immobiliare USA inizio anni '90; Barrack compra asset distressed |
| **Modello** | Acquisto di asset immobiliari sottovalutati, ristrutturazione, rivendita |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1991** | Fondazione Colony Capital da Tom Barrack |
| **1997** | [Sebastien Bazin](../persons/sebastien-bazin.md) diventa CEO Europe |
| **2005** | Acquisizione quota Accor (~€1 mld) |
| **2006** | Acquisizione quota PSG (con Butler Capital) |
| **2007** | Acquisizione 9.1% Carrefour (con gruppo Arnault) |
| **2010** | Acquisto Miramax da Disney (con QIA, $660M) |
| **2011** | Vendita PSG a QSI (mediata da Sarkozy) |
| **2013** | Bazin lascia per diventare CEO Accor; uscita da Accor |
| **2017** | Uscita da Carrefour; Colony diventa REIT quotato |
| **2019** | Acquisizione Digital Bridge Holdings ($325M); Marc Ganzi CEO-elect |
| **2020** | Barrack lascia ruolo di CEO; Ganzi subentra |
| **2021** | Rinominata DigitalBridge; Barrack arrestato per FARA (poi assolto) |
| **2021** | Barrack si dimette dal board |
| **2025** | SoftBank annuncia acquisizione di DigitalBridge per ~$4 mld |

## La Connessione Qatar

Colony Capital ha avuto una relazione profonda con il Qatar:

1. **PSG**: Co-proprietario, poi vendita a QSI (2011)
2. **Miramax**: Co-acquisto con QIA da Disney (2010)
3. **Barrack**: Accusato (e assolto) di agire come agente ARE non registrato

```
         COLONY CAPITAL
               │
    ┌──────────┼──────────┐
    │          │          │
    ▼          ▼          ▼
   PSG     MIRAMAX    BARRACK
    │          │       (FARA)
    │          │          │
    ▼          ▼          ▼
   QSI        QIA       ARE
 (QATAR)   (QATAR)    (Emirati)
```

Il pattern: Colony come ponte tra asset occidentali e capitali del Golfo.
