## Glossario

| Termine | Definizione |
|---------|-------------|
| **Batch** | Coorte di startup che partecipa al programma (es. W24 = Winter 2024) |
| **Demo Day** | Evento finale dove le startup presentano agli investitori |
| **YGL** | Young Global Leader (programma WEF, molti alumni YC) |
| **Unicorn** | Startup con valutazione > $1 miliardo |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Marzo 2005 |
| **Fondatori** | Paul Graham, Jessica Livingston, Robert Morris, Trevor Blackwell |
| **Capitale iniziale** | $200,000 |
| **Contesto** | Rivoluzionare il VC: investimenti piu' piccoli, su hacker non "suits", fondatori giovani |
| **Nome originale** | Cambridge Seed (cambiato subito in Y Combinator) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2005** | Fondazione, prima batch: 8 aziende su 237 (3% acceptance) |
| **2005** | Reddit nella prima batch |
| **2007** | Dropbox (Drew Houston, Arash Ferdowsi) |
| **2009** | Airbnb (Brian Chesky, Joe Gebbia, Nathan Blecharczyk) |
| **2010** | Stripe (Patrick e John Collison) |
| **2011** | Sam Altman entra come Partner |
| **2012** | Coinbase (Brian Armstrong) |
| **2014** | Paul Graham si ritira, Sam Altman diventa President |
| **2019** | Sam Altman lascia per OpenAI |
| **2021** | Continuity Fund raggiunge $2.75B |
| **2023** | Garry Tan diventa CEO, sede spostata a San Francisco |
| **2024** | Michael Seibel lascia come Managing Director |
| **2025** | 4 batch annuali (da 2), portfolio $600B |
