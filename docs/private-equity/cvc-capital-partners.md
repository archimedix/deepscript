## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1981 come Citicorp Venture Capital |
| **Spinoff** | 1993 da Citibank |
| **Fondatori** | Michael Smith, Steve Koltes, Donald Mackenzie, Rolly van Rappard, Hardy McLain, Iain Parham |
| **Origine** | Divisione venture capital di Citibank |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1981** | Fondazione Citicorp Venture Capital (divisione Citibank) |
| **1988** | Donald Mackenzie entra in CVC |
| **1993** | Spinoff da Citibank, nasce CVC Capital Partners |
| **1996** | Rob Lucas entra in CVC |
| **2006 Mar** | Acquisto majority Formula 1 (63.4%) |
| **2013 Gen** | Michael Smith si ritira, Koltes/Mackenzie/van Rappard co-chairmen |
| **2016 Set** | Vendita Formula 1 a Liberty Media ($8B, 4x return) |
| **2021** | Investimento Six Nations Rugby (£365M, 14.3%) |
| **2021** | Investimento La Liga ($3.2B, 10.95%) |
| **2022** | Steve Koltes si ritira |
| **2023** | Fund VIII raccoglie €26B (piu' grande PE fund mai raccolto) |
| **2024 Feb** | Donald Mackenzie si ritira da ruolo attivo |
| **2024 Apr** | IPO Euronext Amsterdam (€2.3B raised) |
| **2024** | Rob Lucas diventa CEO |
| **2024 Giu** | #4 ranking mondiale PEI 300 |
| **2025** | Lancio Global Sport Group ($14B) |
| **2025** | AUM €200B |

## L'Impero Sportivo

### Formula 1 (2006-2016)

| Aspetto | Dettaglio |
|---------|-----------|
| Acquisto | Marzo 2006 |
| Quota | 63.4% (majority) |
| Vendita | Settembre 2016 a Liberty Media |
| Prezzo vendita | $8B |
| Return | ~4x investimento |
| Partner | Bernie Ecclestone |

**Pattern**: CVC ha trasformato F1 da sport a media property globale, monetizzando diritti TV e sponsorizzazioni.
