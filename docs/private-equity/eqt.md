### La Genesi (1993)

Nel 1993, durante una cena a Gamla Stan (centro storico di Stoccolma), Conni Jonsson e Claes Dahlbäck (CEO Investor AB) concepirono l'idea di creare un veicolo PE che applicasse la filosofia Wallenberg su scala globale.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1993** | Cena Jonsson-Dahlbäck, idea di EQT |
| **1994** | Fondazione EQT con backing Investor AB |
| **2013** | Partners possiedono 81%, Investor AB 19% |
| **2014** | Thomas von Koch CEO, Jonsson Chairman |
| **2019 Set** | IPO Stockholm (valuation €7B) |
| **2019** | Christian Sinding diventa CEO |
| **2022 Mar** | Annuncio acquisizione Baring PE Asia (€6.8B) |
| **2022 Ott** | Completata fusione BPEA |
| **2024** | EQT X raccoglie €22B (piu' grande fondo PE 2024) |
| **2024** | BPEA rinominata EQT Private Capital Asia |
| **2025 Feb** | Per Franzén nominato nuovo CEO |
| **2025 Mag** | Franzén diventa CEO, Sinding Chair EQT Council |
| **2025 Ott** | Jean Eric Salata nominato futuro Chairman (dal 2026) |
| **2025** | AUM €266B, #3 PE mondiale |

## La Connessione Wallenberg

### Chi Sono i Wallenberg

La famiglia Wallenberg e' la piu' potente dinastia industriale-finanziaria scandinava, paragonabile ai Rothschild o Rockefeller:

| Generazione | Figure chiave |
|-------------|---------------|
| 1a | Andre Oscar Wallenberg (1816-1886) - fondatore SEB |
| 2a | Knut, Marcus Sr |
| 3a | Jacob (1892), Marcus Jr |
| 4a | Raoul (eroe Olocausto), Peter Sr |
| 5a | Peter Jr, Jacob, Marcus (attuali) |
| 6a | In crescita |

### L'Impero Wallenberg (via Investor AB)

| Partecipazione | Settore |
|----------------|---------|
| ABB | Engineering |
| Atlas Copco | Industrial |
| Ericsson | Telecom |
| Electrolux | Consumer |
| SEB | Banking |
| AstraZeneca | Pharma |
| **EQT** | Private Equity |

### Pattern: Dinastia → PE

```
WALLENBERG FAMILY (6 generazioni)
        │
        ▼
    INVESTOR AB (holding)
        │
        ├── Controllo industrie svedesi
        │
        └── Fonda EQT (1994)
                │
                ▼
        ESPANSIONE GLOBALE
        €266B AUM, #3 mondiale
```

**Implicazione**: EQT e' il braccio PE della dinastia Wallenberg per espandere l'influenza oltre la Scandinavia.
