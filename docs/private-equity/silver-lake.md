## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1999, Silicon Valley |
| **Fondatori** | Jim Davidson, David Roux, Roger McNamee, Glenn Hutchins |
| **Contesto** | Boom tech anni '90, gap nel PE per tech matura |
| **Obiettivo** | PE per tech companies mature (vs VC per startup) |

### Timeline

| Anno | Evento |
|------|--------|
| **1999** | Fondazione |
| **2009** | Acquisizione Skype (65%, $1.9B) |
| **2011** | Exit Skype a Microsoft ($8.5B) |
| **2012** | Primo investimento Endeavor |
| **2013** | Dell take-private ($24.4B) |
| **2019** | Investimento CFG ($500M, 10%) |
| **2019** | Egon Durban e Greg Mondre Co-CEOs |
| **2020** | Investimento Airbnb ($1B) |
| **2022** | Aumento stake CFG (~18%) |
| **2024** | Annuncio take-private Endeavor |
| **2025** | Completamento acquisizione Endeavor ($25B) |

## Modello di Business

### Tech PE Approach

```
IDENTIFY
Mature tech company
undervalued or
needing capital
       │
       ▼
ACQUIRE
Take-private or
significant stake
       │
       ▼
OPERATE
Active board,
management support,
operational improvements
       │
       ▼
SCALE
M&A, international
expansion, product
development
       │
       ▼
EXIT
IPO, strategic sale,
or hold (Endeavor model)
```

### Differenziazione

| Aspetto | Silver Lake | Traditional PE |
|---------|-------------|----------------|
| **Focus** | Tech & tech-enabled | Generalist |
| **Holding period** | Longer (Endeavor: 13+ years) | 3-5 years |
| **Approach** | "Never sold a share" | Exit-focused |
| **Value add** | Tech expertise, network | Financial engineering |
