## Strategia di Investimento

### Stile

| Aspetto | Dettaglio |
|---------|-----------|
| **Approccio** | Value investing + activism |
| **Concentrazione** | Poche posizioni, grandi stake |
| **Confronto** | Paragonato a Warren Buffett (2004) |
| **Differenza** | Activism piu' aggressivo |

### Pattern Sears

| Elemento | Azione |
|----------|--------|
| **Acquisizione** | Controllo via stock accumulation |
| **Management** | Lampert diventa CEO |
| **Asset stripping** | Spin-off di asset di valore |
| **Self-dealing** | ESL mantiene stake negli spin-off |
| **Esito** | Bankruptcy, ESL compra i resti |

## Pattern Rilevante

ESL Investments rappresenta il modello **hedge fund activism**:

1. **Accumulo** → grande stake in azienda undervalued
2. **Controllo** → fondatore diventa CEO
3. **Asset stripping** → spin-off di divisioni di valore
4. **Self-dealing** → hedge fund mantiene stake negli spin-off
5. **Esito** → azienda madre fallisce, asset salvati

Diverso dal PE tradizionale (debt loading), ma stesso risultato: **valore estratto, azienda distrutta**.
