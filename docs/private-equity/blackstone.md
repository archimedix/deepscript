| Dato | Dettaglio |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1985** | Fondazione con $400K da Schwarzman e Peterson (ex-Lehman, ex-Commerce Secretary) |
| **1987** | Primo fondo PE ($850M) |
| **1991** | Lancio divisione Real Estate |
| **2007** | IPO ($4.75B raised, prima PE firm quotata NYSE) |
| **2007** | Acquisizione Hilton Hotels ($26B) - diventa il deal PE piu' profittevole della storia |
| **2018** | Jon Gray nominato President & COO |
| **2018** | Peter Peterson si ritira (muore stesso anno) |
| **2023** | Primo asset manager alternativo a raggiungere $1T AUM |
| **2023** | Ingresso S&P 500 |
| **2024** | Revenue $13B, net income $5.4B |
| **2025** | AUM raggiunge $1.2T |
