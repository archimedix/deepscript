## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1 gennaio 2010 |
| **Fondatori** | Crédit Agricole + Société Générale |
| **Contesto** | Consolidamento asset management francese post-crisi 2008 |
| **Obiettivo** | Creare campione europeo dell'asset management |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2010** | Nascita da fusione CAAM + SGAM, Yves Perrier CEO |
| **2015** | IPO su Euronext Paris (più grande IPO francese da anni) |
| **2017** | Acquisizione Pioneer Investments (Unicredit) per €3.5bn |
| **2020** | Acquisizione Sabadell AM (€430m) |
| **2021** | Acquisizione Lyxor (Société Générale) per €825m |
| **2021** | Valérie Baudson succede a Perrier come CEO |
| **2021** | Superata soglia €2 trillion AUM |
| **2023** | Philippe Brassac diventa Chairman |
| **2025** | Olivier Gavalda diventa Chairman |
| **2025** | Acquisizione 9.9% di ICG plc |
