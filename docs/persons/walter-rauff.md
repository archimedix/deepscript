## I Furgoni della Morte (Gaswagen)

### L'Invenzione

| Aspetto | Dettaglio |
|---------|-----------|
| **Anno** | 1941 |
| **Istituto** | Criminal Technical Institute, RSHA |
| **Ruolo Rauff** | Capo sviluppo e implementazione |
| **Meccanismo** | Gas di scarico del motore incanalato nell'interno |

### Specifiche Tecniche

| Aspetto | Dettaglio |
|---------|-----------|
| **Tipo** | Camion con cassone chiuso |
| **Dimensioni** | 6 metri lunghezza x 1,7 metri altezza |
| **Capienza** | 25-60 persone |
| **Metodo** | Soffocamento da monossido di carbonio |
| **Primo test** | Sachsenhausen, autunno 1941 |

### Dispiegamento

| Numero | Ubicazione |
|--------|------------|
| **20+** | Furgoni sotto comando Rauff |
| - | Fronte orientale (Einsatzgruppen) |
| - | Serbia |
| - | Chelmno (primo campo di sterminio) |

### Vittime

| Stima | Fonte |
|-------|-------|
| **100.000** | Stime storiche |
| - | Ebrei, disabili, prigionieri politici |

### Documento Incriminante

Rauff scrisse memo tecnici sui "miglioramenti" ai furgoni, incluso come rendere piu' "efficiente" il processo di uccisione e facilitare la rimozione dei cadaveri.
