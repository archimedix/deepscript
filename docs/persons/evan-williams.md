# Evan Williams (1972)

> Architetto del web sociale: ha creato Blogger, co-fondato Twitter, fondato Medium — tre piattaforme che hanno ridefinito la comunicazione online

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **~1994** | Co-fondatore | Azienda con il padre (materiale didattico sul WWW) |
| **1994-1999** | Web developer | Varie aziende tech |
| **1999-2003** | Co-fondatore, CEO | Pyra Labs / Blogger |
| **2003-2004** | Employee | [Google](../company/google.md) (via acquisizione Blogger) |
| **2005-2006** | Co-fondatore | Odeo (podcasting) |
| **2006-2007** | Co-fondatore | Obvious Corporation |
| **2006-2019** | Co-fondatore, CEO, Board | [Twitter](../company/twitter.md) |
| **2012-2022** | Fondatore, CEO | [Medium](../media/medium.md) |
| **2014-oggi** | Co-fondatore, Partner | [Obvious Ventures](../private-equity/obvious-ventures.md) |
| **2022-oggi** | Chairman | [Medium](../media/medium.md) |
| **2024-oggi** | Co-fondatore | Mozi (social network privato) |

### Da Blogger a Twitter: il pattern Williams

| Anno | Evento |
|------|--------|
| **1999** | Fonda Pyra Labs con Meg Hourihan per project management software |
| **2000** | Una feature di note-taking diventa Blogger — nasce il blogging di massa |
| **2001** | Crisi: Hourihan e dipendenti lasciano, Williams mantiene Blogger da solo |
| **2003** | Google acquisisce Pyra Labs/Blogger |
| **2004** | Lascia Google |
| **2005** | Fonda Odeo (podcasting), ma Apple lancia podcast su iTunes |
| **2006** | All'interno di Odeo nasce Twitter come side project |
| **2006** | Fonda Obvious Corporation con Biz Stone, acquisisce le proprieta' di Odeo |
| **2007** | Twitter viene scorporata come azienda indipendente |
| **2008** | Williams diventa CEO di Twitter |
| **2010** | Lascia la carica di CEO (sostituito da Dick Costolo) |
| **2012** | Fonda Medium |
| **2019** | Lascia il board di Twitter |

## Potere Esercitato

### Architettura delle piattaforme

| Aspetto | Dettaglio |
|---------|-----------|
| **Blogger** | Democratizzazione della pubblicazione online, conia il termine "blogger" |
| **Twitter** | Comunicazione in tempo reale, impatto su politica e giornalismo globale |
| **Medium** | Alternativa al giornalismo tradizionale, modello ad-free su abbonamento |

### Venture Capital

| Aspetto | Dettaglio |
|---------|-----------|
| **Obvious Ventures** | VC "world-positive", investimenti in Beyond Meat, Blue Bottle Coffee, Nest |
| **Beyond Meat** | 9% alla IPO (maggio 2019), investimento iconico nel settore plant-based |
| **Exit notabili** | Branch (→ Facebook), Nest (→ Google), Blue Bottle (→ Nestle) |
