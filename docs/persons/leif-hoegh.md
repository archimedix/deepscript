## Rilevanza per il Progetto

Leif Høegh rappresenta il **potere marittimo scandinavo** nella rete Bilderberg:

1. **Founding member Bilderberg**: Presente dalla prima riunione, Steering Committee per 20 anni
2. **Continuita' dinastica**: Famiglia mantiene presenza in Bilderberg attraverso generazioni
3. **Ponte business-politica**: Uso sistematico dei forum per costruire network internazionali
4. **Resistenza WWII**: Credenziali anti-naziste consolidate durante l'occupazione
