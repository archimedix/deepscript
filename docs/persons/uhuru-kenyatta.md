# Uhuru Kenyatta (1961)

> Presidente del Kenya 2013-2022, figlio del padre fondatore Jomo Kenyatta, incriminato ICC per violenze post-elettorali 2007, esposto nei Pandora Papers

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 26 ottobre 1961, Nairobi, Kenya |
| **Padre** | Jomo Kenyatta, primo presidente del Kenya (1963-1978) |
| **Madre** | Mama Ngina Kenyatta |
| **Fratello** | Muhoho Kenyatta (gestisce interessi familiari) |
| **Patrimonio familiare** | ~$500M+ (terreni, banche, media, turismo, agricoltura) |
| **Residenza** | Nairobi |

## Controversie

| Tema | Periodo | Dettaglio |
|------|---------|-----------|
| **ICC - Crimini contro l'umanita'** | 2011-2014 | 5 capi d'accusa per violenze post-elettorali 2007-08 (1000+ morti, 600.000 sfollati): omicidio, deportazione, stupro, persecuzione. Accuse ritirate per intimidazione testimoni e mancata cooperazione del governo keniota |
| **Pandora Papers** | 2021 | Famiglia Kenyatta collegata a 10+ societa' offshore (Panama, BVI, Seychelles). Fratello Muhoho con portfolio offshore da $31.6M. Nessuna prova di sottrazione fondi statali |
| **Concentrazione terre** | continuativa | Famiglia possiede 500.000+ acri di terra, gran parte acquisita in epoca post-coloniale |
| **Elezioni 2017** | 2017 | Prima elezione annullata dalla Corte Suprema per irregolarita', rieletto nel secondo turno boicottato dall'opposizione |

## Timeline

| Anno | Evento |
|------|--------|
| **1961** | Nasce a Nairobi, figlio del leader indipendentista Jomo Kenyatta |
| **1963** | Il padre diventa primo presidente del Kenya |
| **1978** | Morte del padre Jomo Kenyatta |
| **1985** | Laurea ad Amherst College, torna in Kenya |
| **2001** | Prima candidatura presidenziale (sconfitto da Kibaki nel 2002) |
| **2007-08** | Violenze post-elettorali in Kenya: 1000+ morti |
| **2008** | Nominato Vice Primo Ministro nel governo di coalizione |
| **2011** | Incriminato dalla ICC per crimini contro l'umanita' |
| **2013** | Eletto Presidente del Kenya nonostante le accuse ICC pendenti |
| **2014** | ICC ritira le accuse per mancanza di prove e intimidazione testimoni |
