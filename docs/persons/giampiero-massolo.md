## Pattern: Il Diplomatico-Spione-Chairman

Massolo rappresenta un profilo unico di "connettore di sistema":

| Caratteristica | Dettaglio |
|----------------|-----------|
| **Carriera diplomatica** | 34 anni nella Farnesina, fino a Segretario Generale |
| **Intelligence** | 4 anni come Direttore DIS (capo coordinamento intelligence) |
| **Aziende strategiche** | Chairman di due campioni nazionali: Fincantieri, Mundys |
| **Think tank** | ISPI (principale think tank italiano di politica estera) |
| **Forum** | Trilateral, Bilderberg, Atlantic Council, Aspen, WEF |

### Il Percorso

```
DIPLOMAZIA → INTELLIGENCE → INDUSTRIA STRATEGICA → INFRASTRUTTURE
Farnesina      DIS           Fincantieri           Mundys
(1978-2012)  (2012-2016)     (2016-2022)         (2022-)
```
