# Reinhard Gehlen (1902-1979)

> "Hitler's Super Spy": generale Wehrmacht che divenne il primo capo dell'intelligence della Germania Ovest, creando il BND su fondamenta naziste con finanziamento CIA

| **Nascita** | 3 aprile 1902, Erfurt, Germania |
| **Morte** | 8 giugno 1979, Berg, Baviera |
| **Nazionalita'** | Germania |
| **Grado militare** | Generalmajor (Wehrmacht) |
| **Soprannome** | "Hitler's Super Spy" |

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1920-1935** | Ufficiale | Reichswehr |
| **1935-1940** | Staff officer | Wehrmacht General Staff |
| **1940** | Liaison officer | Field Marshal von Brauchitsch |
| **1942-1945** | Chief | Fremde Heere Ost (FHO) |
| **1945** | POW | US Army |
| **1946-1956** | Director | [Gehlen Organization](../agency/gehlen-organization.md) |
| **1956-1968** | President | [BND](../agency/bnd.md) |

### Wehrmacht: Fremde Heere Ost (1942-1945)

| Aspetto | Dettaglio |
|---------|-----------|
| **Ruolo** | Capo intelligence militare fronte orientale |
| **Funzione** | Analisi forze armate sovietiche |
| **Subordinati** | Centinaia di ufficiali intelligence |
| **Archivi** | Microfilm su URSS sepolti prima della resa |

### La Resa e l'Offerta (1945)

| Data | Evento |
|------|--------|
| **22 mag 1945** | Gehlen si arrende al US Counter Intelligence Corps (CIC) |
| **Lug 1945** | Offre archivi FHO e servizi agli americani |
| **1945-46** | Negoziati con Gen. Edwin Sibert (G-2) |
| **Lug 1946** | Rilasciato dalla prigionia USA |

Gehlen offri' agli americani:
- **Archivi microfilmati** sull'Unione Sovietica
- **Rete di agenti** nel blocco orientale
- **Expertise** sull'Armata Rossa

### Operation Rusty e Gehlen Organization (1946-1956)

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome in codice** | Operation Rusty |
| **Fusione** | Project KEYSTONE (Abwehr) + Project BOLERO (FHO) |
| **Copertura** | "South German Industrial Development Organization" |
| **Sede** | Pullach, Baviera |
| **Finanziamento** | CIA (budget segreto) |
| **Personale iniziale** | ~350 agenti |
| **Espansione** | 4.000+ agenti (V-men) |
| **Output** | 70%+ dell'intelligence USA sull'URSS |

### BND: Bundesnachrichtendienst (1956-1968)

|---------|-----------|
| **Transizione** | 1 aprile 1956 |
| **Ruolo Gehlen** | Primo Presidente |
| **Durata** | 12 anni |
