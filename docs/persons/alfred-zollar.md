# Alfred W. Zollar (~1955)

> Veterano IBM (34 anni), poi private equity e multi-board director. Siede nei board di IBM, Nasdaq e BNY Mellon simultaneamente — connettore tech-finanza. Tra i piu' influenti dirigenti afroamericani in corporate America.

### Triplo Board: Tech + Finanza + Infrastruttura

Zollar siede simultaneamente nei board di:
- **IBM** — tech/AI
- **Nasdaq** — mercati finanziari

Pattern classico di connettore cross-settoriale.
