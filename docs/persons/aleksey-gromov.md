# Aleksey Gromov (1960-)

> First Deputy Chief of Staff di Putin, architetto del controllo mediatico russo, "zar della televisione", co-creatore di RT

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 1960, Zagorsk (ora Sergiev Posad), vicino Mosca |
| **Formazione** | Moscow State University, Storia (Slavi meridionali e occidentali) |
| **Lingue** | Russo, ceco, slovacco, inglese |
| **Moglie** | Sposato |
| **Figli** | Due maschi: Aleksey Jr., Danila |

### Il Sistema del Controllo Mediatico

| Elemento | Dettaglio |
|----------|-----------|

| **Funzione** | Dettare linea editoriale, cosa mostrare e non mostrare |
| **Esclusivita'** | "Solo Gromov puo' chiamare i canali TV e dare istruzioni" |

### Controllo Televisione Russa

| Aspetto | Dettaglio |
|---------|-----------|

| **Metodo** | Riunioni settimanali con capi TV |
| **Portata** | Tutti i canali federali |
| **Esclusivita'** | Unico funzionario autorizzato a dare ordini ai media |

### Creazione RT (2005)

| Aspetto | Dettaglio |
|---------|-----------|
| **Co-creatori** | Gromov + Mikhail Lesin |
| **Scelta Simonyan** | Gromov insistette per lei come editor-in-chief |
| **Obiettivo** | Proiezione narrativa russa internazionale |
