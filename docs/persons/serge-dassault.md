## Pattern: Concentrazione di Potere

Serge Dassault rappresenta il caso estremo di concentrazione:
1. **Industria difesa** → dipendenza dello Stato dai suoi prodotti
2. **Media** → influenza sull'opinione pubblica
3. **Politica** → voto diretto sulle leggi + immunità
4. **Locale** → controllo clientelare del territorio

Le condanne multiple (Agusta, frode fiscale, corruzione elettorale) mostrano come questo sistema operasse anche illegalmente.
