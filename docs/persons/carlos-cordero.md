### Revolving door legale-diplomatico

| Aspetto | Dettaglio |
|---------|-----------|
| **Pattern** | Studio legale offshore → vice ministro esteri → ambasciatore missioni speciali → ritorno allo studio |
| **Ruolo chiave** | Difesa del sistema dei servizi finanziari internazionali di Panama sul piano diplomatico |
| **Significato** | L'avvocato offshore diventa il difensore istituzionale del modello offshore panamense |
