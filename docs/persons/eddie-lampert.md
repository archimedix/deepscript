## Sears/Kmart

### Acquisizione

| Anno | Evento |
|------|--------|
| **2003** | ESL acquisisce Kmart (post-bankruptcy) |
| **2005** | Kmart acquisisce Sears per $11B |
| **2005** | Lampert diventa Chairman Sears Holdings |

### Strategia

| Elemento | Azione |
|----------|--------|
| **Asset stripping** | Spin-off di asset di valore |
| **Underinvestment** | Negozi trascurati |
| **Financial engineering** | Dividendi, buyback |
| **Self-dealing** | ESL azionista degli spin-off |

## Pattern Rilevante

Eddie Lampert rappresenta il **hedge fund manager come corporate raider**:

1. **Pedigree elite** → Yale (Skull and Bones), Goldman Sachs
2. **Mentor power** → Robert Rubin (futuro Treasury Secretary)
3. **Hedge fund** → ESL, returns 25%
4. **Activist investing** → Kmart, poi Sears
5. **Asset stripping** → Spin-off controllati da ESL
6. **Outcome** → Sears distrutta, Lampert $2B net worth
