### Lotus Notes (1984-1997)

Creazione del primo groupware commerciale di successo:
- **1984**: Fonda Iris Associates con finanziamento $1.3M da Mitch Kapor (Lotus)
- **1989**: Release Lotus Notes - primo sistema integrato email/database/workflow
- **1994**: Iris Associates acquisita da Lotus
- **1995**: Lotus acquisita da IBM
- IBM porta Lotus Notes a 125 milioni di utenti worldwide

### Groove Networks (1997-2005)

- Fondata per software di collaborazione small-team
- Post 9/11: niche critica per connettere agenzie governative USA
- Iraq war e tsunami 2004: necessita' comunicazione decentralizzata
- **2005**: Acquisita da Microsoft, integrata in SharePoint

### Microsoft (2005-2010)

| Anno | Evento |
|------|--------|
| **2005** | Entra come CTO attraverso acquisizione Groove |
| **2006** | Succede a Bill Gates come Chief Software Architect |
| **2008** | Annuncia Microsoft Azure (progetto "Red Dog") |
| **2009** | Crea FUSE Labs (Future Social Experiences) |
| **2010** | Lascia Microsoft |

Ozzie e [Craig Mundie](craig-mundie.md) divisero i compiti di Bill Gates nel 2006:
- Ozzie: direzione tecnica
- Mundie: strategia e policy

## Innovazioni Chiave

### Lotus Notes (1989)

| Aspetto | Impatto |
|---------|---------|
| **Primo groupware commerciale** | Email + database + workflow integrati |
| **Collaborazione asincrona** | Team distribuiti geograficamente |
| **Replicazione** | Sincronizzazione offline/online |
| **Adozione** | 125M utenti (IBM) |

### Microsoft Azure (2008)

Annunciato da Ozzie, Azure diventa il pilastro cloud di Microsoft:
- Progetto originale "Red Dog"
- Lead: Dave Cutler, Amitabh Srivastava
- Oggi: secondo cloud provider mondiale dopo AWS
