# Roy Jenkins (1920-2003)

|---------|-----------|

| **Titolo** | Baron Jenkins of Hillhead |
| **Padre** | Arthur Jenkins, sindacalista minatori, MP Labour |
| **Partiti** | Labour (fino 1981), SDP (1981-1988), Liberal Democrats |

## Potere Esercitato

### Chancellor of the Exchequer (1967-1970)

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Post-svalutazione sterlina (nov 1967) |
| **Strategia** | Politica fiscale restrittiva |
| **Budget 1968** | Aumento tasse £923 milioni (record) |
| **Risultato** | Bilancia dei pagamenti in attivo |
| **Soprannome** | "Surplus Jenkins" |

### Commissione Europea (1977-1981)

| Aspetto | Dettaglio |
|---------|-----------|
| **Nomina** | Primo britannico presidente |
| **Iniziativa chiave** | Sistema Monetario Europeo (SME) |
| **Proposta** | Unione monetaria europea |
| **Risultato** | SME lanciato 1979, precursore dell'Euro |
| **Eredità** | Pose le basi per la moneta unica |

### La Dimbleby Lecture (1979)

| Aspetto | Dettaglio |
|---------|-----------|
| **Titolo** | "Home Thoughts from Abroad" |
| **Data** | Novembre 1979 |
| **Contenuto** | Critica bipartitismo, proposta centro |
| **Conseguenza** | Seme del Social Democratic Party |

### Fondazione SDP (1981)

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 26 marzo 1981 |
| **"Gang of Four"** | Jenkins, Williams, Owen, Rodgers |
| **Causa** | Spostamento a sinistra del Labour |
| **Elezioni 1983** | 25% voti, solo 6 seggi |
| **Fusione** | 1988 con Liberals → Liberal Democrats |
