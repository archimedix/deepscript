## Potere Esercitato

### Hedge Fund - Millennium Management

| Aspetto | Dettaglio |
|---------|-----------|
| **AUM** | $79 miliardi (settembre 2025) |
| **Dipendenti** | 6,300+ |
| **Uffici** | 18 primary offices (USA, Europa, Asia) |
| **Modello** | Multi-strategy, "pod" system |

|---------|-----------|
