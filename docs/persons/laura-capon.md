# Laura Capon Fermi (1907 - 1977)

> Scrittrice e attivista italo-americana, moglie di Enrico Fermi, figlia dell'ammiraglio Augusto Capon ucciso ad Auschwitz — testimone e narratrice dell'era atomica

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 16 giugno 1907, Roma |
| **Morte** | 26 dicembre 1977, Chicago |
| **Famiglia** | Figlia di Augusto Capon (ammiraglio, ucciso ad Auschwitz 1943) e Costanza Romanelli |
| **Coniuge** | Enrico Fermi (dal 1928) |
| **Figli** | Nella (1931-1995), Giulio (1936-1997) |
| **Residenza** | Roma → New York → Chicago |
