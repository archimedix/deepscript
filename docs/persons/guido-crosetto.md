# Guido Crosetto (1963-)

> Il gigante della difesa: da imprenditore a lobbista dell'industria bellica a Ministro, incarna il revolving door tra Confindustria, aziende strategiche e governo.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 19 settembre 1963, Cuneo |
| **Nazionalita'** | Italiana |
| **Famiglia** | Famiglia di imprenditori (macchine agricole, oltre un secolo) |
| **Matrimoni** | Due: 1) pallavolista ceca (1 figlio, 1997); 2) Gaia Saponaro (2015, 2 figli) |
| **Figli** | Tre |
| **Statura** | 2,04 m |

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1988** | Consigliere economico PM | Giovanni Goria (Palazzo Chigi) |
| **1990-2004** | Sindaco | Marene (Cuneo), 3 mandati |
| **1999-2009** | Consigliere provinciale | Provincia di Cuneo (capogruppo FI) |
| **2001-2014** | Deputato | Camera (FI, poi PdL, poi FdI) |
| **2008-2011** | Sottosegretario alla Difesa | [Governo Italia](../government/governo-italia.md) (Berlusconi IV) |
| **2012** | **Co-fondatore e Presidente** | Fratelli d'Italia |
| **2014-2022** | **Presidente** | AIAD (Confindustria) |
| **2014-2022** | **Senior Advisor** | [Leonardo](../defense/leonardo.md) |
| **2020-2022** | **Presidente CdA** | Orizzonte Sistemi Navali (JV [Fincantieri](../company/fincantieri.md)-Leonardo) |
| **2022-** | **Ministro della Difesa** | [Governo Italia](../government/governo-italia.md) (Meloni) |

### Pattern: Politica → Industria Difesa → Politica

| Fase | Periodo | Ruolo |
|------|---------|-------|
| **Politica 1** | 1988-2014 | DC → FI → PdL → FdI, Sottosegretario Difesa |
| **Industria** | 2014-2022 | Presidente AIAD, Advisor Leonardo, Presidente OSN |
| **Politica 2** | 2022- | Ministro della Difesa |

## AIAD - Federazione Aziende Difesa

| Aspetto | Dettaglio |
|---------|-----------|
| **Ruolo** | Presidente (2014-2022) |
| **Membri** | ~200 aziende aerospazio, difesa, sicurezza |
| **Affiliazione** | Confindustria |
| **Potere** | Lobby industria bellica italiana |

### Dimissioni da Deputato (2019)

Nel marzo 2019 Crosetto si dimise da deputato (respinte due volte) per evitare conflitto di interessi, continuando a guidare AIAD.
