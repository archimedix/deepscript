# Bogdan Aurescu (1973)

> Giurista e diplomatico di carriera. Primo romeno giudice alla Corte Internazionale di Giustizia. Due volte Ministro degli Esteri. Architetto della partnership strategica Romania-USA.

## Pattern di Potere

### Giurista di Stato → Giudice Internazionale

1. **Formazione elite** → Sfantul Sava, Giurisprudenza, L'Aia
2. **Ingresso sistema** → MAE a 23 anni (1996)
3. **Specializzazione** → Diritto internazionale
4. **Rappresentanza** → Agent CEDU, poi ICJ
5. **Vittoria strategica** → Mar Nero 2009 (credenziale chiave)
6. **Vertice politico** → Ministro Esteri (x2)
7. **Legittimazione internazionale** → ILC ONU, Venezia, PCA
8. **Coronamento** → Giudice ICJ (primo romeno)
