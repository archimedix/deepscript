# Stefano Pontecorvo (1957-)

> L'ultimo uomo NATO a Kabul: diplomatico di carriera che ha coordinato l'evacuazione dall'Afghanistan prima di diventare Chairman di Leonardo.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 1957 |
| **Nazionalita'** | Italiana |
| **Famiglia** | Figlio di diplomatico italiano (cresciuto a Kabul e Islamabad) |
| **Coniuge** | Sposato |
| **Figli** | Una figlia |
