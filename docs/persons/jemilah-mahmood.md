## Pattern di Potere

### Triangolo Pharma-Finance-Governance

Jemilah Mahmood rappresenta un raro esempio di ponte tra:

1. **Big Pharma**: Board Roche (con Andre Hoffmann, Co-Chair WEF)
2. **Finanza Sovrana**: Board EPF Malaysia (~$250B AUM)
3. **Governance Globale**: WEF, Atlantic Council, UN
4. **Umanitario Istituzionale**: IFRC, Oxfam, MERCY Malaysia

### Connessione Rockefeller

Senior Fellow presso **Adrienne Arsht-Rockefeller Foundation Resilience Center** (Atlantic Council), finanziato con $30M dalla Rockefeller Foundation.
