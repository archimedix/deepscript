### Structured Alpha Scandal (2020-2022)

| Aspetto | Dettaglio |
|---------|-----------|
| **Evento** | Crollo fondi Structured Alpha durante COVID crash (marzo 2020) |
| **Perdite** | $7+ miliardi |
| **Accusa** | Hedge dichiarate mai implementate |
| **Vittime** | Fondi pensione USA (NYC Subway, Milwaukee) |
| **Settlement** | $6B+ (2022), guilty plea securities fraud |
| **Responsabilita' Bäte** | CEO durante lo scandalo, gestione della crisi |
