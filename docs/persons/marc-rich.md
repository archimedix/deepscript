# Marc Rich (1934-2013)

> "King of Oil": fondatore di Glencore, fuggitivo FBI per 17 anni, agente Mossad, graziato da Clinton nell'ultimo giorno di presidenza

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Marcell David Reich |
| **Nascita** | 18 dicembre 1934, Anversa, Belgio |
| **Morte** | 26 giugno 2013, Lucerna, Svizzera |
| **Sepoltura** | Tel Aviv, Israele |
| **Nazionalita'** | USA (emigrato 1941), poi Spagna, Israele |
| **Religione** | Ebreo |
| **Ex-moglie** | Denise Rich (divorzio 1996) |
| **Patrimonio** | ~$2.5B (Forbes) |

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1954-1974** | Oil Trader, Executive | [Philipp Brothers](../company/philipp-brothers.md) |
| **1974-1994** | **Fondatore, CEO** | [Marc Rich + Co AG](../company/glencore.md) (oggi Glencore) |
| **1983-2001** | Fuggitivo FBI | - |
| **2001** | Graziato | Presidente Clinton |

### La Fondazione di Marc Rich + Co (1974)

Nel 1974, Rich lascia Philipp Brothers insieme ai partner:

| Partner | Ruolo | Note |
|---------|-------|------|
| **Marc Rich** | Oil trading | CEO |
| **Pincus Green** | Shipping, finance, admin | Co-fondatore |
| **Alec Hackel** | Metals, minerals | Partner |

La societa' fu fondata a **Zug, Svizzera** per sfruttare la neutralita' svizzera e le basse tasse.

### Buyout e Nascita di Glencore (1993-1994)

| Anno | Evento |
|------|--------|
| **1993** | Rich perde controllo dopo fallimento speculazione zinco |
| **1994** | Management buyout → rinominata **Glencore** |
| **2002** | Ivan Glasenberg diventa CEO |
| **2011** | IPO Glencore |
| **2013** | Fusione Glencore-Xstrata |

## Operazioni Controverse

### 1. Trading con Iran Durante Embargo (1979-1994)

| Aspetto | Dettaglio |
|---------|-----------|
| **Periodo** | 1979-1994 (15 anni) |
| **Contesto** | Embargo USA dopo crisi ostaggi |
| **Operazione** | Acquisto petrolio iraniano, rivendita a USA |
| **Profitto** | Comprare a $12/barile, vendere al doppio |
| **Relazione** | "Relazione speciale" con regime Khomeini |

### 2. Trading con Sudafrica Apartheid

| Aspetto | Dettaglio |
|---------|-----------|
| **Periodo** | Anni '70-'80 |
| **Operazione** | Vendita petrolio a regime apartheid |
| **Profitto totale** | **$2 miliardi** |
| **Posizione Rich** | "Non sono un politico, sono un trader" |
| **Schema** | Petrolio Iran/URSS → Sudafrica → Uranio → URSS |

### 3. Cooperazione con Mossad

| Aspetto | Dettaglio |
|---------|-----------|
| **Conferma** | Rich ammise a biografo Daniel Ammann |
| **Operazioni** | Finanziamento Mossad, uso uffici come copertura |
| **Pipeline segreta** | Iran-Israele (anni '70) |
| **Pardon lobbying** | Avner Azulay (ex-capo Mossad) chiese grazia a Clinton |
| **Supporto** | Shabtai Shavit (ex-capo Mossad) scrisse a Clinton |

|------|--------|

## Il Pardon Clinton (20 gennaio 2001)

### La Grazia

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 20 gennaio 2001, ultime ore presidenza |
| **Beneficiari** | Marc Rich, Pincus Green |
| **Decisione** | Controversa, critiche bipartisan |

### Le Donazioni

| Beneficiario | Ammontare | Note |
|--------------|-----------|------|
| Partito Democratico | $1M+ | Via Denise Rich |

### Reazioni
