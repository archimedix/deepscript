## Digital Public Infrastructure (DPI)

### Il "modello India Stack"

Nilekani è l'architetto del modello DPI indiano, alternativa alla privatizzazione occidentale:

| Layer | Componente | Status |
|-------|------------|--------|
| Identity | Aadhaar | 1.4 miliardi |
| Payments | UPI | Leader mondiale |
| Data | DigiLocker | 150+ milioni |
| Commerce | ONDC | In sviluppo |
| Health | ABHA | In sviluppo |

### Esportazione del modello

| Paese/Regione | Adozione |
|---------------|----------|
| Singapore | UPI linkage |
| UAE | UPI linkage |
| Francia | UPI payments |
| G20 | DPI framework |
| World Bank | Aadhaar reference |

## Significato Storico

### Revolving door "tecnocratico"

A differenza del revolving door occidentale (finanza → governo), Nilekani rappresenta il modello **tech → governo**:

| Aspetto | Nilekani | Modello occidentale |
|---------|----------|---------------------|
| Background | Ingegnere software | Banchiere/Avvocato |
| Ruolo governo | Costruire infrastruttura | Regolare/Deregolare |
| Dopo governo | Torna a tech/filantropia | Torna a finanza |
| Output | Aadhaar, UPI | Policy, leggi |

### "Softpower" digitale

India esporta il modello DPI come alternativa a:
- **USA**: Big Tech privato (Google, Apple Pay)
- **Cina**: Stato centralizzato (WeChat, Alipay controllati)
- **India**: Infrastruttura pubblica, servizi privati
