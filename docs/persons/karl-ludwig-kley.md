## Potere Esercitato

### Revolving Door Germania Inc.

| Pattern | Dettaglio |
|---------|-----------|
| **Settori attraversati** | Chimica/Pharma → Aviazione → Pharma → Energia → Auto |
| **Modello** | CFO → CEO → Chairman multipli |
| **Network** | Ponte tra Bayer, Lufthansa, Merck, BMW, E.ON |
