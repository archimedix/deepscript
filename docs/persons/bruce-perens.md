## Potere Esercitato

### Potere normativo

| Aspetto | Dettaglio |
|---------|-----------|
| **Open Source Definition** | Lo standard che definisce cosa puo' chiamarsi "open source" — adottato globalmente |
| **DFSG** | Le Debian Free Software Guidelines sono la base legale-filosofica della OSD |
| **Legalita' licenze** | Testimone chiave nei casi legali che stabiliscono la validita' delle licenze open source |
| **Corporate strategy** | Ha portato la strategia open source dentro HP, modello per altre aziende |

### La posizione unica di Perens

Perens e' l'unica figura che ha co-fondato OSI e poi l'ha lasciata per tornare verso il free software. Questo lo rende un osservatore critico interno: ha visto il movimento open source dall'interno corporate (Pixar, HP) e dall'interno ideologico (Debian, FSF).
