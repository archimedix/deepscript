# Henry Kravis (1944)

> Pioniere del private equity, co-fondatore di KKR, architetto del leveraged buyout moderno

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 6 gennaio 1944, Tulsa, Oklahoma |
| **Famiglia** | Figlio di Raymond F. Kravis (oil engineer, socio di Joseph P. Kennedy) e Bessie Roberts |
| **Coniuge** | Marie-Josee Kravis (dal 1994), economista canadese, Co-Chair Bilderberg |
| **Figli** | 3 da precedenti matrimoni |
| **Residenza** | New York City, Palm Beach, Southampton, Parigi |
| **Net Worth** | ~$16 miliardi (2025) |

---

## Formazione
