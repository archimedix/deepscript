# Salomon Oppenheim Jr. (1772 - 1828)

> Fondatore di Sal. Oppenheim, banca privata di Colonia, Hofjude degli Arcivescovi-Principi di Colonia

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 19 giugno 1772, Bonn |
| **Morte** | 8 novembre 1828, Colonia |
| **Famiglia** | Hofjuden degli Arcivescovi-Principi di Colonia (dal ~1740) |
| **Coniuge** | Therese Stein (nata Deigen Levi) |
| **Figli** | 12, tra cui Simon (1803), Abraham (1804), Dagobert (1809) |
| **Residenza** | Bonn (fino ~1798) → Colonia |

## Potere Esercitato

### Finanza e Insediamento

| Aspetto | Dettaglio |
|---------|-----------|
| Banca privata | Fondo' Sal. Oppenheim a 17 anni, divenne la principale banca privata renana |
| Pioniere a Colonia | Tra i primi ebrei a ristabilirsi a Colonia dopo 374 anni di espulsione (1424-1798) |
| Esattore fiscale | Nominato dalle autorita' francesi di occupazione |
| Dinastia | I figli Simon e Abraham portarono la banca a livello europeo, finanziando le ferrovie prussiane |
