| Aspetto | Dettaglio |

| Aspetto | Dettaglio |
|---------|-----------|
| **Anno** | 1993 |
| **Partner** | Renault (Francia) |
| **Obiettivo** | Creare gigante auto europeo |
| **Opposizione** | Azionisti svedesi |
| **Esito** | Fallimento, dimissioni Gyllenhammar |
| **Conseguenza** | Fine carriera in Svezia, trasferimento a Londra |

| Aspetto | Dettaglio |
|---------|-----------|
| **Ruolo** | Chairman |
| **Funzione** | Garante dei Trust Principles di Reuters |
| **Controversia** | Approvò merger Thomson-Reuters 2008 |
| **Critica** | Waiver sulla clausola di proprietà diffusa |
| **Giustificazione** | Difficoltà finanziarie di Reuters |
