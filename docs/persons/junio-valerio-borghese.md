### Decima Flottiglia MAS (1941-1943)

| Aspetto | Dettaglio |
|---------|-----------|
| **Tipo unita'** | Commando navale d'assalto |
| **Specializzazione** | Siluri a lenta corsa ("maiali"), uomini rana |
| **Operazione celebre** | Alessandria d'Egitto, dicembre 1941 |
| **Risultato** | Affondamento corazzate HMS Queen Elizabeth e HMS Valiant |

La Decima MAS e' considerata la prima unita' commando navale moderna.

### Decima MAS nella RSI (1943-1945)

Dopo l'armistizio dell'8 settembre 1943:

| Aspetto | Dettaglio |
|---------|-----------|
| **Scelta** | Fedelta' alla Repubblica Sociale Italiana |
| **Forza** | ~10.000 uomini (fine 1943) |
| **Impiego** | Lotta anti-partigiana nel Nord Italia |
| **Collaborazione** | SS tedesche |
