### AKASHA Foundation (2015-oggi)

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondata** | Ottobre 2015 |
| **Sede** | Zug, Switzerland |
| **Advisor** | Vitalik Buterin |
| **Prodotto** | Social network decentralizzato su Ethereum + IPFS |

**AKASHA** = Advanced Knowledge Architecture for Social Human Advocacy

| Tecnologie | Uso |
|------------|-----|
| Ethereum | Smart contracts, governance |
| IPFS | Storage decentralizzato, no server |
| React/Redux | Frontend |

**Ethereum World:** Social network per unificare l'ecosistema Ethereum, primo network su AKASHA World Framework.

### AKASHA Vision

Creare social network senza server centrali, resistente alla censura:
- Nessun single point of failure
- Comunicazione peer-to-peer via IPFS
- Governance decentralizzata via Ethereum
