### Kape Technologies: la trasformazione

Erlichman entra come CEO nel 2016 quando il titolo Kape (allora Crossrider) era crollato a £0.27/share. Guida la trasformazione da piattaforma adware a holding VPN/cybersecurity:

| Anno | Azione |
|------|--------|
| **2016** | Nominato CEO, avvia pivot strategico |
| **2017** | Acquisizione CyberGhost (~$10M) |
| **2018** | Rebrand Crossrider → Kape Technologies |
| **2018** | Acquisizione ZenMate (~$5M) |
| **2019** | Acquisizione Private Internet Access ($127M) |
| **2021** | Acquisizione vpnMentor e Wizcase (review sites) |
| **2021** | Acquisizione ExpressVPN (~$936M) |
| **2023** | Exit: privatizzazione da Teddy Sagi per $1.75B |
