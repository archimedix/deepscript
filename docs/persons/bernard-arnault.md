# Bernard Arnault (1949-)

> Uomo piu' ricco del mondo, architetto dell'impero del lusso LVMH

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 5 marzo 1949, Roubaix, Francia |
| **Famiglia** | Figlio di Jean Leon Arnault (industriale edile) e Marie-Josephe Savinel |
| **Coniuge** | 1) Anne Dewavrin (1973-1990), 2) Helene Mercier (1991-) |
| **Figli** | 5: Delphine, Antoine, Alexandre, Frederic, Jean |
| **Residenza** | Parigi |
| **Patrimonio** | ~$200 miliardi (Forbes 2024, #1 mondo) |
