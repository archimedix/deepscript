# Joseph Trumpeldor (1880 - 1920)

> Eroe militare sionista, co-fondatore della Jewish Legion, simbolo della resistenza ebraica

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 21 novembre 1880, Pyatigorsk, Impero Russo |
| **Morte** | 1 marzo 1920, Tel Hai, Palestina mandataria |
| **Nazionalita'** | RUS |
| **Causa morte** | Ucciso nella battaglia di Tel Hai |
