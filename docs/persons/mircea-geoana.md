# Mircea Geoană (1958)

> Primo Deputy Secretary General NATO dall'Europa orientale post-Guerra Fredda. Fondatore Aspen Romania. Architetto dell'ingresso della Romania in NATO e UE.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 14 luglio 1958, Bucarest |
| **Famiglia** | Padre Ioan Geoană, generale dell'esercito romeno (1965-1992), capo Difesa Civile |
| **Coniuge** | Mihaela Geoană (architetto, ex Presidente Croce Rossa Romania) |
| **Figli** | Ana Maria, Alexandru |
| **Lingue** | Romeno, inglese, francese, spagnolo, italiano |
