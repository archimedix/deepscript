## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|

## Potere Esercitato

### Infrastruttura Cloud

| Aspetto | Dettaglio |
|---------|-----------|
| **Terraform** | Standard de facto per Infrastructure as Code, usato da migliaia di aziende |
| **Vagrant** | Ha rivoluzionato gli ambienti di sviluppo, precursore di Docker |
| **Open source influence** | I tool HashiCorp sono diventati infrastruttura critica del cloud computing |

### Post-HashiCorp

| Aspetto | Dettaglio |
|---------|-----------|
| **Ghostty** | Terminal emulator nonprofit, sfida il modello VC per dev tools |
| **Modello nonprofit** | Via fiscal sponsorship Hack Club, donazioni finanziano contributori |
