# Albert Bourla (1961-)

| **Coniuge** | Myriam Bourla |
| **Figli** | 2 |
| **Nazionalita** | Greco-americana (USA dal 2001) |

## Vaccino COVID: Il Caso

### Timeline Pfizer-BioNTech

| Data | Evento |
|------|--------|
| **Ago 2018** | Partnership Pfizer-BioNTech (influenza mRNA) |
| **Gen 2019** | Bourla diventa CEO |
| **Mar 2020** | Pivot a vaccino COVID |
| **Nov 2020** | Annuncio efficacia 95% |
| **Dic 2020** | Prima approvazione (UK) |
| **Dic 2020** | EUA FDA (Hahn Commissioner) |
| **2021** | $36.8B revenue vaccino |
| **2022** | $37.8B revenue vaccino |

### Decisione "No Government Money"

Bourla ha pubblicamente rifiutato funding diretto Operation Warp Speed per R&D (ma accettato pre-ordini).

> "I wanted to liberate our scientists from any bureaucracy."

**Nota critica**: Pfizer ha comunque beneficiato di:
- Pre-ordini garantiti ($1.95B USA)
- BioNTech funding (CEPI, Germania)
- Infrastruttura regolatori accelerata

## Pattern Identificati

### 1. Carriera Interna

A differenza di molti CEO pharma, Bourla e' cresciuto interamente dentro Pfizer per 30 anni. Non revolving door ma "company man".

### 2. Lobby Leadership

Chairman PhRMA (USA) + ex-President IFPMA (global) = voce principale dell'industria pharma.

### 3. Immigrato → Vertice

Veterinario greco immigrato 2001 → CEO piu' grande pharma 2019. Narrativa "American Dream" usata nel marketing vaccino.

### 4. Heritage come Brand

Origine ebraica sefardita e storia familiare Olocausto usate per costruire credibilita' morale durante campagna vaccinale.
