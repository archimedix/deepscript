## Crimini di Guerra

### Deportazioni

| Crimine | Dettaglio |
|---------|-----------|
| **Treni per Treblinka** | Ottenne i treni dalla Reichsbahn per deportare ebrei |
| **Lettera del 1942** | Documento dove esprime "particolare gioia" per 5.000 deportati al giorno |
| **Italia 1944** | Organizzo' deportazione ebrei italiani |
| **Totale stimato** | Centinaia di migliaia di vittime |

### Il Documento Incriminante

Nel 1942, Wolff scrisse una lettera al Ministero dei Trasporti esprimendo "particolare gioia" per l'ottenimento di treni per deportare 5.000 ebrei al giorno verso Treblinka. Questo documento fu usato contro di lui nel processo del 1964.

## Operation Sunrise (1945)

### Il Tradimento di Hitler

| Data | Evento |
|------|--------|
| **Feb 1945** | Wolff contatta Allen Dulles a Zurigo |
| **8 mar 1945** | Primo incontro diretto Wolff-Dulles |
| **Mar-Apr 1945** | Negoziati segreti in Svizzera |
| **29 apr 1945** | Firma resa a Caserta |
| **2 mag 1945** | Resa effettiva |

### I Negoziatori

| Parte | Rappresentante |
|-------|----------------|
| **Germania** | Karl Wolff (SS) |
| **USA** | [Allen Dulles](allen-dulles.md) (OSS Berna) |
| **USA militare** | Gen. Lyman Lemnitzer |
| **UK** | Gen. Terence Airey |
| **Intermediario** | Max Waibel (intelligence svizzera) |

### Portata della Resa

| Aspetto | Numero |
|---------|--------|
| **Offerta iniziale** | 225.000 uomini (comando Wolff) |
| **Army Group C** | 585.000 uomini |
| **Totale finale** | ~1.000.000 soldati |
| **Anticipo** | 5 giorni prima della resa generale |

### Tensioni con i Sovietici

| Aspetto | Dettaglio |
|---------|-----------|
| **Problema** | URSS informata ma esclusa dai negoziati |
| **Reazione Stalin** | Sospetto di accordo separato anti-sovietico |
| **Conseguenza** | Aumento tensioni tra Alleati |

## Fuga da Norimberga

### Come Sfuggi' al Processo

| Fattore | Dettaglio |
|---------|-----------|
| **Operation Sunrise** | Resa anticipata considerata "servizio" |
| **Testimonianza** | Comparve come testimone dell'accusa |
| **Protezione Dulles** | OSS/CIA lo considero' asset prezioso |
| **Proxy Himmler** | Avrebbe potuto essere processato al posto del suicida Himmler |

> "La mancata accusa di Wolff a Norimberga ha le sue origini nei negoziati per la resa parziale in Italia del Nord."
