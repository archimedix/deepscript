# Lee Kuan Yew (1923-2015)

> Padre fondatore di Singapore: da colonia britannica a potenza finanziaria globale in una generazione

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 16 settembre 1923, Singapore |
| **Morte** | 23 marzo 2015, Singapore |
| **Famiglia** | Hakka cinese, classe media |
| **Coniuge** | Kwa Geok Choo (1920-2010), sposata 1947 |
| **Figli** | 3: Lee Hsien Loong, Lee Wei Ling, Lee Hsien Yang |
| **Residenza** | 38 Oxley Road, Singapore |

---

## Formazione

| Periodo | Istituzione | Dettaglio |
|---------|-------------|-----------|
| **1936-1940** | Raffles Institution | Primo in Singapore e Malaya |
| **1940-1942** | Raffles College | Interrotto da invasione giapponese |
| **1946-1947** | London School of Economics | Breve frequenza |
| **1947-1949** | Fitzwilliam College, Cambridge | Law, Starred First Class Honours |
| **1950** | Middle Temple, Londra | Called to the Bar |

**Mentori:**
- W.S. Thatcher (Censor Fitzwilliam) - Lo ammise a Cambridge
- Colleghi Malayan Forum (Goh Keng Swee, Toh Chin Chye) - Co-fondatori PAP

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1950-1959** | Avvocato, fondatore | Lee & Lee (studio legale) |
| **1954** | Co-fondatore, Secretary-General | People's Action Party (PAP) |
| **1955-1959** | Membro Assemblea Legislativa | Tanjong Pagar |
| **1959-1990** | Prime Minister | Singapore |
| **1990-2004** | Senior Minister | Singapore |
| **2004-2011** | Minister Mentor | Singapore |

### Momenti Chiave

| Anno | Evento |
|------|--------|
| **1954** | Fonda PAP con Goh Keng Swee, Toh Chin Chye, S. Rajaratnam |
| **1959** | Diventa PM dopo vittoria PAP (43/51 seggi) |
| **1963** | Merger con Malesia |
| **1965** | Espulsione dalla Malesia, indipendenza Singapore |
| **1981** | Fonda GIC per gestire riserve estere |
| **1990** | Lascia dopo 31 anni, resta Senior Minister |
| **2011** | Si ritira completamente dalla politica |

## Potere Esercitato

### Costruzione dello Stato

| Aspetto | Dettaglio |
|---------|-----------|
| **Sistema politico** | Democrazia dominata da un partito (PAP) |
| **Economia** | Stato-corporazione, GLCs, free trade |
| **Housing** | HDB - 80% popolazione in case pubbliche |
| **Finanza** | GIC (1981), controllo riserve estere |

### Metodo di Governo

| Aspetto | Dettaglio |
|---------|-----------|
| **Opposizione** | Soppressa via cause per diffamazione |
| **Stampa** | Controllata, no media indipendenti |
| **Meritocrazia** | Sistema di selezione elite (PSC scholarships) |
| **Corruzione** | Zero tolerance, CPIB (Corrupt Practices Investigation Bureau) |
