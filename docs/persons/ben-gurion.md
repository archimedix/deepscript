# David Ben-Gurion (1886-1973)

> Fondatore di Israele, primo Prime Minister, architetto dello Stato ebraico

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 16 ottobre 1886, Plonsk, Polonia (Impero Russo) |
| **Morte** | 1 dicembre 1973, Tel Aviv |
| **Nome nascita** | David Gruen |
| **Coniuge** | Paula Munweis (1892-1968) |
| **Figli** | 3: Geula, Amos, Renana |
| **Residenza** | Sde Boker (kibbutz nel Negev) |
