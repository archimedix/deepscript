# Maurice Saatchi (1946)

> Baron Saatchi, co-fondatore Saatchi & Saatchi e M&C Saatchi, chairman Conservative Party

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 21 giugno 1946, Baghdad, Iraq |
| **Famiglia** | Famiglia ebraica irachena (padre Nathan, mercante tessile), emigrata a Londra 1947 |
| **Coniuge** | Josephine Hart (1984-2011, morta di cancro ovarico) |
| **Titolo** | Baron Saatchi of Staplefield (life peer dal 1996) |
| **Residenza** | Londra |
