# David Ben-Gurion (1886 - 1973)

> Fondatore dello Stato di Israele, primo PM, architetto del sionismo laburista

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 16 ottobre 1886, Plonsk, Impero Russo (ora Polonia) |
| **Morte** | 1 dicembre 1973, Sde Boker, Israele |
| **Nome di nascita** | David Grun |
| **Coniuge** | Paula Munweis (1892-1968), m. 1917 |
| **Figli** | 3: Geula, Amos, Rennana |
| **Sepoltura** | Midreshet Ben-Gurion, Negev |

---

## Formazione

| Periodo | Istituzione | Dettaglio |
|---------|-------------|-----------|
| **~1900** | Scuole ebraiche, Plonsk | Educazione primaria; fondo' club giovanile Ezra (150 membri) |
| **1904** | Universita' di Varsavia | Breve periodo; unito a Poalei Zion |
| **1912-14** | Istanbul University | Studi di giurisprudenza (non completati, espulso dagli Ottomani) |
