## Potere Esercitato

### Governance Open Source Industriale

| Aspetto | Dettaglio |
|---------|-----------|
| Longevita' | 20+ anni alla guida di Eclipse Foundation, caso raro di stabilita' nella leadership open source |
| Transizione Java EE | Ha gestito il trasferimento di Java EE da Oracle a Eclipse come Jakarta EE |
| Ponte industria-community | Bilancia interessi di strategic member corporate (IBM, Microsoft, Oracle, Huawei) con la community di sviluppatori |
| Strategia europea | Ha guidato la ri-incorporazione in Belgio (2020), posizionando Eclipse come alternativa europea alla Linux Foundation |
| Automotive | Ha espanso Eclipse nel settore SDV (Software-Defined Vehicle) con Bosch, Mercedes-Benz, BMW |

### Percorso IBM-Oracle-Eclipse

Il passaggio BNR → OTI → IBM → Oracle → Eclipse Foundation traccia una linea diretta attraverso la storia dell'IDE e della piattaforma Java enterprise. OTI (acquisita da IBM nel 1996) e' l'azienda che ha costruito il nucleo originale di Eclipse per IBM.
