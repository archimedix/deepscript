## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **pre-2011** | Engineer | DaimlerChrysler, Bosch (driver assistance systems) |

| **2021-2025** | CEO | GitHub |
| **2025-** | Founder & CEO | Entire |

### GitHub CEO (2021-2025)

Succede a Nat Friedman nel novembre 2021. Sotto la sua guida:
- GitHub Copilot evolve da autocomplete a coding agent enterprise
- Lancio di GitHub Spark (full-stack app generation)
- Copilot integra code review, voice assistance, workspace

### Entire (2025-)

Lascia GitHub nell'agosto 2025. Lancia Entire, piattaforma open-source per collaborazione sviluppatori-agenti AI. Seed round record da $60M guidato da Felicis, con Madrona, Basis Set e M12 (Microsoft Ventures). Valutazione $300M.

Angel investors: Jerry Yang (Yahoo), Garry Tan (Y Combinator), Olivier Pomel (Datadog).
