# Ilham Aliyev (1961)

> Presidente dell'Azerbaijan dal 2003, architetto del petrostato dinastico Aliyev

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 24 dicembre 1961, Baku |
| **Famiglia** | Figlio di Heydar Aliyev (presidente 1993-2003) |
| **Coniuge** | Mehriban Aliyeva (First VP Azerbaijan) |
| **Figli** | Leyla, Arzu, Heydar Jr |
| **Lingue** | Azero, russo, inglese, francese, turco |
