# George Fu Gao (1961)

> Virologo cinese, architetto della risposta pandemica cinese, ponte tra sistema sanitario cinese e network globali di biosicurezza

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 15 novembre 1961, Ying County, Shanxi, Cina |
| **Nazionalita'** | Cinese |
| **Campo** | Virologia, immunologia, biologia strutturale |
| **Residenza** | Pechino |

### Event 201

Partecipo' a [Event 201](../events/2019-event-201.md) (18 ottobre 2019), esercitazione pandemica organizzata da Johns Hopkins, WEF e Gates Foundation che simulo' uno scenario di coronavirus pandemico, sei settimane prima dell'emergere del COVID-19.
