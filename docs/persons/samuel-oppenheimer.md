# Samuel Oppenheimer (1630 - 1703)

> Hofjude e banchiere imperiale degli Asburgo, finanziatore delle guerre contro l'Impero Ottomano

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 21 giugno 1630, Heidelberg |
| **Morte** | 3 maggio 1703, Vienna |
| **Famiglia** | Famiglia ebraica ashkenazita di Heidelberg |
| **Coniuge** | Sandela Carcassonne (figlia di un ebreo sefardita di Mannheim) |
| **Figli** | Emanuel (Mendel Menachem), Simon Wolf, e altri |
| **Residenza** | Vienna (dal 1676) |
