| **Figli** | 1 (Emma, nata 2018) |
| **Partito** | SDP - Partito Socialdemocratico Finlandese |

### Adesione NATO Finlandia

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Invasione russa Ucraina (feb 2022) |
| **Decisione** | Annuncio candidatura NATO con Pres. Niinisto (maggio 2022) |
| **Risultato** | Finlandia membro NATO (aprile 2023) |
| **Impatto** | Fine di 75 anni di non-allineamento finlandese |
