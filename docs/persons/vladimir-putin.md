# Vladimir Putin (1952)

> Presidente della Federazione Russa, architetto della "verticale del potere", ex ufficiale KGB - figura centrale del sistema di potere russo post-sovietico

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 7 ottobre 1952, Leningrado (San Pietroburgo) |
| **Formazione** | Laurea in Giurisprudenza, Università Statale di Leningrado (1975) |
| **Ex moglie** | Lyudmila Shkrebneva (m. 1983, div. 2014) |
| **Figlie** | Maria Vorontsova (1985), Katerina Tikhonova (1986) |
| **Sanzioni** | USA, EU, UK, Canada, Australia, Giappone (2014-) |

## Potere Esercitato

### Verticale del Potere

| Aspetto | Dettaglio |
|---------|-----------|
| **Consolidamento** | Distruzione oligarchi indipendenti (Khodorkovsky/Yukos 2003) |
| **Media** | Controllo TV nazionali, pressione su media indipendenti |
| **Regioni** | Abolizione elezione diretta governatori (2004-2012) |
| **Opposizione** | Restrizioni ONG, legge "agenti stranieri" |

### Politica Estera

| Anno | Azione |
|------|--------|
| **2008** | Guerra Georgia, riconoscimento Ossezia Sud e Abkhazia |
| **2014** | Annessione Crimea, guerra proxy Donbass |
| **2015** | Intervento militare in Siria |
| **2022** | Invasione su larga scala dell'Ucraina |

### Riforma Costituzionale 2020

La riforma ha resettato i mandati presidenziali di Putin, permettendogli potenzialmente di restare al potere fino al 2036.
