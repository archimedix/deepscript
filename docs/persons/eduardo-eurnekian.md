# Eduardo Eurnekian (1932)

> Miliardario armeno-argentino, re delle concessioni aeroportuali, creatore della piattaforma che ha lanciato Milei

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 4 dicembre 1932, Buenos Aires |
| **Famiglia** | Genitori immigrati armeni in Argentina |
| **Coniuge** | Non sposato |
| **Successore designato** | Martin Eurnekian (nipote) |
| **Patrimonio** | $3.4 miliardi (2024, 4° in Argentina) |
| **Lingue** | Spagnolo, armeno, inglese, italiano |

Formazione imprenditoriale autodidatta, cresciuto nell'azienda tessile familiare.
