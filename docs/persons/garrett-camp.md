## Potere Esercitato

### Venture creation

| Aspetto | Dettaglio |
|---------|-----------|
| **StumbleUpon** | Pioniere della web discovery, acquisita da eBay per $75M (2007) |
| **Uber** | Idea originale 2008, ha concepito il modello di ride-hailing via app |
| **Expa** | Startup studio con $350M raccolti, 20+ aziende create, valore combinato $8B+ |
| **Giving Pledge** | Membro dal 2015, impegno a donare maggioranza patrimonio |

### Modello operativo

| Aspetto | Dettaglio |
|---------|-----------|
| **Pattern** | Ideatore seriale — concepisce l'idea, recluta un CEO operativo (Kalanick per Uber), resta chairman |
| **Expa model** | Non solo investitore: co-crea aziende da zero (design, branding, engineering, recruiting) |
| **Portfolio Expa** | Current ($2.2B valuation), Collective, Metabase, Convoy, Spatial, Aero |
