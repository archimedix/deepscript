## Cambridge Analytica: Lo Scandalo

### Come Funzionava

| Aspetto | Dettaglio |
|---------|-----------|
| **Data harvesting** | App Facebook "thisisyourdigitallife" raccoglieva dati |
| **Profili psicografici** | Modelli OCEAN per targeting comportamentale |
| **Utenti coinvolti** | 87 milioni profili Facebook |
| **Scopo** | Microtargeting elettorale, "psychological warfare" |
