# Linus Torvalds (1969)

> Creatore del kernel Linux e di Git — ha reso possibile il piu' grande progetto collaborativo della storia del software, ridefinendo l'infrastruttura digitale globale.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 28 dicembre 1969, Helsinki, Finlandia |
| **Nazionalita'** | FIN-USA |
| **Famiglia** | Figlio dei giornalisti Anna e Nils Torvalds; nonno materno Leo Tornqvist, statistico |
| **Coniuge** | Tove Torvalds (nata Monni), campionessa finlandese di karate |
| **Figli** | 3 figlie |
| **Residenza** | Portland, Oregon, USA |

---

## Potere Esercitato

### Potere infrastrutturale

| Aspetto | Dettaglio |
|---------|-----------|
| **Linux kernel** | Presente nel 100% dei supercomputer, ~80% degli smartphone (Android), gran parte dei server cloud |
| **Git** | Standard de facto per il version control, base di GitHub/GitLab |
| **Governance tecnica** | "Benevolent dictator" — decide cosa entra nel kernel Linux |
| **Influenza industriale** | Ogni grande azienda tech (Google, Microsoft, Meta, Amazon) contribuisce al kernel |

## Controversie

| Tema | Periodo | Dettaglio |
|------|---------|-----------|
| **Stile comunicativo** | Ongoing | Noto per commenti abrasivi nelle mailing list del kernel |
| **Pausa dal kernel** | 2018 | Si ritira temporaneamente per lavorare sul proprio comportamento, introduce Code of Conduct |
| **GPL v2 vs v3** | 2006-2007 | Rifiuta la GPL v3 di Stallman, mantiene Linux sotto GPL v2 — punto di frattura con FSF |
