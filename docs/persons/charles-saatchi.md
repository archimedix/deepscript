# Charles Saatchi (1943)

> Co-fondatore Saatchi & Saatchi, collezionista d'arte, padrino dei YBA

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 9 giugno 1943, Baghdad, Iraq |
| **Famiglia** | Famiglia ebraica irachena, emigrata a Londra negli anni '40 |
| **Coniugi** | Doris Lockhart (1973-1990), Kay Hartenstein (1990-2001), Nigella Lawson (2003-2013) |
| **Residenza** | Londra |
