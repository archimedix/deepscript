# Alois Brunner (1912-2010)

> Braccio destro di Eichmann, responsabile della deportazione di 128.000 ebrei, fuggito in Siria dove mori' impunito dopo aver addestrato i servizi segreti di Assad

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Alois Brunner |
| **Nascita** | 8 aprile 1912, Rohrbrunn, Austria-Ungheria |
| **Morte** | Dicembre 2001 o 2010, Damasco, Siria |
| **Nazionalita'** | Austria |
| **Grado** | SS-Hauptsturmfuhrer (Capitano) |
| **Alias** | Dr. Georg Fischer (Siria) |
| **Segni distintivi** | Occhio e dita mancanti (lettere bomba Mossad) |
