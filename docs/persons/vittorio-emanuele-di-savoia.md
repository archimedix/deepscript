## Carriera e Ruoli

### Timeline

| Periodo | Evento |
|---------|--------|
| **1937** | Nasce a Napoli, Villa Rosebery |
| **1946** | Esilio della famiglia reale; si trasferisce in Portogallo, poi Svizzera |
| **1960** | Incontra Marina Doria alla Societe' Nautique de Geneve |
| **1969** | Si autoproclamata Re d'Italia, conferisce a Marina il titolo di Duchessa di Sant'Anna di Valdieri |
| **1970** | Matrimonio civile con Marina Doria a Las Vegas |
| **1971** | Matrimonio religioso a Teheran, durante le celebrazioni di Persepoli |
| **1972** | Nascita del figlio Emanuele Filiberto |
| **Anni '70** | Intermediario per Agusta: vendita elicotteri a Iran e paesi arabi grazie all'amicizia con lo Shah |
| **1978** | Spara a Dirk Hamer (19 anni) sull'isola di Cavallo, Corsica — il giovane muore |
| **1981** | Nome nella lista degli iscritti alla P2 di Licio Gelli (tessera n. 1621) |
| **1983** | Morte del padre Umberto II — diventa pretendente al trono |
| **1991** | Assolto dall'accusa di omicidio volontario dalla Corte d'Appello di Parigi; condannato a 6 mesi con la condizionale per porto d'armi |
| **2002** | Rientro in Italia dopo l'abrogazione della XIII disposizione transitoria della Costituzione |
| **2006** | Arrestato a Potenza: accuse di associazione a delinquere, corruzione, sfruttamento della prostituzione (caso Casino' di Campione / Vallettopoli) |
| **2006** | Intercettazione in carcere: ammette di aver sparato a Dirk Hamer |
| **2006** | Inizio disputa dinastica con [Amedeo di Savoia-Aosta](amedeo-di-savoia-aosta.md) |
| **2010** | Assolto da tutte le accuse nel processo di Potenza |
| **2024** | Muore a Ginevra il 3 febbraio. Funerali al Duomo di Torino, ceneri a Superga |
