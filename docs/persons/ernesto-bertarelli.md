### Tre Generazioni

| Gen | Nome | Ruolo | Periodo |

| 3 | Dona Bertarelli | Co-Chair Foundation | 1998- |

### Storia Familiare

| Anno | Evento |
|------|--------|
| **1906** | Cesare Serono fonda Istituto Farmalogico Serono, Roma |
| **1935** | Pietro Bertarelli diventa Managing Director |
| **1965** | Morte Pietro; Fabio inizia a guidare |
| **1970s** | Fabio sfida Michele Sindona, ottiene controllo Serono |
| **1977** | Famiglia si trasferisce in Svizzera (minaccia Brigate Rosse) |
| **1996** | Ernesto diventa CEO |
| **1998** | Morte Fabio; Ernesto e Dona ereditano |
| **2007** | Vendita a Merck KGaA |

### Il Caso Sindona

Negli anni '70, il Vaticano aveva venduto le azioni Serono a Michele Sindona (poi arrestato per bancarotta fraudolenta). Fabio Bertarelli organizzo' un sindacato di investitori per sfidare Sindona e, quando questi fuggi' dall'Italia nel 1974, acquisto' le azioni a prezzo basso ottenendo il controllo.

## Pattern Identificati

### 1. Exit Pharma → Network Filantropia-Accademia

```
SERONO (vendita $13.3B)
         |
         +---> B-Flexion (family office)
         |
         +---> Harvard HMS (Chairman Board)
         |           |
         |           +---> $75M donation
         |
         +---> Campus Biotech Geneva
                    |
                    +---> EPFL, UniGe partnership
```

### 2. Swiss-American Bridge

Bertarelli connette:
- **Svizzera**: EPFL, Campus Biotech, UBS
- **USA**: Harvard (HMS + HBS), Stanford Medicine
- **Pharma globale**: Serono → Merck KGaA

### 3. Dynastia Pharma Europea

Pattern simile a [Hoffmann-La Roche](../family/hoffmann-la-roche.md):
- Famiglia italiana → Svizzera
- Controllo pharma multigenerazionale
- Exit con megadeal ($13.3B)
- Reinvestimento in filantropia e accademia
