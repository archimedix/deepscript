## Potere Esercitato

### Infrastruttura Digitale Globale

| Aspetto | Dettaglio |
|---------|-----------|
| **Unix** | Sistema operativo alla base di Linux, macOS, iOS, Android: >90% dei server internet, tutti gli smartphone |
| **UTF-8** | Encoding universale del testo digitale, >98% delle pagine web |
| **Go** | Linguaggio core dell'infrastruttura cloud (Docker, Kubernetes, Terraform) |
| **Filosofia Unix** | "Do one thing well" - paradigma di design software ancora dominante |

Thompson non ha esercitato potere politico o finanziario diretto, ma le sue invenzioni sono l'infrastruttura invisibile su cui il potere digitale si costruisce.
