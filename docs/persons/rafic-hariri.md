# Rafic Hariri (1944-2005)

> Miliardario libanese-saudita, primo ministro del Libano post-guerra civile, assassinato nel 2005

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 1 novembre 1944, Sidone, Libano |
| **Morte** | 14 febbraio 2005, Beirut (assassinato) |
| **Confessione** | Musulmano sunnita |
| **Prima moglie** | Nidal Bustani (1965) |
| **Seconda moglie** | Nazik Audi (1976) |
| **Figli** | 6: Bahaa, Saad, Houssam†, Ayman, Fahd, Hind |
| **Patrimonio** | ~$4 miliardi (Forbes anni '90) |

### Ascesa in Arabia Saudita

| Anno | Evento |
|------|--------|
| **1965** | Emigra a Jeddah come insegnante |
| **1969** | Prima impresa Ciconest (fallisce) |
| **1970s** | Costruzione hotel Ta'if per Re Khaled - completato in tempo |
| **1970s** | Acquisisce Oger → fonda Saudi Oger |

## Assassinio

### L'Attentato

| Dato | Dettaglio |
|------|-----------|
| **Data** | 14 febbraio 2005, ore 12:55 |
| **Luogo** | Beirut, St. George Hotel |
| **Metodo** | Camion bomba suicida (1,800 kg TNT) |
| **Vittime** | 23 morti (incluso Hariri), 226 feriti |
| **Conseguenze** | Rivoluzione dei Cedri, ritiro truppe siriane |

### Tribunale Speciale ONU (STL)

| Anno | Sviluppo |
|------|----------|
| **2007** | Creazione STL (ONU-Libano) |
| **2020** | Verdetto: **Salim Ayyash** (Hezbollah) colpevole |
| **2020** | Condanna: 5 ergastoli in contumacia |
| **2022** | Altri 2 membri Hezbollah condannati (Merhi, Oneissi) |
| **2023** | Chiusura STL |

**Nota**: Il tribunale non trovò prove di coinvolgimento diretto della leadership Hezbollah o della Siria, pur riconoscendo che "avevano motivi per eliminare Hariri".
