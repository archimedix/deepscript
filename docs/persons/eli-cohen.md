# Eli Cohen (1924 - 1965)

> Spia del Mossad infiltrata in Siria, giustiziato pubblicamente a Damasco

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 26 dicembre 1924, Alessandria, Egitto |
| **Morte** | 18 maggio 1965, Damasco, Siria (impiccagione) |
| **Nome in codice** | Kamal Amin Thaabet |
| **Coniuge** | Nadia Cohen |
| **Figli** | 3 (Sophie, Irit, Shai) |

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1957-1959** | Analista counter-intelligence | Aman (IDF Intelligence) |
| **1959-1960** | Impiegato | Compagnia assicurativa Tel Aviv |
| **1961** | Costruzione copertura | Buenos Aires |
| **1962-1965** | Agente infiltrato | Mossad - Damasco |

### Missione in Siria

| Anno | Evento |
|------|--------|
| **1961** | Reclutato da Meir Amit, inviato a Buenos Aires |
| **1961** | Costruisce identita' come Kamal Amin Thaabet, businessman siriano |
| **Feb 1962** | Si trasferisce a Damasco |
| **1962-64** | Penetra i vertici politico-militari siriani |
| **1963** | Diventa consigliere del Ministro della Difesa |
| **Nov 1964** | Chiede di terminare missione, Mossad insiste per ritorno |
| **Gen 1965** | Catturato durante trasmissione radio |
| **18 Mag 1965** | Impiccato pubblicamente in Piazza Marjeh |

## Impatto Operativo

### Intelligence raccolta

| Aspetto | Dettaglio |
|---------|-----------|
| **Postazioni militari** | Mappatura completa Alture del Golan |
| **Struttura comando** | Organigramma vertici militari siriani |
| **Piani di guerra** | Posizionamento truppe e fortificazioni |
| **Accesso politico** | Consigliere Ministro Difesa siriano |

### Contributo Guerra dei Sei Giorni (1967)

L'intelligence raccolta da Cohen e' ampiamente considerata cruciale per la rapida vittoria israeliana sulle Alture del Golan nel 1967.
