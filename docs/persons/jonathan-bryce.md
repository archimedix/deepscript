### Da Rackspace Cloud a OpenStack

Bryce inizia come web developer a Rackspace nel 2000. Insieme a Todd Morey concepisce una piattaforma di web hosting avanzata che diventa **Rackspace Cloud** (inizialmente nota come Mosso). Questa esperienza lo porta a guidare il progetto **OpenStack**, lanciato nel 2010 come collaborazione tra Rackspace e NASA per creare software open source per cloud computing.

### Leadership Open Source Infrastructure

Nel 2012 diventa Executive Director della OpenStack Foundation, poi rinominata **OpenInfra Foundation**. Nel giugno 2025, la Linux Foundation lo nomina Executive Director per Cloud & Infrastructure, sovrintendendo le principali iniziative open source infrastrutturali inclusa la **CNCF**. Mantiene contemporaneamente il ruolo in OpenInfra Foundation.
