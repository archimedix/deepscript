# Paul Desmarais Jr. (1954-)

> Erede della dinastia Desmarais: Chairman di Power Corporation, ponte Canada-Europa, membro CFR Global Board e Brookings Trustee.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 3 luglio 1954, Sudbury, Ontario, Canada |
| **Nazionalita'** | Canadese |
| **Famiglia** | Figlio maggiore di Paul Desmarais Sr. e Jacqueline Desmarais |
| **Coniuge** | [Helene Desmarais](helene-desmarais.md) (CEO CEIM) |
| **Figli** | 4: Paul, Alexandre, Nicolas, Charles-Edouard |
| **Residenza** | Montreal |
