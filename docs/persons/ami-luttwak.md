### Timeline Professionale

| Periodo | Ruolo | Organizzazione |

### Microsoft (2015-2020)

Come CTO di Microsoft Cloud Security Group, Luttwak ha guidato:
- **Azure Security Center** - Incubazione e sviluppo
- **Azure Sentinel** - SIEM cloud-native
- **Azure Advanced Threat Protection** - Sicurezza identita'
- Supervisione acquisizioni e progetti di incubazione
