# Roberto Calvi (1920 - 1982)

> "Il Banchiere di Dio" — presidente del Banco Ambrosiano, membro P2 n.1519, al centro della piu' grande frode bancaria occidentale, trovato impiccato sotto Blackfriars Bridge

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 13 aprile 1920, Milano |
| **Morte** | 18 giugno 1982, Londra (impiccato sotto Blackfriars Bridge) |
| **Famiglia** | Padre Giacomo Calvi, dirigente Banca Commerciale Italiana; genitori originari di Tremenico (Lecco) |
| **Coniuge** | Clara Canetti (sposata 1952) |
| **Figli** | Carlo Calvi, Anna Calvi |
| **Soprannome** | "Il Banchiere di Dio" (Banchiere di Dio) |

## La Morte a Blackfriars Bridge

### I Fatti

| Dato | Dettaglio |
|------|-----------|
| **Data** | 18 giugno 1982, mattina |
| **Luogo** | Sotto Blackfriars Bridge, Londra, sul Tamigi |
| **Modalita'** | Corpo appeso a un'impalcatura con una corda arancione |
| **Nelle tasche** | Mattoni, calcinacci, $15.000 in contanti, passaporto falso |
| **Simbolismo** | "Blackfriars" = "Frati Neri"; P2 si autodefiniva "i frati neri" |

### Indagini

| Anno | Sviluppo |
|------|----------|
| **1982** | Primo inquest: verdetto di suicidio |
| **1983** | Secondo inquest: verdetto aperto (open verdict) |
| **2002** | Perizia forense: assenza di vernice/ruggine sulle scarpe prova che non cammino' sull'impalcatura → omicidio |
| **2005** | Processo a Roma: 5 imputati (Carboni, Kleinszig, Diotallevi, Vittor, Calo') |
| **2007** | Tutti assolti per insufficienza di prove |
