# Sanae Takaichi (1961)

> Prima donna Primo Ministro del Giappone; protetta di Shinzo Abe, nazionalista conservatrice soprannominata "Iron Lady"

|---------|-----------|
| **Nascita** | 7 marzo 1961, Yamatokoriyama, Nara |
| **Nazionalita'** | JPN |
| **Coniuge** | Divorziata (ex marito: Masataka Yamamoto, parlamentare LDP) |
| **Residenza** | Tokyo |
