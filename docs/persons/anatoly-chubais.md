## Potere Esercitato

### Privatizzazione (1991-1998)

| Aspetto | Dettaglio |
|---------|-----------|
| **Voucher System** | Ogni cittadino riceve voucher come quota aziendale |
| **Risultato** | Pochi imprenditori connessi acquistarono voucher a prezzi stracciati |
| **Oligarchi** | Sistema creò classe di "kleptocrati" |
| **Critica** | "L'uomo più odiato di Russia" |

### Loans-for-Shares (1995)

| Asset | Acquirente | Pagato | Valore reale |
|-------|------------|--------|--------------|
| Norilsk Nickel | Potanin | $170M | $31B |
| Yukos | Khodorkovsky | ~$300M | miliardi |
| Sibneft | Berezovsky/Abramovich | $100M | miliardi |

**Schema:** Oligarchi prestano denaro al governo in cambio di azioni statali. Governo non ripaga, oligarchi tengono le aziende.

### Davos Pact: Salvare Yeltsin (1996)

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Yeltsin al 6-10% nei sondaggi, Zyuganov in testa |
| **Luogo** | World Economic Forum, Davos, febbraio 1996 |
| **Accordo** | Oligarchi finanziano, Chubais gestisce campagna |
| **Spesa stimata** | $700M - $2 miliardi (limite legale: $3M) |
| **Media** | Controllo totale TV (ORT, NTV) pro-Yeltsin |
| **Risultato** | Yeltsin vince 53.8% vs 40.7% |

### RAO UES (1998-2008)

| Aspetto | Dettaglio |
|---------|-----------|
| **Dimensione** | 70% capacità elettrica Russia, 96% rete alta tensione |
| **Riforma** | Diviso in 20 entità per privatizzazione |
| **Investimenti** | Raccolti oltre $30 miliardi privati |
| **Dissoluzione** | Luglio 2008, cessata esistenza giuridica |

### Rusnano (2008-2020)

| Aspetto | Dettaglio |
|---------|-----------|
| **Ruolo** | CEO/General Manager |
| **Missione** | Investimenti nanotecnologie |
| **Controversia** | Progetto chip MRAM fallito |
| **Causa 2025** | Russia richiede $152M danni |
