## Potere Esercitato

### Presidente Parlamento Europeo (2012-2017)

| Aspetto | Dettaglio |
|---------|-----------|
| **Prima elezione** | 17 gennaio 2012 (387 voti) |
| **Rielezione** | 1 luglio 2014 (409 voti) |
| **Significato storico** | Primo presidente rieletto nella storia del PE |
| **Contesto** | Crisi euro, emergenza migranti, Brexit |
| **Stile** | Assertivo, difesa ruolo PE vs Consiglio |

### Scontro con Berlusconi (2003)

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 2 luglio 2003 |
| **Contesto** | Berlusconi presidente di turno UE |
| **Incidente** | Berlusconi paragonò Schulz a un "kapò nazista" |
| **Conseguenze** | Scandalo diplomatico, scuse mai formali |
| **Impatto** | Schulz divenne simbolo anti-populismo |

### Candidatura Cancelliere (2017)

| Aspetto | Dettaglio |
|---------|-----------|
| **Nomina** | Gennaio 2017 |
| **"Schulz-Effekt"** | Impennata sondaggi SPD (+10%) |
| **Elezioni settembre** | SPD 20.5% (peggior risultato dal 1949) |
| **Dopo elezioni** | Rifiutò Große Koalition, poi accettò |
| **Rinuncia** | Febbraio 2018, non entrò nel governo |

### Friedrich-Ebert-Stiftung (2020-)

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondazione** | 1925, più antica fondazione politica tedesca |
| **Budget** | ~170 milioni EUR (2018) |
| **Presenza** | 100+ paesi |
| **Missione** | Formazione politica, promozione socialdemocrazia |
