## Pattern Osservati

### Unit 8200 → Startup Pipeline

Gil Shwed esemplifica il pattern israeliano:
1. **Unit 8200**: Sviluppo tecnologie classificate
2. **Industria civile**: Optrotech come ponte
3. **Startup**: Check Point (1993)
4. **IPO + Scale**: NASDAQ, crescita globale
5. **Successione**: Passaggio a ex-comandante Unit 8200 (Zafrir)

### Network Check Point

Tre co-fondatori, tre percorsi:
- **Shwed**: Unit 8200 → CEO 30 anni
- **Kramer**: Unit 8200 → serial entrepreneur (Imperva, Cato Networks)
- **Nacht**: [Talpiot](../agency/talpiot.md) → Chairman, poi healthtech
