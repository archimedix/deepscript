# Klaus Barbie (1913-1991)

> "Il Boia di Lione": criminale di guerra nazista che torturo' Jean Moulin, protetto dalla CIA, fuggito in Bolivia dove servi' dittature militari

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Nikolaus "Klaus" Barbie |
| **Nascita** | 25 ottobre 1913, Bad Godesberg, Germania |
| **Morte** | 25 settembre 1991, Lione, Francia (prigione) |
| **Nazionalita'** | Germania |
| **Grado** | SS-Hauptsturmfuhrer (Capitano) |
| **Soprannome** | "Boia di Lione" (Butcher of Lyon) |
| **Alias** | Klaus Altmann (Bolivia) |

---

## Crimini di Guerra (1942-1944)

### Gestapo Lione

| Aspetto | Dettaglio |
|---------|-----------|
| **Periodo** | Novembre 1942 - Agosto 1944 |
| **Eta'** | 29 anni all'arrivo |
| **Sede** | Hotel Terminus, Lione |
| **Ruolo** | Capo Gestapo locale |
| **Metodi** | Tortura personale di prigionieri |

### Vittime

| Categoria | Numero |
|-----------|--------|
| **Esecuzioni/omicidi** | 4.000+ |
| **Deportazioni ebrei** | 7.500 (maggioranza ad Auschwitz) |
| **Bambini di Izieu** | 41 bambini ebrei (3-13 anni) deportati 6 aprile 1944 |

### Il Caso Jean Moulin

| Aspetto | Dettaglio |
|---------|-----------|
| **Vittima** | Jean Moulin, eroe della Resistenza francese |
| **Cattura** | Giugno 1943 |
| **Interrogatorio** | 3 settimane di torture quotidiane |
| **Metodi** | Aghi sotto le unghie, dita fratturate, manette a vite |
| **Risultato** | Moulin non parlo' mai |
| **Morte** | 8 luglio 1943, durante trasferimento in Germania |

## Fuga e Protezione USA (1945-1951)

### Reclutamento US Intelligence

| Data | Evento |
|------|--------|
| **1945** | Catturato dagli americani |
| **1947** | Reclutato dal Counter Intelligence Corps (CIC) |
| **1947-51** | Agente per [Gehlen Organization](reinhard-gehlen.md) |
| **1949** | Francia chiede estradizione a John McCloy |
| **1949** | McCloy rifiuta, Barbie troppo "utile" |
| **Dic 1950** | CIC fornisce nuova identita': "Klaus Altmann" |
| **1951** | Fuga via ratlines verso Bolivia |

### La Ratline

| Aspetto | Dettaglio |
|---------|-----------|
| **Organizzatore** | Krunoslav Draganovic (prete croato) |
| **Percorso** | Germania → Austria → Italia → Argentina → Bolivia |
| **Finanziamento** | CIC/CIA |
| **Documenti** | Croce Rossa Internazionale (falsi) |

| **1971-78** | Banzer | Consulente controspionaggio |

| **1980-81** | Garcia Meza | Capo forza paramilitare |

### Attivita' Criminali

| Attivita' | Dettaglio |
|-----------|-----------|
| **Spionaggio** | Per CIA e BND |
| **Traffico armi** | Rete sudamericana |
| **Narcotraffico** | Connessioni con cartelli |
| **Paramilitari** | "Fidanzati della Morte" |
| **Che Guevara** | Possibile ruolo nella cattura (1967) |

### Connessione Cartelli

Secondo Der Spiegel (2022), Barbie aiuto' a creare infrastrutture per il traffico di cocaina e lavoro' con figure legate a Pablo Escobar.

## Cattura e Processo (1983-1987)

### Identificazione

| Data | Evento |
|------|--------|
| **1971** | Serge e Beate Klarsfeld identificano "Altmann" come Barbie |
| **1972** | Campagna pubblica per estradizione |
| **1983** | Nuovo governo boliviano estrada Barbie |
| **1983** | USA si scusano ufficialmente con la Francia |

### Processo di Lione (1987)

| Aspetto | Dettaglio |
|---------|-----------|
| **Inizio** | 11 maggio 1987 |
| **Sede** | Cour d'Assises del Rodano, Lione |
| **Accusa** | Pierre Truche |
| **Difesa** | Jacques Verges |
| **Finanziatore difesa** | Francois Genoud (finanziere pro-nazista svizzero) |
| **Capi d'accusa** | 41 crimini contro l'umanita' |

#### Capi d'Accusa Principali

| Crimine | Dettaglio |
|---------|-----------|
| Bambini di Izieu | 41 bambini deportati |
| Deportazione agosto 1944 | Ultimo convoglio verso Auschwitz |
| Deportazione UGIF Lione | Comunita' ebraica locale |

**Nota**: Barbie NON fu processato per Jean Moulin (prescrizione crimini di guerra, non crimini contro l'umanita').

### Verdetto

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 4 luglio 1987 |
| **Verdetto** | Colpevole |
| **Condanna** | Ergastolo |
| **Morte** | 25 settembre 1991 (cancro, in carcere) |
