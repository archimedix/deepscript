# Henrique Meirelles (1945 - )

> Revolving door paradigmatico: BankBoston → Banca Centrale Brasile → Ministro Finanze → KKR/Lazard

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 31 agosto 1945, Anapolis, Goias, Brasile |

| **Residenza** | Brasile |
