# Lei Jun (1969)

> Fondatore e CEO di Xiaomi, angel investor prolifico, figura chiave dell'ecosistema tech cinese e del modello "hardware + internet + new retail"

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 16 dicembre 1969, Xiantao, Hubei, Cina |
| **Famiglia** | Genitori entrambi insegnanti |
| **Nazionalita'** | Cinese |
| **Patrimonio** | ~$35.3 miliardi (2025), 5° piu' ricco in Cina |
