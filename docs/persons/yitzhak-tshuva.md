# Yitzhak Tshuva (1948 - )

> Miliardario israeliano, fondatore Delek Group e El-Ad, re del gas Leviathan e Tamar

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 1948, Tripoli, Libia |
| **Famiglia** | Famiglia ebraica libica, emigrata in Israele |
| **Coniuge** | Haya Tshuva |
| **Figli** | 5 |
| **Residenza** | Park Bavli, Tel Aviv |

## Formazione

Autodidatta. Non risulta formazione universitaria formale — self-made businessman partito dall'edilizia popolare a Netanya negli anni '70.
