# Josep Borrell (1947-)

> Presidente del Parlamento Europeo 2004-2007, Alto Rappresentante UE per gli Affari Esteri 2019-2024, voce dell'Europa nella crisi ucraina

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 24 aprile 1947, La Pobla de Segur, Lleida, Catalogna |
| **Nazionalità** | ESP |
| **Partito** | PSOE (Partido Socialista Obrero Español) |
| **Professione** | Ingegnere aeronautico, economista, professore |

## Potere Esercitato

### Parlamento Europeo (2004-2007)

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Allargamento UE a 25, Costituzione europea |
| **Sfida** | Referendum falliti Francia e Paesi Bassi (2005) |
| **Approccio** | Mediazione tra eurofederalisti e euroscettici |

### Alto Rappresentante UE (2019-2024)

| Aspetto | Dettaglio |
|---------|-----------|
| **Nomina** | Von der Leyen Commission |
| **Crisi principale** | Guerra Russia-Ucraina (2022-2024) |
| **Azioni** | Coordinamento sanzioni, sostegno militare Ucraina |
| **Stile** | Dichiarazioni dirette, talvolta controverse |
