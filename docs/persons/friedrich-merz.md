# Friedrich Merz (1955)

> Cancelliere tedesco 2025, ex chairman BlackRock Germania e Atlantik-Brücke. Atlantista, revolving door finanza-politica.

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 11 novembre 1955, Brilon, Germania |
| **Famiglia** | Classe media cattolica, padre giudice |
| **Coniuge** | Charlotte Merz (nata Gass), giudice |
| **Figli** | 3 |
| **Residenza** | Arnsberg, Nord Reno-Westfalia |

### Revolving Door

```
POLITICA (1989-2009)
        |
        v
FINANZA/CORPORATE (2009-2020)
  - Mayer Brown (M&A, banking)
  - BlackRock (chairman)
  - Board: BASF, Bayer, Deutsche Borse,
    Airbus, HSBC, Commerzbank, AXA
        |
        v
POLITICA (2018-oggi)
  - CDU Leader 2022
  - Cancelliere 2025
```

### Rivalita' con Merkel

Nel 2002 Angela Merkel lo estromise dalla guida del gruppo parlamentare CDU/CSU. Merkel scrisse nelle sue memorie (2024): *"C'era un problema, fin dall'inizio. Entrambi volevamo essere il capo."*
