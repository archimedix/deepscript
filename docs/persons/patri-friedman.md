## Potere Esercitato

### Governance sperimentale

| Aspetto | Dettaglio |
|---------|-----------|
| **Modello** | Trattare la governance come un prodotto iterabile (come software), finanziando alternative agli stati |
| **Veicolo** | Pronomos Capital: primo VC dedicato a charter cities |
| **Rete** | Pipeline Thiel → Seasteading → Future Cities → Pronomos, con backing costante del network libertario della Silicon Valley |
| **Portfolio** | Prospera (Honduras), Praxis, Itana, Metropolis Global — progetti di citta' autonome in paesi in via di sviluppo |

### Lignaggio intellettuale

Tre generazioni di pensiero economico libertario:
- **Milton Friedman** (nonno): monetarismo, libero mercato, Nobel 1976
- **David D. Friedman** (padre): anarcho-capitalismo, polycentric law
- **Patri Friedman**: applicazione pratica — costruire giurisdizioni alternative con venture capital
