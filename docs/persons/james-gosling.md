# James Gosling (1955)

> Il padre di Java: l'ingegnere canadese che ha creato il linguaggio piu' diffuso al mondo

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 19 maggio 1955, Calgary, Alberta, Canada |
| **Nazionalita'** | Canadese |
| **Residenza** | San Francisco Bay Area |

---

## Formazione

| Periodo | Istituzione | Dettaglio |
|---------|-------------|-----------|
| **1977** | [University of Calgary](../university/university-of-calgary.md) | BSc Computer Science |
| **1983** | [Carnegie Mellon University](../university/carnegie-mellon.md) | PhD Computer Science |

**Note accademiche:**
- A Calgary, scrisse software per analisi dati del satellite ISIS 2 per il dipartimento di fisica
- A CMU, sviluppo' Gosling Emacs (Gosmacs), una variante dell'editor Emacs
- PhD su porting UCSD Pascal p-code e emulatore PERQ su DEC VAX

### Sun Microsystems (1984-2010)

26 anni a Sun, dove creo' le tecnologie piu' influenti:

| Anno | Milestone |
|------|-----------|
| **1984** | Entra a Sun Microsystems |
| **1986** | Crea NeWS (Network extensible Window System) |
| **1991** | Avvia il progetto "Green" (poi Oak, poi Java) |
| **1994** | Completa design e primo compilatore Java |
| **1995** | Lancio pubblico Java: "write once, run anywhere" |
| **2006** | Sun open-source Java sotto GPL |
| **2010** | Lascia dopo acquisizione Oracle (2 aprile) |

### Uscita da Sun/Oracle

Gosling lascio' Sun il 2 aprile 2010, subito dopo l'acquisizione da parte di Oracle, citando riduzioni di stipendio, status e potere decisionale, oltre a sfide etiche con la nuova gestione.
