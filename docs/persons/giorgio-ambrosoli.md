# Giorgio Ambrosoli (1933 - 1979)

> Avvocato milanese, commissario liquidatore della Banca Privata Italiana di Sindona, assassinato dalla mafia per aver scoperto la rete criminale dietro l'impero bancario — "Eroe borghese"

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 17 ottobre 1933, Milano |
| **Morte** | 11 luglio 1979, Milano (assassinato) |
| **Famiglia** | Borghesia milanese, cattolica, conservatrice |
| **Coniuge** | Annalori Ambrosoli |
| **Figli** | Umberto Ambrosoli (avvocato, politico), Francesca, Filippo |
| **Riconoscimenti** | Medaglia d'oro al valor civile (alla memoria) |

## Carriera e Ruoli

### Timeline Professionale

### Incarico alla BPI

| Anno | Sviluppo |
|------|----------|
| **27 set 1974** | Nominato commissario liquidatore unico della BPI dal governatore Banca d'Italia Guido Carli |
| **1974-1979** | Ricostruisce l'intera rete di manipolazioni criminali di Sindona |
| **1978-1979** | Fornisce prove al Dipartimento di Giustizia USA per il processo contro Sindona (crollo Franklin National Bank) |
| **1979** | Riceve minacce telefoniche; in una telefonata registrata, il killer invoca il nome di Giulio Andreotti |
| **11 lug 1979** | Assassinato sotto casa, poche ore dopo aver parlato con le autorita' americane |

## Potere Esercitato

### La Liquidazione che Svelo' il Sistema

| Aspetto | Dettaglio |
|---------|-----------|
| **Buco scoperto** | $180 milioni in perdite da speculazioni valutarie |
| **Rete criminale** | Ricostrui' i flussi mafia → BPI → IOR → Svizzera |
| **Prove internazionali** | Forni' al DOJ USA le prove per condannare Sindona per il crollo della Franklin National Bank |
| **Resistenza** | Rifiuto' pressioni politiche e minacce per 5 anni |
| **Costo personale** | Sapeva di rischiare la vita; scrisse alla moglie: "non ho mai pensato a me ma ho pensato che il mio lavoro era necessario" |
