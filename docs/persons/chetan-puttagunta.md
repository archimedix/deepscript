### Portfolio: L'Architetto dell'Open Source Enterprise

Puttagunta e' una figura chiave nel finanziamento del modello "open source commerciale". I suoi investimenti principali definiscono un pattern:

| Azienda | Modello | Exit/Status |
|---------|---------|-------------|
| **MuleSoft** | API integration, open core | IPO 2017, acquisita da Salesforce $6.5B |
| **MongoDB** | Database, SSPL license | IPO 2017 (NASDAQ: MDB) |
| **Elastic** | Search/analytics, SSPL→AGPL | IPO 2018 (NYSE: ESTC) |
| **Airbyte** | Data integration, open source | Growth stage |
| **LangChain** | LLM framework, open source | Growth stage |
| **Sentry** | Error monitoring, open source | Growth stage |

### Pattern Investimento

Il filo conduttore e' il modello **open source → enterprise SaaS**:
1. Investimento in progetto open source con traction degli sviluppatori
2. Costruzione di un layer enterprise (cloud, security, compliance)
3. IPO o acquisizione

E' significativo che sia MongoDB che Elastic abbiano entrambe adottato la SSPL per proteggersi dagli hyperscaler — un pattern che Puttagunta ha sostenuto e difeso come investitore.
