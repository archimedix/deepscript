# Kiichi Miyazawa (1919-2007)

> Architetto della Trilateral giapponese: PM, doppio Min. Finanze, ponte Giappone-USA dal 1961 al 2001

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 8 ottobre 1919, Tokyo |
| **Morte** | 28 giugno 2007, Tokyo (87 anni) |
| **Famiglia** | Dinastia politica di Hiroshima |
| **Padre** | Parlamentare (Hiroshima) |
| **Madre** | Famiglia politica prominente |
| **Moglie** | Yoko |
| **Figli** | Hiro (architetto), Keiko (moglie di diplomatico USA Christopher LaFleur) |

---

## Formazione

| Periodo | Istituzione | Dettaglio |
|---------|-------------|-----------|
| **-** | Musashi Higher School | Scuola superiore |
| **-1942** | Tokyo Imperial University | Facolta' di Legge |
| **1939** | Japan-America Student Conference | Washington D.C. - ispirazione per inglese fluente |

**Nota:** Continuo' a studiare inglese anche durante la guerra contro USA - divenne interprete chiave.

## Potere Esercitato

### Summit Ikeda-Kennedy (1961)

| Aspetto | Dettaglio |
|---------|-----------|
| **Evento** | "Yacht talks" su Honey Fitz |
| **Ruolo Miyazawa** | Unico traduttore di Ikeda |
| **Significato** | Ponte linguistico Giappone-USA al massimo livello |

### Trilateral Commission (1973-)

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondazione** | Presente a Pocantico Hills (luglio 1972) |
| **Ruolo** | Primo giapponese in forum paritario con USA-Europa |
| **Advocacy** | Peso voto paesi esportatori di capitale |
| **1980** | Discorso a Londra come PM |

### PM 1991-1993

| Decisione | Impatto |
|-----------|---------|
| **Comfort women** | Prima scusa ufficiale (1992, Corea del Sud) |
| **Peacekeeping** | Legge per missioni ONU all'estero |
| **Trade USA** | Negoziato accordo commerciale |
| **Riforme bancarie** | Prime misure contro crisi anni '90 |

**Fine mandato:** Sfiducia 1993 - fine 38 anni di governo LDP ininterrotto.

### New Miyazawa Initiative (1998)

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Crisi finanziaria asiatica 1997-98 |
| **Annuncio** | Ottobre 1998, IMF-World Bank Annual Meeting |
| **Totale** | $30 miliardi |
| **Medio-lungo termine** | $15B per recovery economica Asia |
| **Breve termine** | $15B per capitale emergenza |
| **Controparte USA** | Robert Rubin (critico verso lentezza Giappone) |
