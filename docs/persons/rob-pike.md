## Potere Esercitato

### Infrastruttura Digitale

| Aspetto | Dettaglio |
|---------|-----------|
| **UTF-8** | Co-inventore dell'encoding che ha reso possibile un web multilingue. >98% delle pagine web usano UTF-8 |
| **Go** | Linguaggio alla base di Docker, Kubernetes, Terraform, Prometheus — l'intero stack cloud-native |
| **Plan 9** | Concetti (tutto-e'-un-file, namespaces, 9P) ripresi in Linux (procfs, FUSE, containers) |
| **Filosofia di design** | I principi di Pike sulla semplicita' hanno influenzato una generazione di ingegneri del software |

Pike rappresenta il ponte tra l'era Bell Labs (ricerca pura, sistemi operativi) e l'era Google (cloud, infrastruttura di scala). UTF-8 e Go sono le sue creazioni piu' pervasive: una codifica che il mondo intero usa senza saperlo, e un linguaggio che fa funzionare il cloud.

### Da Bell Labs a Google: continuita'

Pike segui' lo stesso percorso di Ken Thompson: Bell Labs → Google. Insieme portarono la cultura Unix — semplicita', composabilita', tool piccoli — dentro l'infrastruttura di Google, poi la esportarono al mondo con Go.
