# David Mondshine

> Direttore della Ohr Avner Foundation e CEO della Federation of Jewish Communities della CIS: l'uomo che gestisce la piu' grande rete educativa ebraica dell'ex Unione Sovietica per conto di Lev Leviev.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Rabbi Dovid (David) Mondshine |
| **Nazionalita'** | Israeliana |
| **Residenza** | Mosca, Russia |
| **Padre** | Rabbi [Yehoshua Mondshine](https://en.wikipedia.org/wiki/Yehoshua_Mondshine) (1947-2014), studioso Chabad |
| **Famiglia** | Moglie Sarah Chana Mondshine; Shterna Sarah Mondshine (familiare coinvolta nelle operazioni) |
