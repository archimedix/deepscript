## Potere Esercitato

### Infrastruttura Digitale Globale

| Aspetto | Dettaglio |
|---------|-----------|
| **Linguaggio C** | Fondamento di quasi tutti i sistemi operativi, database, compilatori. Linux, Windows, PostgreSQL, Python runtime: tutti scritti in C |
| **Unix** | Architettura alla base di >90% dei server internet, tutti gli smartphone (iOS=derivato BSD, Android=Linux) |
| **Filosofia** | Il modello "small tools, composable" ha definito come si progetta software da 50+ anni |

Ritchie e' stato il ponte tra la matematica pura (Harvard) e l'ingegneria pratica (Bell Labs). Il C ha tradotto le idee in strumenti usabili, democratizzando la programmazione di sistemi.

### Morte e Oblio Mediatico

Ritchie mori' una settimana dopo Steve Jobs (5 ottobre 2011). La morte di Jobs domino' i media globali; quella di Ritchie passo' quasi inosservata — nonostante ogni prodotto Apple funzionasse su fondamenta che Ritchie aveva costruito.
