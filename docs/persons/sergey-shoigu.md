# Sergey Shoigu (1955-)

> Ministro della Difesa russo per 12 anni, co-fondatore di Russia Unita, amico personale di Putin, unico non-russo etnico ai vertici del potere, ricercato dall'ICC per crimini di guerra

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 21 maggio 1955, Chadan, Oblast' Autonoma di Tuva |
| **Etnia** | Meta' tuvano (padre), meta' ucraino-russo (madre) |
| **Padre** | Kuzhuget Shoigu (1921-2010), editore giornale, segretario Partito Comunista Tuva |
| **Madre** | Alexandra Yakovlevna Shoigu (1924-2011), ucraina |
| **Moglie** | Irina Alexandrovna Shoigu (nata Antipina) |
| **Figlie** | Yulia (1977), Kseniya (1991) |
| **Grado** | Generale dell'Esercito (mai servito in esercito) |
| **Status legale** | Mandato di arresto ICC (giugno 2024) |

## Potere Esercitato

### Amicizia con Putin

| Aspetto | Dettaglio |
|---------|-----------|
| **Origine** | Alleanza politica dal 1999-2000 |
| **Attivita' comuni** | Hockey, escursioni in Siberia, vacanze |
| **Speculazioni** | In passato considerato potenziale successore |
| **Perche' non presidente** | Etnia non-russa rende impossibile candidatura |
| **Protezione** | Sopravvissuto a fallimenti guerra Ucraina |

### Ministero della Difesa (2012-2024)

| Aspetto | Dettaglio |
|---------|-----------|
| **Nomina** | 6 novembre 2012, dopo riforme Serdyukov |
| **Paradosso** | Mai servito in esercito, ma Generale dell'Esercito |
| **Obiettivo** | Modernizzazione forze armate |
| **Risultato dichiarato** | "Secondo esercito piu' potente al mondo" |
| **Prova reale** | Invasione Ucraina 2022 |
| **Critica** | Fallimenti logistici, corruzione sistematica |

### Conflitto con Prigozhin e Caduta

| Data | Evento |
|------|--------|
| **2022-2023** | Prigozhin accusa pubblicamente Shoigu di incompetenza |
| **Giu 2023** | Ammutinamento Wagner, marcia su Mosca |
| **Target** | Shoigu e Gerasimov |
| **Ago 2023** | Morte Prigozhin in incidente aereo |
| **Mag 2024** | Putin sostituisce Shoigu con Belousov |
| **Ricollocazione** | Segretario Consiglio di Sicurezza (ruolo meno potente) |
