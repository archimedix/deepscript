# Rob Walton (1944)

> Erede Walmart, ex-chairman del piu' grande retailer del mondo, proprietario Denver Broncos, filantropo ambientalista

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 27 ottobre 1944, Tulsa, Oklahoma |
| **Famiglia** | Walton (figlio maggiore di Sam Walton) |
| **Coniuge** | Melani Lowman-Walton (dal 2005); ex Carolyn Funk, ex Patricia Rawlings |
| **Figli** | 3 (Sam, Carrie Walton Penner, Ben) |
| **Residenza** | Arizona |
| **Patrimonio** | $115-137 miliardi (2025) — 13° al mondo |

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1969-1978** | Partner | Conner & Winters (Tulsa) |
| **1978-1982** | Senior VP, General Counsel | Walmart |
| **1982-1992** | Vice Chairman | Walmart |
| **1992-2015** | Chairman | Walmart |
| **2015-oggi** | Board Member | Walmart |
| **2022-oggi** | Principal Owner | Denver Broncos |

### Tappe Chiave
