# Carlo De Benedetti (1934-)

> Industriale e finanziere italiano, "L'Ingegnere", cofondatore ERT, artefice del rilancio Olivetti, editore di La Repubblica, protagonista dei "100 giorni" alla FIAT

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 14 novembre 1934, Torino, Italia |
| **Nazionalità** | ITA-CHE (doppia cittadinanza) |
| **Soprannome** | "L'Ingegnere" |
| **Origini** | Famiglia ebrea di banchieri e finanzieri |
| **Professione** | Industriale, ingegnere, editore |

## Potere Esercitato

### I "100 Giorni" alla FIAT (1976)

| Aspetto | Dettaglio |
|---------|-----------|
| **Inizio** | 4 maggio 1976 |
| **Fine** | 25 agosto 1976 |
| **Durata** | ~100 giorni |
| **Nomina** | Giovanni Agnelli (6% FIAT) |
| **Motivo uscita** | Divergenze con la famiglia Agnelli |
| **Conseguenza** | Vendette quota, acquisì CIR |
| **Eredità** | Leggenda di rottura con l'establishment torinese |

### Rilancio Olivetti (1978-1996)

| Aspetto | Dettaglio |
|---------|-----------|
| **Acquisizione** | 1978 |
| **Trasformazione** | Da macchine da scrivere a computer |
| **Risultato** | Fatturato da $1 a $5 miliardi (1978-1986) |
| **Metodo** | Ristrutturazione rapida e decisa |
| **Apice** | Leader europeo PC anni '80 |
| **Declino** | Anni '90, uscita 1996 |
