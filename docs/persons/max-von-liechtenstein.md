# Prince Maximilian of Liechtenstein (1969)

> CEO di LGT Group 2006-2020, poi Chairman. Ha trasformato LGT in colosso da $367B AUM. Fondatore di Lightrock (impact investing).

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 16 maggio 1969, Liechtenstein |
| **Famiglia** | Casa del Liechtenstein, secondogenito di Hans-Adam II |
| **Coniuge** | Angela Brown (m. 2000), fashion designer panamense-americana |
| **Figli** | Alfons (2001) |
| **Residenza** | Vaduz, Liechtenstein |
