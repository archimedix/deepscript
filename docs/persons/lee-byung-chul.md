# Lee Byung-chul (1910 - 1987)

> Fondatore Samsung Group, architetto del modello chaebol-Stato sudcoreano

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 12 febbraio 1910, Uiryeong County, Gyeongsang Meridionale |
| **Morte** | 19 novembre 1987, Seoul |
| **Famiglia** | Clan Gyeongju Lee, famiglia yangban proprietaria terriera |
| **Coniuge** | Park Du-eul (prima moglie), Kuroda (seconda moglie) |
| **Figli** | Lee Maeng-hee, Lee Chang-hee, Lee Kun-hee (successore), Lee In-hee + altri |
| **Pseudonimo** | Ho-Am |

## Potere Esercitato

### Il Modello Chaebol

| Aspetto | Dettaglio |
|---------|-----------|
| **Filosofia** | "Governo e industria sono come marito e moglie" |
| **Meccanismo** | Conglomerato familiare con protezione statale |
| **Settori** | Trading, tessile, assicurazioni, elettronica, costruzioni |
| **Controllo** | Partecipazioni incrociate tra societa' del gruppo |
| **Eredita'** | Samsung diventa il piu' grande chaebol coreano, ~20% PIL sudcoreano |

### Costruzione dell'Impero

| Aspetto | Dettaglio |
|---------|-----------|
| **1938** | Samsung Trading Co. (commercio di pesce essiccato e noodles) |
| **1950s** | Espansione in tessile, zucchero, assicurazioni |
| **1960s** | Patto con Park: Samsung esegue piani quinquennali, riceve protezione |
| **1969** | Fondazione Samsung Electronics |
| **1970s-80s** | Elettronica, semiconduttori, costruzioni navali |
