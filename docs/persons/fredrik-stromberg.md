## Potere Esercitato

### Privacy e sicurezza digitale

| Aspetto | Dettaglio |
|---------|-----------|
| **Mullvad VPN** | VPN no-log tra le piu' rispettate al mondo. Nessun investitore esterno, 100% proprietario |
| **System Transparency** | Architettura di sicurezza per server verificabili |
| **Sigsum** | Log di trasparenza crittografico |
| **Tillitis Key** | Spin-off hardware security, chiave di sicurezza open source |

### Filosofia

Fondazione di Mullvad come atto politico diretto contro la legislazione svedese sulla sorveglianza (FRA law). Rifiuto sistematico di acquisizioni e investimenti esterni.
