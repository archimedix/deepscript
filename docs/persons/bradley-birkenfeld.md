## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1990s** | Currency trading | State Street Bank & Trust |
| **1996-1998** | Private banker | Credit Suisse |
| **1998-2001** | Private banker | Barclays Bank |
| **2001-2006** | Private banker | UBS Geneva |
| **2007** | Whistleblower | DOJ, IRS |
| **2010-2012** | Detenuto | Federal prison |
| **2012-oggi** | Autore, attivista | Whistleblower advocacy |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1994** | Prima segnalazione FBI (State Street) |
| **2001** | Entra in UBS Geneva |
| **2007** | "Birkenfeld Disclosure" al DOJ |
| **2008** | UBS paga 780M USD, rivela nomi evasori |
| **2009** | Condannato per conspiracy |
| **2010** | Inizia detenzione (30 mesi) |
| **2012** | Rilasciato |
| **2012** | IRS assegna 104M USD reward |
| **2018** | Recupero totale stimato: 16.19 miliardi USD |

## Reward e Conseguenze Legali

### Per Birkenfeld

| Aspetto | Dettaglio |
|---------|-----------|
| **Condanna** | 1 count fraud conspiracy |
| **Pena** | 30 mesi detenzione + 30.000 USD multa |
| **Motivazione condanna** | Non rivelo' inizialmente cliente principale (Olenicoff) |
| **Reward IRS** | 104 milioni USD (26% di 400M recuperati) |

### Status in Svizzera

| Aspetto | Dettaglio |
|---------|-----------|
| **Considerazione** | "Traditore" dalla comunita' bancaria |
| **Pena potenziale** | 3.5 anni se fosse rimasto in Svizzera |
