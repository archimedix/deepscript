# J. Edgar Hoover (1895-1972)

> Direttore FBI per 48 anni (1924-1972), massone 33° grado e Grand Cross, architetto del COINTELPRO

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 1 gennaio 1895, Washington D.C. |
| **Morte** | 2 maggio 1972, Washington D.C. |
| **Famiglia** | Padre Dickerson Hoover (funzionario federale), madre Annie Scheitlin |
| **Coniuge** | Mai sposato |
| **Figli** | Nessuno |
| **Nazionalita'** | USA |
