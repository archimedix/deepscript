# António Costa (1961-)

> Presidente del Consiglio Europeo 2024-, primo socialista in questo ruolo, Premier portoghese più longevo (9 anni), radici indo-portoghesi (Goa)

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 17 luglio 1961, Lisbona, Portogallo |
| **Nazionalità** | PRT |
| **Origini** | Indo-portoghese (Goa) |
| **Padre** | Orlando da Costa (scrittore, comunista, goanese) |
| **Madre** | Maria Antónia Palla (giornalista, attivista) |
| **Partito** | PS (Partido Socialista) |
| **Professione** | Avvocato |
| **Soprannome** | "Babush" (in Konkani: "giovane amato") |

## Potere Esercitato

### Primo Ministro Portogallo (2015-2024)

| Aspetto | Dettaglio |
|---------|-----------|
| **Inizio** | 26 novembre 2015 |
| **Coalizione** | "Geringonça" (PS + sinistra) |
| **Significato** | Prima alleanza PS-PCP-Bloco |
| **Durata** | Quasi 9 anni (secondo più lungo) |
| **Record** | Più longevo in democrazia portoghese |
| **Fine** | Dimissioni novembre 2023 |
| **Caretaker** | Fino 2 aprile 2024 |

### "Geringonça" (2015-2019)

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome** | "Aggeggio" in portoghese |
| **Composizione** | PS + PCP + Bloco de Esquerda |
| **Innovazione** | Sinistra unita in Portogallo |
| **Risultato** | Stabilità, uscita dalla crisi |
