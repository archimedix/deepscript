# Gennady Timchenko (1952)

> Oligarca dell'energia, co-fondatore Gunvor, azionista Novatek, partner di judo di Putin, architetto del trading petrolifero russo

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 9 novembre 1952, Leninakan, Armenia SSR |
| **Nazionalità** | Russa, finlandese |
| **Formazione** | Ingegneria elettrica, Università Meccanica di Leningrado |
| **Patrimonio** | ~$23 miliardi (2025) |
| **Residenza** | Ginevra, Svizzera (pre-sanzioni) |
| **Sanzioni** | USA (2014), UK, EU (2022) |

---

## Formazione

| Periodo | Istituzione | Dettaglio |
|---------|-------------|-----------|
| **1970s** | Università Meccanica di Leningrado | Laurea in Ingegneria Elettrica |

**Background:**
Figlio di ufficiale militare sovietico. Cresciuto in Ucraina e Germania Est. Dopo la laurea lavorò agli Izhorsk Plants (costruttore di reattori nucleari).

---

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1976-1984** | Ingegnere | Izhorsk Plants, San Pietroburgo |
| **1984-1988** | Funzionario | Ministero Commercio Estero URSS |
| **1988-1991** | Vice Direttore | Kirishineftekhimexport (export petrolio) |
| **1991-1997** | Trader | Export petrolio indipendente |
| **1997-2014** | Co-fondatore | Gunvor Group |
| **2007-oggi** | Fondatore | Volga Group |
| **2009-oggi** | Board Member | Novatek |
| **2010-oggi** | Azionista | Sibur Holding |
| **2011-oggi** | Chairman, Presidente | SKA Saint Petersburg |
| **2012-oggi** | Chairman | KHL (Kontinental Hockey League) |

### Ascesa Oligarchica

| Anno | Evento |
|------|--------|
| **1988** | Vice Direttore Kirishineftekhimexport, accesso export petrolio |
| **1991** | Incontra Putin a San Pietroburgo; riceve licenza export petrolio |
| **1997** | Co-fonda Gunvor con Torbjörn Törnqvist |
| **2000s** | Gunvor gestisce 1/3 delle esportazioni petrolifere russe via mare |
| **2007** | Fonda Volga Resources (poi Volga Group) |
| **2009** | Acquista 5% Novatek per $460M, poi ulteriore 13% |
| **2010-11** | Acquisisce Sibur da Gazprombank con Mikhelson |
| **2013** | Co-acquista Hartwall Arena Helsinki e Jokerit |
| **Marzo 2014** | Vende quota Gunvor (43.5%) un giorno prima delle sanzioni USA |
