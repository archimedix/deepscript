# Bentzi Gopstein (1969 - )

> Fondatore e direttore di Lehava, la piu' grande organizzazione estremista violenta in Israele secondo il Dipartimento di Stato USA. Ex membro Kach, kahanista di seconda generazione. Bandito dalla Corte Suprema, sanzionato da USA, UE, UK e Canada.

### Lehava (2005-presente)

Gopstein fondo' **Lehava** (acronimo di "Prevenzione dell'Assimilazione in Terra Santa"):

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondazione** | 2005 |
| **Membri** | 10.000+ registrati |
| **Missione** | Impedire relazioni tra ebrei e non-ebrei (spec. arabi) |
| **Status USA** | "Largest violent extremist organization in Israel" (Dept. of State, 2024) |

Attivita' principali:
- Campagne contro relazioni miste ebraico-arabe
- Rifugi per donne ebree che lasciano relazioni con arabi (gestiti da Anat Gopstein)
- Proteste, volantinaggi, intimidazioni
- Novembre 2014: tre membri condannati per incendio doloso a scuola bilingue arabo-ebraica a Gerusalemme

### Bando dalla Corte Suprema (2019)

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 25 agosto 2019 |
| **Voto** | 9-0 (unanime) |
| **Motivo** | Incitamento sistematico al razzismo |
| **Giudice** | Presidente Esther Hayut cito' "decine" di dichiarazioni razziste |

### Procedimento penale

| Data | Evento |
|------|--------|
| **1990** | Arrestato per sospetto omicidio coppia araba (rilasciato, caso irrisolto) |
| **1994** | Detenzione amministrativa per Kach |
| **2014** | Arrestato per incitamento dopo incendio scuola bilingue |
| **2017** | Arrestato con 14 membri Lehava per minacce |
| **2019** | Incriminato: incitamento al terrorismo, violenza, razzismo (dichiarazioni 2012-2017) |
| **2024 (gen)** | Condannato per incitamento al razzismo; assolto dagli altri capi |
| **2025 (feb)** | Corte distrettuale di Gerusalemme ribalta la condanna |
