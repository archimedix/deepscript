# King Abdullah II bin Al-Hussein (1962)

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 30 gennaio 1962, Amman, Giordania |
| **Famiglia** | Hashemiti - discendente del Profeta Muhammad (41a generazione) |
| **Padre** | Re Hussein bin Talal |
| **Madre** | Principessa Muna Al Hussein (Antoinette Gardiner, britannica) |
| **Coniuge** | Queen Rania Al Abdullah (dal 1993) |
| **Figli** | 4: Hussein, Iman, Salma, Hashem |
| **Residenza** | Amman |
