# Antonio Tajani (1953-)

> Presidente del Parlamento Europeo 2017-2019, Vice-Presidente Commissione Europea, Ministro degli Esteri italiano, erede politico di Berlusconi

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 4 agosto 1953, Roma, Italia |
| **Nazionalità** | ITA |
| **Partito** | Forza Italia (co-fondatore 1994, Segretario 2023-) |
| **Professione** | Giornalista, politico |
| **Lingue** | Italiano, francese, spagnolo, inglese |
