## Bilderberg

Beatrix rappresenta la continuita' dinastica Orange-Nassau nel Bilderberg:

| Anno | Location | Note |
|------|----------|------|
| **1962** | Prima partecipazione | Accompagnando il padre Bernhard |
| **1997** | - | Partecipazione confermata |
| **2000** | - | Partecipazione confermata |
| **2006** | - | Partecipazione confermata |
| **2008-2015** | Varie | Partecipazioni consecutive |
| **2012** | Chantilly, Virginia | **Golden jubilee**: 50 anni di partecipazione |

> **Pattern dinastico**: Bernhard (fondatore 1954) -> Beatrix (1962-2015) -> Willem-Alexander (2008, 2016, 2022, 2024)

## Club of Rome

Beatrix e' membro onorario del Club of Rome. Ha partecipato a eventi del Club, incluso un incontro ad Amsterdam con Mikhail Gorbachev (anch'egli membro onorario).
