# Edward Snowden (1983-)

> Whistleblower NSA, figura centrale nel dibattito globale su sorveglianza di massa e privacy

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 21 giugno 1983, Elizabeth City, North Carolina |
| **Nazionalita'** | USA (cittadino russo dal 2022) |
| **Coniuge** | Lindsay Mills (2017) |
| **Residenza** | Mosca, Russia (dal 2013) |

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **2004** | Recluta | US Army Special Forces |
| **2005** | Security guard | Center for Advanced Study of Language (NSA) |
| **2006-2009** | Network security technician | CIA |
| **2007-2009** | Operative sotto copertura diplomatica | CIA Geneva |
| **2009-2013** | Infrastructure analyst | NSA (via Dell) |
| **2013** | Systems administrator | NSA (via Booz Allen Hamilton) |
| **2014-2022** | Board member, President (2016-2022) | Freedom of the Press Foundation |

### Eventi Chiave

| Anno | Evento |

| **Giugno 2013** | Fuga a Mosca via WikiLeaks |
| **2014** | Asilo temporaneo Russia |
| **2020** | Residenza permanente Russia |
| **2022** | Cittadinanza russa |

## Rivelazioni 2013

### Programmi NSA Esposti

| Programma | Descrizione |
|-----------|-------------|
| **PRISM** | Data mining da Apple, Google, Facebook, Microsoft |
| **XKeyscore** | Ricerca in tempo reale su comunicazioni internet |
| **Boundless Informant** | Analisi metadati globale |
| **Tempora** | Intercettazione cavi sottomarini (con GCHQ UK) |

### Impatto

| Aspetto | Risultato |
|---------|-----------|
| **USA Freedom Act (2015)** | Limita raccolta metadati NSA |
| **Pulitzer 2014** | Guardian e Washington Post per servizio pubblico |
| **Citizenfour (2015)** | Oscar miglior documentario |
