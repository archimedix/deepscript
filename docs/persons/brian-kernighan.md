## Potere Esercitato

### Influenza Culturale sul Software

| Aspetto | Dettaglio |
|---------|-----------|
| **K&R** | "The C Programming Language" ha formato generazioni di programmatori; lo stile "K&R" e' ancora uno standard di formattazione codice |
| **Filosofia Unix** | I suoi libri hanno codificato e diffuso la filosofia Unix (small tools, pipes, composability) |
| **AWK** | Strumento onnipresente per data processing, precursore di Perl e Python per scripting |
| **Insegnamento** | Dal 2000 a Princeton insegna "Computers in Our World" a non-specialisti — forma la classe dirigente americana |

Kernighan non ha inventato C ne' Unix, ma e' stato il traduttore che ha reso queste tecnologie accessibili al mondo. Senza K&R, il C sarebbe rimasto un linguaggio interno di Bell Labs.
