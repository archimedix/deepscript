# Xabier de Irala (1947-)

> Manager basco-americano che privatizzò Iberia, primo CEO della compagnia proveniente dal settore privato

| Aspetto | Dettaglio |
|---------|-----------|

| **Nazionalita'** | Spagnola-Americana |
| **Origine** | Basco, figlio di esule politico |
| **Padre** | Antón de Irala, segretario del primo Lendakari basco |
| **Lingue** | Spagnolo, Inglese, Francese, Basco |

## Note

### Background Familiare

Figlio di Antón de Irala, figura storica del nazionalismo basco, segretario del primo Lendakari (presidente del governo basco) Jose Antonio Aguirre. Il padre era in esilio a New York, dove nacque Xabier.

### Fondatore Exceltur

Presidente fondatore di Exceltur (Alianza para la Excelencia Turística), associazione per l'eccellenza turistica spagnola.
