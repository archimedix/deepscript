# Otto Wolff von Amerongen (1918 - 2007)

> Industriale tedesco, "ministro ombra del commercio Est-Ovest", architetto delle relazioni economiche tra Germania Ovest e blocco sovietico

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 6 agosto 1918, Colonia |
| **Morte** | 8 marzo 2007, Colonia |
| **Famiglia** | Figlio di Otto Wolff (industriale dell'acciaio) e Else Pieper von Amerongen |
| **Coniuge** | Eva Hehemann (1943-1966, divorziato); Winnie Greger (1966) |
| **Figli** | Claudia, Regine, Jeanne |
