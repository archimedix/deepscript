# Marco Tronchetti Provera (1948)

> Il genero che eredito' Pirelli: da CEO trentennale a protagonista della saga Telecom Italia, oggi in lotta con i cinesi per il controllo del gruppo.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 18 gennaio 1948, Milano |
| **Nazionalita'** | Italiana |
| **Formazione** | Laurea Economia, Bocconi (1971) |
| **Matrimoni** | Tre: Letizia Rittatore Vonwiller, Cecilia Pirelli (1987), Afef Jnifen (2001-2018) |
| **Figli** | Tre (con Cecilia Pirelli) |

|------|--------|

| Anno | Evento |
|------|--------|
| **2015** | ChemChina acquisisce Pirelli per $7 miliardi |
| **2015** | Tronchetti rimane CEO, negozia autonomia |
| **2018** | ChemChina fusa in Sinochem |
| **2023** | Passa a Executive Vice Chairman, Casaluci CEO |
| **2023** | Denuncia interferenze cinesi alla commissione golden power |
| **2023** | Governo italiano esercita golden power su Pirelli |
| **2025** | Tensioni con Sinochem (37%) per governance |

## Pattern: Il Capitalismo Familiare Italiano

Tronchetti Provera rappresenta un classico caso di successione per matrimonio:

| Caratteristica | Dettaglio |
|----------------|-----------|
| **Ingresso** | Manager esterno, sposa la figlia del fondatore |
| **Successione** | Nominato CEO quando il suocero si ritira |
| **Durata** | 30 anni di controllo (1992-2022) |
| **Holding** | Camfin controlla ~26.4% Pirelli |
| **Forum** | Trilateral (non Bilderberg) |

## La Questione Cinese

### Il Deal ChemChina (2015)

| Aspetto | Dettaglio |
|---------|-----------|
| **Valore** | $7 miliardi |
| **Acquirente** | ChemChina (stato cinese) |
| **Quota** | 100% delisting, poi riIPO |
| **Condizione** | Tronchetti rimane CEO |

### La Crisi con Sinochem (2023-2025)

| Aspetto | Dettaglio |
|---------|-----------|
| **Quota cinese** | 37% (Sinochem via Marco Polo International) |
| **Quota Camfin** | ~26.4% (Tronchetti) |
| **Denuncia** | Interferenze operative, richiesta integrazione IT |
| **Golden Power** | Governo protegge tecnologie sensori pneumatici |
| **Esito** | Autonomia gestionale garantita a Pirelli |
