## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| pre-FSF | Software developer | Aziende proprietarie (mentre contribuiva a GNU/Linux nel tempo libero) |
| **~2017-** | Senior Systems Administrator | [Free Software Foundation](../foundation/free-software-foundation.md) |
| **2021-** | Board member (union seat) | [Free Software Foundation](../foundation/free-software-foundation.md) |
| **2025-** | Board President | [Free Software Foundation](../foundation/free-software-foundation.md) |

### Ruolo alla FSF

| Aspetto | Dettaglio |
|---------|-----------|
| **SysAdmin** | Mantiene l'intera infrastruttura FSF usando solo software libero da ~8 anni |
| **Union seat** | Primo rappresentante sindacale nel board (2021), quando il seggio fu creato |
| **Presidente** | Confermato all'unanimita' dal board nel settembre 2025. Primo membro dello staff a diventare presidente |
| **Predecessore** | Geoffrey Knauth (presidente 2020-2025) |

### Contributi tecnici

- Contributore a GNU Emacs
- Contributore alla Free Software Directory
- Speaker alla SeaGL (Seattle GNU/Linux Conference)
- Mantiene infrastruttura FSF interamente con software libero

## Potere Esercitato

### Governance FSF nel 40esimo anno

| Aspetto | Dettaglio |
|---------|-----------|
| **Primo staff-president** | Rappresenta un cambio storico: la FSF guidata da chi ne gestisce l'infrastruttura tecnica quotidiana |

| **Reclutamento board** | Ha contribuito a disegnare e implementare i nuovi processi di selezione board partecipativa |
