# Anthony Albanese (1963)

> 31st Primo Ministro d'Australia, primo PM di origine italiana; carrierista laburista, architetto dell'AUKUS australiano

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 2 marzo 1963, Sydney, NSW, Australia |
| **Famiglia** | Cresciuto dalla madre single Maryanne Albanese in public housing a Camperdown, Sydney; origini italiane |
| **Coniuge** | Jodie Haydon (sposata nov 2025); ex moglie Carmel Tebbutt (2000-2019, ex Vice-Premier NSW) |
| **Figli** | Nathan Albanese |
| **Residenza** | The Lodge, Canberra; Marrickville, Sydney |
