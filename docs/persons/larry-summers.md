## Pattern di Potere

### Il "Rubin Circle"

Summers e' parte del circolo di Robert Rubin:
- **Goldman Sachs** → Treasury → Citigroup (Rubin)
- **Harvard** → Treasury → D.E. Shaw → Harvard (Summers)
- **Treasury** → NY Fed → Treasury → Warburg Pincus (Geithner)

Questo network ha dominato la politica economica USA 1993-2010.

### Deregulation Legacy

```
1999: Glass-Steagall abolito (Summers Treasury)
      ↓
2000: Derivati deregolamentati (Summers Treasury)
      ↓
2008: Crisi finanziaria globale
      ↓
2009: Summers gestisce risposta (NEC)
```
