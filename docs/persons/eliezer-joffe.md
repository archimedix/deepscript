| **Morte** | 1944, Eretz Israel |

## Potere Esercitato

### Il Modello Moshav

| Aspetto | Dettaglio |
|---------|-----------|
| **Teoria** | "Yissud Moshevei Ovedim" (1918): cooperazione + proprieta' individuale |
| **Differenza dal kibbutz** | Moshav: terra individuale, cooperazione nella vendita; Kibbutz: proprieta' collettiva totale |
| **Impatto** | Il moshav divenne la forma piu' diffusa di insediamento agricolo in Israele |
| **Tnuva** | La cooperativa che fondo' divenne monopolista del mercato alimentare israeliano |
