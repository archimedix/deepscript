# Steven Sinofsky (1965)

> Ex presidente della divisione Windows di Microsoft, technical assistant di Bill Gates, ora board partner di Andreessen Horowitz. Figura chiave nello sviluppo di Office e Windows 7/8.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 1965, New York City, USA |
| **Nazionalita'** | Americana |
| **Famiglia** | Genitori Marsha e David Sinofsky |
| **Residenza** | USA |
