# Dmitry Medvedev (1965-)

> Presidente russo 2008-2012, tandemocracy con Putin, ora falco del Security Council

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 14 settembre 1965, Leningrado (San Pietroburgo) |
| **Formazione** | PhD in Diritto, Universita' Statale di Leningrado |
| **Moglie** | Svetlana Vladimirovna Medvedeva (compagna di scuola) |
| **Figlio** | Ilya (1995), lavora per Rostec (difesa) |
