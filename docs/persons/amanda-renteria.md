### Revolving Door

Percorso classico **finanza → governo → politica → nonprofit**:
1. **Finanza** → Goldman Sachs (analyst)
2. **Governo** → Senate staff (Feinstein, Stabenow)
3. **Politica** → Candidature Congresso e governatrice, campagna Clinton
4. **Governo statale** → COO California AG
5. **Civic tech** → CEO Code for America
