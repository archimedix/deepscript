# Stephen A. Schwarzman (1947)

> Co-fondatore e CEO di Blackstone, il piu' grande fondo di private equity al mondo, advisor presidenziale, mega-donatore repubblicano

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 14 febbraio 1947, Philadelphia, Pennsylvania |
| **Nazionalita'** | Americana |
| **Famiglia** | Figlio di Joseph Schwarzman (commerciante, Wharton) |
| **Coniuge** | Christine Hearst Schwarzman (1995-oggi) |
| **Figli** | 2 dal primo matrimonio (Teddy, Elizabeth), 1 figliastro |
| **Residenza** | 740 Park Avenue, NYC (ex appartamento Rockefeller) |
| **Patrimonio** | ~$50 miliardi (ottobre 2025, #32 Forbes) |
