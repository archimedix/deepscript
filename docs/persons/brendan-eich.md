### Creazione di JavaScript (1995)

Eich entro' in Netscape nell'aprile 1995 con l'intenzione di portare Scheme nel browser. Il management insistette per una sintassi simile a Java. Eich creo' il linguaggio in 10 giorni per rispettare la release di Navigator 2.0 Beta. Il linguaggio fu chiamato prima Mocha, poi LiveScript, infine JavaScript (annuncio congiunto con Sun Microsystems, dicembre 1995).

Fonda Brave Software nel 2015. Il browser Brave blocca tracker e pubblicita'. Nel 2017 lancia il Basic Attention Token (BAT), raccogliendo $35M via ICO. Investitori iniziali: Founders Fund, Foundation Capital, Digital Currency Group.

## Potere Esercitato

### Infrastruttura del Web

JavaScript e' il linguaggio di programmazione piu' usato al mondo, presente in ogni browser web. Eich ha plasmato l'infrastruttura fondamentale di Internet con una creazione nata in 10 giorni nel 1995.

### Open Source e Privacy

Con Mozilla ha contribuito a mantenere il web aperto contro il monopolio Microsoft (Internet Explorer). Con Brave sfida il modello di business basato sulla sorveglianza pubblicitaria di Google/Chrome.
