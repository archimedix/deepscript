# Grand Duke Jean di Lussemburgo (1921-2019)

> Granduca del Lussemburgo per 36 anni, veterano WWII, membro IOC per 52 anni

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Jean Benoit Guillaume Robert Antoine Louis Marie Adolphe Marc d'Aviano |
| **Nascita** | 5 gennaio 1921, Castello di Berg, Colmar-Berg |
| **Morte** | 23 aprile 2019 (98 anni) |
| **Famiglia** | [Nassau-Weilburg](../family/nassau-weilburg.md) |
| **Padre** | Principe Felix di Borbone-Parma |
| **Madre** | [Granduchessa Charlotte](grand-duchess-charlotte.md) |
| **Coniuge** | Principessa Josephine-Charlotte del Belgio (m. 1953) |
| **Figli** | 5: Marie-Astrid, [Henri](grand-duke-henri.md), Jean, Margaretha, Guillaume |
| **Regno** | 1964-2000 (36 anni) |

## Servizio Militare WWII

### Irish Guards

| Data | Evento |
|------|--------|
| **Nov 1942** | Arruolamento volontario British Army (Irish Guards) |
| **Lug 1943** | Commissione come Lieutenant |
| **1944** | Promosso Captain |
| **11 giu 1944** | Sbarco in Normandia |
| **Estate 1944** | Battaglia per Caen |
| **Set 1944** | Liberazione di Bruxelles |
| **10 set 1944** | Partecipa alla liberazione del Lussemburgo |
| **Set 1944** | Operazione Market Garden (Arnhem) |
| **1944-45** | Invasione della Germania |
| **Giu 1947** | Congedo dall'esercito britannico |
| **1984-2000** | Colonel of the Regiment, Irish Guards |

Il Principe Jean si arruolo' volontario nell'esercito britannico su consiglio di Re Giorgio VI. Fu tra i primi liberatori a rientrare in Lussemburgo il 10 settembre 1944.

---

## Regno (1964-2000)

### Successione

| Data | Evento |
|------|--------|
| **12 nov 1964** | Abdicazione di Charlotte; Jean diventa Granduca |
| **7 ott 2000** | Abdicazione in favore del figlio Henri |

Jean regno' per 36 anni, periodo di grande stabilita' e crescita economica per il Lussemburgo, con l'espansione del settore finanziario e il consolidamento del ruolo europeo del paese.

### Comitato Olimpico Internazionale

| Aspetto | Dettaglio |
|---------|-----------|
| **Membro** | 1946-1998 (52 anni) |
| **Status post-1998** | Honorary Member, Honorary Doyen |
| **Doyen IOC** | 1982-1998 (secondo mandato piu' lungo nella storia) |
| **Chairman Rules Reviewing Commission** | 1973-1976 |
| **Commission of Enquiry Winter Games** | 1968-1969 |

Jean fu co-optato nel [IOC](../agency/ioc.md) nel settembre 1946 a soli 25 anni. Fu il secondo membro IOC con maggiore anzianita' nella storia (dopo il greco-egiziano Angelos Bolanaki).
