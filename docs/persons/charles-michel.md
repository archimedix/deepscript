# Charles Michel (1975-)

> Presidente del Consiglio Europeo 2019-2024, più giovane Premier belga dal 1841, figlio di Louis Michel (ex Commissario UE), artefice del Recovery Fund

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 21 dicembre 1975, Namur, Belgio |
| **Nazionalità** | BEL |
| **Padre** | Louis Michel (ex Commissario Europeo, MEP) |
| **Partito** | MR (Mouvement Réformateur) |
| **Professione** | Avvocato |
| **Lingue** | Francese, olandese, inglese |
