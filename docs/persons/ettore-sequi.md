# Ettore Francesco Sequi (1956)

> Diplomatico di carriera: ambasciatore a Kabul e Pechino, Rappresentante UE per Afghanistan, Segretario Generale Farnesina

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 13 febbraio 1956, Ghilarza (Sardegna) |
| **Nazionalita'** | Italiana |
| **Carriera** | Diplomatico di carriera (dal 1985) |
| **Grado** | Ambasciatore (dal 2015) |
