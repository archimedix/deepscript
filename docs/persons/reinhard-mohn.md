# Reinhard Mohn (1921-2009)

> Il patriarca che trasformo' una tipografia di provincia nell'impero mediatico Bertelsmann

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 29 giugno 1921, Gutersloh, Germania |
| **Morte** | 3 ottobre 2009, Gutersloh |
| **Famiglia** | Quinta generazione famiglia Mohn/Bertelsmann |
| **Prima moglie** | Magdalene Rassfeld (1948-1982, 3 figli) |
| **Seconda moglie** | [Liz Mohn](liz-mohn.md) (dal 1982, 3 figli adottati) |
| **Figli** | 6 totali: Johannes, Susanne, Christiane (1° matr.) + Brigitte, Christoph, Andreas (con Liz) |
| **Patrimonio** | Controllava 76.9% Bertelsmann via Stiftung |

---

## Formazione

| Periodo | Istituzione | Dettaglio |
|---------|-------------|-----------|
| **-1939** | Evangelisch-Stiftisches-Gymnasium, Gutersloh | Scuola fondata da Carl Bertelsmann |
| **1939** | Reichsarbeitsdienst | Servizio lavoro obbligatorio |
| **1939-1943** | Wehrmacht | Volontario Afrika Korps |
| **1943-1945** | POW Camp Concordia, Kansas | Prigioniero di guerra USA |

**Influenza americana**: Gli anni da POW in Kansas influenzarono profondamente il suo stile manageriale: profit-sharing, partecipazione dipendenti, struttura decentralizzata.

## Il Padre: Heinrich Mohn e il Nazismo

| Aspetto | Dettaglio |
|---------|-----------|
| **Heinrich Mohn** | Padre di Reinhard, dirigeva Bertelsmann durante III Reich |
| **SS Sponsors Circle** | Membro finanziatore (Forderndes Mitglied SS) |
| **Donazioni naziste** | Contributi a varie organizzazioni del partito |
| **Produzione bellica** | Bertelsmann = piu' grande fornitore libri Wehrmacht (~19M copie) |
| **Pubblicazioni antisemite** | Pubblicate anche dopo che i campi erano noti |
| **Iscrizione NSDAP** | Mai iscritto formalmente al partito |

### La Falsa Narrativa Post-Bellica

| Aspetto | Dettaglio |
|---------|-----------|
| **Versione ufficiale (1945-1998)** | "Editore cristiano perseguitato dai nazisti" |
| **Storico aziendale** | Dirk Bavendamm, storia ufficiale 1985 |
| **Verita' (2002)** | Commissione storica indipendente smentisce tutto |
| **Chiusura 1944** | Non per "resistenza" ma per accaparramento carta illegale |
