## SwissLeaks: Il Leak

### Dimensioni

| Aspetto | Dato |
|---------|------|
| **Clienti coinvolti** | 130.000+ |
| **Societa' offshore** | 20.000+ |
| **Volume transazioni** | 180.6 miliardi EUR (nov 2006 - mar 2007) |
| **Paesi coinvolti** | 200+ |

### Impatto per Paese

| Paese | Clienti | Volume |
|-------|---------|--------|
| Svizzera | - | Base operativa |
| Francia | - | Prima destinazione dati |
| **Italia** | 7.499 | 7.4 miliardi USD |
| UK | - | Sede HSBC Holdings |
| Spagna | - | Caso Emilio Botin (Santander) |

### "Lista Lagarde"

I dati consegnati a Christine Lagarde (Min. Finanze Francia 2007-2011) furono poi distribuiti ai governi europei:
- Italia: Guardia di Finanza indaga 5.439 nomi
- Grecia: Scandalo politico (lista ignorata)
- Spagna: Perseguiti evasori di alto profilo
