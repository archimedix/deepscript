### Revolving Door

Il percorso di Pahlka rappresenta un caso emblematico di **revolving door tech-governo**:
1. **Tech/Media** → O'Reilly Media (Gov 2.0 Summit)
2. **Nonprofit** → Fonda Code for America
3. **Governo** → Deputy CTO Casa Bianca
4. **Governo (creazione)** → Co-fonda USDS
5. **Nonprofit** → Torna a CfA, poi USDR
6. **Think tank** → Niskanen Center, FAS
