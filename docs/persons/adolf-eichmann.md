# Adolf Eichmann (1906-1962)

> Architetto logistico dell'Olocausto: organizzo' la deportazione di milioni di ebrei verso i campi di sterminio, catturato dal Mossad e giustiziato in Israele

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Otto Adolf Eichmann |
| **Nascita** | 19 marzo 1906, Solingen, Germania |
| **Morte** | 31 maggio 1962, Ramla, Israele (impiccato) |
| **Nazionalita'** | Germania |
| **Grado** | SS-Obersturmbannfuhrer (Tenente Colonnello) |
| **Ufficio** | RSHA Sezione IV B4 (Affari Ebraici) |

---

## Ascesa nel Reich (1932-1941)

### Carriera SS

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **Apr 1932** | Membro | Partito Nazista (Linz) |
| **Nov 1932** | Membro | SS |

| **1938** | Direttore | Ufficio Emigrazione Ebraica Vienna |
| **1939** | Direttore | Ufficio Emigrazione Praga |
| **1941** | Capo | RSHA IV B4 |

### L'Ufficio di Vienna (1938)

| Aspetto | Dettaglio |
|---------|-----------|
| **Apertura** | 20 agosto 1938 |
| **Funzione** | "Emigrazione forzata" ebrei austriaci |
| **Metodo** | Esproprio beni, coercizione |
| **Risultato** | 110.000 ebrei "emigrati" (ago 1938 - giu 1939) |
| **Innovazione** | Modello replicato in Germania e Praga |

## La Soluzione Finale (1941-1945)

### Conferenza di Wannsee (20 gennaio 1942)

| Aspetto | Dettaglio |
|---------|-----------|
| **Luogo** | Villa Am Grossen Wannsee 56-58, Berlino |
| **Convocata da** | Reinhard Heydrich |
| **Ruolo Eichmann** | Preparazione documenti, segretario |
| **Obiettivo** | Coordinare "Soluzione Finale della Questione Ebraica" |
| **Stima** | 11 milioni di ebrei da eliminare |

### Organizzazione delle Deportazioni

| Funzione | Dettaglio |
|----------|-----------|
| **Ruolo** | Coordinatore logistico |
| **Responsabilita'** | Treni, orari, destinazioni |
| **Rete** | Ferrovie Reich + paesi occupati |
| **Destinazioni** | Auschwitz, Treblinka, Sobibor, altri |

## Deportazioni per Paese

### Austria (1938-1945)

| Aspetto | Dettaglio |
|---------|-----------|
| **Emigrazione forzata** | 110.000 (1938-39) |
| **Deportazioni Auschwitz** | 71.000+ |
| **Primo trasporto** | 18 luglio 1942 (Vienna → Auschwitz) |

### Grecia - Salonicco (1943)

| Aspetto | Dettaglio |
|---------|-----------|
| **Periodo** | Marzo - Agosto 1943 |
| **Vittime** | 43.000-46.000 |
| **Destinazione** | Auschwitz-Birkenau |
| **Esito** | Maggioranza gasati all'arrivo |
| **Supervisore locale** | [Alois Brunner](alois-brunner.md) |

### Ungheria (1944)

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Invasione tedesca marzo 1944 |
| **Popolazione ebraica** | 825.000 (la piu' grande in Europa) |
| **Ruolo Eichmann** | Supervisione diretta sul campo |
| **Periodo deportazioni** | 15 maggio - 9 luglio 1944 |
| **Vittime** | 437.000-440.000 |
| **Treni** | 147 |
| **Destinazione** | Auschwitz |
| **Gasati all'arrivo** | ~80% |
| **Marce della morte** | Ott-Nov 1944, Budapest → Vienna |

> L'Ungheria fu l'unico caso in cui Eichmann si coinvolse direttamente sul terreno.

### Totale Stimato

| Aspetto | Numero |
|---------|--------|
| **Deportati sotto coordinamento Eichmann** | 1.500.000+ |
| **Paesi coinvolti** | Germania, Austria, Cecoslovacchia, Paesi Bassi, Francia, Grecia, Ungheria, altri |

## "La Banalita' del Male"

### Hannah Arendt

| Aspetto | Dettaglio |
|---------|-----------|
| **Autrice** | Hannah Arendt, filosofa |
| **Testata** | The New Yorker (reportage) |
| **Libro** | *Eichmann in Jerusalem* (1963) |
| **Concetto** | "La banalita' del male" |
| **Tesi** | Eichmann non era un mostro fanatico |
| **Ritratto** | Burocrate ordinario, abdico' giudizio morale |

### La Difesa di Eichmann

| Argomento | Dettaglio |
|-----------|-----------|
| **"Solo ordini"** | Eseguiva ordini superiori |
| **"Piccolo ingranaggio"** | Non decise le politiche |
| **"Nessun odio"** | Non odiava personalmente gli ebrei |

### Controversia

La tesi di Arendt fu contestata: documenti mostrano che Eichmann era ideologicamente motivato e prese iniziative personali per accelerare lo sterminio.
