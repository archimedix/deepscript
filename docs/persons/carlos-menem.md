# Carlos Saúl Menem (1930 - 2021)

> Presidente argentino (1989-99), artefice delle privatizzazioni neoliberiste e del Washington Consensus in Argentina

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 2 luglio 1930, Anillaco, La Rioja, Argentina |
| **Morte** | 14 febbraio 2021, Buenos Aires |
| **Famiglia** | Origine siriana (famiglia di immigrati sunniti) |
| **Coniuge** | Zulema Yoma (1966-1991), Cecilia Bolocco (2001-2011) |
| **Figli** | Carlos Menem Jr. (1968-1995), Zulemita Menem, Maximo Menem |
| **Religione** | Nato musulmano sunnita, convertito al cattolicesimo |

## Potere Esercitato

### Privatizzazioni

| Aspetto | Dettaglio |
|---------|-----------|
| **Scala** | 200+ imprese statali privatizzate |
| **YPF** | Petrolifera nazionale venduta a Repsol |
| **Ferrovie** | Rete ferroviaria statale smantellata |
| **Telecomunicazioni** | ENTel venduta a Telefonica e France Telecom |
| **Beneficiari** | Gruppi imprenditoriali locali (Macri, Perez Companc, Rocca) e multinazionali |

### Washington Consensus

| Aspetto | Dettaglio |
|---------|-----------|
| **Convertibilidad** | Parita fissa peso-dollaro (1991-2001) |
| **Deregulation** | Apertura mercati, rimozione barriere commerciali |
| **FMI** | Allineamento totale con le ricette del Fondo |
| **Risultato** | Stabilita anni '90, poi collasso 2001 |

| **Corruzione** | 1989-99 | Arricchimento illecito; stile di vita sfarzoso |
