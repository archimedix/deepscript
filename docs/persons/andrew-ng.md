## Rilevanza per DeepScript

Andrew Ng rappresenta pattern chiave:

1. **AI Education Pioneer**: Ha democratizzato l'accesso all'AI education
2. **US-China Bridge**: Stanford → Baidu → Silicon Valley
3. **Triple ecosystem**: Academia (Stanford) + Big Tech (Google, Baidu) + Startup (LandingAI, AI Fund)
4. **Talent pipeline**: I suoi studenti sono ora in posizioni chiave (Zico Kolter → OpenAI board)
5. **Corporate governance**: Amazon board member
