# Donald Tusk (1957-)

> Il polacco che ha guidato l'Europa - Presidente Consiglio Europeo, Presidente EPP, PM piu' longevo della Polonia democratica

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 22 aprile 1957, Danzica, Polonia |
| **Nazionalita'** | Polacca (minoranza Kashubian) |
| **Coniuge** | Malgorzata Sochacka (dal 1978) |
| **Figli** | Michal (1982), Katarzyna (1987) |
| **Nipoti** | 5 |
| **Residenza** | Sopot, Polonia |
