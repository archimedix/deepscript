## Contributi Tecnici

### Hashcash (1997)

| Aspetto | Dettaglio |
|---------|-----------|
| **Anno** | 1997 |
| **Scopo originale** | Anti-spam per email |
| **Meccanismo** | Proof-of-work |
| **Impatto** | Base del mining Bitcoin |

Hashcash aggiunge un costo computazionale all'invio di email, rendendo lo spam non economico. Il concetto di "proof-of-work" diventa fondamentale per Bitcoin.

### Citazione nel Whitepaper Bitcoin

Adam Back e' **l'unico individuo citato direttamente** nel whitepaper di Satoshi Nakamoto:

> "To implement a distributed timestamp server on a peer-to-peer basis, we will need to use a proof-of-work system similar to Adam Back's Hashcash"

### Email con Satoshi (2008-2009)

Nel caso legale contro Craig Wright, Back ha rilasciato le email scambiate con Satoshi:
- Discussione sul whitepaper Bitcoin
- Back riferisce Satoshi al lavoro di Wei Dai (b-money)
- Ultima email: 10 gennaio 2009, Satoshi informa Back del lancio di Bitcoin

## Attivismo Cypherpunk

### Crypto Wars (1990s)

Durante le "Crypto Wars", il governo USA classificava la crittografia come "munizione" soggetta a restrizioni all'export.

### La Maglietta RSA

**1995**: Back crea la celebre "Munitions T-shirt" con 3 righe di codice Perl che implementano RSA:

```perl
#!/bin/perl -sp0777i<X+d*lMLa^*lN%0teleType>teleType
```

**Significato**: Indossare ed esportare la maglietta era tecnicamente illegale - una forma di disobbedienza civile.

**Esito**: L'amministrazione Clinton alla fine rimuove le restrizioni sull'export di crittografia.

## Filosofia Bitcoin

Back e' un "Bitcoin maximalist" che crede in:
- Bitcoin come base layer immutabile
- Innovazione sui layer 2 (Lightning, Liquid)
- Resistenza a modifiche del protocollo base
- Decentralizzazione come valore primario
