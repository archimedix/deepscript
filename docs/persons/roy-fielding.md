## Potere Esercitato

### Architettura del Web

| Aspetto | Dettaglio |
|---------|-----------|
| **HTTP** | Co-autore del protocollo su cui si basa l'intero web |
| **REST** | Stile architetturale che domina le API moderne (ogni app mobile, ogni servizio cloud) |
| **Apache HTTP Server** | Web server dominante per 20+ anni |
| **Apache License** | Contributo al modello di licenza permissiva |

### Impatto Strutturale

REST, definito nella sua dissertazione del 2000, e' diventato lo standard de facto per la comunicazione tra sistemi software. Ogni volta che un'app accede a dati da un server, quasi certamente usa un'API REST progettata secondo i principi di Fielding. Questo rende il suo lavoro accademico uno dei piu' influenti nella storia pratica dell'informatica.
