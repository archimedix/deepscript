## Potere Esercitato

### Infrastruttura Digitale Globale

| Aspetto | Dettaglio |
|---------|-----------|
| Apache HTTP Server | Creazione del web server che ha dominato Internet per decenni |
| Apache License 2.0 | Modello di licenza permissiva adottato dall'industria tech globale |
| Hyperledger | Governance della blockchain enterprise (IBM, Intel, SAP) |
| OpenSSF | Sicurezza della supply chain open source dopo Log4Shell |

### Revolving Door: Open Source - Governo - WEF

Behlendorf incarna il ponte tra il mondo open source e le istituzioni del potere globale:
- **Apache** (1999): crea l'infrastruttura del web
- **White House OSTP** (2009): porta l'open source nel governo USA
- **WEF** (2011): digitalizza il forum dell'elite globale per Klaus Schwab
- **Linux Foundation** (2016): governa la blockchain enterprise
- **OpenSSF** (2021): protegge la supply chain software globale
