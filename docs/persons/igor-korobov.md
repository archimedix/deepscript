# Igor Korobov (1956-2018)

> Direttore GRU 2016-2018, sotto il cui comando avvennero hacking DNC, avvelenamento Skripal, secondo direttore consecutivo a morire in carica

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 3 agosto 1956, Vyazma, Oblast' di Smolensk |
| **Morte** | 21 novembre 2018 (62 anni) |
| **Famiglia** | Sposato, due figlie |
| **Grado** | Colonnello Generale |
| **Onorificenze** | Eroe della Federazione Russa (maggio 2017) |
| **Status legale** | Sanzionato USA (dic 2016, mar 2018) |

## Potere Esercitato

### Operazioni Sotto il Suo Comando

| Operazione | Anno | Target | Esito |
|------------|------|--------|-------|
| **Hacking DNC** | 2016 | Partito Democratico USA | Email pubblicate via WikiLeaks |
| **Interferenza elezioni USA** | 2016 | Elezioni presidenziali | Sanzioni USA |
| **Avvelenamento Skripal** | Mar 2018 | Ex spia GRU, UK | Fallito, Skripals sopravvissuti |
| **Hacking OPCW** | 2018 | Org. armi chimiche, L'Aia | Fallito, 4 agenti espulsi |

### Hacking Elezioni USA (2016)

| Aspetto | Dettaglio |
|---------|-----------|
| **Unita'** | APT28/Fancy Bear (Unit 26165) |
| **Target** | DNC, DCCC, campagna Clinton |
| **Metodo** | Spear phishing, leak via DCLeaks e WikiLeaks |
| **Attribuzione** | CrowdStrike, poi confermata da intelligence USA |
| **Conseguenze** | Sanzioni Obama dic 2016, 35 diplomatici espulsi |

### Avvelenamento Skripal (Marzo 2018)

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 4 marzo 2018 |
| **Target** | Sergei Skripal (ex GRU) e figlia Yulia |
| **Luogo** | Salisbury, Regno Unito |
| **Agente** | Novichok (agente nervino militare) |
| **Esecutori** | Anatoly Chepiga e Alexander Mishkin (Unit 29155) |
| **Esito** | Fallito - Skripals sopravvissuti |
| **Vittime collaterali** | Dawn Sturgess (morta), Charlie Rowley (ferito) |
| **Conseguenze** | 150+ diplomatici russi espulsi da 20+ paesi |

| Aspetto | Versione Ufficiale | Speculazioni |
