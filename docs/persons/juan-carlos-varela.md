## Controversie

### Caso Odebrecht

| Aspetto | Dettaglio |
|---------|-----------|
| **Incriminazione** | Luglio 2020, money laundering |
| **Accuse** | Fondi Odebrecht ricevuti come donazioni per campagne 2009 e 2014 |
| **Difesa** | Ammette fondi come donazioni, nega tangenti durante il governo |
| **Divieto viaggio** | Bandito dagli USA dal luglio 2023 (Sec. Blinken) |
| **Motivazione USA** | Accettazione di tangenti in cambio di contratti governativi |
| **Processo** | Rinviato post-elezioni 2024, da giudicare dalla Corte Suprema |
| **Immunita'** | Deputato del Parlamento Centroamericano |

### Rapporto con Martinelli

| Data | Evento |
|------|--------|
| **2009** | Alleanza: Varela VP di Martinelli |
| **2009-2011** | Ministro degli Esteri nel governo Martinelli |
| **Ago 2011** | Martinelli lo licenzia come ministro |
| **2014** | Varela batte il candidato di Martinelli alle presidenziali |
