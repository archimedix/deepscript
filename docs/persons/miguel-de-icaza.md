| Periodo | Istituzione | Dettaglio |

## Carriera e Ruoli

### Contributi fondamentali all'open source

**Midnight Commander (1994):** File manager per Linux, uno dei primi tool user-friendly.

**GNOME (1997):** Desktop environment libero per Linux/Unix. Risposta al problema di licenza di KDE/Qt. Diventa lo standard per molte distribuzioni Linux.

**Mono (2001):** Implementazione open source di .NET su Linux. Progetto controverso per la dipendenza da tecnologia Microsoft, ma cruciale per la cross-platform development.

**Xamarin (2011):** Dopo che Attachmate (acquirente di Novell) abbandona Mono, de Icaza rifonda il progetto come startup. Permette sviluppo mobile cross-platform con C#. Acquisita da Microsoft nel 2016 per integrare .NET nel mondo mobile.

### Periodo Microsoft (2016-2022)

Distinguished Engineer. Ultimo progetto: ONNX (Open Neural Network Exchange), standard aperto per modelli di machine learning. Lascia Microsoft nel marzo 2022.

### Xibbon (2022-oggi)

Startup a Boston che sviluppa game development tools. Ritorno alle radici imprenditoriali dopo un decennio in corporate.

## Potere Esercitato

### Il ponte open source-Microsoft

| Aspetto | Dettaglio |
|---------|-----------|
| **GNOME** | Ha creato il desktop environment libero dominante, influenzando milioni di utenti Linux |
| **Mono/.NET** | Ha portato la tecnologia Microsoft nel mondo open source, aprendo la strada alla futura apertura di .NET |
| **Xamarin** | Ha dimostrato che open source e' un modello di business viable, influenzando la strategia di Microsoft verso l'open source |
| **Effetto culturale** | Il suo percorso (critico di MS → collaboratore → impiegato) simboleggia la trasformazione di Microsoft da nemico a sostenitore dell'open source |
