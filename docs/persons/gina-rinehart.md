# Gina Rinehart (1954)

> La donna piu' ricca d'Australia e tra le piu' ricche al mondo: dall'eredita' mineraria del Pilbara alla costruzione di un impero privato

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 9 febbraio 1954, Perth, Western Australia |
| **Nazionalita'** | AUS |
| **Padre** | [Lang Hancock](../persons/lang-hancock.md), fondatore Hancock Prospecting |
| **Primo marito** | Greg Hayward (nee Milton), sposato ~1973, divorziato 1981 |
| **Secondo marito** | Frank Rinehart, avvocato ed ex military (Harvard Law), sposato 1983, morto 1990 |
| **Figli** | 4: John Langley Hancock (1976), Bianca Hope (1977), Hope (1986), Ginia (1987) |
| **Patrimonio** | A$38,11 miliardi (AFR Rich List 2025) |
