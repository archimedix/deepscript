# Claudio Descalzi (1955)

> Il manager dell'energia africana: da ingegnere petrolifero a CEO di ENI per quattro mandati, architetto della transizione energetica e ponte tra Italia e Africa.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 27 febbraio 1955 |
| **Nazionalita'** | Italiana |
| **Formazione** | Laurea in Fisica, Universita' di Milano (1979) |
| **Onorificenze** | Cavaliere del Lavoro (2025), Charles F. Rand Memorial Gold Medal (2012) |

### La Vicenda

| Aspetto | Dettaglio |
|---------|-----------|
| **Oggetto** | Acquisizione giacimento OPL 245 Nigeria (2011) |
| **Valore** | $1.3 miliardi |
| **Accusa** | Corruzione internazionale |
| **Imputati** | ENI, Shell, Descalzi, Scaroni, dirigenti |
| **Richiesta PM** | 8 anni per Descalzi e Scaroni |
