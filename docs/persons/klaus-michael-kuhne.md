# Klaus-Michael Kühne (1937-)

> L'uomo più ricco della Germania, ultimo erede di una dinastia della logistica

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 2 giugno 1937, Amburgo |
| **Nazionalità** | Tedesca |

| **Figli** | Nessuno |
| **Residenza** | Schindellegi, Svizzera (dal 1975) |
| **Patrimonio** | $37-41B (2024) |
