## Pattern di Potere

### 1. Eton-Oxford-Westminster Pipeline

Percorso classico elite britannica:
- Eton → Oxford (Bullingdon) → CCHQ → Westminster → Downing Street

### 2. Revolving Door Estremo

Post-premiership:
- 6+ ruoli corporate simultanei
- Sfruttamento network governativo
- Lobbying ex-colleghi ministri

### 3. Chipping Norton Set

Incrocio élite:
- Politica (Cameron, Osborne)
- Media (Brooks, Murdoch)
- Entertainment (Clarkson, Freud)
- Stessa area geografica Cotswolds

### 4. Fallimento Exit Strategy

Tutti i business ventures post-PM sono finiti male:
- Greensill: collasso + scandal
- Afiniti: dimissioni per scandal sessuale fondatore
- UK-China Fund: abortito per tensioni geopolitiche
- Solo Illumina rimane
