# Isser Harel (1912-2003)

> Padre dell'intelligence israeliana: fondatore dello Shin Bet, direttore del Mossad per 11 anni, architetto della cattura di Eichmann e del reclutamento di Otto Skorzeny

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Isser Halperin (poi Harel) |
| **Nascita** | 1912, Vitebsk, Russia (oggi Bielorussia) |
| **Morte** | 18 febbraio 2003, Israele |
| **Nazionalita'** | Israele |
| **Ruolo chiave** | Unico nella storia a dirigere Shin Bet E Mossad |
| **Altezza** | 1,52 m (soprannome "Isser il Piccolo") |

---

## Origini e Immigrazione

| Aspetto | Dettaglio |
|---------|-----------|
| **Famiglia** | Agiata, fabbrica di aceto a Vitebsk |
| **Data nascita** | Sconosciuta (registri persi durante Rivoluzione) |
| **Aliyah** | 1930, kibbutz Shfayim |
| **Attivita'** | Fondo' azienda di arance |
| **Cambio nome** | Halperin → Harel |

## Carriera Intelligence

### Haganah (1942-1948)

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **1942** | Capo intelligence | Haganah |
| **1944-48** | Dipartimento informazioni | Shai (servizio segreto pre-statale) |

### Shin Bet (1948-1952)

| Aspetto | Dettaglio |
|---------|-----------|
| **Ruolo** | Fondatore e primo direttore |
| **Periodo** | 1948-1952 |
| **Funzione** | Sicurezza interna |
| **Risultato** | Creo' le fondamenta del controspionaggio israeliano |

### Mossad (1952-1963)

| Aspetto | Dettaglio |
|---------|-----------|
| **Ruolo** | Direttore (Memuneh) |
| **Periodo** | 1952-1963 |
| **Predecessore** | Reuven Shiloah |
| **Successore** | Meir Amit |
| **Crescita** | Da 80 a 620 dipendenti |

**Unicita' storica:** Harel fu l'unico individuo nella storia israeliana a controllare simultaneamente Shin Bet e Mossad, detenendo potere senza precedenti nei primi 15 anni dello stato.

## Operazioni Principali

### Cattura di Adolf Eichmann (1960)

| Aspetto | Dettaglio |
|---------|-----------|
| **Target** | Adolf Eichmann, architetto della Soluzione Finale |
| **Localizzazione** | Buenos Aires, Argentina |
| **Alias Eichmann** | Ricardo Klement |
| **Data cattura** | 11 maggio 1960 |
| **Metodo** | Rapimento, esfiltrazione via El Al |
| **Processo** | Gerusalemme, 1961 |
| **Esecuzione** | 1 giugno 1962 |

> Harel diresse personalmente l'operazione dall'Argentina. Il suo libro *The House on Garibaldi Street* (1975) racconta i dettagli.

### Operation Damocles (1962-1963)

| Aspetto | Dettaglio |
|---------|-----------|
| **Target** | Scienziati tedeschi in Egitto |
| **Luogo** | Factory 333, programma missilistico egiziano |
| **Minaccia** | Missili con potenziale capacita' nucleare/chimica |
| **Metodo** | Intimidazione, lettere bomba, assassinii |
| **Vittima** | Heinz Krug (settembre 1962) |
| **Esito** | Operazione fermata da Ben-Gurion |

### Reclutamento Otto Skorzeny (1962)

| Aspetto | Dettaglio |
|---------|-----------|
| **Target** | Otto Skorzeny, ex-SS, "uomo piu' pericoloso d'Europa" |
| **Piano iniziale** | Assassinio |
| **Decisione Harel** | Reclutamento invece di eliminazione |
| **Logica** | Serviva un nazista infiltrato tra gli scienziati |
| **Incontro** | Skorzeny portato segretamente in Israele |
| **Handler** | Yosef Raanan, Rafi Eitan, Avraham Ahituv |

> "There are people who are marked to die." — Isser Harel

## Dimissioni e Conflitto con Ben-Gurion (1963)

### La Crisi

| Data | Evento |
|------|--------|
| **Mar 1963** | Due agenti Mossad arrestati in Svizzera |
| **Accusa** | Minacce alla figlia di uno scienziato |
| **Reazione** | Ben-Gurion ordina stop operazioni |
| **Conflitto** | Harel considera gli scienziati minaccia esistenziale |
| **Mar 1963** | Harel si dimette |

### Le Due Visioni

| Harel | Ben-Gurion |
|-------|------------|
| Scienziati = minaccia esistenziale | Minaccia sopravvalutata |
| Azione aggressiva necessaria | Relazioni con Germania Ovest prioritarie |
| Ossessionato dall'Olocausto | Pragmatismo diplomatico |

## Potere Esercitato

### Intelligence Consolidata

| Aspetto | Dettaglio |
|---------|-----------|
| **Controllo** | Shin Bet + Mossad simultaneamente |
| **Periodo** | 1952-1963 |
| **Precedenti** | Nessuno prima, nessuno dopo |
| **Accesso** | Diretto ai primi ministri |

### Eredita' Operativa

- Trasformo' Mossad da piccola agenzia a servizio di classe mondiale
- Stabili' metodologie per operazioni extraterritoriali
- Creo' precedente per reclutamento asset "impossibili" (Skorzeny)
