# Adolph Freiherr von Knigge (1752 - 1796)

> Scrittore, massone e organizzatore chiave degli Illuminati; nome in codice "Philo"; riorganizzo' l'ordine sul modello massonico reclutando logge intere

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome completo** | Adolph Franz Friedrich Ludwig Freiherr von Knigge |
| **Nascita** | 16 ottobre 1752, Bredenbeck (Wennigsen), Elettorato di Hannover |
| **Morte** | 6 maggio 1796, Brema |
| **Famiglia** | Nobile; orfano a 14 anni; debiti ereditari, proprieta' sequestrate |
| **Nome in codice** | Philo (nell'Ordine degli Illuminati) |
| **Professione** | Scrittore, cortigiano, amministratore civile |

## Carriera e Ruoli

### Timeline Professionale

| Periodo | Ruolo | Organizzazione |
|---------|-------|----------------|
| **~1772** | Massone | Iniziato a Kassel |
| **~1772-1777** | Gentiluomo di corte | Corti tedesche |
| **1777** | Ritiro a vita privata | - |
| **1780-1784** | Membro chiave ("Philo") | [Ordine degli Illuminati](../secret-society/illuminati.md) |
| **1791-1796** | Oberhauptmann (amministratore civile) | Brema |

### Ruolo nell'Ordine degli Illuminati (1780-1784)

| Aspetto | Dettaglio |
|---------|-----------|
| **Ingresso** | 1780: gia' massone esperto, porto' competenze organizzative e rete di contatti |
| **Contributo principale** | Riorganizzo' il sistema dei gradi dell'ordine (13 gradi), modellandolo sulle logge massoniche |
| **Reclutamento** | Recluto' logge massoniche intere, convincendole ad adottare gli insegnamenti illuministi |
| **Espansione** | Sotto la sua guida l'ordine passo' da poche centinaia a 2.000-3.000 membri |
| **Area** | Espansione nella Germania del Nord e oltre la Baviera |
| **Recluta notabile** | [Johann Wolfgang von Goethe](../persons/johann-wolfgang-goethe.md) (iniziato 1783, nome in codice "Abaris") |

### Il Conflitto con Weishaupt (1783-1784)

| Aspetto | Dettaglio |
|---------|-----------|
| **Causa** | Divergenze sugli obiettivi e le procedure dell'ordine |
| **Weishaupt** | Voleva mantenere il controllo centralizzato e l'orientamento anticlericale |
| **Knigge** | Piu' pragmatico; voleva un'organizzazione piu' aperta, con rituali massonici |
| **Esito** | Knigge costretto a lasciare l'ordine nel 1784 |
| **Tempistica** | La rottura coincise con il primo editto bavarese contro le societa' segrete (giugno 1784) |
| **Autodifesa** | Scrisse *Philo's Reply* per difendersi pubblicamente dopo il bando |

### Carriera Letteraria

| Opera | Dettaglio |
|-------|-----------|
| **Uber den Umgang mit Menschen** (1788) | Trattato sulle relazioni umane; divenne il testo di riferimento tedesco sul comportamento e l'etichetta; il cognome "Knigge" e' ancora sinonimo di buone maniere in Germania |
| **Philo's Reply** | Autodifesa pubblica dopo l'uscita dall'ordine |
| **Romanzi e saggi** | Autore prolifico dell'Illuminismo tedesco |
