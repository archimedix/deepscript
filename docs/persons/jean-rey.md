## Potere Esercitato

### Kennedy Round (1964-1967)

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Negoziati GATT per riduzione tariffe |
| **Significato** | Prima volta che la CEE negozia come entità unica |
| **Ruolo Rey** | Commissario Relazioni Esterne, negoziatore capo |
| **Risultato** | Riduzione media tariffe del 35% |

### Fusione degli Esecutivi (1967)

| Aspetto | Dettaglio |
|---------|-----------|
| **Trattato** | Trattato di Fusione (8 aprile 1965) |
| **Entrata in vigore** | 1 luglio 1967 |
| **Prima** | 3 esecutivi separati (CECA, CEE, Euratom) |
| **Dopo** | Una Commissione unica delle Comunità Europee |

### Unione Doganale (1968)

| Aspetto | Dettaglio |
|---------|-----------|
| **Completamento** | 1 luglio 1968 |
| **Anticipo** | 18 mesi prima del previsto |
| **Significato** | Abolizione dazi doganali interni tra i Sei |

### Vertice dell'Aia (1969)

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 1-2 dicembre 1969 |
| **Decisioni** | Rilancio integrazione europea |
| **Iniziative** | UEM (Unione Economica Monetaria), CPE (Cooperazione Politica) |
| **Allargamento** | Via libera a negoziati con UK |
