| Anno | Evento |

| **2023** | Fonda Dream Security con Sebastian Kurz e Gil Dolev |
| **2024** | Fonda The Institute per AI a Ben-Gurion University |
| **2024** | Investitore in IntelEye (dark web monitoring) |
| **2025** | Dream raggiunge valutazione $1.1B (unicorn) |
