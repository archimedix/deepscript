# Willem-Alexander (1967-)

> Re dei Paesi Bassi, pilota KLM, chairman UN Water: il monarca che continua la tradizione Bilderberg di famiglia

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 27 aprile 1967, Utrecht, Paesi Bassi |
| **Famiglia** | [Orange-Nassau](../family/orange-nassau.md) (17a generazione) |
| **Nazionalita'** | Olandese |
| **Coniuge** | [Maxima Zorreguieta](queen-maxima.md) (dal 2002) |
| **Figli** | 3: Catharina-Amalia (2003), Alexia (2005), Ariane (2007) |
| **Residenza** | Huis ten Bosch, L'Aia |
