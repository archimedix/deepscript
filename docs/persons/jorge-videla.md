## Potere Esercitato

### Repressione di Stato

| Aspetto | Dettaglio |
|---------|-----------|
| **Desaparecidos** | Circa 30.000 secondo organizzazioni per i diritti umani |
| **Metodo** | Sequestro, tortura, esecuzioni clandestine, voli della morte |
| **Bambini rubati** | Centinaia di neonati sottratti a prigioniere e dati a famiglie di militari |
| **Operacion Condor** | Coordinamento repressivo con Cile, Uruguay, Paraguay, Brasile, Bolivia |

### Politica Economica

| Aspetto | Dettaglio |
|---------|-----------|
| **Orientamento** | Liberismo, smantellamento politiche peroniste |
| **Ministro economia** | Jose Alfredo Martinez de Hoz |
| **Risultati** | Deindustrializzazione, aumento debito estero |
