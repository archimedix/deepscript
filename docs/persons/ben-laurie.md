## Potere Esercitato

### Sicurezza dell'Infrastruttura Web Globale

| Aspetto | Dettaglio |
|---------|-----------|
| **Apache-SSL** | Ha reso possibile il commercio elettronico sicuro creando la prima implementazione SSL per il web server dominante |
| **OpenSSL** | Come core team member, ha mantenuto la libreria crittografica su cui si basa la sicurezza di Internet |
| **Certificate Transparency** | A Google, ha creato il sistema che rende trasparente e verificabile l'emissione di certificati digitali |
| **Google Security** | Director of Security and Transparency nella piu' grande azienda tech del mondo |

### Pattern: il Crittografo Istituzionale

Laurie opera all'intersezione tra crittografia, open source e big tech. Unico fondatore Apache britannico, ha costruito la sicurezza del web in tre fasi:
1. **Apache-SSL** (anni '90): rende possibile HTTPS
2. **OpenSSL** (2000s): mantiene l'infrastruttura crittografica globale
3. **Certificate Transparency** (2010s): a Google, crea il sistema di audit dei certificati
