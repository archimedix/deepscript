# Israel Eldad (1910 - 1996)

> Filosofo sionista revisionista, ideologo del Lehi. Ponte intellettuale tra il massimalismo di Jabotinsky e la destra radicale israeliana contemporanea.

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 11 novembre 1910, Pidvolochysk, Galizia (oggi Ucraina) |
| **Morte** | 22 gennaio 1996 |
| **Nome di nascita** | Israel Scheib |
| **Nazionalita'** | ISR |
| **Figli** | [Aryeh Eldad](aryeh-eldad.md) (medico e politico) |

## Potere Esercitato

### Ideologia e Influenza

| Aspetto | Dettaglio |
|---------|-----------|
| **Dottrina** | Massimalismo revisionista: Malkhut Yisrael (regno d'Israele) dal Nilo all'Eufrate |
| **Rivista *Sulam*** | 15 anni (1949-1964) di analisi politico-culturale dalla destra massimalista |
| **Insegnamento** | Professore al Technion e BGU, formazione di generazioni |
| **Eredita'** | Idee riprese da Magshimey Herut, Zionist Freedom Alliance, partito Hatikva |

### Linea Underground → Stato

Eldad rappresenta un nodo cruciale nella trasmissione ideologica:

```
Jabotinsky (Revisionism) → Betar → Lehi (Eldad, Shamir, Yellin-Mor)
                                         ↓
                                   Stato d'Israele (1948)
                                         ↓
                              Herut → Likud (Begin, Shamir PM)
                                         ↓
                              Aryeh Eldad → Otzma Yehudit (Ben Gvir)
```
