# Rick Warren (1954)

> Pastore evangelico, fondatore Saddleback Church, autore di The Purpose Driven Life, membro Dialog Forum

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 28 gennaio 1954, San Jose, California |
| **Famiglia** | Figlio di Jimmy Warren (pastore battista) e Dot Warren (bibliotecaria) |
| **Coniuge** | Kay Warren |
| **Residenza** | Lake Forest, California |
