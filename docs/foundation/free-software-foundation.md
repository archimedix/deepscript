## Glossario

| Termine | Definizione |
|---------|-------------|
| **Copyleft** | Licenza che impone ai derivati di mantenere le stesse liberta' |
| **GPL** | GNU General Public License, la licenza copyleft piu' diffusa |
| **GNU** | Progetto di sistema operativo libero avviato da Stallman nel 1983 |
| **Four Freedoms** | Uso, studio, modifica, distribuzione: le 4 liberta' del software libero |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 4 ottobre 1985 |
| **Fondatore** | [Richard Stallman](../persons/richard-stallman.md) |
| **Contesto** | Stallman annuncio' il GNU Project nel 1983 per creare un OS libero. La FSF fu fondata per dare struttura legale e finanziaria al movimento |
| **Obiettivo dichiarato** | Promuovere le liberta' degli utenti di computer, difendere il diritto di usare, studiare, modificare e distribuire software |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1983** | Stallman annuncia il GNU Project |
| **1985** | Fondazione della FSF a Boston |
| **1989** | Pubblicazione della GPL v1 |
| **1991** | GPL v2; Linux adotta la GPL |
| **1992** | GNU + Linux = sistema operativo completo |
| **1999** | Lancio della Free Software Directory |
| **2001** | GNU Project riceve USENIX Lifetime Achievement Award |
| **2001** | Creazione del GPL Compliance Lab (enforcement formale) |
| **2007** | Pubblicazione della GPL v3 |
| **2019** | Stallman si dimette da presidente dopo controversie |
| **2020** | Geoffrey Knauth eletto presidente (5 agosto) |
| **2021** | Stallman rientra nel board. Riforme governance: union seat, Board Agreement, Code of Ethics |
| **2022** | Zoë Kooyman nominata executive director (1 marzo), succede a John Sullivan |
| **2023** | 40 anni del GNU Project |
| **2024** | Tre nuovi board members: Gilmore, Pievatolo, Haralanova (23 giugno). Processo partecipativo con 83 nomination |
| **2025** | Ian Kelling presidente (settembre). Knauth si dimette da board. FSF celebra il 40esimo anniversario |
| **2025** | Eko K. A. Owen entra nel board come union staff pick (dicembre) |

### Executive Director

[Zoë Kooyman](../persons/zoe-kooyman.md) (NLD), nominata nel 2022, succede a John Sullivan (ED 2010-2022). Background nell'industria musicale e project management internazionale. Alla FSF dal 2019 come program manager. University of Amsterdam (BA Media and Culture, MA Moving Image).

### Organi Direttivi

- Board of Directors (8 membri): governa strategia e policy
- Dal 2021, nuova governance partecipativa: gli associate members possono nominare e valutare candidati al board
- Executive Director gestisce operazioni quotidiane
- Board Member Agreement e Code of Ethics introdotti nel 2021-2022 sotto la presidenza Knauth
- Union seat nel board dal 2021 (primo occupante: Ian Kelling)

## Progetti e Campagne

### Progetti principali

| Progetto | Descrizione |
|----------|-------------|
| **GNU Project** | Sistema operativo libero (stewardship) |
| **GPL** | Licenze copyleft (v1, v2, v3, LGPL, AGPL) |
| **Free Software Directory** | Catalogo di 5000+ programmi liberi |
| **LibrePlanet** | Rete globale di attivismo + conferenza annuale |
| **GPL Compliance Lab** | Enforcement della GPL |

### Campagne attive

- Contro software patents
- Contro DRM (Digital Restrictions Management)
- Contro Trusted Computing
- Per standard aperti e adozione di software libero

## Critiche

| Tema | Dettaglio |
|------|-----------|
| **Stallman controversies** | Dimissioni 2019 per commenti su caso Epstein, ritorno contestato nel 2021 |
| **Governance opaca** | Fino al 2021 il board era auto-selezionato, riforme successive |
| **Rigidita' ideologica** | Rifiuto del termine "open source", tensioni con pragmatisti |
| **GPL enforcement** | Critiche per enforcement troppo aggressivo o troppo debole |
