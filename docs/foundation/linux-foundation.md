# The Linux Foundation

> Hub centrale dell'ecosistema open source globale, dove le maggiori corporation tech convergono per governare infrastruttura critica

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2007 (da fusione OSDL + FSG) |
| **Sede** | San Francisco, USA |
| **Tipo** | Foundation |
| **Leadership** | Jim Zemlin, Executive Director |

## Agenda/Focus

1. **Infrastruttura critica**: Linux kernel governa >90% dei server cloud, >80% degli smartphone (Android)
2. **Sicurezza supply chain**: OpenSSF dopo Log4Shell e SolarWinds
3. **AI open source**: LF AI & Data, modelli aperti
4. **Standard aperti**: RISC-V, OpenChain (compliance licenze)
5. **Bridge pubblico-privato**: punto d'incontro tra Big Tech, governi e accademia

## Potere Esercitato

| Aspetto | Dettaglio |
|---------|-----------|
| **Governance infrastruttura** | Linux kernel e' la base di Android, cloud (AWS, Azure, GCP), supercomputer (Top 500) |
| **Standard de facto** | Kubernetes (CNCF) e' lo standard per orchestrazione container |
| **Neutralita' competitiva** | Spazio dove competitor diretti (Google, Microsoft, Amazon) collaborano |
| **Talent pipeline** | Certificazioni Linux, training, eventi (Open Source Summit) |
| **Lobbying** | Rappresenta l'open source presso governi e regolatori |

## Critiche

| Tema | Dettaglio |
|------|-----------|
| **Corporate capture** | Accusata di favorire gli interessi dei Platinum member rispetto alla community |
| **Proliferazione sub-foundations** | Modello "umbrella" criticato come frammentazione |
| **Costi membership** | $500K Platinum esclude piccole aziende dalla governance |
| **Trasparenza finanziaria** | Budget non completamente pubblico |
