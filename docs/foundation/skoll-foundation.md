## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1999 |
| **Fondatore** | Jeff Skoll, con proventi dalla vendita di azioni eBay (~$1B donati nel tempo) |
| **Contesto** | Skoll lascio' la presidenza eBay (1998) e creo' la fondazione per applicare principi imprenditoriali a problemi sociali |
| **Obiettivo dichiarato** | Investire in social entrepreneurs che affrontano i problemi piu' urgenti del mondo |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1999** | Jeff Skoll fonda la Skoll Foundation con azioni eBay |
| **2001 feb** | [Sally Osberg](../persons/sally-osberg.md) diventa prima President & CEO (16 anni). BA English Scripps College, MA Claremont. Ex founding director Children's Discovery Museum. |
| **2003** | Fonda Skoll Centre for Social Entrepreneurship a Oxford Said ($7.5M) |
| **2004** | Primo Skoll World Forum a Oxford. Skoll fonda Participant Productions (cinema) |
| **2006** | Participant produce An Inconvenient Truth (Al Gore, clima) |
| **2008** | Skoll crea Skoll Global Threats Fund (SGTF) con $100M: pandemie, clima, nucleare, acqua, Medio Oriente |
| **2009** | Larry Brilliant diventa presidente fondatore di SGTF |
| **2010** | Jeff Skoll firma il Giving Pledge (5o firmatario) |
| **2011** | Participant produce Contagion - thriller pandemico su virus zoonotico da pipistrelli |
| **2017** | SGTF si chiude come previsto. Spin-off: [Ending Pandemics](../foundation/ending-pandemics.md) (nonprofit) e Climate Advocacy Lab |
| **2017** | [Sally Osberg](../persons/sally-osberg.md) lascia dopo 16 anni. Co-fonda [Co-Impact](../foundation/co-impact.md) ($500M) con Gates, Rockefeller Foundation, Olivia Leland, Nilekani |
| **2019 apr** | [Don Gips](../persons/don-gips.md) diventa CEO. Harvard BA, Yale MBA. McKinsey -> Al Gore advisor -> FCC -> Level 3 Communications -> Obama WH Dir. Presidential Personnel -> Ambasciatore USA Sudafrica -> Albright Stonebridge |
| **2020** | Jeff Skoll dona $100M aggiuntivi per risposta COVID-19. Grantmaking quadruplicato a $200M |
| **2021 giu** | Nancy Messonnier (ex CDC NCIRD Director) assunta come Executive Director Pandemic Prevention |
| **2021** | Ulteriori $100M impegnati per pandemia/sistemi sanitari su 5 anni |
| **2022** | Messonnier lascia per diventare Dean UNC Gillings |
| **2024 apr** | Participant Media chiude dopo 20 anni (135 film, $3.3B box office) |
| **2025** | Marla Blow diventa CEO. Jeff Skoll torna come Board Chair. Dona $25M aggiuntivi |

## Critiche

| Tema | Dettaglio |
|------|-----------|
| **Timing pandemico** | Infrastruttura pandemica costruita 2008-2018 (SGTF -> Ending Pandemics), film Contagion 2011, poi COVID-19 2020. La 115 Hotline cambogiana (finanziata da SGTF) rilevo' il 90% dei primi casi COVID nel paese |
| **Revolving door Messonnier** | CDC NCIRD Director dimessa maggio 2021, assunta immediatamente da Skoll. Sorella di Rod Rosenstein (Deputy AG) |
| **Revolving door Gips** | McKinsey -> FCC -> Al Gore -> Level 3 Comms (telecom CSO) -> Obama WH (3,000+ nomine) -> Ambasciata Sudafrica -> Albright Stonebridge (lobbying) -> Skoll Foundation -> Blackstone (PE) |
| **Co-Impact circuito chiuso** | Osberg (Skoll CEO) co-fonda Co-Impact con stessi donatori dell'ecosistema Giving Pledge. Leland (ex dir. Giving Pledge) ne diventa CEO. Core partners: Gates, Skoll, Rockefeller, Nilekani, MacKenzie Scott. |
| **Struttura fiscale** | Doppia entita' (Foundation + Fund via SVCF) per efficienza fiscale. $1B+ in stock eBay donato con significativi vantaggi fiscali |
| **Entertainment come influenza** | 135 film con "social action campaigns" integrate - intrattenimento e advocacy come leve di potere culturale |
