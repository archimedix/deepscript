### Programmi Chiave

- **Community Fellowship**: tecnologi inseriti per 3-6 mesi in governi locali
- **GetCalFresh**: applicazione digitale per food stamps in California (ridotto tempo da 45 min a <10 min)
- **Clear My Record**: automazione espunzione precedenti penali
- **Brigades**: network di volontari civic tech in citta' USA

## Connessioni PowerLink

### Revolving Door governo-tech

Code for America e' il nodo centrale del **civic tech pipeline** USA:
- Pahlka fonda CfA → entra alla Casa Bianca → co-fonda USDS → torna al nonprofit
- Il modello CfA ispira la creazione di unita' digitali governative (USDS, 18F)
- La trasformazione di USDS in DOGE (2025) rappresenta una discontinuita' rispetto alla missione originale
