# Thiel Fellowship

> Programma che paga giovani talenti per abbandonare l'universita' e costruire startup

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2011 |
| **Sede** | San Francisco, USA |
| **Tipo** | Foundation (Fellowship Program) |
| **Leadership** | Blake Masters (President, Thiel Foundation) |
| **Fellows totali** | 310+ (2011-2025) |
| **Grant** | $200,000 (2 anni) |
| **Status** | active |

## Glossario

| Termine | Definizione |
|---------|-------------|

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2011 |
| **Fondatori** | [Peter Thiel](../persons/peter-thiel.md), Michael Gibson, Danielle Strachman |
| **Contesto** | Critica di Thiel al sistema universitario americano come "bolla" |
| **Obiettivo dichiarato** | Supportare giovani innovatori a costruire invece di studiare |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2010** | Peter Thiel annuncia l'idea, recluta Gibson e Strachman |
| **2011** | Prima classe di 24 Fellows (grant: $100k) |
| **2012** | Dylan Field e Chris Olah tra i Fellows |
| **2013** | Austin Russell, Lucy Guo, Ritesh Agarwal selezionati |
| **2014** | Vitalik Buterin diventa Fellow dopo aver fondato Ethereum |
| **2015** | Gibson e Strachman lasciano per fondare 1517 Fund |
| **2015** | Blake Masters diventa President della Thiel Foundation |
| **2020** | Luminar (Russell) va in borsa, valutazione miliardaria |
| **2022** | Adobe annuncia acquisizione Figma per $20B |
| **2025** | Grant aumentato a $200k, classe di 20 Fellows |

### Meccanismo di Selezione

- Applicazioni aperte tutto l'anno
- Eta' massima: 22 anni
- Requisito: non avere laurea (o abbandonare se selezionati)
- Selezione basata su progetto/idea imprenditoriale
- Nessuna equity richiesta dalla Foundation

## Impatto

### Statistiche (2011-2025)

| Metrica | Valore |
|---------|--------|
| Valore totale creato | $750B+ |
| Unicorn hit rate | 13.79% |
| Fellows diventati billionaires | 5+ |
| Aziende fondate | 200+ |

### Tesi Dimostrata

Il programma ha validato la tesi di Thiel: talenti eccezionali possono creare piu' valore abbandonando l'universita' che completandola. I critici iniziali (che vedevano il programma come irresponsabile) sono stati smentiti dai risultati.
