## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2001 |
| **Fondatori** | Lawrence Lessig, Hal Abelson, Eric Eldred |
| **Contesto** | Nasce dalla sconfitta in *Eldred v. Ashcroft* (2003): Lessig rappresento' l'editore Eric Eldred alla Supreme Court contro il Sonny Bono Copyright Term Extension Act (1998). La Corte decise 7-2 a favore del governo. La risposta costruttiva fu CC: se non si puo' cambiare la legge, si creano licenze alternative |
| **Obiettivo dichiarato** | Fornire strumenti legali gratuiti per condividere opere creative con condizioni personalizzabili |
| **Supporto iniziale** | Center for the Public Domain |
| **Framework teorico** | Lessig (*Code*, 1999; *The Future of Ideas*, 2001) + Boyle (teoria della "seconda enclosure" della conoscenza) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1998** | Sonny Bono Copyright Term Extension Act; Eldred avvia la sfida legale |
| **2001** | Fondazione da parte di Lessig (Stanford), Abelson (MIT), Eldred; Boyle nel founding board |
| **2002** | Primo set di licenze CC pubblicato (16 dicembre) |
| **2003** | Supreme Court decide *Eldred v. Ashcroft* 7-2 contro Eldred (gennaio); Joi Ito entra nel board; Science Commons avviato con Boyle |
| **2005** | CC raggiunge copertura globale con porting delle licenze in 70+ giurisdizioni |
| **2006** | Lessig annuncia svolta verso anticorruzione; Joi Ito Chairman |
| **2008** | Joi Ito CEO; James Boyle Board Chair; Lessig lascia per fondare poi Equal Citizens |
| **2009** | Boyle lascia il board; pubblica *The Public Domain* |
| **2012** | Joi Ito lascia per MIT Media Lab |
| **2019** | Stihler Director of Product; Tumadottir entra come Director of Product |
| **2020** | Catherine Stihler (ex MEP, ex CEO Open Knowledge Foundation) nominata CEO (agosto) |
| **2021** | Tumadottir promossa COO |
| **2023** | Stihler lascia (dicembre) |
| **2024** | Tumadottir nominata CEO (aprile); Delia Browne completa mandato Board Chair (ottobre); Angela Oduor Lungati nuova Chair |

## Glossario

| Termine | Definizione |
|---------|-------------|
| CTEA / Sonny Bono Act | Copyright Term Extension Act (1998): estese il copyright di 20 anni, bloccando opere dal pubblico dominio |
| Seconda enclosure | Concetto di Boyle: privatizzazione della conoscenza digitale analoga alle enclosures agrarie inglesi |
| Science Commons | Progetto CC per applicare licenze aperte alla ricerca scientifica |
| CC0 | Licenza "zero diritti riservati" â€” equivalente funzionale del pubblico dominio |

## Critiche

| Tema | Dettaglio |
|------|-----------|
| Complessita' licenze | Proliferazione iniziale di versioni (6 combinazioni + CC0), semplificata nel tempo |
| Connessione Joi Ito-Epstein | Ito ricevette fondi da Epstein per MIT Media Lab; Lessig lo difese in email interna (2019), poi si scuso' |
| Enforcement debole | Licenze CC dipendono dalla buona fede, enforcement costoso per singoli creatori |
| Sconfitta Eldred v. Ashcroft | La nascita stessa di CC fu una "ritirata strategica" dopo il fallimento della via giudiziaria |
