## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Luglio 2010, annunciato a OSCON Portland |
| **Fondatori** | Rackspace Hosting + NASA |
| **Contesto** | Convergenza tra Nebula (NASA) e Cloud Files (Rackspace) per creare un'alternativa open source ad AWS |
| **Obiettivo dichiarato** | Produrre piattaforma cloud ubiqua e open source |

### Origini del Codice

Il progetto nasce dalla fusione di due basi di codice:
- **NASA Nebula**: piattaforma cloud interna, il cui modulo compute diventa **Nova**
- **Rackspace Cloud Files**: sistema di object storage che diventa **Swift**

Anso Labs (contractor NASA) pubblica il codice beta di Nova nei primi mesi del 2010. Rackspace contribuisce il codice storage. Il primo Design Summit si tiene ad Austin il 13-14 luglio 2010.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2010** | Lancio a OSCON Portland (21 luglio). Prima release "Austin" (ottobre) |
| **2011** | Governance rivista: Project Policy Board, PTL eletti dalla community |
| **2012** | Creazione della OpenStack Foundation (settembre) |
| **2013-2015** | Crescita esplosiva: centinaia di aziende contributrici, summit globali |
| **2016** | Consolidamento, focus su containers e bare metal |
| **2020** | OpenStack Foundation si rinomina OpenInfra Foundation |
| **2025** | 15 anni di OpenStack, consolidamento sotto Linux Foundation Cloud & Infrastructure |

### Organi Direttivi

- **Board of Directors** (OpenInfra Foundation): 1/3 Platinum Members (nominati), 1/3 Gold Members (eletti), 1/3 Individual Members (eletti)
- **Technical Committee (TC)**: eletto dai contribuenti attivi, governa le decisioni tecniche
- **PTL**: leader eletti per ogni progetto (Nova, Neutron, Cinder, etc.)
