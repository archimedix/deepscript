# Cloud Native Computing Foundation (CNCF)

> Governance dell'infrastruttura cloud-native globale: da Kubernetes ai 200+ progetti che definiscono lo standard del cloud computing

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2015 |
| **Sede** | San Francisco, USA |
| **Tipo** | Foundation (sussidiaria Linux Foundation) |
| **Leadership** | Jonathan Bryce (Executive Director, dal 2025) |
| **Progetti ospitati** | 200+ (31 graduated) |
| **Contributori** | 244.000+ in 193 paesi |
| **Status** | active |

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Cloud-native** | Architettura software basata su container, microservizi, orchestrazione automatizzata |
| **Graduated project** | Progetto CNCF maturo, adottato in produzione, con governance solida |
| **Incubating** | Progetto in fase di crescita con adozione significativa |
| **Sandbox** | Progetto sperimentale in fase iniziale |
| **Governing Board** | Organo decisionale per budget, marketing, strategia |
| **TOC** | Technical Oversight Committee, governance tecnica dei progetti |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 21 giugno 2015 |
| **Fondatori** | Linux Foundation + Google, Docker, Mesosphere, Red Hat, Twitter, Huawei, Intel, Cisco, IBM, VMware |
| **Contesto** | Necessita' di governance neutrale per container orchestration dopo il rilascio di Kubernetes |
| **Obiettivo dichiarato** | Rendere il cloud-native computing universale e sostenibile |
| **Seed technology** | Kubernetes 1.0, donato da Google (21 luglio 2015) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2015** | Fondazione CNCF; Kubernetes 1.0 donato da Google come primo progetto |
| **2016** | Prometheus entra come secondo progetto |
| **2017** | Microsoft e AWS diventano platinum members |
| **2018** | Prometheus diventa il primo progetto graduated dopo Kubernetes |
| **2020** | Dan Kohn muore; Priyanka Sharma diventa GM poi Executive Director |
| **2023** | Istio graduated; superati 150 progetti ospitati |
| **2025** | Priyanka Sharma lascia; Jonathan Bryce nominato Executive Director, Chris Aniszczyk CTO Cloud & Infrastructure |

### Organi Direttivi

- **Governing Board (GB)**: decisioni budget, marketing, strategia. Rappresentanti da platinum members + eletti
- **Technical Oversight Committee (TOC)**: governance tecnica, ammissione e promozione progetti
- **End User Technical Advisory Board**: input da aziende che usano le tecnologie in produzione

## Rilevanza Geopolitica

La CNCF controlla di fatto gli standard dell'infrastruttura cloud globale. La presenza di Huawei tra i founding members platinum e' significativa: la Cina partecipa alla governance degli standard cloud attraverso CNCF nonostante le tensioni tech USA-CHN. La struttura "open governance" della Linux Foundation permette influenza tecnica senza il controllo diretto di un singolo governo o azienda.
