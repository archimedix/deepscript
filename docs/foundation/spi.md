## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1997 |
| **Fondatori** | [Ian Murdock](../persons/ian-murdock.md), [Bruce Perens](../persons/bruce-perens.md) |
| **Contesto** | Creata per dare a [Debian](debian.md) un'entita' legale |
| **Obiettivo** | Fiscal sponsor per progetti di software libero |

SPI gestisce donazioni, marchi e asset legali per Debian e altri progetti open source (PostgreSQL, LibreOffice, Arch Linux, etc.).
