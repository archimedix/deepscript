## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|

| **Fondatore** | George Soros |
| **Prima Direttrice** | Sandra Pralong (ex communications director Newsweek) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1990** | Fondazione Soros Foundation Romania |
| **1990** | Creazione Group for Social Dialogue (GDS) con Andrei Pleșu |
| **1994** | Fondazione New Europe College (NEC) da Andrei Pleșu |
| **1996** | Alina Mungiu-Pippidi fonda Romanian Academic Society (SAR) |
| **1998** | Renate Weber diventa Presidente del Consiglio Nazionale |
| **~2000** | Budget raggiunge picco di $16 milioni |
| **2004** | Renate Weber advisor costituzionale del Presidente Romania |
| **2007** | Renate Weber eletta al Parlamento Europeo (PNL) |
| **2012** | Inizio riduzione finanziamenti diretti Soros |
| **2014** | Fine finanziamenti diretti, solo project-based |
| **2015** | Spin-off Center for Public Innovation (Ovidiu Voicu) |

## Pattern di Potere

### Formazione Elite Liberale

1. **Borse studio** → Giovani formati in Occidente
2. **Think tank** → Produzione intellettuale (SAR, NEC)
3. **Media** → Rivista "22" (GDS)
4. **Advocacy** → Coalition for Clean Parliament
5. **Politica** → Weber, Pleșu in governo

### Strategia di Sostenibilità

- Creazione spin-off indipendenti
- Diversificazione fonti (USAID, World Bank, UE)
- Membri a vita nel Foundation Council
