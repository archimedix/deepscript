## Glossario

| Termine | Definizione |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1982, New York |
| **Fondatrice** | [Barbara Ribakove Gordon](../persons/barbara-ribakove.md) |
| **Contesto** | Prima missione ebraico-americana in Etiopia (1981) rivela condizioni critiche degli ebrei etiopi |
| **Obiettivo dichiarato** | Aiutare gli ebrei etiopi a sopravvivere in Etiopia, raggiungere Israele, integrarsi, preservare la cultura |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1981** | Prima missione americana in Etiopia |
| **1982** | Fondazione NACOEJ nell'appartamento di Barbara Ribakove Gordon |
| **1983-84** | 18 missioni in Etiopia con medici, medicine, materiali scolastici |
| **1984** | Operation Moses: evacuazione segreta di ~8.000 ebrei etiopi via Sudan |
| **1987** | Lancio programma Adopt-A-Student per universitari etiopi-israeliani |
| **1989** | Programma Bar/Bat Mitzvah Twinning (gemellaggio giovani americani-etiopi) |
| **1990** | Compound ad Addis Abeba con strutture ricreative e formative |
| **1991** | Operation Solomon: ponte aereo di ~14.000 ebrei etiopi in 36 ore |
| **1993** | Primo Limudiah (doposcuola intensivo) aperto a Ramla |
| **1995** | Limudiah a Lod |
| **1997** | High School Sponsorship Program |
| **1998** | Compound Beta Israel a Gondar; Limudiah a Rehovot |
| **2004-07** | Espansione Limudiah: Rishon LeZion, Nes Ziona, Kiryat Ekron, Gedara |
| **2005** | Partnership con [Jewish Agency](../agency/jewish-agency.md) e United Jewish Communities |
| **2009** | Limudiah a Gerusalemme |
| **2011** | Trasferimento programmi Gondar alla Jewish Agency |

## Programmi Principali

1. **Limudiah** - Doposcuola intensivo in 8+ citta' israeliane per studenti etiopi-israeliani
2. **Adopt-A-Student** - Sponsorizzazione universitaria per giovani etiopi-israeliani
3. **High School Sponsorship** - Supporto studenti liceali
4. **Bar/Bat Mitzvah Twinning** - Gemellaggio giovani americani ed etiopi-israeliani
