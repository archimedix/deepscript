## Glossario

| Termine | Definizione |
|---------|-------------|
| **Amana** | "Patto" in ebraico — societa' cooperativa per l'insediamento |
| **Outpost** | Avamposto non autorizzato dal governo israeliano, spesso legalizzato retroattivamente |
| **Al-Watan** | Sussidiaria di Amana (fondata 2002) per l'acquisizione di terre in Cisgiordania |
| **Binyanei Bar Amana** | Sussidiaria per progetti di costruzione |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1976, come braccio insediativo di Gush Emunim |
| **Contesto** | Transizione di Gush Emunim da movimento di protesta a organizzazione di insediamento pratico |
| **Obiettivo** | Costruire ed espandere le comunita' ebraiche in Giudea e Samaria |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1976** | Fondazione come braccio operativo di Gush Emunim |
| **1980** | Amana si dichiara di fatto portavoce di Gush Emunim dopo lo scioglimento del segretariato informale |
| **1989** | Ze'ev Hever (Zambish) diventa Segretario Generale |
| **Anni '90** | Facilitazione di oltre 100 outpost non autorizzati |
| **2002** | Fondazione di Al-Watan per acquisizione terreni |
| **2014-2016** | Indagini di polizia: falsificazione documenti in 14 su 15 transazioni esaminate |
| **2018** | Trasferimento della sede a Gerusalemme Est |
| **2021** | Supporto a ~30 fattorie agricole in Cisgiordania |
| **2024 (giu)** | Sanzioni dal Canada |
| **2024 (ott)** | Sanzioni dal Regno Unito |
| **2024 (nov)** | Sanzioni dagli USA |
| **2025 (gen)** | Revoca sanzioni USA (24 gennaio 2025) |
| **2025 (feb)** | Tribunale ordina indagine di polizia per assistenza illegale da consigli regionali |
