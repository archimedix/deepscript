# AVI CHAI Foundation

> La fondazione che ha investito $1.2B nell'educazione ebraica mondiale prima di chiudere volontariamente nel 2019: creata dal fondatore di Sanford C. Bernstein & Co., connette Wall Street, Lehman Brothers, la Rothschild Foundation e la rete filantropica Rohr-Chabad.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1984 |
| **Chiusura** | 31 dicembre 2019 (spend-down pianificato) |
| **Sede** | New York, USA + Gerusalemme, Israele |
| **Tipo** | Foundation (filantropia ebraica) |
| **Fondatore** | [Zalman Bernstein](../persons/zalman-bernstein.md) (1926-1999) |
| **Chairman finale** | Mem Dryan Bernstein (vedova del fondatore) |
| **Endowment picco** | ~$600M (2010) |
| **Grants totali** | ~$1.2B (lifetime) |
| **Status** | defunct (residuo per Beit Avi Chai, Gerusalemme) |

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Spend-down** | Strategia deliberata di esaurire l'endowment entro un termine, anziche' operare in perpetuo |
| **Beit Avi Chai** | Centro culturale ebraico a Gerusalemme; unico beneficiario post-chiusura |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1984 (originariamente "T'Shuva Foundation") |
| **Fondatori** | [Zalman Bernstein](../persons/zalman-bernstein.md), Rabbi Shlomo Riskin, Samuel J. Silberman, George W. Kaufman |
| **Contesto** | Conversione religiosa di Bernstein (1978): dopo la morte del padre, diventa ortodosso sotto guida di Rabbi Riskin |
| **Nome** | "Avi Chai" = "Mio padre vive" (ebraico) |
| **Obiettivo** | Tre pilastri: educazione ebraica, unita' tra ebrei, legame con Israele |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1967** | Bernstein fonda Sanford C. Bernstein & Co. con Roger Hertog |
| **1978** | Conversione religiosa di Bernstein; adotta nome ebraico "Zalman Chaim" |
| **1984** | Fondazione AVI CHAI |
| **1992** | Bernstein fonda anche il Tikvah Fund (diagnosticato con linfoma) |
| **1999** | Morte di Zalman Bernstein in Israele; sepolto sul Monte degli Ulivi |
| **~1999** | Infusione massiccia dall'eredita' di Bernstein |
| **2000** | Alliance Capital acquisisce Sanford C. Bernstein → AllianceBernstein |
| **~2005** | Board decide formalmente lo spend-down (~20 anni dalla morte del fondatore) |
| **2016** | Creazione di Prizmah (fusione associazioni day school) |
| **31 dic 2019** | Chiusura grantmaking generale dopo 35 anni |
| **Post-2019** | Fondi residui per grants perpetui a Beit Avi Chai |

### Il Modello Bernstein

|------|---------|-----------|
