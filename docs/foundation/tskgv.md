## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 26 settembre 1987 |
| **Legge** | Law No. 3388 |
| **Origine** | Unificazione Land, Naval, Air Force Foundations |
| **Contesto** | Risposta all'embargo USA post-Cipro 1974 |
| **Obiettivo** | Rafforzare capacita' warfare TSK attraverso industria difesa nazionale |

### Contesto Storico

Dopo l'Operazione Pace a Cipro (1974), gli USA imposero un embargo sulle armi alla Turchia. Questo spinse il paese a sviluppare un'industria della difesa nazionale. Le fondazioni delle tre forze armate furono unificate nel 1987 per creare TSKGV.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1974** | Embargo USA post-Cipro, spinta verso autonomia |
| **1975** | Fondazione Aselsan |
| **1978** | Fondazione Isbir |
| **1981** | Fondazione Aspilsan |
| **1982** | Fondazione Havelsan |
| **1984** | Fondazione TAI |
| **1987** | Creazione TSKGV (Law 3388) |
| **1988** | Fondazione Roketsan |
| **Dic 2017** | **Decreto 696**: Erdogan diventa Chairman del board |
| **2017** | TSKGV passa dal Min. Difesa alla Presidenza |
| **2019** | 42% vendite difesa turca, 33% export, 74% R&D |

## Il Decreto 696 (Dicembre 2017)

### Contesto

Dopo il tentato golpe del luglio 2016, Erdogan governo' con decreti di stato di emergenza. Nel dicembre 2017, il Decreto 696 trasformo' la governance dell'industria difesa turca.

### Cambiamenti Chiave

| Prima | Dopo Decreto 696 |
|-------|------------------|
| TSKGV sotto Min. Difesa | TSKGV sotto Presidenza |
| Chairman: Min. Difesa | Chairman: Presidente |
| SSB sotto Min. Difesa | SSB sotto Presidenza |
| SSIK convocato da Min. | SSIK convocato da Presidente |

### Significato

> "The decree only formalizes a de facto situation in which Erdogan was the procurement boss."
> — Western defense executive (2017)

## Significato nel Sistema di Potere

### Impero Industriale-Militare

TSKGV rappresenta il cuore del complesso militare-industriale turco:

| Aspetto | Dettaglio |
|---------|-----------|
| **Scala** | 42% vendite difesa, 74% R&D nazionale |
| **Verticale** | Dall'elettronica (Aselsan) ai missili (Roketsan) |
| **Autonomia** | Risposta storica all'embargo USA |

### Concentrazione del Potere (2017)

Il Decreto 696 ha consolidato sotto Erdogan:

1. **TSKGV**: Holding industria difesa
2. **SSB**: Procurement difesa
3. **SSIK**: Comitato decisionale programmi

```
              ERDOGAN (Presidente)
                      |
         +------------+------------+
         |            |            |
       TSKGV        SSB         SSIK
    (Chairman)   (reports)   (convoca)
         |
   +-----+-----+-----+-----+-----+
   |     |     |     |     |     |
Aselsan TAI Havelsan Roketsan Aspilsan Isbir
```

### Pattern: Fondazione come Veicolo di Controllo

| Pattern | Descrizione |
|---------|-------------|
| **Forma giuridica** | Fondazione (non ministeriale) |
| **Flessibilita'** | Meno vincoli di trasparenza pubblica |
| **Controllo** | Board nominato, ora presieduto da Presidente |
| **Scala** | Holding di fatto del settore difesa |
