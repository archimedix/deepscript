# Jewish Defense League (JDL)

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1968, New York City |
| **Fondatori** | Meir Kahane, Bertram Zweibon, Morton Delinsky, Irving Calderon, Chaim Bieber |
| **Contesto** | Scioperi degli insegnanti di NYC 1968; tensioni razziali nei quartieri ebraici di Brooklyn |
| **Obiettivo dichiarato** | Protezione delle comunita' ebraiche dall'antisemitismo |
| **Slogan** | "Never Again!" |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1968** | Fondazione a New York da Meir Kahane e altri |
| **1969-1970** | Campagne contro l'antisemitismo nei quartieri di NYC |
| **1970-1971** | Focus sulla causa degli ebrei sovietici; azioni contro istituzioni sovietiche negli USA |
| **1971** | Kahane emigra in Israele; David Fisch diventa leader temporaneo |
| **1974** | Kahane torna negli USA, nomina Russel Kelner chairman internazionale |
| **1985** | Alex Odeh, direttore dell'American-Arab Anti-Discrimination Committee, ucciso con bomba attribuita a membri JDL |
| **1990** | Assassinio di Kahane a New York da El Sayyid Nosair |
| **1994** | Baruch Goldstein (membro JDL e Kach) compie il massacro di Hebron |
| **2001** | FBI classifica la JDL come gruppo terroristico di destra |
| **2001** | Irv Rubin e Earl Krugel arrestati per complotto terroristico |
| **~2015** | Organizzazione considerata sostanzialmente inattiva |

## Ideologia

La JDL promuoveva una concezione militante della difesa ebraica, basata su:

1. **Autodifesa armata** delle comunita' ebraiche
2. **Attivismo aggressivo** contro l'antisemitismo
3. **Causa degli ebrei sovietici** (dal 1970)
4. **Sionismo radicale** e sostegno incondizionato a Israele
