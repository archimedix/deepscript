| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 16 agosto 1993 |
| **Fondatore** | [Ian Murdock](../persons/ian-murdock.md) |
| **Contesto** | Creata come alternativa comunitaria alle distribuzioni commerciali Linux |
| **Obiettivo dichiarato** | Distribuzione libera, gestita democraticamente dalla comunita' |

| Anno | Evento |
|------|--------|
| **1993** | Ian Murdock fonda Debian, pubblica il Debian Manifesto (gennaio 1994) |
| **1996** | [Bruce Perens](../persons/bruce-perens.md) diventa DPL, introduce Debian Social Contract e DFSG |

| **2015** | Morte di Ian Murdock (28 dicembre) |

| **2023** | Debian 12 "Bookworm" - 30 anni del progetto |
| **2025** | Andreas Tille rieletto DPL |

### Modello di Governance

- **DPL**: eletto annualmente dai Debian Developers, rappresentante ufficiale
- **Technical Committee**: decisioni tecniche di ultima istanza
- **Debian Constitution**: ratificata 1998, definisce poteri e limiti
- **General Resolution**: meccanismo di voto per decisioni importanti
- **Governance decentralizzata**: decisioni collettive, il DPL deve agire in linea con il consenso degli sviluppatori (clausola 5.3)

## Rilevanza per il Potere

### Infrastruttura digitale globale

Debian e' la base di Ubuntu (Canonical), di gran parte dell'infrastruttura cloud e server mondiale, e dei sistemi Raspberry Pi. La sua governance comunitaria rappresenta un modello alternativo al controllo corporate del software.

### Connessioni ecosistema open source

- **[Free Software Foundation](free-software-foundation.md)**: radici ideologiche nel software libero di Stallman
- **[Linux Foundation](linux-foundation.md)**: ecosistema Linux piu' ampio
- **[Open Source Initiative](open-source-initiative.md)**: Bruce Perens fondo' OSI dopo essere stato DPL
- **[Canonical](../companies/canonical.md)**: Ubuntu e' basata su Debian

### SPI (Software in the Public Interest)

Organizzazione no-profit fondata nel 1997 da membri Debian per gestire donazioni e asset legali del progetto. Serve anche altri progetti open source.
