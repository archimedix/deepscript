## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2004 (progetto originale IBM: novembre 2001) |
| **Fondatore** | IBM |
| **Contesto** | IBM crea Eclipse come IDE open source per competere con strumenti proprietari; nel 2004 trasferisce la governance a una fondazione indipendente |
| **Obiettivo dichiarato** | Ambiente vendor-neutral per collaborazione e innovazione open source |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2001** | IBM lancia Eclipse Project come IDE open source |
| **2004** | Fondazione della Eclipse Foundation come entita' indipendente |
| **2004** | Mike Milinkovich nominato Executive Director |
| **2017** | Oracle trasferisce Java EE a Eclipse (diventa Jakarta EE) |
| **2020** | Incorporazione come AISBL in Belgio, sede a Bruxelles |
| **2021** | Microsoft diventa strategic member |
| **2021** | Lancio Eclipse Software-Defined Vehicle (SDV) Working Group |
| **2024** | OpenHW Group (RISC-V) si unisce alla Eclipse Foundation |

## Progetti Chiave

1. **Eclipse IDE** - L'IDE originale, ancora ampiamente usato
2. **Jakarta EE** - Piattaforma enterprise Java (ex Java EE)
3. **Eclipse MicroProfile** - Microservizi per Jakarta EE
4. **Eclipse Adoptium** - Distribuzione OpenJDK (Temurin)
5. **Eclipse SDV** - Software-Defined Vehicle per automotive
6. **Eclipse Oniro** - OS per dispositivi IoT (collaborazione con OpenAtom/Huawei)
7. **Eclipse Theia** - IDE cloud-native, alternativa a VS Code
8. **OpenHW Group** - Processori RISC-V open source

## Rilevanza Geopolitica

- **Spostamento in Europa (2020)**: la ri-incorporazione come AISBL belga segnala una strategia di autonomia dall'ecosistema USA, in linea con la sovranita' digitale europea
- **Ponte Cina-Europa**: la collaborazione Oniro con Huawei/OpenAtom connette l'ecosistema open source europeo con quello cinese
- **Automotive tedesco**: forte presenza di Bosch, Mercedes-Benz e BMW nel SDV Working Group, posizionando Eclipse come standard de facto per il software automotive europeo
- **Assenza italiana**: nessun strategic member italiano nel board
