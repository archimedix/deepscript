> Organizzazione che difende il software libero in Europa attraverso advocacy politica, supporto legale e campagne pubbliche

| Metrica | Valore |

## Glossario

| Termine | Definizione |
|---------|-------------|
| Free Software | Software che garantisce liberta' di uso, studio, modifica e distribuzione (4 liberta' FSF) |
| PMPC | Public Money, Public Code - campagna FSFE per il software pubblico open source |
| REUSE | Iniziativa FSFE per standardizzare le informazioni di licenza nei progetti software |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2001 |
| **Fondatori** | [Georg C. F. Greve](../persons/georg-greve.md) |
| **Contesto** | Necessita' di rappresentanza del movimento free software a livello europeo, con focus su policy EU |
| **Obiettivo dichiarato** | Empower users to control technology; promuovere liberta', trasparenza e autodeterminazione attraverso il software libero |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2001** | Fondazione da parte di Georg Greve come sister organization della [FSF](free-software-foundation.md) |
| **2004** | Intervento nel caso antitrust EU contro Microsoft (Samba/interoperabilita') |
| **2007** | Supporto legale nei processi antitrust Microsoft in Europa |
| **2009** | Georg Greve lascia la presidenza; Karsten Gerloff diventa presidente |
| **2010** | Greve riceve la Croce al Merito tedesca per il lavoro su FSFE |
| **2010** | FSFE premiata con la Medaglia Theodor Heuss |
| **2015** | Matthias Kirschner eletto presidente |
| **2017** | Lancio campagna "Public Money? Public Code!" con 31 organizzazioni firmatarie |
| **2019** | Kirschner rieletto presidente e vice-presidente |
| **2025** | Caso contro Apple alla Corte di Giustizia Europea; quarta edizione Youth Hacking 4 Freedom |

## Agenda/Focus

1. **Public Money, Public Code** - Software finanziato pubblicamente deve essere open source
2. **REUSE** - Standard per informazioni di licenza nei progetti software
3. **Router Freedom** - Diritto degli utenti di scegliere il proprio router
4. **Device Neutrality** - Contro il lock-in dei dispositivi
5. **Youth Hacking 4 Freedom** - Contest di programmazione per teenager europei
6. **Advocacy EU** - Intervento nei processi legislativi europei su software e diritti digitali
