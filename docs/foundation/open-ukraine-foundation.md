## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Luglio 2007 |
| **Fondatori** | [Arseniy Yatsenyuk](../persons/arseniy-yatsenyuk.md), Zbigniev Drzymala |
| **Contesto** | Creata durante il mandato di Yatsenyuk come Ministro degli Esteri ucraino per promuovere l'immagine internazionale dell'Ucraina |
| **Obiettivo dichiarato** | Diplomazia pubblica, promozione internazionale dell'Ucraina, dialogo sulla sicurezza |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2007** | Fondazione da parte di Yatsenyuk e Drzymala |
| **2007** | Lancio del Kyiv Security Forum (KSF) |
| **2014** | Yatsenyuk diventa Primo Ministro dell'Ucraina (2014-2016) |
| **2023** | Creazione KSF Security Council (5 maggio) |
| **2024** | 16° KSF "Ukraine's Call to the World: Together, We Prevail" (21-22 marzo) |
| **2025** | 17° KSF "UA: UNITE AGAIN TO DEFEAT THE GLOBAL AGGRESSOR" (8-9 maggio) |

## Programmi

1. **International Dialogue** — dialogo diplomatico e sicurezza
2. **Cultural Horizons** — promozione culturale ucraina
3. **Young Leaders** — formazione giovani leader
4. **Kyiv Security Forum** — conferenza annuale sulla sicurezza europea e del Mar Nero
