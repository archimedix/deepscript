## Fondazione e Storia

### Timeline

| Anno | Evento |
|------|--------|
| **1810** | Fonderia Peugeot (acciaio) |
| **1885** | Prima bicicletta Peugeot |
| **1889** | Prima automobile Peugeot (vapore) |
| **1919** | Fondazione Citroën (André Citroën) |
| **1974** | Peugeot acquisisce 38.2% Citroën (in crisi) |
| **1976** | Fusione completa: nasce PSA Peugeot Citroën |
| **1978-79** | Acquisizione Chrysler Europe (Simca, Talbot) |
| **1980-85** | Crisi finanziaria post-Chrysler |
| **1987** | Fine marchio Talbot (auto) |
| **2012** | Crisi: governo francese + Dongfeng entrano nel capitale |
| **2014** | Carlos Tavares CEO: inizio turnaround |
| **2015** | Ritorno al profitto |
| **2017** | Acquisizione Opel/Vauxhall da GM (€2.2B) |
| **2018** | Opel torna in utile (12 mesi, 2 anni prima del previsto) |
| **2019** | Annuncio fusione 50-50 con FCA |
| **2021** | Fusione completata: nasce Stellantis |

### Pattern: Stato + Capitale Straniero = Salvataggio

La crisi 2012 mostrò il modello francese di salvataggio industriale:
- Stato (Bpifrance) entra con 12.2%
- Partner straniero (Dongfeng) con 12.2%
- Famiglia mantiene 12.2%
- Manager esterno (Tavares) per turnaround

---

## Eredità

PSA dimostrò che:
1. **Manager di crisi** possono ribaltare situazioni disperate
2. **Acquisizioni di asset in perdita** possono funzionare (Opel)
3. **Parità 50-50** nelle fusioni è possibile (con FCA)

Il gruppo confluì in Stellantis mantenendo:
- Marchi intatti (Peugeot, Citroën, DS, Opel)
- Management (Tavares CEO Stellantis)
- Azionisti (Peugeot 7.2%, Bpifrance 6%)
