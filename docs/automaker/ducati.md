## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 4 luglio 1926 |
| **Nome originale** | Societa' Scientifica Radio Brevetti Ducati |
| **Fondatori** | Antonio Cavalieri Ducati (con figli Adriano, Bruno, Marcello) |
| **Primo prodotto** | Condensatore "Manens" per radio |
| **Prima moto** | Cucciolo (1946) |

### La Famiglia Ducati

Nel 1924 Adriano Cavalieri Ducati, studente di fisica, realizzo' un collegamento radio Italia-America con apparecchiatura a onde corte da lui ideata. Due anni dopo la famiglia fondo' l'azienda per sfruttare questa competenza.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1926** | Fondazione Societa' Scientifica Radio Brevetti Ducati |
| **1935** | Inizio costruzione stabilimento Borgo Panigale |
| **1936** | Inaugurazione stabilimento Borgo Panigale |
| **1946** | Nasce reparto motociclistico, primo Cucciolo |
| **1948** | Fratelli Ducati cedono a IRI/partecipazioni statali |
| **1954** | Scissione: Ducati Meccanica e Ducati Elettrotecnica |
| **1983** | Cagiva acquisisce Ducati Meccanica |
| **1996** | TPG acquisisce da Cagiva |
| **1999** | IPO Borsa Milano |
| **2006** | Investindustrial acquisisce 29.9% |
| **2008** | Investindustrial sale al 100%, delisting |
| **2012** | Lamborghini/Audi/VW acquisisce per €747M |
| **2013** | Claudio Domenicali diventa CEO |
| **2025** | Centenario aziendale |
