## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 11 luglio 1899 |
| **Fondatori** | Giovanni Agnelli e altri soci torinesi |
| **Capitale iniziale** | $400 quota di Agnelli |
| **Prima sede** | Torino |
| **Obiettivo** | Produzione automobili |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1899** | Fondazione FIAT a Torino |
| **1900** | Giovanni Agnelli diventa amministratore delegato |
| **1910** | Fiat e' il piu' grande produttore italiano |
| **1920** | Agnelli diventa Chairman |
| **1922-23** | Costruzione Lingotto, la piu' grande fabbrica europea |
| **1925** | Fiat controlla 87% del mercato italiano |
| **1927** | Fondazione IFI (holding Agnelli, poi Exor) |
| **1945** | Morte Giovanni Agnelli Sr.; Vittorio Valletta guida azienda |
| **1966** | [Gianni Agnelli](../persons/gianni-agnelli.md) diventa Chairman |
| **1969** | Acquisizione 50% Ferrari |
| **1976** | Carlo De Benedetti CEO per 100 giorni |
| **1986** | Acquisizione Alfa Romeo |
| **1988** | Morte Enzo Ferrari; Fiat sale al 90% Ferrari |
| **2003** | Morte Gianni Agnelli; [Umberto Agnelli](../persons/umberto-agnelli.md) Chairman |
| **2004** | Morte Umberto; crisi successoria |
| **2004** | [Sergio Marchionne](../persons/sergio-marchionne.md) CEO, salva Fiat |
| **2009** | Alleanza con Chrysler |
| **2014** | Fusione Fiat-Chrysler → FCA |
| **2016** | Spin-off Ferrari |
| **2018** | Morte Marchionne |
| **2021** | Fusione FCA-PSA → [Stellantis](stellantis.md) |

## Controllo Agnelli

La famiglia [Agnelli](../family/agnelli.md) ha controllato Fiat per 122 anni attraverso una struttura piramidale di holding.

### Struttura di Controllo

```
Famiglia Agnelli
      ↓
Giovanni Agnelli BV (cassaforte)
      ↓
   Exor N.V.
      ↓
Stellantis (14.2%) | Ferrari (24.65%) | CNH | Juventus
```

### Successione

| Periodo | Leader | Note |
|---------|--------|------|
| 1899-1945 | [Giovanni Agnelli Sr.](../persons/giovanni-agnelli-sr.md) | Fondatore |
| 1945-1966 | Vittorio Valletta | Manager, non famiglia |
| 1966-2003 | [Gianni Agnelli](../persons/gianni-agnelli.md) | "L'Avvocato" |
| 2003-2004 | [Umberto Agnelli](../persons/umberto-agnelli.md) | Fratello di Gianni |
| 2004- | [John Elkann](../persons/john-elkann.md) | Nipote di Gianni |

### Tragedie Dinastiche

| Anno | Evento |
|------|--------|
| 1997 | Morte Giovanni Alberto Agnelli (33 anni), primo erede designato |
| 2000 | Morte Edoardo Agnelli (46 anni), figlio di Gianni |

## Impatto Economico

Al suo apice, Fiat rappresentava:
- 4.4% del PIL italiano
- 3.1% della forza lavoro industriale
- 16.5% degli investimenti in R&D industriali
