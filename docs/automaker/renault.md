### Pattern: Stato Azionista Strategico

Lo Stato francese mantiene una posizione di blocco:
- **Loi Florange (2014)**: Voto doppio per azioni >2 anni
- **22.5% voting**: Può bloccare decisioni straordinarie
- **Interesse nazionale**: Protezione occupazione, R&D in Francia

## L'Alleanza Renault-Nissan-Mitsubishi

### Struttura (post-2023)

```
RENAULT
   │
   │ 15%
   ▼
NISSAN ◄──── 15% ──── RENAULT
   │
   │ 34%
   ▼
MITSUBISHI
```

### Evoluzione

| Anno | Struttura | Contesto |
|------|-----------|----------|
| **1999** | Renault 36.8% → Nissan | Nissan in bancarotta, Ghosn "cost killer" |
| **2002** | Renault 44.4% → Nissan | Consolidamento |
| **2016** | + Mitsubishi (Nissan 34%) | Scandalo emissioni Mitsubishi |
| **2018** | Crisi Ghosn | Arresto, tensioni franco-giapponesi |
| **2023** | Renault 15% ↔ Nissan 15% | Riequilibrio, più autonomia |

### Il Caso Ghosn

[Carlos Ghosn](../persons/carlos-ghosn.md) (1954-) incarna l'ascesa e caduta del manager globale:

```
MICHELIN (Brasile)
       │
       ▼
RENAULT (1996)
Head of Purchasing
       │
       ▼
NISSAN COO (1999)
"Le Cost Killer"
       │
       ▼
NISSAN CEO (2000) + RENAULT CEO (2005)
Dual leadership unica al mondo
       │
       ▼
+ MITSUBISHI CHAIRMAN (2016)
       │
       ▼
ARRESTO TOKYO (Nov 2018)
Accuse: false dichiarazioni, uso fondi
       │
       ▼
FUGA IN LIBANO (Dic 2019)
Nascosto in flight case per strumenti musicali
       │
       ▼
LATITANTE A BEIRUT (2020-)
Mandato arresto internazionale Francia
```

**Interpretazioni contrastanti:**
- **Giappone/Nissan**: Frode fiscale, abuso fondi aziendali
- **Ghosn**: Colpo di stato aziendale per bloccare fusione Renault-Nissan

## I Tre Turnaround

### 1. Schweitzer (1992-2005): Privatizzazione

| Sfida | Soluzione |
|-------|-----------|
| Azienda statale inefficiente | Privatizzazione 1996 |
| Nessuna scala globale | Alliance Nissan 1999 |
| Gamma limitata | Acquisizione Dacia (low-cost) |

### 2. Ghosn (2005-2018): Globalizzazione

| Sfida | Soluzione |
|-------|-----------|
| Nissan in bancarotta | Cost cutting radicale |
| Duplicazioni | Piattaforme comuni |
| Mercati emergenti | Expansion Cina, India, Russia |

### 3. De Meo (2020-2025): Renaulution

| Sfida | Soluzione |
|-------|-----------|
| Perdita €8B (2020) | Profitto €1B in 18 mesi |
| Gamma obsoleta | Nuovi modelli (R5 elettrica) |
| EV transition | Creazione Ampere (divisione EV) |
| Alliance in crisi | Riequilibrio 15-15% |

## Pattern: Il Manager Globale

Ghosn rappresentò il primo tentativo di CEO simultaneo di due major automaker in continenti diversi:

| Aspetto | Innovazione | Problema |
|---------|-------------|----------|
| **Dual CEO** | Sinergie, velocità decisionale | Conflitti governance |
| **Cross-cultural** | Franco-giapponese | Tensioni nazionali |
| **Compensation** | Pacchetti globali | Accuse di occultamento |
| **Integration** | Piattaforme comuni | Resistenze interne |

La caduta di Ghosn segnò la fine del modello "global CEO" nell'automotive.
