### Double Voting Rights

Stellantis usa il sistema olandese di **loyalty shares**: chi detiene azioni per 3+ anni ottiene voti doppi. Questo amplifica il potere delle famiglie fondatrici.

### Patto di Consultazione

Nel 2021, Agnelli e Peugeot hanno firmato un **patto di consultazione** per coordinare le decisioni strategiche.

## Storia

### Timeline

| Anno | Evento |
|------|--------|
| 1899 | Giovanni Agnelli fonda FIAT |
| 1810 | Famiglia Peugeot inizia (mulini) |
| 1889 | Prima auto Peugeot |
| 2009-14 | Fiat acquisisce Chrysler |
| 2014 | Nasce FCA (Fiat Chrysler Automobiles) |
| 2019 | Annuncio merger FCA + PSA |
| **2021** | **Nasce Stellantis** |
| 2024 | Profit warning, -50% market cap |
| 2024 | **Tavares si dimette** |

### Logica del Merger

1. **Scala**: 4° costruttore mondiale
2. **Sinergie**: €5B/anno target
3. **Complementarieta'**:
   - FCA forte in USA (Jeep, Ram)
   - PSA forte in Europa (Peugeot, Opel)
4. **EV transition**: Condivisione piattaforme elettriche

## Pattern

**Fusione di dinastie rivali sotto pressione industriale**

1. Due famiglie centenarie (Agnelli 1899, Peugeot 1810)
2. Entrambe in difficolta' (transizione EV, scala)
3. Merger "tra pari" ma controllo condiviso
4. CEO esterno (Tavares) come arbitro
5. Crisi → CEO sacrificato, famiglie restano

**Confronto con altri merger auto:**
- Daimler-Chrysler (1998): Fallimento, Chrysler venduta
- Renault-Nissan (1999): Alliance funziona, ma tensioni
- FCA-PSA (2021): In corso, primi problemi emersi
