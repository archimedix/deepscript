# Ford Otosan - Ford Otomotiv Sanayi A.S.

> Joint venture 50-50 tra Ford Motor e Koc Holding, maggior produttore automotive turco

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1959 (Otosan); 1977 (Ford Otosan) |
| **Sede** | Kocaeli, Turchia |
| **Tipo** | Automaker (JV) |
| **Proprieta'** | Ford 41%, Koc 41%, Public 18% |
| **Dipendenti** | 20.000+ |
| **Capacita'** | 934.500 veicoli/anno |
| **Status** | active |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origini** | 1928 - Vehbi Koc diventa dealer Ford ad Ankara |
| **Prima fabbrica** | 1929-1934 - Assemblaggio Ford Model A/B a Tophane, Istanbul |
| **Fondazione Otosan** | 1959, Istanbul |
| **JV con Ford** | 1977 - Accordo con Henry Ford II |
| **Nome attuale** | Ford Otosan (dal 1977) |

## Significato Strategico

Ford Otosan rappresenta il modello di successo della partnership tra capitale turco e multinazionali occidentali:

1. **Longevita'**: 95+ anni di relazione Ford-Koc (dal 1928)
2. **Localizzazione**: Prima auto turca (Anadol 1966)
3. **Export hub**: Transit esportato in tutta Europa
4. **Transizione EV**: JV batterie per elettrificazione
