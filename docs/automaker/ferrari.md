## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1947 (prima auto con nome Ferrari) |
| **Fondatore** | [Enzo Ferrari](../persons/enzo-ferrari.md) |
| **Predecessore** | Scuderia Ferrari (1929), Auto Avio Costruzioni (1939) |
| **Prima auto** | Ferrari 125 S (1947) |
| **Obiettivo** | Finanziare le corse vendendo auto stradali |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1929** | Enzo Ferrari fonda Scuderia Ferrari come team corse Alfa Romeo |
| **1939** | Enzo lascia Alfa Romeo, fonda Auto Avio Costruzioni |
| **1940** | Prima vettura (AAC 815), senza nome Ferrari per clausola non-compete |
| **1945** | L'azienda assume il nome Ferrari |
| **1947** | Prima Ferrari ufficiale: 125 S |
| **1951** | Inizio partnership con [Pininfarina](../company/pininfarina.md) |
| **1956** | Morte di Alfredo "Dino" Ferrari, figlio di Enzo |
| **1963** | Trattativa fallita con Ford |
| **1969** | Fiat acquisisce 50% |
| **1988** | Morte di Enzo Ferrari; Fiat sale al 90% |
| **1991** | [Luca di Montezemolo](../persons/luca-di-montezemolo.md) diventa Chairman |
| **2000-2004** | 5 titoli mondiali consecutivi F1 con Schumacher |
| **2014** | Di Montezemolo lascia; Sergio Marchionne CEO/Chairman |
| **2015** | IPO NYSE ($1B), 10% venduto |
| **2016** | Spin-off completo da FCA (3 gennaio) |
| **2017** | Fine partnership con Pininfarina (F12berlinetta ultima) |
| **2018** | Morte di Marchionne; John Elkann Chairman, Louis Camilleri CEO |
| **2021** | Benedetto Vigna diventa CEO |
| **2025** | Prima Ferrari full-electric annunciata |

## Il mancato accordo Ford (1963)

Nel 1963, Ford Motor Company tratto' l'acquisizione di Ferrari per $18M. L'accordo falli' quando Enzo Ferrari scopri' che non avrebbe mantenuto il controllo del programma corse. La rivalita' Ford-Ferrari culmine' nella vittoria Ford a Le Mans 1966.
