### L'Origine Leggendaria

Ferruccio Lamborghini (1916-1993), gia' affermato produttore di trattori, caldaie e condizionatori, decise di costruire auto sportive dopo un diverbio con Enzo Ferrari sulla qualita' delle sue GT.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1963** | Fondazione, stabilimento Sant'Agata Bolognese |
| **1964** | 350 GT, prima auto di serie |
| **1966** | Miura, prima supercar a motore centrale |
| **1971** | Countach, icona design anni '70-'80 |
| **1974** | Ferruccio vende le quote |
| **1978-1987** | Vari proprietari (Mimran, etc.) |
| **1987** | Chrysler acquisisce Lamborghini |
| **1990** | Diablo sostituisce Countach |
| **1994** | Chrysler vende a Mycom/V'Power |
| **1998** | Audi/Volkswagen acquisisce |
| **2003** | Gallardo, entry-level V10 |
| **2011** | Aventador, successore Murcielago |
| **2018** | Urus, primo SUV |
| **2020** | Winkelmann torna come CEO |
