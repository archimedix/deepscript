# Regno di Sardegna

> Stato sabaudo 1720-1861, nucleo del futuro Regno d'Italia

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1720 (cessione Sicilia per Sardegna) |
| **Dissoluzione** | 1861 (diventa Regno d'Italia) |
| **Sede** | Torino |
| **Sistema** | Monarchia assoluta, poi costituzionale (1848) |
| **Dinastia** | Casa Savoia |
| **Status** | defunct |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| Fusione perfetta | Unificazione amministrativa del 1847 tra Sardegna e Stati di Terraferma |
| Statuto Albertino | Costituzione concessa da Carlo Alberto nel 1848, rimasta in vigore fino al 1948 |
| Stati di Terraferma | Piemonte, Savoia, Nizza, Liguria sotto dominio sabaudo |
| Vicere' | Governatore dell'isola di Sardegna per conto del sovrano |

---

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Monarchia |
| **Forma di Governo** | Assoluta (1720-1848), costituzionale (1848-1861) |
| **Legislativo** | Parlamento bicamerale dal 1848 (Senato + Camera dei Deputati) |
| **Territorio** | Piemonte, Sardegna, Savoia, Nizza, Liguria (dal 1815) |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1720-1730** | Acquisizione | Vittorio Amedeo II ottiene la Sardegna in cambio della Sicilia |
| **1730-1796** | Consolidamento | Riforme amministrative, partecipazione guerre europee |
| **1796-1814** | Crisi napoleonica | Perdita del Piemonte, sovrani in esilio in Sardegna |
| **1814-1831** | Restaurazione | Ritorno a Torino, annessione di Genova (1815) |
| **1831-1848** | Riforme | Carlo Alberto: Statuto, codice civile, apertura liberale |
| **1848-1861** | Risorgimento | Guerre d'indipendenza, Cavour, unificazione italiana |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1720** | Trattato dell'Aia: Savoia ottengono la Sardegna |
| **1747** | Battaglia dell'Assietta: vittoria contro la Francia |
| **1793** | Invasione francese della Sardegna respinta |
| **1798** | Carlo Emanuele IV perde Torino, si rifugia in Sardegna |
| **1815** | Congresso di Vienna: annessione di Genova |
| **1821** | Moti costituzionali, abdicazione di Vittorio Emanuele I |
| **1847** | Fusione perfetta: unione amministrativa Sardegna-Terraferma |
| **1848** | Statuto Albertino, Prima guerra d'indipendenza |
| **1849** | Sconfitta di Novara, abdicazione di Carlo Alberto |
| **1852** | Cavour presidente del Consiglio |
| **1855** | Partecipazione alla guerra di Crimea |
| **1859** | Seconda guerra d'indipendenza, alleanza con Napoleone III |
| **1860** | Spedizione dei Mille, annessioni Italia centrale e meridionale |
| **1861** | Proclamazione del Regno d'Italia (17 marzo) |
