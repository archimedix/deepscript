# Belt and Road Initiative (BRI)

> Il piu' ambizioso progetto infrastrutturale della storia: la visione cinese per ridefinire la governance globale attraverso connettivita' fisica e finanziaria.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2013 (annuncio Kazakhstan) |
| **Sede** | Beijing, China |
| **Tipo** | Iniziativa strategica governativa |
| **Leadership** | [Xi Jinping](../persons/xi-jinping.md) |
| **Paesi partecipanti** | 149 (MoU firmati, 2024) |
| **Investimenti cumulativi** | $1.175 trilioni (2013-2024) |
| **Status** | active |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 7 settembre 2013 |
| **Luogo annuncio** | Nazarbayev University, Astana, Kazakhstan |
| **Fondatore** | [Xi Jinping](../persons/xi-jinping.md) |
| **Contesto** | Crescita economica cinese, eccesso capacita' industriale |
| **Obiettivo dichiarato** | Connettivita' infrastrutturale Eurasia-Africa |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2013 Set** | Xi annuncia "Silk Road Economic Belt" in Kazakhstan |
| **2013 Ott** | Xi annuncia "Maritime Silk Road" in Indonesia |
| **2014 Dic** | Creazione [Silk Road Fund](../swf/silk-road-fund.md) ($40B) |
| **2015** | Creazione Leading Group for BRI |
| **2016 Gen** | Lancio [AIIB](../bank/aiib.md) ($100B capitale) |
| **2017** | BRI inserito nella Costituzione CCP |
| **2017 Mag** | Primo Belt and Road Forum, Silk Road Fund aumentato a $55B |
| **2019** | Secondo Belt and Road Forum |
| **2023** | Decimo anniversario, $1T in contratti |
| **2024** | Record: $71B costruzione + $51B investimenti |

## Struttura e Governance

### Leadership

| Ruolo | Nome | Note |
|-------|------|------|
| Architetto | [Xi Jinping](../persons/xi-jinping.md) | Decisore supremo |
| Coordinamento | State Council | Supervisione generale |
| Operativo | NDRC + MFA + MOFCOM | Implementazione |

### Organi Direttivi

| Organo | Funzione |
|--------|----------|
| **Leading Group for BRI** | Coordinamento centrale (dal 2014) |
| **NDRC** | National Development and Reform Commission |
| **MFA** | Ministry of Foreign Affairs |
| **MOFCOM** | Ministry of Commerce |
| **Gruppi provinciali** | Implementazione locale |

**Nota**: A differenza di G20 o altre iniziative, BRI **non ha secretariato formale** ne' principi di governance condivisi. E' un'iniziativa unilaterale cinese con partecipazione bilaterale.

## Critiche

| Tema | Dettaglio |
|------|-----------|
| **Debt trap diplomacy** | Prestiti insostenibili per leverage geopolitico (Sri Lanka, Zambia) |
| **Mancanza trasparenza** | Termini prestiti non pubblici, corruzione |
| **Standard ambientali** | Progetti carbone, deforestazione |
| **Standard lavoro** | Uso lavoratori cinesi, condizioni locali |
| **Sovranita'** | Controllo infrastrutture strategiche (porti) |
| **Qualita' costruzione** | Progetti incompiuti o difettosi |

### Casi Controversi

| Paese | Progetto | Problema |
|-------|----------|----------|
| **Sri Lanka** | Porto Hambantota | Ceduto a Cina per 99 anni (2017) |
| **Pakistan** | CPEC | Debito insostenibile |
| **Montenegro** | Autostrada Bar-Boljare | Debito 80% PIL |
| **Kenya** | Ferrovia Mombasa-Nairobi | Costi gonfiati |

## Confronto con Iniziative Occidentali

| Iniziativa | Proponente | Budget | Focus |
|------------|------------|--------|-------|
| **BRI** | Cina | $1T+ | Infrastrutture, connettivita' |
| **B3W** | G7 (2021) | $600B (obiettivo) | Clima, salute, digitale |
| **Global Gateway** | UE (2021) | EUR300B | Standard democratici |
| **PGII** | USA (2022) | $600B (G7) | Alternativa a BRI |

**Differenza chiave**: BRI e' operativo dal 2013 con $1T+ disbursato; le alternative occidentali sono annunci recenti con funding limitato.

## Significato nel Sistema di Potere

### Obiettivi Strategici

| Obiettivo | Meccanismo |
|-----------|------------|
| **Egemonia regionale** | Infrastrutture creano dipendenza |
| **Internazionalizzazione RMB** | Prestiti in yuan |
| **Accesso risorse** | Mining, energia |
| **Mercati export** | Eccesso capacita' industriale |
| **Governance globale** | Alternativa a Bretton Woods |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Bilateralismo asimmetrico** | Cina negozia con singoli paesi, non blocchi |
| **Infrastruttura come leverage** | Porti, ferrovie creano dipendenza |
| **Financing as foreign policy** | Prestiti = influenza politica |
| **Standard-setting** | Cina definisce regole tecniche |
