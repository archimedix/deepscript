> Agenzia USA che regola farmaci e dispositivi medici, al centro del revolving door sistemico con Big Pharma - ogni Commissioner dal 2000 e' finito nell'industria

### I 3 Vaccini COVID e i loro Ex-Commissioner

```
VACCINO COVID         FDA COMMISSIONER         DESTINAZIONE
     |                      |                       |
     +-- Pfizer       →   Gottlieb (2017-19)  →   Board Pfizer
     |
     +-- J&J          →   McClellan (2002-04) →   Board J&J
     |
     +-- Moderna      →   Hahn (2019-21)      →   Flagship (incubatore Moderna)
```

**Pattern**: Tutte e 3 le aziende con vaccini COVID approvati da FDA hanno ex-Commissioner nei rispettivi board o incubatori.

## Funding Model: Chi Paga FDA?

### User Fees (PDUFA)

| Fonte | % Budget |
|-------|----------|
| **Industria (user fees)** | 65% |
| Governo federale | 35% |

**Implicazione**: L'industria che FDA deve regolare paga 2/3 del budget dell'agenzia.

### PDUFA (Prescription Drug User Fee Act)

| Anno | Evento |
|------|--------|
| **1992** | Introdotto PDUFA |
| **Effetto** | Aziende pagano per accelerare review |
| **Critica** | "Chi paga il pifferaio sceglie la musica" |

## Pattern Identificati

### 1. Regulatory Capture

FDA e' finanziata al 65% dall'industria che deve regolare. Commissioners finiscono sistematicamente in board pharma. Pattern classico di cattura del regolatore.

### 2. COVID Accelerator

Emergency Use Authorization per vaccini COVID:
- Pfizer: approvato da agency con Gottlieb-to-Pfizer precedent
- J&J: McClellan gia' nel board
- Moderna: Hahn approva, poi va in Flagship

### 3. Forum Connections

Hamburg collega FDA a network filantropico-transnazionale:
- GAVI (board)
- CFR (board)
- Gates Foundation (advisory)
- Rockefeller Foundation (ex-board)

### 4. Behind-the-Scenes Loophole

Ex-funzionari possono:
- Lavorare "behind the scenes"
- Dirigere campagne lobbying
- Influenzare senza contatto diretto

Solo il contatto diretto e' vietato.
