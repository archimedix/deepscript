# Indonesian Bank Restructuring Agency (IBRA)

> L'agenzia che nazionalizzo' il sistema bancario indonesiano durante la crisi 1998 e poi lo riprivatizz a prezzi di saldo.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 26 gennaio 1998 |
| **Dissoluzione** | 30 aprile 2004 |
| **Sede** | Jakarta, Indonesia |
| **Tipo** | Agenzia governativa (Agency) |
| **Nome locale** | Badan Penyehatan Perbankan Nasional (BPPN) |
| **Supervisione** | Ministero delle Finanze + IMF/World Bank/ADB |
| **Asset gestiti** | ~$65 miliardi |
| **Recovery** | ~$20 miliardi |
| **Status** | defunct |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 26 gennaio 1998 (Decreto Presidenziale No. 27/1998) |
| **Fondatore** | Governo Suharto (sotto pressione IMF) |
| **Contesto** | Crisi finanziaria asiatica 1997-98 |
| **Obiettivo dichiarato** | Ristrutturare sistema bancario, amministrare blanket guarantee |
| **Dissoluzione** | 27 febbraio 2004 (Decreto No. 15/2004, Megawati) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **Ott 1997** | Inizio crisi bancaria Indonesia |
| **Nov-Dic 1997** | Bank Indonesia fornisce prestiti BLBI di emergenza |
| **Gen 1998** | Istituzione IBRA e blanket guarantee |
| **Mag 1998** | Bank run su BCA durante riots anti-cinesi |
| **28 Mag 1998** | IBRA prende controllo BCA (92.8%) |
| **21 Mag 1998** | Caduta Suharto |
| **Ago 1998** | Creazione FSPC per supervisione |
| **1998-99** | 65 banche chiuse, 9 fuse, 4 nazionalizzate |
| **2000** | IPO BCA, IBRA vende 22.5% |
| **2002** | Vendita 51% BCA a FarIndo (Hartono) |
| **2004** | Dissoluzione IBRA |

## Il Caso BCA

Il sequestro e vendita di BCA rappresenta il caso piu' significativo:

| Data | Evento |
|------|--------|
| **1957** | BCA fondata dalla famiglia Salim |
| **Mag 1998** | Bank run durante riots: $5B ritirati in 2 settimane |
| **28 Mag 1998** | IBRA nazionalizza BCA (92.8%) |
| **2000** | IPO: IBRA vende 22.5% |
| **2002** | FarIndo (Hartono) acquisisce 51% |
| **2005** | Governo esce completamente |

> "Ci furono accuse di manipolazione nell'acquisizione del 51% di BCA"

La vendita ai Hartono a valutazioni depresse - mentre BCA era ancora sotto stress - rappresenta il pattern classico: liquidita' da business stabile (tabacco Djarum) usata per acquisire asset strategici durante crisi.

## Critiche e Controversie

| Tema | Dettaglio |
|------|-----------|
| **Lentezza** | Ristrutturazione troppo lenta |
| **Trasparenza** | Mancanza di trasparenza nelle operazioni |
| **Irregolarita'** | Accuse di irregolarita' nelle vendite |
| **Stipendi** | Compensi dirigenti "molto alti" |
| **Interferenze** | Interferenza politica costante |
| **Recovery basso** | $20B recuperati su $65B spesi (30%) |
| **Scandalo Bank Bali** | 1999, fondi dirottati per campagna elettorale |

### Problemi Strutturali

- 7 chairman in 6 anni (instabilita' leadership)
- Conflitti di competenza legale
- Problemi documentazione
- Pressioni politiche durante transizione democratica

## Eredita'

IBRA rappresenta un caso studio di:

1. **Nazionalizzazione di emergenza** - risposta a crisi sistemica
2. **Trasferimento di ricchezza** - da vecchi oligarchi (Salim) a nuovi (Hartono)
3. **Supervisione internazionale** - IMF/World Bank come garanti
4. **Governance debole** - instabilita' politica compromette efficacia
5. **Recovery limitato** - 30% degli asset recuperati

> "Il dottore delle banche" - soprannome popolare di IBRA
