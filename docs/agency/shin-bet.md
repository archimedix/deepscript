# Shin Bet (Shabak)

> Servizio di sicurezza interna israeliano: controspionaggio, antiterrorismo, protezione personalita'

| Aspetto | Dettaglio |
|---------|-----------|

| **Sede** | Tel Aviv, Israele |
| **Status** | Attivo |
| **Tipo** | Agenzia sicurezza interna |
| **Supervisione** | Primo Ministro israeliano |

## Missione

| Funzione | Dettaglio |
|----------|-----------|
| **Controspionaggio** | Protezione da spionaggio straniero |
