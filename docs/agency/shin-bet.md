# Shin Bet (Shabak)

> Servizio di sicurezza interna israeliano: controspionaggio, antiterrorismo, protezione personalita'

## Dati Essenziali

| Aspetto | Dettaglio |
|---------|-----------|
| **Nome ufficiale** | Sherut haBitachon haKlali (שירות הביטחון הכללי) |
| **Nome inglese** | Israel Security Agency (ISA) |
| **Acronimo** | Shin Bet, Shabak |
| **Fondazione** | 1948 |
| **Sede** | Tel Aviv, Israele |
| **Status** | Attivo |
| **Tipo** | Agenzia sicurezza interna |
| **Supervisione** | Primo Ministro israeliano |
| **Mandato direttore** | 5 anni, non rinnovabile |

---

## Missione

| Funzione | Dettaglio |
|----------|-----------|
| **Controspionaggio** | Protezione da spionaggio straniero |
| **Antiterrorismo** | Prevenzione attacchi interni |
| **Protezione VIP** | Sicurezza leader israeliani |
| **Sicurezza informazioni** | Protezione segreti di stato |
| **Territori** | Operazioni in Cisgiordania e Gaza |

## Storia

### Fondazione (1948)

| Aspetto | Dettaglio |
|---------|-----------|
| **Origine** | Shai (servizio segreto Haganah) |
| **Fondatore** | [Isser Harel](../persons/isser-harel.md) |
| **Primo direttore** | Isser Harel (1948-1952) |
| **Contesto** | Creazione Stato di Israele |

### Evoluzione

| Periodo | Sviluppo |
|---------|----------|
| **1948-1952** | Fondazione, Harel dirige Shin Bet e poi Mossad |
| **1953-1963** | Era Amos Manor, focus su spionaggio sovietico |
| **Anni '70** | Antiterrorismo post-Monaco |
| **1984** | Affare Bus 300, crisi di credibilita' |
| **Anni '90** | Prima Intifada, penetrazione Hamas |
| **2000-2005** | Seconda Intifada, targeted killings |
| **2023** | Fallimento 7 ottobre, crisi storica |
| **2025** | Licenziamento Ronen Bar, primo nella storia |

### Caso Ronen Bar (2021-2025)

| Data | Evento |
|------|--------|
| **Ott 2021** | Nominato direttore |
| **7 Ott 2023** | Attacco Hamas, fallimento intelligence |
| **16 Ott 2023** | Assume responsabilita' pubblica |
| **Mar 2025** | Licenziato dal governo Netanyahu |
| **Apr 2025** | Alta Corte: licenziamento illegittimo |
| **28 Apr 2025** | Annuncia dimissioni volontarie |
| **16 Giu 2025** | Lascia dopo 35 anni nei servizi |

Bar affermo' che Netanyahu chiese un parere di sicurezza per evitare di testimoniare al proprio processo, e che lui rifiuto'.
