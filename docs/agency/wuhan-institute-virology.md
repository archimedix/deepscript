# Wuhan Institute of Virology

> Primo laboratorio BSL-4 della Cina continentale, al centro della controversia sulle origini COVID-19

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1956 |
| **Sede** | Wuhan, Hubei, Cina |
| **Tipo** | Agency (Istituto di ricerca) |
| **Leadership** | Wang Yanyi (Director General, 2018-) |
| **Dipendenti** | ~300 ricercatori |
| **Parent** | [Chinese Academy of Sciences](chinese-academy-sciences.md) |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **BSL-4** | Biosafety Level 4, massimo livello di contenimento per patogeni letali |
| **BSL-3** | Biosafety Level 3, per patogeni trasmissibili per via aerea |
| **Gain-of-function** | Ricerca che modifica patogeni per aumentarne trasmissibilita' o virulenza |
| **Coronavirus** | Famiglia di virus a RNA, include SARS-CoV-1, MERS, SARS-CoV-2 |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1956 |
| **Fondatori** | [Gao Shangyin](../persons/gao-shangyin.md), [Chen Huagui](../persons/chen-huagui.md) |
| **Contesto** | Parte della "Marcia verso la Scienza" post-rivoluzionaria cinese |
| **Obiettivo dichiarato** | Ricerca virologica per industria, agricoltura e difesa nazionale |
| **Costo BSL-4** | 300 milioni yuan (~$44M), costruito 2014 con assistenza francese |

### Evoluzione del Nome

| Anno | Nome |
|------|------|
| **1956** | Wuhan Microbiology Laboratory, CAS |
| **1961** | South China Institute of Microbiology |
| **1962** | Wuhan Microbiology Institute |
| **1970** | Microbiology Institute of Hubei Province |
| **1978** | Wuhan Institute of Virology (nome attuale) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1956** | Fondazione da Gao Shangyin e Chen Huagui |
| **1978** | Torna sotto CAS, assume nome attuale |
| **2003** | CAS approva costruzione laboratorio BSL-4 |
| **2004** | Creato State Key Laboratory of Virology (con Wuhan University) |
| **2005** | Paper Science: pipistrelli identificati come reservoir SARS |
| **2014** | Costruzione BSL-4 con assistenza francese (CIRI Lyon) |
| **2017** | Accreditamento CNAS per BSL-4 |
| **2018** | BSL-4 operativo; Wang Yanyi diventa Director General |
| **2019 set** | Database WIV messo offline |
| **2019 dic** | Primi casi COVID-19 a Wuhan |
| **2020** | Pandemia COVID-19, controversia origini |
| **2022 ago** | NIH termina definitivamente subaward WIV |
| **2023 set** | HHS esclude WIV da finanziamenti federali USA per 10 anni |
| **2025 apr** | Casa Bianca dichiara ufficialmente origine lab leak |

## La Controversia COVID-19

### Ipotesi Lab Leak

| Aspetto | Dettaglio |
|---------|-----------|
| **RaTG13** | Coronavirus 96.2% identico a SARS-CoV-2, campionato da WIV nel 2013 |
| **Database offline** | Settembre 2019, prima dei casi noti |
| **Minatori Mojiang** | 6 malati nel 2012, 3 morti; WIV campiono' nella stessa miniera |
| **Project DEFUSE** | Proposta DARPA 2018 respinta per inserire siti clivaggio furina |

### Posizione Ufficiale WIV

> "We didn't even know the virus existed." — Wang Yanyi, 2020

### Indagini USA

| Anno | Azione |
|------|--------|
| **2021** | Rapporto IC non conclusivo su origini |
| **2022** | NIH termina subaward per mancata compliance |
| **2023** | DOE e FBI propendono per lab leak |
| **2023 set** | HHS esclude WIV da fondi federali per 10 anni |
| **2025 apr** | Casa Bianca dichiara ufficialmente origine lab leak |

### Critiche Principali

| Critica | Fonte |
|---------|-------|
| Mancata consegna lab notebooks | NIH, Congress |
| Database offline senza spiegazioni | Investigatori indipendenti |
| Ritardi nella condivisione sequenze | Comunita' scientifica |
| Classificazione ricerca militare | State Department 2021 |
