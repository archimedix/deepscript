# International AIDS Vaccine Initiative (IAVI)

> Pioniere globale della ricerca su vaccini HIV e malattie infettive neglette

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1996 |
| **Sede** | New York, USA |
| **Tipo** | Non-profit scientific research organization (501c3) |
| **Leadership** | Mark Feinberg (President & CEO) |
| **Staff** | 7 paesi, 3 laboratori (New Jersey, India, California) |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| Broadly Neutralizing Antibodies (bnAbs) | Anticorpi capaci di neutralizzare molteplici varianti HIV |
| Germline-targeting vaccine | Vaccino che stimola precursori di cellule B per produrre bnAbs |
| PEPFAR | President's Emergency Plan for AIDS Relief (programma USA) |
| ADVANCE | Cooperative agreement USAID-IAVI per vaccini HIV |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1996 |
| **Fondatore** | [Seth Berkley](../persons/seth-berkley.md) |
| **Contesto** | Gap tra ricerca accademica e sviluppo vaccini HIV accessibili |
| **Obiettivo dichiarato** | Tradurre scoperte scientifiche in soluzioni di salute pubblica globalmente accessibili |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1996** | Fondazione da Seth Berkley |
| **1998** | Primo grant da Gates Foundation |
| **1999** | UK e Olanda diventano founding government donors |
| **2001** | Inizio partnership USAID/PEPFAR |
| **2007** | Inizio supporto EDCTP per programma HIV |
| **2010** | Partnership con World Bank e Giappone |
| **2015** | Mark Feinberg diventa President & CEO (da Merck) |
| **2018** | Linda-Gail Bekker entra nel board (ex-IAS President) |
| **2018** | Partnership con CEPI per vaccino Lassa fever |
| **2021** | Breakthrough: trial IAVI G001 dimostra proof-of-concept per germline-targeting HIV vaccine |
| **2021** | Lancio consorzio LEAP4WA (Lassa fever) con CEPI |
| **2022** | Susan Silbermann, Wafaa El-Sadr, Alexis Pinto entrano nel board |
| **2023** | Espansione a TB vaccines |
| **2025** | Fine partnership USAID/PEPFAR (USAID dismantled) |

### Pattern di Board Composition

Il board IAVI e' un concentrato dell'elite global health:
- **3 CFR members**: Feinberg, Venkayya (life), Abboud (ex)
- **2 White House veterans**: Venkayya (Bush biodefense), Goosby (Clinton/Obama AIDS)
- **2 CEPI connections**: Feinberg (Scientific Advisory Chair), Venkayya (board)
- **2 Gates Foundation alumni**: Venkayya (Vaccine Delivery), Berkley (founder)
- **Big Pharma pipeline**: Merck (Feinberg), Pfizer (Silbermann, 34 anni), Takeda (Venkayya), Teva (Blumberg)
- **Gavi overlap**: Silbermann (board IFPMA 2017-20), Wijnroks (board NL gov), Berkley (CEO 2011-23)
- **Global Fund connection**: Wijnroks (Chief of Staff, Interim ED, Head Strategy), Goosby (ex-US board member)
- **Unica voce Global South**: Bekker (sudafricana, Zimbabwe-born, CEO Desmond Tutu Foundation, IAS President)
- **Triple multilateral board**: Wijnroks ha seduto in Global Fund + Gavi + UNAIDS contemporaneamente
- **WEF network**: Bekker (GLT 1998), Venkayya (Agenda Contributor)

## Critiche

| Tema | Dettaglio |
|------|-----------|
| HIV vaccine delay | 30+ anni senza vaccino efficace (problema scientifico, non organizzativo) |
| USAID dependency | 2025: fine funding USAID crea incertezza |
| Pharma influence | Board members da Merck (Feinberg 11 anni), Pfizer (Silbermann 34 anni), Takeda (Venkayya President), Teva (Blumberg) |
| Government revolving door | Chair ex-PEPFAR (Goosby), board ex-White House (Venkayya), CEO ex-Merck (Feinberg), board ex-Ambassador NL (Wijnroks) |
| CFR concentration | Almeno 3 figure chiave membri CFR |
| Gavi interlocking | 3 board members IAVI con legami diretti a Gavi (Silbermann board IFPMA, Wijnroks board NL gov, Berkley founder→CEO Gavi) |
| Global South underrepresentation | Su 11 board members, una sola voce dal Global South (Bekker, Sudafrica); dominanza USA/Europa |
| Multilateral concentration | Wijnroks ha seduto in Global Fund + Gavi + UNAIDS + IAVI: concentrazione governance in poche figure |
