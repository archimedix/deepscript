# Korean Central Intelligence Agency (KCIA)

> Servizio segreto sudcoreano 1961-1981, strumento di Park Chung Hee, rete APACL/WACL e Unification Church

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 13 giugno 1961 |
| **Sede** | Seoul, Corea del Sud |
| **Tipo** | Agency |
| **Fondatore** | Park Chung Hee (tramite Kim Jong-pil) |
| **Dipendenti** | 3.000+ (iniziali, poi espansione massiccia) |
| **Status** | Defunct (rinominata ANSP 1981, poi NIS 1999) |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 13 giugno 1961, un mese dopo il golpe del 16 maggio |
| **Fondatore** | [Park Chung Hee](../persons/park-chung-hee.md) |
| **Primo direttore** | [Kim Jong-pil](../persons/kim-jong-pil.md), nipote acquisito di Park |
| **Contesto** | Prevenire contro-golpe, reprimere l'opposizione interna |
| **Modello** | CIA americana, ma con poteri interni molto piu' ampi |
| **Obiettivo dichiarato** | Intelligence estera e controspionaggio |
| **Obiettivo reale** | Controllo politico totale, operazioni economiche, propaganda all'estero |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1961** | Fondazione dopo il golpe di Park. Kim Jong-pil primo direttore |
| **1961** | Personale iniziale: 3.000, reclutato dal KCIC (controintelligence militare) |
| **1963** | Report CIA: KCIA "organizza" la Unification Church come strumento politico |
| **1963** | Kim Jong-pil lascia la direzione, sostituito da Kim Hyong-uk |
| **1966** | Supporto logistico alla conferenza APACL/WACL a Seoul |
| **1970** | Lee Hu-rak diventa direttore |
| **1972** | Costituzione Yushin: KCIA rafforza il controllo interno |
| **1973** | Rapimento di Kim Dae-jung a Tokyo. Lee Hu-rak rimosso |
| **1976** | Kim Jae-kyu nominato direttore |
| **1976-78** | Scandalo Koreagate: 115 congressisti USA corrotti tramite Tongsun Park |
| **1977** | Commissione Fraser espone legami KCIA-Unification Church |
| **1979** | 26 ottobre: Kim Jae-kyu assassina Park Chung Hee durante una cena |
| **1980** | Chun Doo-hwan usa la direzione ad interim KCIA per consolidare il potere |
| **1981** | KCIA rinominata ANSP (Agency for National Security Planning) |
| **1999** | ANSP rinominata NIS (National Intelligence Service) |

## Operazioni Chiave

### Unification Church come asset

| Aspetto | Dettaglio |
|---------|-----------|
| **Origine** | Report CIA 1963: la UC fu "organizzata" dal direttore KCIA Kim Jong-pil |
| **Funzione** | Strumento politico con 27.000 membri |
| **Operativo chiave** | [Bo Hi Pak](../persons/bo-hi-pak.md), attache' militare a Washington, liaison KCIA-USA |
| **Attivita'** | Volontari negli uffici congressuali, raccolta fondi, propaganda anticomunista |
| **Fonte** | Commissione Fraser 1977, House Subcommittee on International Organizations |

### Koreagate (1976-1978)

| Aspetto | Dettaglio |
|---------|-----------|
| **Agente** | [Tongsun Park](../persons/tongsun-park.md), uomo d'affari e operativo KCIA |
| **Metodo** | Tangenti, contributi elettorali illegali, favori |
| **Target** | 115 membri del Congresso USA implicati |
| **Incriminazione** | 36 capi d'accusa: corruzione, frode, racketeering, mancata registrazione come agente straniero |
| **Investigatore** | Rep. Donald Fraser (D-Minnesota) |
| **Risultato** | Esposizione rete KCIA negli USA e legami con Unification Church |

### Rapimento Kim Dae-jung (1973)

| Aspetto | Dettaglio |
|---------|-----------|
| **Vittima** | Kim Dae-jung, leader dell'opposizione |
| **Luogo** | Tokyo, Giappone |
| **Esecutore** | KCIA sotto Lee Hu-rak |
| **Risultato** | Kim Dae-jung salvato (pressioni USA), Lee Hu-rak rimosso |
| **Seguito** | Kim Dae-jung diventa presidente 1998-2003 |

### Assassinio di Park Chung Hee (1979)

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | 26 ottobre 1979 |
| **Esecutore** | Kim Jae-kyu, direttore KCIA |
| **Luogo** | Cena privata, Seoul |
| **Contesto** | Tensioni interne sul livello di repressione |
| **Conseguenza** | Vuoto di potere, Chun Doo-hwan prende il controllo |
