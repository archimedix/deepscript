# Banca Europea per gli Investimenti (BEI/EIB)

> La banca di sviluppo dell'UE: fondata 1958 con il Trattato di Roma, sede a Lussemburgo, bilancio EUR 600 miliardi, primo finanziatore multilaterale per il clima

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1 gennaio 1958 |
| **Sede** | Luxembourg City, Luxembourg |
| **Tipo** | Banca multilaterale di sviluppo |
| **Presidente** | Nadia Calvino (dal 2024) |
| **Azionisti** | 27 Stati Membri UE |
| **Total Assets** | ~EUR 600 miliardi |
| **Financing 2024** | EUR 88.8 miliardi |
| **Rating** | AAA (Moody's, S&P, Fitch) |
| **Status** | Active |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1 gennaio 1958 |
| **Base legale** | Trattato di Roma (1957) |
| **Prima sede** | Bruxelles, Belgio |
| **Sede dal 1968** | Luxembourg City (Place de Metz, poi Kirchberg dal 1980) |
| **Primo presidente** | Pietro Campilli (Italia) |
| **Capitale iniziale** | 1 miliardo unita' di conto |
| **Obiettivo** | Finanziamento infrastrutture europee |

### Fondatori e Capitale Iniziale (1958)

| Paese | Quota |
|-------|-------|
| **Francia** | 300M u.a. |
| **Germania Ovest** | 300M u.a. |
| **Italia** | 240M u.a. |
| **Belgio, Lussemburgo, Paesi Bassi** | 160M u.a. (resto) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1957** | Trattato di Roma istituisce la BEI |
| **1958** | Prima riunione Board of Governors (25 gennaio, Bruxelles) |
| **1958** | Pietro Campilli primo presidente |
| **1962** | Primi prestiti fuori dalla CEE |
| **1968** | Trasferimento sede a Lussemburgo |
| **1980** | Nuova sede a Kirchberg |
| **1986** | Ingresso Spagna e Portogallo |
| **1994** | Creazione EIF (European Investment Fund) |
| **2000** | Formazione EIB Group (BEI + EIF) |
| **2000** | Philippe Maystadt presidente |
| **2009** | Aumento capitale EUR 67B (risposta crisi) |
| **2012** | Werner Hoyer presidente |
| **2015** | Lancio EFSI (Piano Juncker) |
| **2019** | Stop finanziamento combustibili fossili (dal 2022) |
| **2024** | Nadia Calvino presidente |
| **2024** | Finanziamenti record EUR 88.8B |
| **2025** | Target EUR 100B; focus difesa e sicurezza |

## EIB Group

Il Gruppo BEI comprende due entita':

### BEI (European Investment Bank)

- Prestiti diretti a grandi progetti
- Finanziamento infrastrutture, energia, innovazione
- Bilancio ~EUR 600B

### EIF (European Investment Fund)

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1994 |
| **Focus** | SME, venture capital, garanzie |
| **Azionariato** | BEI 62.2%, Commissione 30%, altri 7.8% |
| **Chair** | Nadia Calvino (dal 2024) |

---

## Dati Finanziari (2024)

| Metrica | Valore | Variazione |
|---------|--------|------------|
| **Total Balance Sheet** | ~EUR 600B | - |
| **Financing Signed** | EUR 88.8B | Record |
| **EIB Lending** | EUR 76.6B | - |
| **Climate & Environment** | EUR 51B | 57% del totale |
| **Cohesion** | EUR 38B | Record |
| **Energy (grids, storage)** | EUR 31B | Record |
| **Rating** | AAA | Stabile |
| **Leverage Ratio** | 210% | (limite 290%) |

### Target 2025

| Metrica | Valore |
|---------|--------|
| **Total Financing** | EUR 100B (approvato) |
| **Focus** | Grids, difesa, tech leadership |
| **Difesa/Sicurezza** | 3.5% del totale |

---

## Priorita' Strategiche 2024-2027

1. **Transizione climatica** - 57% investimenti green
2. **Sicurezza energetica** - Grids, storage, rinnovabili
3. **Coesione territoriale** - Riduzione divari regionali
4. **Innovazione e tech** - Leadership tecnologica UE
5. **Difesa e sicurezza** - Nuovo mandato 2025
6. **PMI** - Via EIF
