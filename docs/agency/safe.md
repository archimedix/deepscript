# State Administration of Foreign Exchange (SAFE)

> Il custode dei $3.3 trilioni di riserve valutarie cinesi: l'agenzia che finanzia silenziosamente la proiezione globale di Pechino.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1979 |
| **Sede** | Beijing, China |
| **Tipo** | Agenzia governativa |
| **Riserve gestite** | $3.3 trilioni (2024) |
| **Administrator** | Zhu Hexin (dal 2023) |
| **Supervisione** | State Council / PBOC |
| **Status** | active |

---

| Termine | Definizione |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1979 |
| **Nome originale** | State Central Administration of Foreign Exchange (SCAFE) |
| **Contesto** | Riforma sistema bancario cinese |
| **Rinominazione** | 1983 (diventa SAFE) |
| **Supervisione** | State Council, amministrato da PBOC |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1979** | Creazione SCAFE (riserve allora minime) |
| **1983** | Diventa SAFE, sussidiaria PBOC |
| **1997 Giu** | Creazione SAFE Investment Company (Hong Kong) |
| **2007** | Riserve superano $1 trilione |
| **2008 Set** | Creazione Investment Company PRC (Singapore) |
| **2009 Dic** | Creazione Gingko Tree Investment (Londra) |
| **2012** | Uffici Francoforte e New York |
| **2014** | Riserve raggiungono picco $4 trilioni |
| **2014 Dic** | SAFE contribuisce 65% a Silk Road Fund |
| **2016** | Pan Gongsheng diventa Administrator |
| **2023 Lug** | Pan Gongsheng promosso PBOC Governor |
| **2023 Dic** | Zhu Hexin diventa Administrator |
| **2024** | Riserve stabili ~$3.3 trilioni |

## Significato nel Sistema di Potere

### Ruolo Strategico

| Aspetto | Dettaglio |
|---------|-----------|
| **Stabilita' RMB** | Difende cambio yuan |
| **Proiezione globale** | Finanzia BRI via Silk Road Fund |
| **Leverage geopolitico** | $782B in US Treasuries = arma finanziaria |
| **Diversificazione** | Riduce dipendenza da USD |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Opacita'** | Investimenti offshore poco trasparenti |
| **Dual use** | Riserve = stabilita' + proiezione potere |
| **Interconnessione** | SAFE → SRF → BRI → influenza globale |
| **Policy banks** | Controlla finanziamento sviluppo via Buttonwood |
