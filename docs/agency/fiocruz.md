## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 25 maggio 1900 |
| **Nome originale** | Instituto Soroterápico Federal (Istituto Sieroterapico) |
| **Fondatore** | Oswaldo Cruz (direttore dal 1902) |
| **Contesto** | Produzione sieri e vaccini contro la peste bubbonica |
| **Obiettivo iniziale** | Campagne sanitarie a Rio de Janeiro |

### Oswaldo Cruz

Batteriologo brasiliano che implementò la vaccinazione di massa nel paese. Sotto la sua direzione (1902-1917) l'istituto si espanse verso la ricerca e medicina sperimentale.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1900** | Fondazione come Instituto Soroterápico Federal |
| **1902** | Oswaldo Cruz diventa direttore |
| **1908** | Rinominato Instituto Oswaldo Cruz |
| **1920** | Base per creazione Dipartimento Nazionale Sanità Pubblica |
| **1930** | Trasferimento al Ministero Educazione e Salute |
| **1970** | "Massacro di Manguinhos" - ricercatori forzati al ritiro |
| **1974** | Rinominato Fundação Oswaldo Cruz (Fiocruz) |
| **2009** | Associazione con Pasteur Network |
| **2015** | Membro ufficiale Pasteur Network |
| **2017** | Nísia Trindade prima donna presidente |
| **2020** | Accordo AstraZeneca per vaccino COVID |
| **2021** | Inizio produzione vaccino COVID Bio-Manguinhos |
| **2022** | Registrazione vaccino COVID 100% brasiliano |
| **2022** | Hub WHO per vaccini mRNA |
| **2023** | Mario Moreira presidente |
| **2025** | Moreira eletto presidente Pasteur Network |

## Pattern Identificati

### 1. Revolving Door Fiocruz-Governo

Nísia Trindade: Fiocruz (presidente) → Governo Lula (Min. Salute 2023).

Pattern di passaggio da istituzioni tecniche a ruoli politici.

### 2. Sovranità Vaccinale

Fiocruz rappresenta la strategia brasiliana per autonomia nella produzione di vaccini:
- Transfer tecnologico accelerato COVID
- Hub mRNA per future pandemie
- Riduzione dipendenza da import

## Massacro di Manguinhos (1970)

Nel 1970, durante la dittatura militare, ricercatori di spicco furono forzati al ritiro e persero i diritti politici - episodio noto come "Massacro di Manguinhos". Momento buio nella storia dell'istituzione, poi recuperata con la democratizzazione.
