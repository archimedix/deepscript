## Glossario

| Termine | Definizione |
|---------|-------------|
| **CEE** | Comunita' Economica Europea (1957-1993) |
| **CECA** | Comunita' Europea Carbone e Acciaio (1951-2002) |
| **Euratom** | Comunita' Europea Energia Atomica |
| **Eurozona** | 20 stati membri che usano l'euro |
| **Acquis** | Corpo legislativo UE che i candidati devono adottare |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 25 marzo 1957 (Trattati di Roma) |
| **Fondatori** | Belgio, Francia, Germania Ovest, Italia, Lussemburgo, Paesi Bassi |
| **Contesto** | Post-WWII, interdipendenza per prevenire guerre |
| **Obiettivo dichiarato** | Mercato comune, integrazione economica, pace |

### I Padri Fondatori

| Nome | Paese | Ruolo |
|------|-------|-------|
| [Jean Monnet](../persons/jean-monnet.md) | Francia | Architetto integrazione, primo presidente Alta Autorita' CECA |
| [Robert Schuman](../persons/robert-schuman.md) | Francia | Dichiarazione Schuman 9 maggio 1950 |
| [Alcide De Gasperi](../persons/alcide-de-gasperi.md) | Italia | PM Italia, padre fondatore |
| [Konrad Adenauer](../persons/konrad-adenauer.md) | Germania | Cancelliere Germania Ovest |
| [Paul-Henri Spaak](../persons/paul-henri-spaak.md) | Belgio | Conferenza Messina 1955, Trattati di Roma |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1950** | Dichiarazione Schuman (9 maggio) |
| **1951** | Trattato di Parigi, nasce la CECA |
| **1957** | Trattati di Roma: CEE ed Euratom |
| **1958** | Entrano in vigore i Trattati di Roma |
| **1973** | Primo allargamento: UK, Irlanda, Danimarca |
| **1979** | Prime elezioni dirette Parlamento Europeo |
| **1981** | Grecia entra nella CEE |
| **1986** | Spagna e Portogallo entrano; Atto Unico Europeo |
| **1992** | Trattato di Maastricht: nasce l'Unione Europea |
| **1993** | Mercato unico europeo operativo |
| **1995** | Austria, Finlandia, Svezia entrano |
| **1999** | Euro introdotto (valuta scritturale) |
| **2002** | Euro in circolazione |
| **2004** | "Big Bang": 10 nuovi membri (Europa dell'Est + Malta/Cipro) |
| **2007** | Romania e Bulgaria entrano |
| **2009** | Trattato di Lisbona |
| **2013** | Croazia entra (28 membri) |
| **2016** | Referendum Brexit |
| **2020** | UK esce dall'UE (27 membri) |
| **2022** | Ucraina e Moldova candidati |
| **2024** | Apertura negoziati Ucraina e Moldova |
