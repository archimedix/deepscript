## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 4 novembre 1952 |
| **Predecessore** | Armed Forces Security Agency (AFSA) |
| **Creatore** | Presidente Harry Truman (direttiva segreta) |
| **Contesto** | Guerra Fredda, necessita' intelligence centralizzata |
| **Obiettivo dichiarato** | Crittografia, intercettazione comunicazioni straniere |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1952** | Creazione NSA (segreta fino agli anni '70) |
| **1967-73** | Operazione Minaret: sorveglianza illegale cittadini USA |
| **1972** | Creazione Central Security Service (CSS) |
| **1978** | FISA Act: limiti legali alla sorveglianza domestica |
| **2001** | Post-9/11: President's Surveillance Program |
| **2007** | Lancio programma PRISM |
| **2009** | Creazione US Cyber Command (dual-hat con NSA) |
| **2013** | Rivelazioni Snowden: scandalo sorveglianza globale |
| **2015** | USA Freedom Act: riforme post-Snowden |
| **2024** | Nakasone lascia, Haugh nuovo direttore |

## Struttura e Governance

### Direttori Recenti

| Periodo | Nome | Note |
|---------|------|------|
| **1999-2005** | Michael Hayden | Poi direttore CIA |
| **2005-2014** | Keith Alexander | Piu' lungo mandato, era Snowden |
| **2014-2018** | Michael Rogers | Post-Snowden reforms |
| **2018-2024** | Paul Nakasone | Poi board OpenAI |
| **2024-** | Timothy Haugh | Attuale |

### Dual-Hat Command

Dal 2010, il direttore NSA comanda anche US Cyber Command:
- **NSA**: intelligence (raccolta, analisi)
- **Cyber Command**: operazioni offensive/difensive

## Programmi di Sorveglianza

### PRISM (2007-)

| Aspetto | Dettaglio |
|---------|-----------|
| **Target** | Comunicazioni straniere via provider USA |
| **Provider** | Microsoft, Google, Apple, Facebook, Yahoo, etc. |
| **Dati** | Email, video, foto, VoIP, file |
| **Rivelato** | Snowden 2013 |

### ECHELON

| Aspetto | Dettaglio |
|---------|-----------|
| **Alleanza** | Five Eyes (USA, UK, Canada, Australia, NZ) |
| **Capacita'** | Intercettazione globale comunicazioni |
| **Stazioni** | Menwith Hill (UK), Pine Gap (Australia), etc. |

### Metadata Collection

- Raccolta massiva metadati telefonate USA
- Limitata da USA Freedom Act 2015
- Controversie ongoing su privacy

## Rivelazioni Snowden (2013)

### Impatto

| Aspetto | Dettaglio |
|---------|-----------|
| **Leaker** | Edward Snowden, contractor NSA |
| **Documenti** | 1+ milione stimati |
| **Rivelazioni** | PRISM, raccolta massiva, spionaggio alleati |
| **Conseguenze** | Riforme, USA Freedom Act, tensioni diplomatiche |

### Figure coinvolte

- **Keith Alexander**: direttore durante leak
- **Michael Rogers**: successore, gestione crisi
- **James Clapper**: DNI, accusato di mentire al Congresso
