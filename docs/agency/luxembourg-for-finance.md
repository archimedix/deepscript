## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2008 |
| **Origine strategia** | 2004 (strategia "Luxembourg for Finance" proposta da ABBL) |
| **Contesto** | Globalizzazione mercati, necessita' promozione centro finanziario |
| **Modello** | Public-private partnership |
| **Presidenza** | Ministro delle Finanze (ex officio) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2000** | ABBL lancia PROFIL |
| **2004** | ABBL propone strategia "Luxembourg for Finance" |
| **2008** | Lancio ufficiale LFF come agenzia |
| **2013** | Nicolas Mackel nominato CEO |
| **2014-2019** | Tom Theobald Deputy CEO (primo mandato) |
| **2019** | Theobald passa al Ministero Finanze |
| **2021** | Mackel inizia terzo mandato |
| **2022** | Philipp von Restorff nominato Deputy CEO |
| **2024** | Mackel nominato Ambasciatore UE; Theobald diventa CEO |

## Missione e Attivita'

### Obiettivi

1. Promozione expertise del centro finanziario lussemburghese all'estero
2. Diversificazione dei servizi finanziari
3. Comunicazione istituzionale
4. Coordinamento federazioni settoriali
