## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1 agosto 1953 |
| **Creatore** | Dwight D. Eisenhower |
| **Strumento legale** | Reorganization Plan No. 8, Executive Order 10477 |
| **Contesto** | Guerra Fredda, contenimento comunismo |
| **Obiettivo dichiarato** | Informare e influenzare opinione pubblica estera a favore USA |
| **Abolizione** | Foreign Affairs Reform and Restructuring Act 1998 |

### Origini

L'USIA nacque dalla necessita' di centralizzare le attivita' di informazione internazionale sparse tra varie agenzie. Eisenhower consolido' in USIA tutti i programmi informativi precedentemente sotto il Dipartimento di Stato, eccetto gli scambi educativi.

All'estero l'agenzia operava come **U.S. Information Service (USIS)** per evitare l'associazione con attivita' di intelligence.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1942** | Creazione Voice of America (OWI) |
| **1948** | Smith-Mundt Act autorizza propaganda estera |
| **1953** | Eisenhower crea USIA (1 agosto) |
| **1953** | VOA passa sotto USIA |
| **1961** | [Edward R. Murrow](../persons/edward-murrow.md) diventa Director |
| **1962** | Copertura Crisi Missili Cuba |
| **1965** | Murrow si dimette per malattia |
| **1978** | USIA diventa USICA (Carter) |
| **1981** | [Charles Z. Wick](../persons/charles-wick.md) Director (fino 1989) |
| **1982** | Torna nome USIA |
| **1989** | Caduta Muro Berlino - cambia missione |
| **1998** | Foreign Affairs Reform Act ordina abolizione |
| **1999** | USIA abolita (1 ottobre) |
| **1999** | Funzioni broadcasting a BBG (poi [USAGM](usagm.md)) |
| **1999** | Altre funzioni a State Department |

## Mission e Attivita'

### Strumenti di Soft Power

| Strumento | Descrizione |
|-----------|-------------|
| **Voice of America** | Broadcasting radiofonico/TV internazionale |
| **Worldnet** | Satellite television (dal 1983) |
| **Fulbright Program** | Scambi accademici internazionali |
| **International Visitor Program** | Leader stranieri in visita USA |
| **USIS Libraries** | Biblioteche americane all'estero |
| **Film Service** | Documentari e film propagandistici |
| **Press Service** | News service per media esteri |

### Target Geografici

Durante la Guerra Fredda, focus su:
- Europa Orientale (oltre cortina)
- Unione Sovietica
- America Latina
- Asia (Vietnam, Cina)
- Medio Oriente

### Evoluzione della Missione

| Era | Focus |
|-----|-------|
| **1953-1989** | Contenimento comunismo, Guerra Fredda |
| **1989-1999** | Transizione post-Guerra Fredda, promozione democrazia |

## Eredita' e Impatto

### Pattern stabiliti

| Pattern | Descrizione |
|---------|-------------|
| **Revolving door** | Media → governo → think tank |
| **Editorial independence** | Tensione tra propaganda e giornalismo |
| **Soft power doctrine** | Broadcasting come strumento geopolitico |
