| Dato | Dettaglio |

| **Predecessore** | KGB First Chief Directorate |

| **Obiettivo** | Intelligence civile all'estero |

### Origini Storiche

L'SVR fa risalire le proprie origini al 20 dicembre 1920, con la creazione della Sezione Speciale della Cheka. La linea di successione:

| Periodo | Organizzazione |
|---------|----------------|
| 1920-1922 | INO (Foreign Dept) Cheka |
| 1922-1934 | GPU/OGPU |
| 1934-1946 | NKVD |
| 1946-1991 | KGB First Chief Directorate |
| 1991- | SVR |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1920** | Creazione Sezione Speciale Cheka |
| **Dic 1991** | Fondazione SVR, Primakov primo direttore |
| **1996** | Primakov diventa Min. Esteri, Trubnikov direttore |
| **2000** | Lebedev direttore, Putin presidente |
| **2007** | Fradkov direttore (nominato personalmente da Putin) |
| **2010** | **Illegals Program**: 10 agenti arrestati USA (Anna Chapman) |
| **2016** | Naryshkin direttore |
| **2020** | **SolarWinds**: attacco cyber supply-chain a 18.000 organizzazioni |
| **Apr 2021** | USA/UK attribuiscono SolarWinds a SVR/APT29 |
| **Mag 2023** | SVR sanzionato da US Treasury |
| **2024** | 750+ diplomatici russi espulsi dall'Europa |

- Direttore SVR riporta direttamente al Presidente
- Membro permanente Consiglio di Sicurezza russo
- Dal 2012: Presidente puo' emettere ordini segreti senza consultare parlamento

## Operazioni Notevoli

### Illegals Program (Operation Ghost Stories)

| Aspetto | Dettaglio |
|---------|-----------|
| **Periodo** | Anni 2000-2010 |
| **Arrestati** | 10 agenti SVR negli USA (27 giugno 2010) |
| **Figura nota** | Anna Chapman |
| **Indagine FBI** | Operation Ghost Stories (10+ anni) |
| **Comunicazioni** | Steganografia, inchiostro simpatico, reti ad-hoc |
| **Scambio** | 10 illegals per 4 prigionieri (incl. Sergei Skripal) |
| **Talpa** | Col. Aleksandr Poteyev (Vice Directorate S) |

### SolarWinds Cyberattack (2020)

| Aspetto | Dettaglio |
|---------|-----------|
| **Data** | Scoperto dicembre 2020 |
| **Metodo** | Supply-chain attack su SolarWinds Orion |
| **Vittime** | ~18.000 organizzazioni, 9 agenzie federali USA |
| **Attribuzione** | USA/UK: SVR/APT29/Cozy Bear (aprile 2021) |
| **Sanzioni** | 6 aziende tech russe, espulsione 10 diplomatici |

### Altre Operazioni Attribuite

| NotPetya (supporto) | 2017 | Ucraina, globale |
| Attacchi a think tank | Continuo | Europa, USA |

## Relazione con Altri Servizi Russi

| Servizio | Relazione | Focus |
|----------|-----------|-------|
| [FSB](fsb.md) | Coordinamento | FSB = interno, SVR = estero |
| GRU | Rivalita'/complementarita' | GRU = militare, SVR = civile |
| [KGB](kgb.md) | Predecessore | SVR = First Chief Directorate |

### Differenze SVR vs GRU

| Aspetto | SVR | GRU |
|---------|-----|-----|
| **Tipo** | Civile | Militare |
| **Focus** | Politico, economico, tecnologico | Militare |
| **Stile** | Operazioni lunghe, pazienza | Aggressivo, rischi maggiori |
| **Cyber** | APT29/Cozy Bear | APT28/Fancy Bear |
| **Assassinii** | Rari | Skripal (Novichok), altri |
