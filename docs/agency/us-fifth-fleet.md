# United States Fifth Fleet / NAVCENT

> La presenza navale americana piu' longeva nel Golfo Persico: dal 1948 a Bahrain, 2.5 milioni di miglia quadrate di controllo sulle rotte petrolifere mondiali.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1943 (WW2), riattivata 1995 |
| **Sede** | NSA Bahrain, Manama |
| **Tipo** | Flotta numerata US Navy |
| **Comando parent** | US Central Command (CENTCOM) |
| **Area di responsabilita'** | 2.5M miglia quadrate |
| **Coalizione** | Combined Maritime Forces (46 nazioni) |
| **Commander attuale** | VADM Curt Renshaw (2025-) |
| **Status** | Active |

Il Commander e' simultaneamente:

## Significato nel Sistema di Potere

### Proiezione USA nel Golfo

| Aspetto | Dettaglio |
|---------|-----------|
| **Permanenza** | 77 anni di presenza continua (1948-2025) |
| **Funzione** | Contenimento Iran, protezione rotte petrolio |
| **Soft power** | Integrazione militare con monarchie GCC |
| **Hard power** | Capacita' strike immediata su Iran |

### Dipendenza Bahrain

Il Bahrain e' l'unico stato del Golfo con base navale USA permanente. Questa relazione e':
- **Esistenziale** per la sicurezza del regime Al Khalifa
- **Economica** (affitto base, personale)
- **Politica** (protezione implicita USA)
