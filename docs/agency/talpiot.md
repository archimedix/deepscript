> IDF elite program che fonde scienza, tecnologia e leadership militare - incubatore della "Start-Up Nation"

| Metrica | Valore |
|---------|--------|

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1979 |
| **Fondatori** | [Felix Dothan](../persons/felix-dothan.md), [Shaul Yatziv](../persons/shaul-yatziv.md) |
| **Sponsor** | Rafael Eitan (Capo di Stato Maggiore IDF) |
| **Contesto** | Trauma della Guerra dello Yom Kippur (1973), necessita' di autosufficienza tecnologica dopo embargo francese |
| **Obiettivo dichiarato** | Sviluppare talenti per R&D militare, colmare gap tecnologico |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1973** | Guerra dello Yom Kippur - shock iniziale israeliano |
| **1978** | Dothan e Yatziv presentano proposta a Rafael Eitan |
| **1979** | Prima classe Talpiot: 25 cadetti |
| **1980s** | Programma si espande, ~50 cadetti/anno |
| **1990s** | Alumni iniziano a fondare startup (Check Point 1993) |
| **2000s** | Focus su cybersecurity e AI |
| **2015** | Microsoft acquisisce Adallom (fondata da Talpionim) per $320M |
| **2020** | Fondazione Wiz da team Talpiot |
| **2025** | Google acquisisce Wiz per $32B - record acquisizione tech israeliana |

## Struttura e Governance

### Organizzazione

| Aspetto | Dettaglio |
|---------|-----------|
| Supervisione | Ministry of Defense / MAFAT |
| Partner accademico | [Hebrew University](../university/hebrew-university.md) |
| Durata | 40 mesi accademici + 6 anni servizio = 9 anni totali |
| Grado finale | Segen (First Lieutenant) + B.Sc. |

### Processo di Selezione

1. ~10.000 candidati (top scorer test nazionale)
2. 300 selezionati per 2 giorni di test
3. Esami IQ, dinamiche di gruppo, valutazione psicologica
4. ~50 ammessi per classe
5. Criteri: eccellenza accademica, leadership, integrita' morale

### Curriculum

- B.Sc. in Fisica, Matematica o Computer Science (Hebrew University)
- Training militare in tutte le branche IDF
- Specializzazione R&D
- Post-corso: assegnazione a unita' R&D, Unit 8200, o posizioni combattimento

## Pattern Osservati

### Talpiot-to-Startup Pipeline

```
Talpiot (3 anni) → Unit 8200 (servizio) → Microsoft/Big Tech → Startup
```

Esempio Wiz founders:
1. 2001: Entrano in Talpiot
2. 2004: Assegnati a Unit 8200
3. 2012: Fondano Adallom
4. 2015: Exit a Microsoft, guidano Cloud Security Group
5. 2020: Fondano Wiz
6. 2025: Exit a Google per $32B

### Network Effect

- Talpionim tendono a co-fondare aziende insieme
- Investono reciprocamente nelle startup
- Compugen aveva piu' Talpionim di qualsiasi altra azienda israeliana
