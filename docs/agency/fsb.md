## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Predecessore** | Financial Stability Forum (FSF), 1999 |
| **Fondazione FSB** | Aprile 2009, G20 London Summit |
| **Contesto** | Crisi finanziaria 2008 - necessita' coordinamento globale |
| **Mandato** | Coordinare autorita' finanziarie nazionali e standard-setter |
| **Status legale** | Associazione no-profit di diritto svizzero |

### Da FSF a FSB

| Aspetto | FSF (1999-2009) | FSB (2009-) |
|---------|-----------------|-------------|
| **Membri** | G7 + pochi altri | G20 + tutti i centri finanziari |
| **Mandato** | Discussione e cooperazione | Coordinamento e raccomandazioni |
| **Potere** | Consultivo | Soft law, peer review |
| **Segretariato** | Piccolo, presso BIS | Ampliato, presso BIS |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1999** | Creazione Financial Stability Forum (G7) |
| **2008** | Crisi finanziaria globale |
| **2009** | **G20 London Summit**: FSF trasformato in FSB con mandato ampliato |
| **2009** | Mario Draghi primo Chairman |
| **2011** | Mark Carney Chairman |
| **2012** | Pubblicata prima lista G-SIBs |
| **2018** | Randal Quarles Chairman (primo USA) |
| **2021** | Klaas Knot Chairman |
| **2025** | Andrew Bailey Chairman |

## Funzioni e Mandato

### Mandate (dal G20)

1. **Identificare vulnerabilita'** nel sistema finanziario globale
2. **Sviluppare policy** per la stabilita' finanziaria
3. **Coordinare** autorita' nazionali e standard-setter
4. **Monitorare implementazione** degli standard concordati
5. **Gestire contingency planning** per crisi cross-border

### Output Principali

| Output | Descrizione |
|--------|-------------|
| **Lista G-SIBs** | 30 banche "too big to fail" con requisiti extra |
| **Lista G-SIIs** | Assicurazioni sistemiche |
| **TLAC** | Total Loss-Absorbing Capacity standard |
| **OTC derivatives** | Riforme mercati derivati |
| **Shadow banking** | Monitoraggio finanza non bancaria |
| **Crypto framework** | Regolamentazione crypto-asset (2023) |

## Il Pattern Draghi-Carney

Due Chairman FSB sono diventati capi di governo:

```
         FSB Chair
             |
    +--------+--------+
    |                 |
    v                 v
Mario Draghi      Mark Carney
(2009-11)         (2011-18)
    |                 |
    v                 v
ECB President     BoE Governor
(2011-19)         (2013-20)
    |                 |
    v                 v
PM Italia         PM Canada
(2021-22)         (2025-)
```

> **Osservazione**: La presidenza FSB come credenziale per ruoli politici apicali. Entrambi erano "tecnici" diventati leader politici.

## Critiche

| Tema | Dettaglio |
|------|-----------|
| **Soft law** | Raccomandazioni non vincolanti, dipende da implementazione nazionale |
| **Accountability** | Non risponde a parlamenti, solo a G20 |
| **Cattura regolamentare** | Membri sono anche regolati (banche centrali supervisori) |
| **Trasparenza** | Processi decisionali poco visibili |
