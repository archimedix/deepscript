## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 7 novembre 1985 |
| **Legge** | Law No. 3238 |
| **Nome originale** | Defence Industry Development and Support Administration Office (SaGeB) |
| **Contesto** | Risposta all'embargo USA post-Cipro 1974, necessita' di industria difesa nazionale |
| **Obiettivo** | Costituire industria difesa moderna, modernizzare Forze Armate Turche |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1974** | Embargo USA post-operazione Cipro, spinta verso autonomia |
| **Nov 1985** | Fondazione SaGeB (Law 3238) |
| **1989** | Ristrutturazione come SSM (Undersecretariat for Defence Industries) |
| **Apr 2014** | Ismail Demir nominato Undersecretary |
| **Dic 2017** | **Decreto 696**: SSM passa sotto Presidenza della Repubblica |
| **Lug 2018** | Rinominato SSB (Presidency of Defence Industries) |
| **Dic 2020** | **Sanzioni USA CAATSA** su SSB e Ismail Demir per acquisto S-400 |
| **Mag 2022** | Cambio nome inglese in "Defence Industry Agency" |
| **Giu 2023** | Prof. Dr. Haluk Gorgun succede a Ismail Demir |
| **2024** | Export record $7.154 miliardi (+29%) |
| **2025** | Target export $7.3 miliardi, 200+ progetti in corso |

## Sanzioni USA (2020)

### CAATSA Sanctions

| Data | Evento |
|------|--------|
| **14 Dic 2020** | USA impongono sanzioni CAATSA |
| **Motivo** | Acquisto sistema S-400 da Russia (Rosoboronexport) |
| **Soggetti** | SSB come entita', Ismail Demir personalmente |
| **Effetti** | Divieto licenze export USA, Demir su SDN List OFAC |

### Impatto

| Aspetto | Dettaglio |
|---------|-----------|
| **Export licenses** | Bloccate verso SSB |
| **F-35 program** | Turchia esclusa |
| **Demir** | Rimane su OFAC SDN List anche dopo uscita da SSB |

## Significato nel Sistema di Potere

### Architettura del Controllo

```
              ERDOGAN (Presidente)
                      |
         +------------+------------+
         |            |            |
       SSIK         TSKGV        Governo
    (presiede)   (chairman)        |
         |            |            |
         v            v            v
        SSB ←----→ Aziende ←---→ TSK
    (coordina)    (producono)   (acquista)
```

### Pattern: Centralizzazione 2017

Il Decreto 696 (Dicembre 2017) ha creato un sistema in cui:

| Prima | Dopo |
|-------|------|
| SSM sotto Min. Difesa | SSB sotto Presidenza |
| SSIK convocato da PM | SSIK convocato da Presidente |
| Controllo ministeriale | Controllo presidenziale diretto |

### Dual-Hatting Pattern

| Persona | Ruolo 1 | Ruolo 2 |
|---------|---------|---------|
| **Haluk Gorgun** | President SSB | Board TSKGV |
| **Ihsan Kaya** | Deputy President SSB | Chairman STM |

Questo pattern crea:
- Coordinamento pubblico-privato
- Potenziali conflitti di interesse
- Allineamento strategico verticale
