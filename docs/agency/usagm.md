## Mission e Focus

1. **Broadcasting** in 62 lingue verso regioni con scarsa liberta' di stampa
2. **Audience globale** di oltre 350 milioni (pre-2025)
3. **Soft power** americano durante e dopo Guerra Fredda
4. **Strumenti anti-censura** (Open Technology Fund)

### Target Geografici e Lingue

| Rete | Target | Lingue | Audience |
|------|--------|--------|----------|
| VOA | Globale | 48 | 350M+ settimanali |
| RFE/RL | Europa Est, Russia, Asia Centrale, Iran | 27 | 50M settimanali |
| RFA | Cina, Tibet, Xinjiang, Myanmar, Vietnam, NK | 9 | - |
| MBN | Mondo arabo | Arabo | - |
| OCB | Cuba | Spagnolo | - |

**Totale pre-2025**: 62 lingue, 100+ paesi, 400M+ audience

### Revolving Door Pattern

I leader USAGM mostrano un pattern classico di circolazione tra media, governo e think tank:

| Persona | Percorso |
|---------|----------|
| [Henry Loomis](../persons/henry-loomis.md) | MIT/USIA → **VOA** → CPB (pattern originale) |
| [John Lansing](../persons/john-lansing.md) | TV locale → Scripps Networks → **USAGM** → NPR |
| [Amanda Bennett](../persons/amanda-bennett.md) | WSJ/Bloomberg → **VOA/USAGM** → GMF board |
| [Michael Abramowitz](../persons/michael-abramowitz.md) | Washington Post → Holocaust Museum → Freedom House → **VOA** |
| [Jeff Shell](../persons/jeff-shell.md) | FOX/Comcast → **BBG** → NBCUniversal → Paramount |
| [Jamie Fly](../persons/jamie-fly.md) | CFR → NSC/Rubio → GMF → **RFE/RL** → Palantir |
| [Michael Pack](../persons/michael-pack.md) | RAI/Pathe' → CPB → Claremont → **USAGM** → Claremont (conservative media pipeline) |

**Nota storica:** Henry Loomis stabili' il pattern nel 1965 quando si dimise da VOA per proteggere l'indipendenza editoriale dall'interferenza della Casa Bianca (Johnson/Vietnam). Dichiarazione storica: *"The Voice of America is not the voice of the administration."*

**Contrasto 2020:** Michael Pack rappresenta il pattern opposto - nomina politica per controllare le reti. Il report 2023 di tre esperti indipendenti concluse che Pack "abused his authority" e "engaged in gross mismanagement." Collaboratore di Steve Bannon in documentari (Rickover, Last 600 Meters).
