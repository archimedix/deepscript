# Eastern Seaboard Development Program (ESDP)

> Il megaprogetto infrastrutturale degli anni '80 che trasformo' la costa orientale thailandese in hub industriale, finanziato dal Giappone quando la World Bank rifiuto'.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1982 (5째 Piano Nazionale) |
| **Sede** | Bangkok, Thailand |
| **Tipo** | Programma di sviluppo governativo |
| **Province** | Chonburi, Rayong, Chachoengsao |
| **Finanziamento** | ~$1.6 miliardi (JICA/OECF Giappone) |
| **PM di riferimento** | [Prem Tinsulanonda](../persons/prem-tinsulanonda.md) (1980-1988) |
| **Successore** | [EEC](eec.md) (2017) |
| **Status** | completed |

## Fondazione e Storia

### Contesto

| Fattore | Dettaglio |
|---------|-----------|
| **Scoperta gas naturale** | Grandi riserve nel Golfo di Thailandia (anni '70) |
| **Obiettivo** | Industrializzazione export-led, decentramento da Bangkok |
| **Ideazione** | Fine anni '70, NESDB |
| **Piano formale** | 5째 Piano Nazionale (1982-1986) |

### Il Rifiuto della World Bank

| Evento | Dettaglio |
|--------|-----------|
| **Richiesta iniziale** | Governo Thai chiede finanziamento a World Bank |
| **Risposta WB** | Rifiuto - progetto "mancante di razionalita' economica" |
| **Crisi 1985** | WB suggerisce di abbandonare Map Ta Phut e Laem Chabang |
| **Decisione Thai** | Rifiuto proposta WB, conferma piano originale |

### L'Intervento Giapponese

| Anno | Evento |
|------|--------|
| **1981** | PM Suzuki visita Thailandia, annuncia cooperazione |
| **1982 Maggio** | JICA avvia studio di fattibilita' per Map Ta Phut |
| **1982 Ottobre** | ESB incorporato nel 5째 Piano Nazionale |
| **1982 Novembre** | Approvata costruzione Laem Chabang e Map Ta Phut |
| **1982-1990s** | Giappone finanzia 16 progetti (~$1.6B) |

### Timeline Completa

| Anno | Evento |
|------|--------|
| **Fine '70s** | Scoperta gas naturale Golfo di Thailandia |
| **1980** | Prem Tinsulanonda diventa PM |
| **1981** | Visita PM Suzuki, impegno Giappone |
| **1982** | Lancio ESDP nel 5째 Piano Nazionale |
| **1985** | Crisi: WB chiede abbandono, Thai rifiuta |
| **1987** | Svalutazione baht, boom investimenti giapponesi |
| **1988** | Fine governo Prem |
| **1991** | Apertura Porto Laem Chabang |
| **1992** | Apertura Porto Map Ta Phut |
| **1997** | Crisi asiatica interrompe crescita |
| **2017** | EEC lanciato come successore |

## Infrastrutture Chiave

### I Due Porti

| Porto | Provincia | Tipo | Funzione |
|-------|-----------|------|----------|
| **Laem Chabang** | Chonburi | Commerciale | Import/export merci industriali |
| **Map Ta Phut** | Rayong | Industriale | Petrolchimica, adiacente gasdotto |

### Progetti Finanziati dal Giappone (16 totali)

| Categoria | Progetti |
|-----------|----------|
| **Porti** | Laem Chabang, Map Ta Phut |
| **Strade** | Rete stradale regionale |
| **Acquedotti** | Infrastrutture idriche |
| **Zone industriali** | Industrial estates |

### Zone Industriali

| Zona | Settore |
|------|---------|
| **Map Ta Phut Industrial Estate** | Petrolchimica (primo grande parco chimico Thai) |
| **Laem Chabang Industrial Estate** | Manifattura, automotive |
| **Amata City** | Elettronica, automotive |

## Impatto Economico

### Settori Sviluppati

| Settore | Note |
|---------|------|
| **Petrolchimica** | Basata su gas naturale Golfo |
| **Automotive** | Hub produzione Toyota, Honda, altri |
| **Elettronica** | Elettrodomestici, componenti |
| **Esportazioni** | Base per export-led growth |

### Investimenti Giapponesi

| Periodo | Trend |
|---------|-------|
| **Pre-1987** | Limitati |
| **1987-1997** | Boom dopo svalutazione baht |
| **Post-crisi** | Consolidamento, legacy |

### Risultati

| Metrica | Impatto |
|---------|---------|
| **Crescita** | Pilastro export-led growth per oltre un decennio |
| **Decentramento** | Spostamento industria da Bangkok |
| **Occupazione** | Creazione base manifatturiera moderna |

## Significato Storico

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Alternativa al Washington Consensus** | Giappone finanzia quando WB rifiuta |
| **Relazione Giappone-Thailandia** | Base per decenni di investimenti giapponesi |
| **Modello replicato** | EEC (2017) tenta di replicare successo ESB |
| **Shift geopolitico** | Nel 2020s, Cina sostituisce Giappone come partner principale |

### Da ESB a EEC: Continuita' e Cambiamento

| Aspetto | ESB (1982) | EEC (2017) |
|---------|------------|------------|
| **Finanziatore** | Giappone | Cina (BRI) |
| **Settori** | Petrolchimica, auto ICE | EV, data center, hi-tech |
| **Province** | Stesse 3 | Stesse 3 |
| **Contesto politico** | Governo semi-militare Prem | Giunta NCPO Prayuth |
