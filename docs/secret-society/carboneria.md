# Carboneria

> Societa' segreta risorgimentale, predecessore della Giovine Italia

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | ~1807 |
| **Sede** | Regno di Napoli (poi diffusa in tutta Italia) |
| **Tipo** | SecretSociety |
| **Leadership** | Struttura piramidale con capi segreti |
| **Status** | defunct (~1834) |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origine** | Regno di Napoli, sotto Gioacchino Murat (1808-1815) |
| **Radici** | Movimento giacobino napoletano, massoneria illuminata |
| **Denominazione** | Dalla Francia (simbolismo dei carbonai) |
| **Finanziamenti** | Aiuti economici dall'Inghilterra |
| **Obiettivo iniziale** | Trasformare governi assoluti in costituzionali |
| **Evoluzione** | Post-1830: ideali repubblicani e democratici |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **~1807** | Nascita nel Regno di Napoli sotto Murat |
| **1815** | Diffusione dopo la Restaurazione |
| **1820** | Moti di Napoli: Morelli e Silvati ottengono governo costituzionale |
| **1820** | Cospirazione in Lombardia: Silvio Pellico e Pietro Maroncelli arrestati |
| **1821** | Intervento austriaco annulla le conquiste napoletane |
| **1827** | Mazzini affiliato a Genova |
| **1830** | Moti di Modena: Ciro Menotti tradito e giustiziato |
| **1830** | Mazzini arrestato |
| **1831** | Mazzini fonda la Giovine Italia per superare i limiti della Carboneria |
| **~1834** | Declino irreversibile |

## Struttura e Governance

### Organizzazione Piramidale

| Livello | Funzione |
|---------|----------|
| **Vertice segreto** | Ristretto numero di capi, identita' ignota |
| **Comitato esecutivo** | Coordinamento tra vendite |
| **Vendite** | Cellule di ~10 adepti |
| **Compartimentazione** | I livelli inferiori non conoscono gli affiliati superiori |

## Limiti e Declino

| Critica | Dettaglio |
|---------|-----------|
| **Segretezza eccessiva** | Impediva coordinamento interno ed esterno |
| **Vulnerabilita'** | Arresto dei capi paralizzava l'organizzazione |
| **Base ristretta** | Mai superato il cerchio militare-borghese |
| **Critica di Mazzini** | Le masse popolari non furono mai mobilitate |
| **Ingenuita'** | Speranza di coinvolgere i sovrani stessi |
