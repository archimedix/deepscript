## Glossario

| Termine | Definizione |
|---------|-------------|
| **Obbedienza** | Organizzazione che federa logge massoniche sotto un'autorità comune |
| **Gran Maestro** | Massima carica elettiva dell'obbedienza |
| **Loggia** | Unità base della massoneria, gruppo locale di fratelli |
| **Laïcité** | Principio francese di separazione Stato-Chiesa |
| **Massoneria liberale** | Corrente continentale che non richiede fede in un Essere Supremo |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1773, ristrutturazione della Grande Loge de France (origini 1733) |
| **Fondatore** | Louis Philippe II, Duc d'Orléans (primo Gran Maestro) |
| **Contesto** | Riorganizzazione della massoneria francese sotto un'unica autorità centralizzata |
| **Obiettivo dichiarato** | Perfezionamento morale, progresso dell'umanità, libertà di coscienza |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1733** | Prime logge riconosciute in Francia dalla Grand Lodge di Londra |
| **1773** | Nascita ufficiale del GODF sotto Louis Philippe d'Orléans |
| **1778** | Voltaire iniziato nella Loggia Les Neuf Sœurs |
| **1789** | Rivoluzione Francese — numerosi membri tra i leader rivoluzionari |
| **1877** | Abolizione obbligo di fede in Dio — rottura con massoneria anglosassone (UGLE) |
| **1875-1893** | Jules Ferry e Léon Gambetta: leggi sulla scuola laica, obbligatoria e gratuita |
| **1905** | Legge di separazione Stato-Chiesa — forte influenza massonica nella stesura |
| **1940-44** | Vichy scioglie le logge, persecuzione dei massoni |
| **1945** | Ricostituzione post-Liberazione |
| **2010** | Ammissione delle donne (apertura alla mixité) |

## Caratteristiche Distintive

### Massoneria Liberale vs Anglosassone

| Caratteristica | GODF (Liberale) | UGLE (Regolare) |
|----------------|-----------------|------------------|
| Fede in Dio | Non richiesta (dal 1877) | Obbligatoria |
| Discussioni politiche | Ammesse | Vietate |
| Discussioni religiose | Ammesse | Vietate |
| Donne | Ammesse (dal 2010) | Escluse |
| Riconoscimento reciproco | No | Sì (tra obbedienze regolari) |

### Principi

1. **Libertà assoluta di coscienza** — nessun dogma religioso
2. **Laïcité** — separazione Stato-Chiesa come fondamento
3. **Progresso sociale** — impegno diretto in dibattiti politici
4. **Universalismo** — fraternità oltre confini nazionali
