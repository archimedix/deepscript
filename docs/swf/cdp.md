# Cassa Depositi e Prestiti (CDP)

> Il braccio finanziario dello Stato italiano: dalla ricostruzione post-unitaria alla difesa degli asset strategici nazionali.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 20 novembre 1850 |
| **Sede** | Roma, Italia |
| **Tipo** | Development Bank / Sovereign Wealth Fund |
| **Leadership** | Giovanni Gorno Tempini (Presidente), Dario Scannapieco (CEO) |
| **Azionista** | MEF 82.77%, Fondazioni bancarie 15.93% |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| CDP Equity | Veicolo per partecipazioni strategiche in imprese di interesse nazionale |
| CDP RETI | Veicolo per partecipazioni in società di reti (Snam, Terna, Italgas) |
| Fondazioni bancarie | Ex casse di risparmio trasformate in fondazioni, azioniste minoritarie CDP |
| Golden power | Potere di veto del governo su acquisizioni straniere in settori strategici |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 20 novembre 1850 |
| **Luogo** | Torino, Regno di Sardegna |
| **Nome originale** | Cassa Piemontese |
| **Contesto** | Finanziamento opere pubbliche nel Regno di Sardegna |
| **Sponsor politico** | Camillo Benso di Cavour |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1850** | Fondazione come Cassa Piemontese a Torino |
| **1857** | Prima riorganizzazione sotto governo Cavour |
| **1861** | Unità d'Italia, espansione nazionale |
| **1863** | Incorporazione casse degli altri stati pre-unitari |
| **1876** | Trasferimento sede a Roma |
| **2003** | Trasformazione in S.p.A. (70% MEF, 30% Fondazioni) |
| **2011** | Creazione Fondo Strategico Italiano (oggi CDP Equity) |
| **2021** | Dario Scannapieco diventa CEO |
| **2023-24** | Operazioni di "rimpatrio" asset strategici |
| **2024** | Riconferma Scannapieco |

## Ruolo Strategico

### Missione

1. **Finanziamento enti locali**: Prestiti a comuni, regioni, province
2. **Infrastrutture nazionali**: Sviluppo reti e trasporti
3. **Supporto imprese**: Crescita e internazionalizzazione PMI
4. **Difesa asset strategici**: Mantenimento controllo italiano su settori chiave

### Operazioni recenti (2023-2025)

- **Rimpatrio asset**: Acquisizioni per riportare sotto controllo italiano aziende cedute a stranieri
- **AI Fund**: Fondo per startup intelligenza artificiale
- **PNRR**: Ruolo centrale nell'attuazione del Piano Nazionale Ripresa Resilienza

## Critiche

| Tema | Dettaglio |
|------|-----------|
| Capitalismo di Stato | CDP usata per mantenere controllo pubblico su settori che potrebbero essere privatizzati |
| Inefficienza | Partecipazioni in aziende in difficoltà (es. Telecom) |
| Politicizzazione | Nomine legate a cicli politici |
| State Grid | Presenza cinese in CDP RETI criticata |
