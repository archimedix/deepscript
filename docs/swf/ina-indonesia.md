# Indonesia Investment Authority (INA)

> Il primo fondo sovrano indonesiano: dove il capitale di Abu Dhabi, Singapore e Cina converge per finanziare le infrastrutture del quarto paese piu' popoloso al mondo.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | Febbraio 2021 |
| **Sede** | Jakarta, Indonesia |
| **Tipo** | Sovereign Wealth Fund (SWF) |
| **Capitale iniziale** | $5 miliardi |
| **AUM target** | $24.5 miliardi ($200B obiettivo Jokowi) |
| **AUM Oct 2025** | IDR 163.4 trilioni |
| **Rating Fitch** | BBB (stable) / AAA (idn) |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **SWF** | Sovereign Wealth Fund - fondo sovrano statale |
| **IFSWF** | International Forum of Sovereign Wealth Funds |
| **Santiago Principles** | Standard internazionali per governance SWF |
| **Sui generis** | Entita' giuridica unica, indipendente dalla burocrazia |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Base legale** | Job Creation Law (2020) |
| **Fondazione** | Dicembre 2020 (legge), Febbraio 2021 (operativo) |
| **Capitale iniziale** | $5B governo ($2B cash + $3B azioni SOE) |
| **Commitment esterni** | $10B+ pre-lancio (DFC USA, JBIC, UAE) |
| **Target Jokowi** | $200 miliardi |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **Dic 2020** | INA istituita come "sui generis institution" |
| **Feb 2021** | Lancio operativo, $1B cash injection |
| **Mag 2021** | MoU ADIA/APG/CDPQ per toll platform $3.75B |
| **Nov 2021** | Secondo $1B cash injection |
| **Dic 2021** | $3B in azioni SOE trasferite |
| **2022** | Investimento Mitratel $800M con ADIA/GIC |
| **Set 2022** | Full member IFSWF |
| **2022** | Investimento Kimia Farma $150M con Silk Road Fund |
| **2022** | Investimento Pertamina Geothermal $500M con Masdar |
| **2023** | Investimento Traveloka $300M |
| **Ott 2025** | AUM IDR 163.4T (+92% dal 2021) |

## Confronto con Altri SWF ASEAN

| SWF | Paese | Fondazione | AUM |

| [GIC](../swf/gic.md) | Singapore | 1981 | $770B+ |

| Khazanah | Malaysia | 1993 | $35B+ |

**Pattern**: INA e' il piu' giovane ma copre la piu' grande economia ASEAN (GDP $1.4T).
