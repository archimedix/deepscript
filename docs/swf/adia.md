## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origine** | 1967 - Financial Investments Board |
| **Fondazione ADIA** | 1976 |
| **Fondatore** | Sheikh Zayed bin Sultan Al Nahyan |
| **Obiettivo** | Investire surplus petroliferi con basso rischio |

### Timeline Storica

| Anno | Evento |

## Struttura e Governance

### Famiglia Al Nahyan

ADIA e' controllata dalla **famiglia Al Nahyan**, la dinastia regnante di Abu Dhabi e dominante negli UAE.

```

    ┌─────────┴─────────────────────────────────┐
    │                                           │

(Presidente UAE         (Presidente UAE 2022-oggi)
2004-2022, m. 2022)              │
    │                            │
    └── Sheikh Hamed ◄───────────┘
        (MD ADIA)            (fratellastro)
```

## Stile di Investimento

### Caratteristiche

| Aspetto | ADIA |
|---------|------|
| Approccio | **Passivo**, diversificato |
| Orizzonte | Lungo termine (generazionale) |
| Trasparenza | Bassa |
| Trophy assets | No |
| Operativita' | Minima |

### Confronto con altri SWF UAE

| Fondo | AUM | Stile | Focus |
|-------|-----|-------|-------|
| **ADIA** | $1.06T | Passivo | Global diversified |
| [Mubadala](mubadala.md) | $330B | Attivo | Tech, AI, co-investments |
| ADQ | $249B | Operativo | Domestic champions |
| [MGX](mgx.md) | $100B target | Attivo | AI-focused |

### Allocazione Tipica

ADIA non pubblica il portfolio, ma stima:

| Asset Class | Range |
|-------------|-------|
| Equities | 35-55% |
| Fixed Income | 10-25% |
| Real Estate | 5-10% |
| Private Equity | 2-8% |
| Infrastructure | 1-5% |
| Alternatives | 5-10% |
