## Portfolio Investimenti

| Metrica | Valore |

## Integrazione Verticale: QSI + beIN

```
CONTENT                    DISTRIBUTION
   │                            │
   ▼                            ▼
┌─────────┐              ┌─────────────┐
│   PSG   │──────────────│ beIN MEDIA  │
│ (club)  │   Rights     │   GROUP     │
└─────────┘              └──────┬──────┘
                                │
                    ┌───────────┼───────────┐
                    │           │           │
                    ▼           ▼           ▼
               beIN Sports  Digiturk   Miramax
               (40+ paesi)  (Turkey)   (Hollywood)
```

**Vantaggio strategico:**
- QSI controlla il content (PSG, padel)
- beIN controlla la distribuzione (broadcasting)
- Stesso chairman (Al-Khelaifi) = decisioni coordinate
- Primo club a trasmettere partita live in VR (USA)
