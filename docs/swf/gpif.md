## Gestione Asset

### Modello Operativo

GPIF non gestisce internamente ma **delega a manager esterni**:
- 80% equity in strategie passive (index)
- 20% equity in strategie attive
- BlackRock e' il maggiore manager

### Manager Esterni Principali

| Manager | Ruolo |
|---------|-------|
| [BlackRock](../asset-manager/blackrock.md) | Largest asset manager |
| State Street | Passive equities |
| Legal & General | Passive equities |
| Mitsubishi UFJ Trust | Domestic equities |
| Nomura Asset Management | Domestic |

### Selezione Manager

| Criterio | Peso 2017+ |
|----------|------------|
| Stewardship activities | 30% |
| Track record | - |
| Fees | - |
| ESG integration | Obbligatorio |

## ESG e Stewardship

### Rivoluzione Mizuno (2015-2020)

Hiro Mizuno ha trasformato GPIF in leader globale ESG:

| Iniziativa | Anno | Impatto |
|------------|------|---------|
| Firma PRI | 2015 | Primo grande fondo pensione giapponese |
| Japan Stewardship Code | 2014 | Adozione |
| ESG nei mandati | 2017 | Peso stewardship da 5% a 30% |
| Taglio manager | 2018-19 | Ridotto BlackRock Japan per stewardship insufficiente |
| Climate Action 100+ | - | Membro fondatore |

### Requisiti per Manager

1. Firmare PRI o spiegare perche' no
2. Integrare ESG nelle decisioni di investimento
3. Engagement con aziende su temi ESG seri

### Dopo Mizuno

Mizuno (2020-):
- UN Special Envoy per Finanza Innovativa e Investimenti Sostenibili
- Special Advisor CEO di MSCI (ESG ratings)
- Board Tesla (2020-2024)
