## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1971 |
| **Scopo originale** | Gestire partecipazioni statali |
| **Trasformazione** | 2015-16, MBS ne prende controllo |
| **Obiettivo attuale** | Diversificare economia post-petrolio |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1971** | Fondazione |
| **2015** | MBS diventa chairman |
| **2016** | Lancio Vision 2030 |
| **Set 2016** | Meeting Son-MBS, nasce Vision Fund |
| **2017** | $45B commitment in SoftBank Vision Fund |
| **2018** | FII "Davos del Deserto" inaugurato |
| **Ott 2018** | Omicidio Khashoggi, boicottaggio FII |
| **2019** | IPO Saudi Aramco ($29B) |
| **2021** | $2B in Affinity Partners (Kushner) |
| **2021** | LIV Golf lanciato |
| **2024** | $26.7B US equity holdings |
| **2025** | Escluso da Stargate |
