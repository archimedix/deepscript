## Pattern Identificati

### 1. Hub AI degli Emirati

```
MUBADALA ($330B)
       │
       ├──► G42 (majority) ◄── Microsoft $1.5B
       │         │
       │         └──► MGX (co-founder)
       │                    │
       │                    ├──► OpenAI
       │                    ├──► Anthropic
       │                    └──► Stargate
       │
       └──► Silver Lake (partner)
                    │
                    └──► Endeavor ($25B)
```

### 2. Ponte Sport-Finance

Khaldoon Al Mubarak connette:
- **Finance**: CEO Mubadala ($330B)
- **Sport**: Chairman CFG ($6B+, 13 club)
- **AI**: Vice Chair MGX, Board G42

### 3. Co-Investment Model

Mubadala non investe solo come LP, ma prende **minority stakes in PE firms** (Silver Lake) per co-investire deal by deal. Modello replicato con:
- Silver Lake → Endeavor, Waymo
- SoftBank → Vision Fund
- KKR → Vari
