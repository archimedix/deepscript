## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1862 (Bank in Winterthur) |
| **Fusione UBS** | 1998 (Union Bank of Switzerland + Swiss Bank Corporation) |
| **Contesto** | Consolidamento bancario svizzero |
| **Obiettivo** | Wealth management globale, investment banking |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1862** | Fondazione Bank in Winterthur |
| **1912** | Fusione con Toggenburger Bank → Schweizerische Bankgesellschaft |
| **1998** | Fusione UBS: Union Bank of Switzerland + Swiss Bank Corporation |
| **2008** | Crisi subprime, bailout $60 miliardi da governo svizzero |
| **2009** | Scandalo evasione fiscale USA (Bradley Birkenfeld whistleblower) |
| **2011** | Sergio Ermotti diventa CEO |
| **2012** | Axel Weber diventa Chairman |
| **2020** | Ralph Hamers sostituisce Ermotti come CEO |
| **2022** | Colm Kelleher diventa Chairman |
| **2023** | Acquisizione Credit Suisse (CHF 3 miliardi) |
| **2023** | Ermotti ritorna come CEO per gestire integrazione |
| **2024** | Completata fusione UBS AG + Credit Suisse AG |

## Acquisizione Credit Suisse (2023)

### Timeline Crisi

| Data | Evento |
|------|--------|
| **Ott 2022** | Inizio deflusso depositi CS |
| **Mar 2023** | Saudi National Bank rifiuta ulteriori investimenti |
| **15 Mar 2023** | Iniziano negoziati emergenza |
| **19 Mar 2023** | Consiglio Federale approva acquisizione |
| **12 Giu 2023** | UBS completa acquisizione |
| **31 Mag 2024** | Fusione UBS AG + Credit Suisse AG |
| **1 Lug 2024** | Fusione entità svizzere |

### Dettagli Deal

| Aspetto | Valore |
|---------|--------|
| Prezzo | CHF 3 miliardi (~$3.2B) |
| Rapporto conversione | 1 UBS : 22.48 CS |
| Garanzie governative | Terminate agosto 2023 |
| Risparmi previsti | $13 miliardi entro 2026 |
| Deflusso CS pre-deal | CHF 171 miliardi |
