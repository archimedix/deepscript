## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Licenza originale** | 19 febbraio 1957 |
| **Nome originale** | PT Bank Bali Tbk |
| **Merger 5 banche** | 2002 |
| **Rebranding** | Ottobre 2002 (Permata = "gemma") |

### Le Cinque Banche Fuse (2002)

Permata Bank nasce dalla fusione di cinque banche sotto gestione IBRA dopo la crisi asiatica 1997-98:

| Banca | Note |
|-------|------|
| PT Bank Bali Tbk | Fondata 1954, centro scandalo corruzione |
| PT Bank Universal Tbk | - |
| PT Bank Prima Express | - |
| PT Bank Artamedia | - |
| PT Bank Patriot | - |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1954** | Fondazione Bank Bali |
| **1957** | Licenza bancaria commerciale |
| **1997-98** | Crisi finanziaria asiatica |
| **2001** | Programma ristrutturazione IBRA |
| **2002** | Fusione 5 banche, rebranding "Permata" |
| **2004** | Acquisizione 51% da Standard Chartered + Astra |
| **2006** | Quota SC/Astra sale a 89.01% |
| **2019** | SC/Astra a 89.12% (44.56% ciascuno) |
| **Dic 2019** | Bangkok Bank annuncia acquisizione |
| **Mag 2020** | Completamento acquisizione ($2.3B) |
| **Dic 2020** | Integrazione filiali Bangkok Bank Indonesia |
| **Gen 2021** | Status BUKU IV (OJK) |
| **2021-22** | Bangkok Bank sale a 98.71% (tender offer) |
| **Set 2024** | Adozione logo Bangkok Bank |

## Integrazione Bangkok Bank

### Razionale Strategico

Bangkok Bank era presente in Indonesia dal 1968 con solo 3 filiali (Jakarta, Surabaya, Medan). L'acquisizione di Permata ha trasformato la presenza:

| Prima | Dopo |
|-------|------|
| 3 filiali | 300+ filiali |
| Presenza limitata | Top 10 banca Indonesia |
| 50 anni crescita lenta | Scala immediata |

### Sinergie

- **Dicembre 2020**: Integrazione operazioni Bangkok Bank Indonesia in Permata
- **Settembre 2024**: Adozione logo Bangkok Bank
- Accesso a rete internazionale Bangkok Bank (32 filiali, 15 economie)
