## Fondazione e Storia

### Origini (1822-2003)

| Data | Evento |
|------|--------|
| **1822** | Fondazione Christiania Sparebank |
| **1847** | Fondazione Gjensidige |
| **1855** | Fondazione Bergens Privatbank |
| **1857** | Fondazione Den norske Creditbank |
| **1990s** | Crisi bancaria norvegese, intervento statale |
| **2003** | Fusione DnB + Gjensidige NOR = DnB NOR |
| **2011** | Rebranding a DNB |
| **2024** | Acquisizione Carnegie (banca svedese) per $1.14B |

### Crisi Bancaria 1990s

La crisi bancaria norvegese dei primi anni '90 porto' il governo a nazionalizzare temporaneamente diverse banche. Nel 2003, il governo Conservative decise di mantenere una quota del 34% in DnB per garantire la proprieta' nazionale della maggiore banca del paese.

## Pattern di Potere

### State Capitalism Norvegese

DNB e' parte del modello norvegese di "state capitalism":
- **34% statale**: Sufficiente per controllo nazionale, non maggioranza
- **Management privato**: Governance corporate indipendente
- **Obiettivo strategico**: Mantenere proprieta' nazionale della finanza

### Revolving Door

| Pattern | Esempio |
|---------|---------|
| Fondo sovrano → Banca | Svarva: Folketrygdfondet CEO → DNB Chairman |
| Carriera interna | Braathen: 20+ anni in DNB, shipping → CFO → CEO |
