## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1998 |
| **Predecessori** | Credito Italiano, Unicredito (CRT, CRV, Cassamarca), Rolo Banca |
| **Architetto fusione** | Andrea Orcel (allora Merrill Lynch) |
| **Obiettivo** | Creare la prima mega-banca italiana |

### Origini (1870-1998)

Le radici risalgono al 1870 con la fondazione della Banca di Genova, poi Credito Italiano. Nel 1998 la fusione con Unicredito (banche di Torino, Verona, Treviso) creo' UniCredito Italiano, la piu' grande banca italiana.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1870** | Fondazione Banca di Genova |
| **1895** | Diventa Credito Italiano |
| **1998** | Fusione Credito Italiano + Unicredito → UniCredito Italiano |
| **2005** | Acquisizione HypoVereinsbank (Germania) |
| **2007** | Acquisizione Capitalia (Banca di Roma) |
| **2008** | Crisi finanziaria, perdite miliardarie |
| **2010** | Dimissioni Profumo (scandalo Libia) |
| **2016-2017** | Aumento capitale €13 mld, ristrutturazione Mustier |
| **2021** | Andrea Orcel CEO |
| **Settembre 2024** | Inizio acquisizione Commerzbank |
| **Novembre 2024** | OPA su Banco BPM (€10 mld) |
| **Marzo 2025** | ECB approva 29.9% Commerzbank |
| **Luglio 2025** | Ritiro OPA Banco BPM (Golden Power) |
| **Luglio 2025** | Quota Commerzbank sale al 28% |

## L'Offensiva Europea (2024-2025)

### Commerzbank (Germania)

| Fase | Data | Quota |
|------|------|-------|
| Inizio acquisti | Settembre 2024 | 9% |
| Conversione derivati | Luglio 2025 | 20% |
| Quota attuale | Agosto 2025 | ~28% |
| Obiettivo | In corso | 29.9% (soglia OPA) |

**Opposizione tedesca**:
- Cancelliere Scholz contrario
- Premier Assia Rhein: "Ritiratevi!"
- Governo tedesco mantiene 12% (non vende)
- Sindacati: timore tagli 6.000+ posti

**Approvazioni ottenute**:
- ECB (marzo 2025): via libera fino a 29.9%
- Bundeskartellamt (aprile 2025): clearance antitrust

### Banco BPM (Italia)

| Fase | Data | Evento |
|------|------|--------|
| OPA lanciata | Novembre 2024 | €10.1 miliardi (all-share) |
| Rifiuto BPM | Novembre 2024 | "Sottovaluta il potenziale" |
| Golden Power | 2025 | Governo italiano impone condizioni |
| Approvazione UE | 2025 | Commissione approva con condizioni |
| **Ritiro OPA** | **Luglio 2025** | Orcel rinuncia |

**Risultato**: Orcel abbandona BPM, si concentra su Commerzbank. Utile Q2 2025: +25% a €3.3 miliardi.

## Pattern: La Banca Paneuropea

UniCredit rappresenta il tentativo italiano di creare un campione bancario europeo:

| Caratteristica | Dettaglio |
|----------------|-----------|
| **Strategia** | Consolidamento cross-border |
| **Germania** | HVB (2005) + Commerzbank (in corso) |
| **Italia** | Capitalia (2007), BPM (fallita 2025) |
| **CEE** | Network Europa Centro-Orientale |
| **Modello** | "Banca per il futuro dell'Europa" (Orcel) |

### Confronto con Altre Banche Italiane

| Banca | CEO | Strategia |
|-------|-----|-----------|
| UniCredit | Orcel | Espansione europea aggressiva |
| Intesa Sanpaolo | Messina | Consolidamento domestico |
| [Mediobanca](mediobanca.md) | Nagel | Boutique, wealth management |
