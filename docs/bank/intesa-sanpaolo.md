### Governance Model

Dal 2016, Intesa Sanpaolo ha adottato il **sistema monistico** (one-tier), abbandonando il sistema dualistico usato dalla fondazione.

## L'Acquisizione UBI Banca (2020-2021)

| Fase | Data | Dettaglio |
|------|------|-----------|
| Annuncio OPA | Febbraio 2020 | €4.9 miliardi, 17 azioni ISP per 10 UBI |
| Rifiuto UBI | Febbraio 2020 | "Offerta inadeguata" |
| Lancio formale | Luglio 2020 | |

| Controllo | 5 agosto 2020 | |
| Fusione | 12 aprile 2021 | Incorporazione |

**Risultato**: Intesa Sanpaolo diventa seconda banca Eurozona per capitalizzazione.

## Pattern: La Banca delle Fondazioni

Intesa Sanpaolo rappresenta il modello italiano di banca controllata dalle fondazioni:

| Caratteristica | Dettaglio |
|----------------|-----------|
| **Azionisti** | Fondazioni bancarie (~18% coordinato) |
| **Governance** | Patto parasociale fondazioni |
| **Leadership** | Stabile, conservatrice |
| **Strategia** | Consolidamento domestico (vs UniCredit paneuropea) |
| **Politica** | Vicinanza al sistema, "banca di sistema" |

### Confronto con UniCredit

| Aspetto | Intesa Sanpaolo | UniCredit |
|---------|-----------------|-----------|
| CEO | Messina (2013-) | Orcel (2021-) |
| Strategia | Domestica | Paneuropea aggressiva |
| Stile | Conservatore | Deal-maker |
| Azionisti | Fondazioni | Free float |
| M&A recenti | UBI (Italia) | Commerzbank, BPM (fallita) |
