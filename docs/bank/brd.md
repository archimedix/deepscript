## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origini** | 1923 - Societatea Nationala de Credit Industrial |
| **Nazionalizzazione** | 1948 - diventa Banca de Credit pentru Investitii |
| **Rifondazione** | 1990 - Banca Romana pentru Dezvoltare (BRD) |
| **Privatizzazione** | 1999 - Société Générale acquisisce 51% |
| **Obiettivo** | Retail banking, corporate banking, investment banking |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1923** | Fondazione Societatea Nationala de Credit Industrial |
| **1948** | Nazionalizzazione comunista |
| **1990** | Rifondazione come BRD - banca commerciale |
| **1998** | Accordo con Société Générale per cessione 51% |
| **1999** | Société Générale completa acquisizione, EBRD acquisisce 4.99% |
| **2001** | Quotazione Bucharest Stock Exchange (BVB) |
| **2004** | Société Générale acquisisce quota residua dello Stato |
| **2011** | EBRD vende intera partecipazione |
| **2016** | François Bloch diventa CEO |
| **2023** | Maria Rousseva prima donna CEO |
