## Glossario

| Termine | Definizione |
|---------|-------------|
| **Bancassurance** | Modello integrato banca + assicurazione |
| **KBC Ancora** | Holding quotata che detiene ~18.6% di KBC Group |
| **Cera** | Cooperativa degli ex-clienti CERA Bank |
| **MRBB** | Maatschappij voor Roerend Bezit van de Boerenbond (veicolo Boerenbond) |
| **Boerenbond** | Associazione agricoltori cattolici fiamminghi |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione fusione** | 1998 |
| **Componenti fusione** | Kredietbank (KB) + CERA Bank + ABB Insurance + Fidelitas Insurance |
| **Nome** | KBC = KredietBank + CERA |
| **Radici** | 1889 (Volksbank van Leuven) |

### Origini Cattoliche Fiamminghe

| Anno | Evento |
|------|--------|
| **1889** | Fondazione Volksbank van Leuven (cooperativa cattolica) |
| **1890s** | Boerenbond sviluppa rete di casse rurali cooperative |
| **1935** | Fusione ABV + BHN Kortrijk → Kredietbank |
| **1938** | Fernand Collin presidente, definisce KB come "banca fiamminga indipendente" |

### Kredietbank (1935-1998)

| Aspetto | Dettaglio |
|---------|-----------|
| **Identita'** | Unica istituzione finanziaria fiamminga sopravvissuta alle crisi anni '30 |
| **Motto** | "Una banca indipendente per una clientela indipendente" |
| **Espansione** | Lussemburgo (1949), Vallonia (1961) |
| **Innovazione** | Primo ATM in Belgio (fine anni '60), co-fondatore Bancontact |
| **1970** | Co-fondatore Inter-Alpha Group of Banks |

### Era KBC (1998-oggi)

| Anno | Evento |
|------|--------|
| **1998** | Fusione KB + CERA + ABB → KBC |
| **2008-09** | Crisi finanziaria, aiuti di stato belgi |
| **2012** | Johan Thijs CEO |
| **2020** | Koenraad Debackere Chairman |
| **2025** | Acquisizione 365.bank (Slovacchia) per EUR 761 milioni |

## Pattern di Potere

### Capitalismo Cooperativo Fiammingo

| Aspetto | Dettaglio |
|---------|-----------|
| **Origine** | Movimento cattolico sociale fine '800 |
| **Struttura** | Cooperative (Cera, Boerenbond) → holding → banca |
| **Effetto** | Azionariato stabile, governance long-term |
| **Contrasto** | Diverso da modello anglosassone shareholder-driven |

### Banca "Radicata"

| Aspetto | Dettaglio |
|---------|-----------|
| **Filosofia** | "Rooted in local community" |
| **Accordo azionisti** | 30+ anni di stabilita' |
| **Vantaggio** | Resistenza a takeover ostili |

### Espansione CEE

| Aspetto | Dettaglio |
|---------|-----------|
| **Strategia** | Acquisizioni in ex-blocco sovietico |
| **Paesi** | Repubblica Ceca, Slovacchia, Ungheria, Bulgaria |
| **Logica** | Mercati in crescita, meno saturati |
