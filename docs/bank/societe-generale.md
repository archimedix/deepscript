## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 4 maggio 1864 |
| **Decreto** | Firmato da Napoleone III |
| **Fondatori** | Joseph-Eugène Schneider, Paulin Talabot, Edward Blount |
| **Obiettivo** | Sviluppo di commercio e industria in Francia |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1864** | Fondazione con decreto di Napoleone III |
| **1870** | 15 filiali a Parigi, 32 in Francia |
| **1871** | Apertura ufficio permanente Londra |
| **1945** | Nazionalizzazione |
| **1987** | Privatizzazione |
| **2008** | Scandalo Kerviel (perdita €4.9bn), Oudéa diventa CEO |
| **2010** | Co-fondazione Amundi (fusione SGAM con CAAM) |
| **2015** | Separazione ruoli Chairman/CEO, Bini Smaghi Chairman |
| **2021** | Vendita Lyxor ad Amundi |
| **2023** | Slawomir Krupa succede a Oudéa come CEO |
| **2025** | Alexis Kohler (ex capo gabinetto Macron) entra come EVP |
| **2026** | William Connelly succederà a Bini Smaghi come Chairman |
