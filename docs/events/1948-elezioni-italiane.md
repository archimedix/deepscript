# Elezioni Italiane 1948

> La prima grande interferenza elettorale CIA: il modello per decenni di operazioni coperte

## Cosa Accadde

Il 18 aprile 1948 si tennero le prime elezioni politiche della Repubblica Italiana. La CIA intervenne massicciamente per impedire la vittoria del **Fronte Democratico Popolare** (PCI-PSI), finanziando la **Democrazia Cristiana** con $10-20 milioni e orchestrando una campagna di propaganda senza precedenti.

### Timeline

| Data | Evento |
|------|--------|
| **Dic 1947** | NSC approva operazioni coperte Italia |
| **Gen 1948** | CIA inizia finanziamenti DC |
| **Mar 1948** | Campagna lettere italo-americani |
| **18 apr 1948** | Elezioni: DC 48.5%, FDP 31% |
| **1948-60s** | CIA interviene in ogni elezione |

| Metrica | Valore |
