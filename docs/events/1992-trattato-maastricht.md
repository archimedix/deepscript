# Trattato di Maastricht (1992)

> Il patto che creo' l'Euro e la governance tecnocratica europea

## Cosa Accadde

Il 7 febbraio 1992, i dodici paesi della Comunita' Europea firmarono a Maastricht il Trattato sull'Unione Europea. Il trattato creo' l'Unione Europea, stabili' i criteri per l'unione monetaria e la futura Banca Centrale Europea.

### I Punti Chiave

| Aspetto | Contenuto |
|---------|-----------|
| **Unione Europea** | Creazione formale dell'UE |
| **Unione monetaria** | Percorso verso moneta unica |
| **Criteri di convergenza** | Deficit <3%, debito <60% PIL |
| **BCE** | Istituzione banca centrale indipendente |
| **Tre pilastri** | CE, PESC, GAI |

### Timeline

### Il Patto Franco-Tedesco

| Aspetto | Dettaglio |
|---------|-----------|
| **Contesto** | Caduta Muro, riunificazione tedesca |
| **Paura francese** | Germania troppo potente |
| **Richiesta Mitterrand** | Abbandono del Marco per moneta comune |
| **Accettazione Kohl** | Senza consultare Bundesbank |
| **Contropartita tedesca** | BCE indipendente, modello Bundesbank |

> Mitterrand, gravemente malato, partecipo' a un dibattito televisivo sostenendo che il trattato aveva "grande valore simbolico".

### Il Ruolo di Delors

| Aspetto | Dettaglio |
|---------|-----------|
| **1988** | Incaricato dal Consiglio di Hannover |
| **Rapporto Delors** | Tre fasi verso unione monetaria |
| **Visione** | Europa federale |
| **Delusione** | Maastricht meno federale di quanto sperava |

## Chi Beneficio'

| Beneficiario | Come |
|--------------|------|
| **Germania** | BCE su modello Bundesbank, stabilita' |
| **Francia** | "Controllo" sulla politica monetaria tedesca |
| **Commissione UE** | Piu' poteri sovranazionali |
| **Tecnocrati** | Governance per regole, non democrazia |
| **Grandi imprese** | Mercato unico, stabilita' cambi |
| **Esportatori tedeschi** | Euro "sottovalutato" rispetto al Marco |

### I Criteri di Convergenza

| Criterio | Requisito |
|----------|-----------|
| **Inflazione** | Max 1.5% sopra media 3 paesi piu' stabili |
| **Deficit** | Max 3% del PIL |
| **Debito** | Max 60% del PIL |
| **Tassi** | Max 2% sopra media 3 paesi piu' stabili |
| **Cambi** | 2 anni nello SME senza svalutazioni |

---

## Chi Perse

| Perdente | Come |
|----------|------|
| **Sovranita' nazionale** | Politica monetaria ceduta a BCE |
| **Paesi con alto debito** | Austerita' obbligata (Italia, Belgio, Grecia) |
| **Democrazie** | Decisioni economiche de-politicizzate |
| **Lavoratori** | Svalutazione interna invece che monetaria |
| **Sud Europa** | Moneta troppo forte, deindustrializzazione |

---

## Il Contesto: Riunificazione Tedesca

### La "Cattura" della Germania

| Aspetto | Dettaglio |
|---------|-----------|
| **Paura europea** | Germania riunificata troppo potente |
| **Strategia francese** | Vincolare Germania a progetto europeo |
| **Marco** | Simbolo di potenza economica tedesca |
| **Euro** | "Europeizzazione" del potere monetario tedesco |

### L'Ironia

La Germania ottenne una BCE sul modello Bundesbank, con sede a Francoforte, parlante tedesco come lingua di lavoro, e con mandato esclusivo sulla stabilita' dei prezzi.
