# Scoperta della P2 (1981)

> Il raid di Villa Wanda: quando l'Italia scopri' la loggia segreta che controllava il potere

## Cosa Accadde

Il 17 marzo 1981, la Guardia di Finanza perquisendo **Villa Wanda**, la residenza di [Licio Gelli](../persons/licio-gelli.md) ad Arezzo, scopri' la lista dei membri della loggia massonica segreta **Propaganda Due (P2)**: 962 nomi che includevano ministri, generali, banchieri, giornalisti e imprenditori.

### Timeline

| Data | Evento |
|------|--------|
| **Mar 1981** | Raid Villa Wanda, lista scoperta |
| **21 mag 1981** | Lista pubblicata |
| **Giu 1981** | Governo Forlani cade |
| **Dic 1981** | P2 sciolta per legge |
| **1984** | Commissione Anselmi |
| **2015** | Morte Gelli |
