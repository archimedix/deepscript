# Crisi Finanziaria 2008

> Il crollo che dimostro' "too big to fail" e socializzo' le perdite di Wall Street

## Cosa Accadde

Nel settembre 2008, il sistema finanziario globale arrivo' sull'orlo del collasso. Lehman Brothers falli', altre banche furono salvate con soldi pubblici, e i mercati crollarono. Fu la peggiore crisi finanziaria dalla Grande Depressione.

### Timeline

| Data | Evento |
|------|--------|
| **2007** | Inizio crisi subprime |
| **Marzo 2008** | Bear Stearns venduta a JPMorgan ($2→$10/azione) |
| **7 settembre** | Fannie Mae e Freddie Mac nazionalizzate |
| **15 settembre** | Fallimento Lehman Brothers |
| **16 settembre** | AIG salvata ($85 miliardi) |
| **18 settembre** | Paulson/Bernanke chiedono $700 miliardi |
| **29 settembre** | Camera respinge TARP (prima volta) |
| **3 ottobre** | TARP approvato |
| **Ottobre 2008** | Crollo mercati globali |

### L'Impatto

| Metrica | Valore |
|---------|--------|
| **Crollo Dow Jones** | -54% dal picco |
| **Ricchezza persa** | $12.8 trilioni |
| **Disoccupazione USA** | Da 4.7% a 10% |
| **Pignoramenti** | 3.8 milioni di case |

### La Riunione del 18 Settembre 2008

| Aspetto | Dettaglio |
|---------|-----------|
| **Dove** | Capitol Hill |
| **Chi** | Paulson, Bernanke, Pelosi, leader Congresso |
| **Messaggio Bernanke** | "If we don't do this, we may not have an economy on Monday" |
| **Richiesta** | $700 miliardi |
| **Risultato** | TARP approvato dopo primo rifiuto |

### Il Caso Lehman

| Aspetto | Dettaglio |
|---------|-----------|
| **Decisione** | Lasciata fallire |
| **Giustificazione** | "Nessuna autorita' legale" |
| **Critica** | Bear Stearns salvata, perche' Lehman no? |
| **Conseguenze** | Panico globale |
| **Dibattito** | Errore o necessita'? |

> Paulson aveva detto a Lehman mesi prima che non ci sarebbe stato bailout, ma Bear Stearns fu salvata.

## Le Cause

### Il Sistema

| Fattore | Descrizione |
|---------|-------------|
| **Deregulation** | Glass-Steagall abolita (1999) |
| **Derivati** | Non regolati (Commodity Futures Modernization Act 2000) |
| **Rating agencies** | AAA a spazzatura |
| **Moral hazard** | Rischi presi sapendo del bailout |
| **Leva** | 30:1 e oltre |

### La Catena

```
Prestiti subprime
       ↓
Cartolarizzazione (MBS)
       ↓
Tranche "AAA" (rating falsi)
       ↓
Venduti a investitori globali
       ↓
Default → Crollo catena
```
