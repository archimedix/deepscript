# Scandalo Wirecard (2020)

> La piu' grande frode contabile europea: €1.9 miliardi spariti, un COO spia del GRU in fuga a Mosca, e un regolatore che perseguitava chi denunciava

## Cosa Accadde

Wirecard, azienda fintech tedesca di pagamenti digitali quotata al DAX 30, collasso' nel giugno 2020 quando emerse che €1.9 miliardi nei suoi bilanci non esistevano. Il CEO Markus Braun fu arrestato; il COO Jan Marsalek fuggi' in Russia, dove si scopri' essere un agente del GRU da almeno un decennio. La BaFin, regolatore tedesco, aveva attivamente protetto l'azienda perseguendo giornalisti e short-seller invece di indagare le irregolarita'.

### Timeline

| Data | Evento |
|------|--------|
| **1999** | Wirecard fondata a Monaco di Baviera come processore di pagamenti |
| **2002** | Markus Braun diventa CEO |
| **2005** | Quotazione alla Borsa di Francoforte |
| **2010** | Jan Marsalek diventa COO; stesso anno del reclutamento da parte del GRU |
| **2010-11** | Marsalek ordina a Oliver Bellenhaus di creare shell companies a Hong Kong e BVI per deviare fondi |
| **2013** | Bellenhaus si trasferisce a Dubai come MD di Cardsystems Middle East |
| **2015** | Dan McCrum (Financial Times) pubblica le prime inchieste sulle irregolarita'; Bellenhaus intensifica le transazioni fittizie |
| **Feb 2016** | Rapporto Zatarra (101 pagine) documenta frode e riciclaggio |
| **2018** | Wirecard entra nel DAX 30, sostituendo Commerzbank; capitalizzazione €24 miliardi |
| **Feb 2019** | BaFin apre indagine penale contro McCrum per presunta manipolazione di mercato |
| **Apr 2019** | SoftBank investe €900M in Wirecard |
| **Ott 2019** | FT pubblica serie investigativa decisiva basata su documenti interni |
| **Gen 2020** | KPMG incaricata di audit speciale |
| **18 Giu 2020** | EY rifiuta di certificare il bilancio 2019; €1.9B non localizzabili |
| **19 Giu 2020** | Braun si dimette da CEO |
| **22 Giu 2020** | Braun arrestato |
| **22 Giu 2020** | Marsalek licenziato; fugge via Manila e Bielorussia verso la Russia |
| **25 Giu 2020** | Wirecard dichiara insolvenza |
| **6 Lug 2020** | Oliver Bellenhaus si consegna alla procura di Monaco |
| **Gen 2021** | Presidente BaFin Hufeld e vice Roegele rimossi |
| **Dic 2022** | Inizio processo penale a Monaco; Bellenhaus deposita confessione di 100 pagine |
| **2023** | Cinque bulgari arrestati a Londra come parte di rete spionistica diretta da Marsalek |
| **Feb 2024** | Mandato d'arresto di Bellenhaus sospeso con condizioni |
| **Mar 2024** | Egisto Ott (ex BVT austriaco) arrestato per spionaggio a favore di Marsalek/Russia |
| **2024** | Marsalek localizzato a Mosca: telefono agganciato 304 volte alla cella FSB in Lubjanka |
| **Gen 2025** | Braun condannato civilmente a €140M di danni per prestito non garantito |
| **Feb 2025** | Processo ridotto ai 10 capi d'accusa principali; verdetto atteso 2026 |
| **Ago 2025** | Austria apre processo penale formale contro Egisto Ott |

| Aspetto | Dettaglio |
|---------|-----------|
| **Reclutamento** | Marsalek reclutato dal GRU almeno dal 2010 tramite handler Stanislav Petlinsky |
| **Copertura** | Wirecard usata come copertura per operazioni di intelligence russa |
| **Finanziamento** | Milioni sottratti a Wirecard per finanziare Wagner in Siria e Libia |
| **Fuga** | Rete di esfiltrazione via Manila-Minsk-Mosca organizzata da servizi russi |
| **Spionaggio** | Marsalek diresse rete spionistica in UK (5 bulgari arrestati nel 2023) |
| **Austria** | Rete nel BVT: Martin Weiss (ex capo operazioni) reclutato da Marsalek dopo uscita nel 2018; Egisto Ott forni' laptop SINA-S ai russi per €20,000 |
| **Libia** | Marsalek pianifico' forza di 15,000 mercenari per controllare rotte migratorie; acquisto' la RSB Group |
| **Controspionaggio** | Rami El Obeidi (ex intelligence libico) incaricato di operazioni sting contro McCrum e short-seller |
| **Ucraina** | Marsalek viaggiò in zona di guerra e a Mariupol occupata post-2022 |
