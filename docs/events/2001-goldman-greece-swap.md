# Goldman Sachs - Greece Swap (2001)

> Come una banca d'affari aiuto' uno stato sovrano a nascondere €2.8 miliardi di debito per entrare nell'Euro

## Cosa Accadde

Nel 2001, Goldman Sachs strutturò una serie di cross-currency swap per la Grecia che permisero di nascondere €2.8 miliardi di debito pubblico. L'operazione consentì alla Grecia di rispettare formalmente i parametri di Maastricht ed entrare nell'Eurozona, ma creo' una bomba a orologeria finanziaria che sarebbe esplosa nel 2009.

### Timeline

| Data | Evento |
|------|--------|
| **1992** | Trattato di Maastricht fissa parametri (debito <60% PIL, deficit <3%) |
| **1999** | Grecia esclusa dalla prima ondata Euro per parametri non rispettati |
| **2000** | Antigone Loudiadis (Goldman) inizia negoziati con Atene |
| **2001** | Strutturati 13 cross-currency swap yen-euro |
| **2001** | Grecia ammessa nell'Eurozona |
| **2002** | Dracma sostituita dall'Euro |
| **2005** | Debito swap lievita a €5.1B (dopo crollo rendimenti post-9/11) |
| **2008** | Goldman trasferisce deal a National Bank of Greece via Titlos Plc |
| **2009** | Papandreou rivela deficit reale: scoppia crisi |

### L'Impatto

| Metrica | Valore |
|---------|--------|
| **Debito nascosto** | €2.8 miliardi |
| **Fee Goldman** | ~€600 milioni |
| **% revenue Goldman trading unit** | 12% |
| **Debito 2005** | €5.1 miliardi (quasi raddoppiato) |
| **Numero swap** | 13 contratti |
| **Scadenza finale** | 2037 (dopo ristrutturazione 2005) |

### Il Meccanismo Tecnico

| Aspetto | Dettaglio |
|---------|-----------|
| **Strumento** | Cross-currency swap |
| **Trucco** | Tasso di cambio "storico" invece di mercato |
| **Effetto** | Euro debole → Grecia riceve €1B upfront |
| **Contabilita'** | Swap, non loan → fuori bilancio |
| **Regole** | Eurostat pre-2008 non lo vietava esplicitamente |

Il tasso di cambio usato non era quello di mercato ma uno "historical implied foreign exchange rate" con Euro piu' debole. Questo genero' un guadagno immediato di circa €1 miliardo per la Grecia, mascherando debito come strumento derivato.

### La Bomba a Orologeria

Dopo gli attacchi dell'11 settembre 2001, i rendimenti obbligazionari crollarono. La formula usata da Goldman per calcolare i rimborsi greci produsse perdite massicce per Atene:

| Anno | Debito Swap |
|------|-------------|
| **2001** | €2.8 miliardi |
| **2005** | €5.1 miliardi |
| **Aumento** | +82% |

## Le Cause

### Il Sistema

| Fattore | Descrizione |
|---------|-------------|
| **Maastricht rigido** | Parametri binari (dentro/fuori) incentivavano trucchi |
| **Eurostat debole** | Nessun potere di audit reale pre-2008 |
| **Goldman incentivi** | Fee enormi per strutturare deal complessi |
| **Politica greca** | Entrata Euro priorita' assoluta |

## Conseguenze

### A Breve Termine (2001-2005)

| Effetto | Dettaglio |
|---------|-----------|
| **Grecia in Euro** | 1 gennaio 2002 dracma abolita |
| **Tassi bassi** | Accesso a credito a tassi tedeschi |
| **Debito swap cresce** | Da €2.8B a €5.1B |
| **Opacita'** | Deal segreto, Eurostat ignaro |

### A Lungo Termine (2009+)

| Trasformazione | Dettaglio |
|----------------|-----------|
| **Crisi debito 2009** | Papandreou rivela buco: deficit 12.7% (poi 15.4%) |
| **Bailout €330B** | Tre programmi 2010-2015 |
| **Troika** | Commissariamento de facto |
| **Pattern replicato** | Scrutinio su altri deal Goldman-stati |
| **Regole Eurostat** | Riformate per vietare swap simili |
