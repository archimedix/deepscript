# Guerra Iran-Iraq (1980-1988)

> Otto anni di guerra totale che ridisegnarono il Medio Oriente e rivelarono le alleanze opportunistiche delle grandi potenze

## Cosa Accadde

Il 22 settembre 1980, l'Iraq di Saddam Hussein invase l'Iran rivoluzionario dell'Ayatollah Khomeini, dando inizio alla piu' lunga guerra convenzionale del XX secolo. Otto anni di combattimenti, uso di armi chimiche, guerra ai tanker petroliferi e oltre mezzo milione di morti terminarono in uno stallo con il cessate il fuoco dell'agosto 1988.

## Conseguenze

### A Breve Termine

| Effetto | Dettaglio |
|---------|-----------|
| Stallo militare | Nessun vincitore chiaro |
| Devastazione economica | Entrambi i paesi in rovina |
| Debito iracheno | $80 miliardi (causa invasione Kuwait) |
| Trauma iraniano | Generazione segnata, diffidenza verso Occidente |

| Trasformazione | Dettaglio |
|----------------|-----------|
| Invasione Kuwait | Iraq cerca di ripagare debiti (1990) |
| Isolamento Iraq | Sanzioni, no-fly zones |
| Programma nucleare Iran | Deterrenza contro future aggressioni |
| Espansione iraniana | Post-2003, Iran riempie vuoto iracheno |
