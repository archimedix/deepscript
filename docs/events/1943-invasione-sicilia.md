# Invasione della Sicilia - Operation Husky (1943)

> Il primo teatro europeo: dove l'OSS dimostro' il suo valore e un soldato siciliano di 20 anni pianifico' l'operazione

## Cosa Accadde

Il 9 luglio 1943 inizio' l'**Operation Husky**, l'invasione alleata della Sicilia. L'OSS ebbe un ruolo cruciale nella preparazione, grazie al lavoro di [Max Corvo](../persons/max-corvo.md), un italo-americano di 20 anni che scrisse il piano di intelligence per l'operazione.

### Timeline

| Data | Evento |

| **Gen 1943** | Conferenza Casablanca: decisione invasione |
| **Feb 1943** | Corvo presenta piano OSS |
| **Mag 1943** | "I sei di Middletown" addestrati |
| **9 lug 1943** | Inizio Operation Husky |
| **17 ago 1943** | Sicilia conquistata |
| **Set 1943** | Armistizio italiano |

### L'Impatto

| Metrica | Valore |
|---------|--------|

| **Truppe Asse** | 300.000 |

| **Perdite alleate** | ~23.000 |
