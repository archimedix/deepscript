# Predator Gate (2022)

> Scandalo sorveglianza in Grecia: lo spyware Predator di Intellexa usato per spiare almeno 87 persone tra politici, giornalisti, giudici e ministri, con il coinvolgimento dell'intelligence nazionale EYP

## Cosa Accadde

Nel luglio 2022, il leader dell'opposizione greca Nikos Androulakis rivelo' un tentativo di hack del suo telefono tramite lo spyware Predator. L'indagine successiva sveleo' un sistema di sorveglianza illegale che coinvolgeva l'intelligence nazionale (EYP), il contractor Krikel e il consorzio spyware Intellexa. Lo scandalo, ribattezzato "Predator Gate" o "Greek Watergate", porto' alle dimissioni del capo dell'EYP e del segretario generale del primo ministro Mitsotakis.

### Timeline

| Data | Evento |
|------|--------|
| **2019** | Mitsotakis vince le elezioni, pone EYP sotto controllo diretto del PM, nomina Kontoleon capo EYP |
| **Giu-Ago 2020** | EYP intercetta le comunicazioni del giornalista Thanasis Koukakis |
| **2020** | Sorveglianza di Androulakis inizia tramite servizi segreti |
| **Mar 2021** | Governo cambia la legge: elimina retroattivamente il diritto dei cittadini di essere informati della sorveglianza |
| **2021** | Telefono di Androulakis infettato da Predator tramite 8 SMS |
| **Lug 2022** | Androulakis rivela pubblicamente il tentativo di hack |
| **Lug 2022** | Kontoleon ammette in Parlamento che EYP monitorava Koukakis |
| **5 Ago 2022** | EFSYN collega il segretario gen. Dimitriadis (nipote di Mitsotakis) a Intellexa; Dimitriadis si dimette |
| **5 Ago 2022** | Kontoleon si dimette meno di un'ora dopo Dimitriadis |
| **8 Ago 2022** | PM Mitsotakis ammette sorveglianza EYP su Androulakis, nega di esserne stato a conoscenza |
| **Nov 2022** | Settimanale Documento pubblica lista di 87+ vittime: oppositori, giornalisti, giudici, militari, ministri |
| **Mag 2023** | Comitato PEGA del Parlamento Europeo pubblica rapporto sullo scandalo |
| **Nov 2023** | ICIJ pubblica "Cyprus Confidential": espone rete offshore di Intellexa |
| **Mar 2024** | Tesoro USA sanziona Intellexa, Dilian e Hamou |
| **Set 2024** | Seconda ondata sanzioni USA: Bitzios, Harpaz, Karaoli, Gambazzi |
| **Apr 2025** | Inizio processo ad Atene: Lavranos, Bitzios, Dilian, Hamou |

### L'Impatto

| Metrica | Valore |
|---------|--------|
| Vittime confermate | 87+ persone |
| Categorie target | Politici, giornalisti, giudici, militari, ministri del governo |
| Dimissioni | Capo EYP (Kontoleon), segretario gen. PM (Dimitriadis) |

## Le Cause

### Il Sistema

| Fattore | Descrizione |
|---------|-------------|
| EYP sotto controllo PM | Mitsotakis nel 2019 pose l'intelligence sotto suo controllo diretto |
| Legge retroattiva 2021 | Eliminato il diritto dei cittadini di sapere se sorvegliati |
| Mercato spyware | Disponibilita' commerciale di strumenti di sorveglianza di livello statale |
| Rete offshore | Struttura societaria opaca (Cipro, BVI, Irlanda) per occultare la filiera |
