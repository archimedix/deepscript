## Glossario

| Termine | Definizione |
|---------|-------------|
| MPAA | Motion Picture Association of America (nome 1945-2019) |
| MPPDA | Motion Picture Producers and Distributors of America (nome 1922-1945) |
| Production Code | Sistema di autocensura 1930-1968 (Hays Code) |
| Rating System | Sistema di classificazione film introdotto 1968 |
| ACE | Alliance for Creativity and Entertainment (anti-pirateria) |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1922 |
| **Fondatori** | Major Hollywood studios |
| **Contesto** | Scandali Hollywood anni '20, pressioni censura |
| **Obiettivo dichiarato** | Proteggere e promuovere l'industria cinematografica |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1922** | Fondazione come MPPDA, Will H. Hays primo presidente |
| **1930** | Adozione Production Code (Hays Code) |
| **1945** | Eric Johnston presidente, rinominata MPAA |
| **1966** | Jack Valenti presidente (ex aide LBJ) |
| **1968** | Abolizione Production Code, introduzione rating system (G, M, R, X) |
| **1984** | Aggiunta rating PG-13 (proposta Spielberg) |
| **1990** | Rating NC-17 sostituisce X |
| **2004** | Valenti si ritira dopo 38 anni |
| **2017** | Charles Rivkin Chairman & CEO |
| **2019** | Rinominata MPA (Motion Picture Association), riflette streaming |
| **2019** | Netflix diventa membro |
| **2024** | Prime Video & Amazon MGM Studios diventa membro |

## Agenda/Focus

1. **Protezione copyright** - Lobby per leggi anti-pirateria
2. **Classificazione film** - Gestione rating system
3. **Apertura mercati** - Trade negotiations per accesso mercati esteri
4. **Anti-pirateria** - ACE (Alliance for Creativity and Entertainment)
5. **Politiche digitali** - Regolamentazione streaming

### Revolving Door

La MPA ha una tradizione di leader con connessioni politiche:
- **Jack Valenti**: Special Assistant to President Lyndon Johnson (1963-1966)
- **Chris Dodd**: U.S. Senator from Connecticut (1981-2011)
- **Charles Rivkin**: Ambassador to France (2009-2013), Assistant Secretary of State (2014-2017)

## Critiche

| Tema | Dettaglio |
|------|-----------|
| **Lobby aggressivo** | Pressioni per leggi anti-pirateria controverse (SOPA, PIPA) |
| **Rating opaco** | Processo di classificazione non trasparente |
| **Conflitto interessi** | Rating board finanziato da studios membri |
| **Anti-competitive** | Barriere ingresso per studios indipendenti |
