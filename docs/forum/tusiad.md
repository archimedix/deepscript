## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1971 |
| **Fondatori** | Elite industriale turca |
| **Primo Chairman** | Feyyaz Berker (1971-1979) |
| **Primo High Advisory Council Chair** | Vehbi Koc (1971-1979) |
| **Contesto** | Transizione da economia chiusa a competizione globale |
| **Obiettivo** | Promuovere impresa privata, democrazia, economia di mercato |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1971** | Fondazione TUSIAD |
| **1970s** | Advocacy per liberalizzazione commercio estero |
| **1970s** | Inserimento membership CEE nell'agenda nazionale |
| **1979** | Discussione indipendenza Banca Centrale |
| **1980s** | Advocacy riduzione intervento statale, liberalizzazione capitali |
| **1987** | Ingresso in BusinessEurope |
| **1990s** | Report su sostenibilita' previdenza, riforma costituzionale |
| **1999** | Ingresso in BIAC (OECD) |
| **2002** | Fondatore BusinessMed (Mediterraneo) |
| **2020s** | Focus su Industry 4.0, inclusione giovani/donne/PMI |

## Posizioni Politiche

TUSIAD ha storicamente promosso:

| Area | Posizione |
|------|-----------|
| **EU Membership** | Forte sostenitore dall'ingresso agenda anni '70 |
| **Democrazia** | Promuove democrazia partecipativa |
| **Economia** | Economia di mercato competitiva |
| **Banca Centrale** | Indipendenza (dal 1979) |
| **Riforme** | Costituzionali e previdenziali |

### Tensioni con Governo Erdogan

Storicamente TUSIAD ha avuto rapporti tesi con l'AKP su temi come:
- Stato di diritto
- Liberta' di stampa
- Indipendenza giudiziaria
- Politica monetaria
