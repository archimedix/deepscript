# Sun Valley Conference

> "Summer camp for billionaires" - ritiro annuale esclusivo dove si decidono le maggiori M&A dei media e del tech

## Executive Summary

| Metrica | Valore |
|---------|--------|

| **Status** | active |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1983 |
| **Fondatore** | Herbert Allen Jr. (Allen & Company) |
| **Prima edizione** | 35 partecipanti, Allen unico speaker |
| **Location** | Sun Valley Lodge (resort sciistico) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1983** | Prima edizione con 35 invitati |
| **primi 90s** | Warren Buffett diventa partecipante regolare |
| **1995** | Deal Disney-ABC discusso qui |
| **2001** | Merger AOL-Time Warner emerge dal forum |
| **2006** | Acquisition Google-YouTube |
| **2012** | 628 partecipanti (record) |
| **2013** | Jeff Bezos acquista Washington Post |
| **2018** | AT&T-Time Warner |
| **2022** | Elon Musk speaker (Twitter acquisition) |
| **2025** | Musk non invitato, Buffett ritirato |
