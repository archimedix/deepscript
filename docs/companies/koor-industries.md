# Koor Industries Ltd.

> Braccio industriale dell'Histadrut, il più grande conglomerato israeliano del XX secolo

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1944 |
| **Scioglimento** | 2013 (fusione in DIC) |
| **Sede** | Tel Aviv, Israel |
| **Tipo** | Conglomerato industriale |
| **Settori** | Defense electronics, agrochemicals, telecom, construction |
| **Peak workforce** | 32,000 dipendenti (1987) |
| **Status** | Dissolved |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1944 |
| **Fondatore** | Histadrut (via Solel Boneh) |
| **Contesto** | Creata come braccio industriale per preparare l'indipendenza israeliana |
| **Primi impianti** | Phoenicia Glass, Vulcan Foundries (Haifa) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1924** | Solel Boneh fondata dall'Histadrut |
| **1944** | Koor Industries creata come divisione industriale |
| **1945** | JV Nesher Cement con privati |
| **1951** | Entra in telecom con Telrad (Lod) |
| **1958** | Separazione da Solel Boneh, entità autonoma |
| **1962** | Crea Tadiran (JV con Ministero Difesa) |
| **1963** | Acquisisce Makhteshim (chimica) |
| **1974** | Fonda Agan Chemical nel Negev |
| **1987-89** | Crisi finanziaria: perdite $845M |
| **1988** | Benjamin Gaon CEO, ristrutturazione |
| **1995** | Shamrock Holdings (Disney) acquisisce 22.5% per $252M |
| **1997** | Claridge Group (Bronfman) acquisisce da Shamrock |
| **1998** | Jonathan Kolber CEO |
| **2007** | IDB Group/DIC acquisisce maggioranza |
| **2013** | Fusione in Discount Investment Corporation |

## Crisi e Ristrutturazione (1987-1992)

| Metrica | Prima | Dopo |
|---------|-------|------|
| **Dipendenti** | 32,000 | 17,000 |
| **Società controllate** | 130 | 35 |
| **Debito** | $1.2B | Ridotto |
| **Focus** | Diversificato | 3 settori core |

La crisi fu causata da:
- Interferenza politica dell'Histadrut
- Diversificazione eccessiva
- Inefficienze del modello socialista

### Connessioni Intelligence

Koor rappresenta il classico esempio di **revolving door** israeliano intelligence-industria:
- Meir Amit: Mossad → Koor CEO
- Tadiran: JV con Ministero Difesa per elettronica militare

## Significato Storico

Koor incarna la storia economica di Israele:

1. **Era socialista (1944-1980s)**: Economia controllata da Histadrut, industria al servizio del nation-building
2. **Crisi del modello (1987-89)**: Inefficienze del sistema, debito insostenibile
3. **Privatizzazione (1990s)**: Ingresso capitali esteri (Disney, Bronfman)
4. **Consolidamento (2000s)**: Assorbimento nei grandi gruppi israeliani (IDB)
