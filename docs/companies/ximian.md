## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 19 ottobre 1999 |
| **Fondatori** | Miguel de Icaza, Nat Friedman |
| **Nome originale** | International GNOME Support, poi Helix Code |
| **Contesto** | Commercializzazione del desktop GNOME su Linux/Unix |
| **Obiettivo dichiarato** | Sviluppo e supporto software per Linux/Unix basato su GNOME |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1999** | Fondata come International GNOME Support, rinominata Helix Code |
| **2001 gen** | Rinominata Ximian (nome precedente non registrabile come trademark) |
| **2001** | Annuncio del Mono Project (.NET su Linux), guidato da de Icaza |
| **2001** | David Patrick sostituisce Friedman come CEO; Friedman diventa VP Product Management |
| **2003 ago** | Acquisita da [Novell](novell.md) per rafforzare l'offerta Linux enterprise |
| **2011 mag** | Attachmate (nuovo proprietario Novell) licenzia il team Mono USA |
| **2011 mag** | De Icaza e Friedman fondano Xamarin per continuare lo sviluppo di Mono |

## Significato

Ximian rappresenta un nodo chiave nella transizione dell'open source da progetto comunitario a business enterprise. Il ponte tra mondo Microsoft (.NET/Mono) e Linux creato da Ximian genero' controversie nella comunita' FOSS ma apri' la strada all'adozione enterprise di tecnologie cross-platform. La traiettoria Ximian -> Novell -> Xamarin -> Microsoft illustra il pattern di assorbimento delle startup open source da parte delle Big Tech.
