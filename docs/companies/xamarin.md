## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 16 maggio 2011 |
| **Fondatori** | Miguel de Icaza, Nat Friedman |
| **Contesto** | Attachmate acquista Novell e licenzia il team Mono USA |
| **Obiettivo dichiarato** | Supporto commerciale per Mono, sviluppo mobile cross-platform (.NET su iOS/Android) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2011 mag** | [Attachmate](attachmate.md) licenzia il team Mono di [Novell](novell.md) |
| **2011 mag 16** | De Icaza annuncia la fondazione di Xamarin; Friedman CEO |
| **2011** | Rilascio di Xamarin.iOS e Xamarin.Android |
| **2013** | Lancio Xamarin 2.0 con IDE dedicato (Xamarin Studio) |
| **2014** | Partnership con [Microsoft](microsoft.md) per integrazione Visual Studio |
| **2016 feb 24** | Microsoft annuncia l'acquisizione definitiva |
| **2016** | Xamarin reso gratuito e integrato in Visual Studio |
| **2020** | Xamarin.Forms evolve in .NET MAUI (Multi-platform App UI) |

### Catena di continuita'

```
Ximian (1999) → Novell (2003) → Attachmate layoffs (2011) → Xamarin (2011) → Microsoft (2016)
```

Il progetto Mono ha attraversato quattro proprietari mantenendo lo stesso nucleo tecnico (de Icaza), illustrando la resilienza dei progetti open source rispetto alle acquisizioni corporate.

---

## Significato

Xamarin rappresenta un caso emblematico di come il talento open source possa rinascere dopo acquisizioni distruttive. La traiettoria Ximian->Novell->Xamarin->Microsoft segue il pattern ricorrente: startup open source innovativa -> acquisizione corporate -> ristrutturazione/layoff -> spin-off -> ri-acquisizione da Big Tech. L'acquisizione da parte di Microsoft nel 2016 segnalo' il pivot di Satya Nadella verso l'open source, culminato nell'acquisizione di GitHub (2018) con Friedman stesso come CEO.
