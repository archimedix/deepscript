## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Aprile 2008 (sviluppo iniziato ottobre 2007) |
| **Fondatori** | Tom Preston-Werner, Chris Wanstrath, P.J. Hyett, Scott Chacon |
| **Background fondatori** | Tutti autodidatti/dropout; Preston-Werner (Harvey Mudd, dropout), Wanstrath (U. Cincinnati, dropout per CNET), Hyett (BS CS, North Central College). Wanstrath e Hyett si conoscono da CNET Networks e gestivano una consulenza Ruby on Rails |
| **Contesto** | Necessita' di una piattaforma social per Git, il sistema di version control di Linus Torvalds |
| **Obiettivo dichiarato** | Rendere la collaborazione sul codice semplice e accessibile |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2007** | Inizio sviluppo da Preston-Werner e Wanstrath |
| **2008** | Lancio pubblico (aprile) |
| **2009** | 46.000 repository pubblici nel primo anno |
| **2010** | 1 milione di repository |
| **2012** | Andreessen Horowitz investe $100M |
| **2014** | Preston-Werner lascia da CEO dopo accuse di molestie; Wanstrath diventa CEO |
| **2018** | Microsoft acquisisce GitHub per $7.5B in azioni; Nat Friedman diventa CEO |
| **2019** | Acquisizione di Semmle (CodeQL) e Dependabot |
| **2020** | Acquisizione di npm (package manager JavaScript) |
| **2021** | Thomas Dohmke diventa CEO (novembre) |
| **2021** | Lancio GitHub Copilot (preview tecnica, basato su OpenAI Codex) |
| **2024** | Copilot supera 20M utenti; revenue run rate $2B |
| **2025** | Dohmke lascia (agosto); GitHub integrata in Microsoft CoreAI division |

### Significato strategico

GitHub rappresenta il punto di controllo chiave nell'ecosistema software globale:
- **150M+ sviluppatori** dipendono dalla piattaforma
- **Copilot** posiziona Microsoft al centro dell'AI per sviluppatori
- **npm** controlla la supply chain JavaScript
- Acquisizione ha trasformato la percezione di Microsoft da antagonista a campione dell'open source

### Diaspora dei fondatori

Dopo l'acquisizione Microsoft (2018), i quattro fondatori hanno disperso il capitale (~$7.5B totali) in direzioni diverse: venture capital tech/AI (Preston-Werner), gaming indie e browser indipendenti (Wanstrath), motorsport professionistico (Hyett), language learning (Chacon). Tutti restano attivi nell'ecosistema tech come investitori o fondatori.

## Critiche

| Tema | Dettaglio |
|------|-----------|
