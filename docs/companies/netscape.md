## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Aprile 1994 (come Mosaic Communications Corp.) |
| **Fondatori** | [Jim Clark](../persons/jim-clark.md), [Marc Andreessen](../persons/marc-andreessen.md) |
| **Contesto** | Clark (ex professore Stanford, fondatore [Silicon Graphics](../companies/silicon-graphics.md)) + Andreessen (22 anni, creatore NCSA Mosaic alla [UIUC](../university/uiuc.md)) unirono forze per commercializzare il web browser |
| **Investimento iniziale** | $4M da Jim Clark |
| **Obiettivo dichiarato** | Sviluppare software per la navigazione e i servizi web |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1993** | Andreessen sviluppa NCSA Mosaic alla [University of Illinois](../university/uiuc.md) ($6.85/ora) |
| **Apr 1994** | Clark lascia [Silicon Graphics](../companies/silicon-graphics.md) e fonda Mosaic Communications Corp. con $4M propri |
| **1994** | Rinominata Netscape Communications dopo obiezioni NCSA sul nome |
| **Gen 1995** | [Jim Barksdale](../persons/jim-barksdale.md) reclutato come CEO (ex COO FedEx, ex CEO AT&T Wireless) |
| **Apr 1995** | [Brendan Eich](../persons/brendan-eich.md) entra in azienda (ex [Silicon Graphics](../companies/silicon-graphics.md)) |
| **Mag 1995** | Eich crea JavaScript in 10 giorni (Mocha → LiveScript → JavaScript) |
| **Ago 1995** | IPO: le azioni raddoppiano il primo giorno, market cap $2.2B. Andreessen (24 anni) in copertina su Time |
| **1995-1998** | Browser War contro Microsoft Internet Explorer |
| **Gen 1998** | Netscape rilascia il codice sorgente del browser (progetto Mozilla, co-fondato da Eich e [Mitchell Baker](../persons/mitchell-baker.md)) |
| **Apr 1998** | Acquisizione di Kiva Software |
| **Nov 1998** | AOL annuncia acquisizione per $4.2B |
| **Mar 1999** | Acquisizione da [AOL](../companies/aol.md) completata (valore finale ~$10B). Clark esce con $1.2B personali |

## Prodotti

- **Netscape Navigator** — browser web dominante 1994-1997
- **Netscape Communicator** — suite integrata (browser, email, editor)
- **JavaScript** — linguaggio di programmazione per il web
- **SSL/HTTPS** — protocollo di sicurezza per il web
- **Netscape Enterprise Server** — web server commerciale

### Eredita'

Netscape e' stata un **incubatore di potere**: i suoi alumni hanno fondato o guidato alcune delle istituzioni piu' influenti del tech:
- **[Andreessen Horowitz](../private-equity/andreessen-horowitz.md)** — primo fondo VC USA ($90B+ AUM), fondato da Andreessen + Horowitz
- **[Mozilla Foundation](../foundation/mozilla-foundation.md)** — dal codice sorgente rilasciato nel 1998, co-fondata da Eich e Baker
- **[Brave Software](../companies/brave-software.md)** — browser anti-sorveglianza, fondato da Eich
- **Barksdale → intelligence** — dal CEO di un'azienda tech al PFIAB e board In-Q-Tel (CIA venture fund)
- **Andreessen → politica** — da VP Technology a advisor informale di Trump e reclutatore DOGE

Il pattern Netscape illustra come una singola azienda tech possa generare nodi di potere in venture capital, open source, privacy e intelligence.
