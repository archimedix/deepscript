## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Marzo 2004 |
| **Fondatore** | [Mark Shuttleworth](../persons/mark-shuttleworth.md) |
| **Contesto** | Shuttleworth, arricchitosi con la vendita di Thawte a VeriSign ($575M nel 1999), recluta sviluppatori [Debian](../foundation/debian.md) per creare una distribuzione Linux user-friendly |
| **Obiettivo dichiarato** | Promuovere e supportare commercialmente il software libero, in particolare Ubuntu |
| **Finanziamento** | Interamente autofinanziata da Shuttleworth, nessun investitore esterno |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2004** | Shuttleworth fonda Canonical; Ubuntu 4.10 "Warty Warthog" (ottobre) |
| **2005** | Lancio di Ubuntu Foundation ($10M endowment) |
| **2006** | Primo LTS release: Ubuntu 6.06 "Dapper Drake" |
| **2010** | Ubuntu diventa la distro Linux desktop piu' popolare |
| **2011** | Lancio di Unity desktop, controverso nella community |
| **2013** | Ubuntu Touch per smartphone (poi abbandonato) |
| **2016** | Snap package manager, focus su IoT e cloud |
| **2017** | Abbandono di Unity, ritorno a GNOME; primi rumors IPO |
| **2018** | Focus su enterprise: Kubernetes, OpenStack, cloud |
| **2024** | Revenue $292M, profitto operativo $15.5M |
| **2025** | IPO ancora non realizzata nonostante rumors dal 2017 |

## Prodotti e Business

| Prodotto | Descrizione |
|----------|-------------|
| **Ubuntu Desktop** | Distribuzione Linux desktop (gratuita) |
| **Ubuntu Server** | OS server, base per cloud e data center |
| **Ubuntu Pro** | Supporto enterprise e security patching |
| **Juju** | Orchestrazione applicazioni |
| **MAAS** | Metal as a Service (provisioning bare metal) |
| **Snap** | Package manager universale |
| **MicroK8s** | Kubernetes leggero |
| **Charmed OpenStack** | Distribuzione OpenStack enterprise |

### Modello di business

Ubuntu e' gratuito; Canonical genera ricavi da supporto enterprise (Ubuntu Pro), servizi cloud, consulting e formazione. Principale base clienti: telco, finanza, governi, cloud provider.
