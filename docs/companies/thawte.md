## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1995 |
| **Fondatori** | [Mark Shuttleworth](../persons/mark-shuttleworth.md) |
| **Contesto** | Creata nel garage dei genitori di Shuttleworth a Cape Town, per produrre server sicuri senza le restrizioni USA sull'export di crittografia |
| **Obiettivo dichiarato** | Certificate authority per certificati X.509 e SSL |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1995** | Fondazione da parte di Shuttleworth, studente alla University of Cape Town |
| **~1997** | Sviluppa Sioux, web server sicuro, poi pivota verso certificati SSL |
| **~1998** | Seconda CA al mondo per quota di mercato (~50%, dietro solo a VeriSign) |
| **1999** | [VeriSign](verisign.md) acquisisce Thawte per $575M in azioni |
| **2010** | Symantec acquisisce la divisione sicurezza di VeriSign (inclusa Thawte) |
| **2017** | DigiCert acquisisce gli asset web security di Symantec (inclusa Thawte) |

## Significato per DeepScript

La vendita di Thawte a VeriSign nel 1999 e' l'evento fondante della fortuna di Shuttleworth (~$575M), che gli permise di:
- Diventare il secondo turista spaziale (2002, ~$20M)
- Fondare e finanziare interamente Canonical/Ubuntu (2004-oggi)

Thawte era nata nel garage dei genitori — non da capitali familiari o ereditati.
