# Stack Overflow

> La piattaforma Q&A che ha ridefinito la knowledge-sharing per sviluppatori, acquisita da Prosus per $1.8B

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2008 |
| **Sede** | New York, USA |
| **Tipo** | Company (tech platform) |
| **Leadership** | Prashanth Chandrasekar, CEO |
| **Utenti** | 100M+ visitatori/mese |
| **Revenue** | ~$125M (2024) |
| **Status** | active (sussidiaria Prosus) |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Settembre 2008 |
| **Fondatori** | Joel Spolsky, Jeff Atwood |
| **Contesto** | Alternativa aperta a Experts-Exchange |
| **Entita' legale** | Stack Exchange Inc. |
| **Obiettivo dichiarato** | Piattaforma Q&A aperta per sviluppatori |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2008** | Lancio Stack Overflow da Joel Spolsky e Jeff Atwood |
| **2009** | Espansione a Stack Exchange (network multi-topic) |
| **2010** | Series A — Union Square Ventures |
| **2011** | Series B — Index Ventures, Spark Capital |
| **2012** | Jeff Atwood lascia la company |
| **2015** | Series D ($40M) — led by Andreessen Horowitz |
| **2019** | Prashanth Chandrasekar nominato CEO (ottobre) |
| **2020** | Series E ($85M) — led by GIC, Silver Lake Waterman |
| **2021** | Acquisizione da Prosus (Naspers) per $1.8B |
| **2025** | Lancio Enterprise AI Products |
