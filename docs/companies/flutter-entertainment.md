## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2016 |
| **Origine** | Merger tra Paddy Power plc e Betfair Group plc |
| **Nome originale** | Paddy Power Betfair plc |
| **Rebranding** | Flutter Entertainment plc (2019) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1958** | Incorporazione in Irlanda come Corcoran's Management Limited |
| **1988** | Diventa Paddy Power plc (merger 3 bookmakers irlandesi) |
| **2015** | Annuncio merger Paddy Power-Betfair |
| **2016** | Completamento merger (52% Paddy Power, 48% Betfair) |
| **2018** | Acquisizione FanDuel per $158M (post-PASPA) |
| **2019** | Rebranding a Flutter Entertainment |
| **2019** | Acquisizione The Stars Group per $6.95B |
| **2022** | Acquisizione Sisal per EUR 1.9B |
| **2024** | Quotazione NYSE (FLUT) |
| **2025** | Full ownership FanDuel (~$31B valuation) |
| **2025** | Acquisizione NSX Group (Betnacional, Brasile) |
