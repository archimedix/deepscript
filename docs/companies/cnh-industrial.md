## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2013 |
| **Origine** | Fusione Fiat Industrial + CNH Global N.V. |
| **Eredità** | Case (1842) e New Holland (1895) |
| **Controllo** | Exor N.V. (famiglia Agnelli) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1842** | Jerome Increase Case fonda Case Corporation |
| **1895** | Abe Zimmerman fonda New Holland |
| **1999** | Fusione Case + New Holland → CNH Global N.V. |
| **2013** | Fusione Fiat Industrial + CNH Global → CNH Industrial N.V. |
| **2018** | Suzanne Heywood diventa Chairman |
| **2022** | Spin-off di Iveco Group (veicoli commerciali) |
| **2024** | Gerrit Marx nominato CEO (luglio) |
