## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2 settembre 1992 |
| **Fondatori** | Roland Dyroff, Thomas Fehr, Burchard Steinbild, Hubert Mantel |
| **Nome originale** | S.u.S.E. — Software und System-Entwicklung |
| **Prodotto chiave** | SUSE Linux Enterprise Server (SLES) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1992** | Fondazione a Norimberga come S.u.S.E. |
| **1994** | Prima distribuzione Linux propria (basata su Slackware, poi indipendente) |
| **2003** | Acquisita da Novell |
| **2006** | Accordo patent/interoperability Microsoft-Novell copre SUSE Linux |
| **2011** | Novell acquisita da Attachmate; SUSE diventa business unit separata |
| **2014** | Micro Focus acquisisce Attachmate; SUSE scorporata come entita' distinta |
| **2018** | EQT AB acquisisce SUSE da Micro Focus per $2.535B |
| **2019** | Melissa Di Donato nominata CEO |
| **2020** | Acquisizione Rancher Labs (Kubernetes management); K3s donato a CNCF |
| **2021** | IPO alla Borsa di Francoforte (19 maggio, prezzo €30); EQT mantiene 75.7% |
| **2023** | EQT ritira SUSE dal mercato (delisting); Dirk-Peter van Leeuwen nuovo CEO |

## Significato Strategico

SUSE e' un caso unico di azienda open source passata attraverso cinque proprietari in due decenni, mantenendo continuita' di prodotto. La catena Novell → Attachmate → Micro Focus → EQT illustra come il private equity abbia progressivamente estratto valore dal software open source enterprise.

L'accordo Microsoft-Novell/SUSE (2006) ha segnato un punto di svolta: Microsoft per la prima volta riconosceva commercialmente Linux enterprise, aprendo la strada alla successiva apertura verso l'open source.
