## Glossario

| Termine | Definizione |
|---------|-------------|
| RFE | Radio Free Europe (1950) - target: satelliti sovietici |
| RL | Radio Liberty (1953) - target: Unione Sovietica |
| NCFE | National Committee for a Free Europe (front CIA) |
| Amcomlib | American Committee for Liberation (creatore RL) |
| Grantee | Organizzazione finanziata ma non direttamente controllata |

## Timeline Storica

| Anno | Evento |
|------|--------|
| **1949** | NCFE (front CIA) crea Radio Free Europe |
| **1950** | Prime trasmissioni RFE verso Europa Est |
| **1953** | Radio Liberty inizia trasmissioni verso URSS |
| **1956** | Rivolta ungherese: RFE accusata di fomentare |
| **1971** | Fine finanziamento CIA, fondi pubblici |
| **1976** | Merger RFE + RL = RFE/RL Inc. |
| **1981** | Bomba servizi segreti Romania a sede Monaco |
| **1989** | Caduta Muro: picco audience |
| **1995** | Trasferimento sede da Monaco a Praga |
| **2020** | Michael Pack licenzia Jamie Fly |
| **2021** | Biden reinstalla Fly |
| **2024** | Russia dichiara RFE/RL "organizzazione indesiderabile" |
| **Mar 2025** | USAGM termina finanziamenti |
| **Mar 2025** | RFE/RL fa causa a USAGM |
| **Mar 2025** | Rep. Ceca promette supporto |
| **Mag 2025** | EU (Kaja Kallas) stanzia $6.2M |
| **Nov 2025** | Chiude servizio ungherese |
