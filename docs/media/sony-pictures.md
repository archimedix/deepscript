## Storia

| Anno | Evento |
|------|--------|
| **1918** | Fondazione CBC Film Sales (futura Columbia Pictures) |
| **1924** | Ridenominazione in Columbia Pictures Corporation |
| **1982** | Fondazione Tri-Star Pictures (JV Columbia/CBS/HBO) |
| **1987** | Merger Columbia + Tri-Star = Columbia Pictures Entertainment |
| **1989** | Sony acquista da Coca-Cola per $3.4 miliardi |
| **1991** | Rinominata Sony Pictures Entertainment |
| **2014** | Sony Pictures hack (attacco nordcoreano) |
| **2017** | Acquisizione Funimation ($143M) |
| **2021** | Acquisizione Crunchyroll da AT&T ($1.175B) |
| **2024** | Acquisizione Alamo Drafthouse Cinema |
