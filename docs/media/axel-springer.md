## Proprieta' - Storia

| Periodo | Proprietario | Quota | Note |
|---------|--------------|-------|------|
| 1946-1985 | Axel Springer | 100% | Fondatore |
| 1985-2020 | Friede Springer | Majority | Vedova, quotata in borsa |
| 2020-2025 | KKR + CPP Investments + Springer/Dopfner | 48.5% + 22% + 22% | Take-private €6.8B |
| **2025-oggi** | **Friede Springer + Mathias Dopfner** | **95%** | **Exit KKR, debt-free** |

### Deal KKR (2020-2025)

```
2020: Take-private
├── KKR + CPP Investments: 48.5%
├── Friede Springer: ~22%
└── Mathias Dopfner: ~22%
    Valutazione: €6.8B

2024: Accordo split
├── Media assets → Dopfner/Springer
├── Classifieds → KKR/CPP (majority)
│   ├── Stepstone (recruiting)
│   └── AVIV (real estate)
└── Valutazione totale: €13.5B

2025 (aprile): Closing
├── Axel Springer: debt-free, family-owned
├── Dopfner/Springer: 95%
└── KKR return: ~2x in 5 anni
```

**Pattern PE classico**: take-private → ottimizzazione → split → exit profittevole.
