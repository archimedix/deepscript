# Maariv (מעריב)

> Dal quotidiano piu' letto d'Israele a campo di battaglia di oligarchi: cinque proprietari in quattro anni

## Executive Summary

| Campo | Valore |
|-------|--------|
| **Tipo** | testata (quotidiano) |
| **Fondazione** | 15 febbraio 1948 |
| **Sede** | Tel Aviv, Israel |
| **Proprietario** | Eli Azur (dal 2014) |
| **Direttori** | Doron Cohen, Golan Bar-Yosef (co-editors dal 2014) |
| **Status** | active |

---

## Proprieta' - Storia

| Periodo | Proprietario | Note |
|---------|--------------|------|
| 1948-1989 | Cooperativa lavoratori | Fondato come giornale indipendente worker-owned |
| 1989-1991 | [Robert Maxwell](../persons/robert-maxwell.md) | Acquisto' 87% quote; mori' nel novembre 1991 |
| 1992-2010 | Yaakov Nimrodi | Trafficante d'armi; figlio Ofer come publisher |
| 2010-2011 | Zaki Rakib | 50% stake |
| 2011-2012 | Nochi Dankner (IDB Holding) | ~300M shekels; tentativo fallito di stabilizzazione |
| 2012-2014 | Shlomo Ben-Tzvi | 80-85M shekels; publisher di Makor Rishon; svolta a destra |
| 2014-oggi | Eli Azur | 4M shekels; proprietario anche di Jerusalem Post, Sport1, 103FM |

## Eventi Significativi

| Anno | Evento | Impatto |
|------|--------|---------|
| 1948 | Fondazione dopo scissione da Yedioth Aharonot | Nasce come cooperativa worker-owned |
| 1950s-1970s | Quotidiano piu' letto d'Israele | Dominio del mercato |
| 1989 | Acquisizione Robert Maxwell (87%) | Fine della cooperativa |
| 1991 | Morte Maxwell | Crisi proprietaria |
| 1992 | Acquisizione Yaakov Nimrodi | Passaggio a proprietario controverso |
| 1999 | Incriminazione Ofer Nimrodi | Scandalo cospirazione per omicidio |
| 2007+ | Perdite operative NIS 70-100M annui | Declino finanziario inarrestabile |
| 2012 | Sospensione edizione cartacea | Crisi di liquidita'; 2.000 dipendenti ridotti a 350-500 |
| 2014 | Vendita a Eli Azur per 4M shekels | Quinto cambio proprietario in 4 anni |
| 2023 | Hack iraniano | Attacco informatico a Maariv e Jerusalem Post |

## Influenza e Potere

| Aspetto | Dettaglio |
|---------|-----------|
| **Agenda setting** | Per decenni il quotidiano piu' influente d'Israele; oggi marginalizzato |
| **Rapporti politici** | Centro-destra con posizioni hawkish su sicurezza; svolta destra accentuata dopo 2012 |
| **Rapporti economici** | Passato attraverso mani di arms dealer (Nimrodi), tycoon (Dankner), magnate britannico (Maxwell) |
| **Pattern** | Esempio paradigmatico di come il potere economico determina la linea editoriale |
