## Struttura e Controllo

### Governance

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma legale** | Ente pubblico autonomo (de jure) |
| **Controllo reale** | Presidenza della Repubblica |
| **Affiliazione** | Presidential Communication Authority (dal 2018) |
| **Regolatore** | RTÜK (Radio and Television Supreme Council) |
| **Nomina DG** | Presidente della Repubblica |

### Finanziamento (2023)

| Fonte | Quota | Note |
|-------|-------|------|
| Bandrol fee (tassa dispositivi) | 86.3% | TRY 18.24B |
| Pubblicità e commerciale | 13.7% | In calo da 17% (2022) |
| **Totale** | 100% | TRY 21.14B |

> La electricity fee (dal 1984) abolita nel 2021 per proteste pubbliche sui costi utilities.

## Storia

### Timeline

| Anno | Evento |
|------|--------|
| **1926** | Prime trasmissioni radio test |
| **1950** | Türkiye Radyoları tra i 23 fondatori EBU |
| **1964** | Fondazione TRT (1 maggio) |
| **1968** | Prime trasmissioni TV test (TRT 1) |
| **1971** | Programmazione TV nazionale regolare |
| **1990** | Fine monopolio radio |
| **1992** | Fine monopolio TV (broadcasting commerciale) |
| **2010** | Lancio TRT Haber, TRT Arabi |
| **2015** | Lancio TRT World (inglese) |
| **2016** | Tentato golpe - aumento restrizioni media |
| **2018** | TRT affiliato a Presidential Communication Authority |
| **2020** | TRT World registrato FARA (USA) |
| **2021** | Abolita electricity fee |
| **2023** | Emmy Internazionale per documentario Ucraina |
| **2025** | Sobacı riconfermato DG |
