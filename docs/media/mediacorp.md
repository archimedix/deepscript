## Storia della Proprieta'

| Periodo | Entita' | Note |
|---------|---------|------|
| **1936** | British Malaya Broadcasting Corp | Licenza dalla corona britannica |
| **1955** | Singapore Broadcasting Service | Post-autogoverno |
| **1980** | Singapore Broadcasting Corporation | Corporatizzazione |
| **1994** | Singapore International Media (SIM) | Privatizzazione → Temasek |
| **1999** | Mediacorp | Ristrutturazione per competizione |

|---------|-----------|
