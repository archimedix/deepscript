## Storia

### Timeline

| Anno | Evento |
|------|--------|
| **2003** | Lanciato come Al Jazeera Sports |
| **2012** | Brand beIN Sports debutta in Francia |
| **2013** | Spin-off da Al Jazeera Media Network |
| **2014** | Incorporazione beIN Media Group |
| **2016** | Acquisizione Miramax ($1B+) |
| **2016** | Acquisizione Digiturk (Turchia) |
| **2017-2021** | Blocco saudita (Qatar diplomatic crisis) |
| **2021** | Ritorno mercato saudita |
| **2022** | Lancio TOD (OTT service) |
| **2025** | Estensione diritti Premier League (£550M) |

### Origine Al Jazeera

beIN nasce come divisione sportiva di Al Jazeera, il network di news qatariota. Lo spin-off del 2013-2014 separa le attivita' sportive dal brand news, ma entrambi rimangono sotto controllo statale qatariota.

## Integrazione Verticale: Content + Distribution

### Il Pattern Al-Khelaifi

Nasser Al-Khelaifi controlla simultaneamente:
- **Content**: PSG (calcio), Premier Padel
- **Distribution**: beIN Media (broadcasting)

| Dimensione | Asset |
|------------|-------|
| **Club** | PSG (87.5% QSI) |
| **Tour** | Premier Padel (100% QSI) |
| **Broadcasting** | beIN Sports (40+ paesi) |
| **Film** | Miramax (Hollywood) |
| **OTT** | TOD (streaming) |

**Vantaggio competitivo**: Puo' negoziare diritti sapendo di avere sia il contenuto che la distribuzione.

## Controversie

### beoutQ Piracy (2017-2020)

| Aspetto | Dettaglio |
|---------|-----------|
| **Cosa** | Network pirata saudita che ritrasmetteva contenuti beIN |
| **Periodo** | 2017-2020 (durante blocco Qatar) |
| **Perdite** | >$1B (stima beIN) |
| **Azione legale** | $1B arbitrato internazionale vs Saudi Arabia (2018) |
| **Risoluzione** | Blocco rimosso 2021 (Al-Ula agreement) |

### Qatar Diplomatic Crisis Impact

- 2017-2021: beIN bloccato in Saudi Arabia, UAE, Bahrain, Egypt
- Perdita del mercato piu' grande della regione
- 2021: Ritorno dopo risoluzione crisi diplomatica
