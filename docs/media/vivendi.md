## La Grande Scissione (Dicembre 2024)

Il 9 dicembre 2024, Vivendi si e' divisa in **4 entita' quotate**:

```
VIVENDI (pre-split)
        │
        ├─────────────────────────────────────────────┐
        │                                             │
        ▼                                             ▼
   CANAL+ (LSE)                              HAVAS (Amsterdam)
   €6.85B                                    €3.44B
   TV, streaming, CNews                      Advertising, comms
        │                                             │
        └──────────────┬──────────────────────────────┘
                       │
        ┌──────────────┴──────────────┐
        │                             │
        ▼                             ▼
 LOUIS HACHETTE GROUP           VIVENDI SE
 (Euronext Paris)               (holding residua)
 €2.15B                         Gameloft, minority stakes
 Lagardere, Prisma, Europe 1
```

### Perche' lo Split?

| Motivazione | Spiegazione |
|-------------|-------------|
| **Conglomerate discount** | Vivendi valeva meno della somma delle parti |
| **Focus strategico** | Ogni entita' puo' perseguire M&A indipendente |
| **Controllo rafforzato** | Amsterdam da' voting rights multipli (2x, 4x) |
| **Canal+ globale** | LSE per espansione UK, streaming internazionale |

## Influenza Politica: "France's Murdoch"

### Strategia Mediatica

| Fase | Azione | Effetto |
|------|--------|---------|
| 2015 | Acquisisce Canal+ | Controllo TV premium |
| 2016 | Trasforma iTele in **CNews** | Crea "Fox News francese" |
| 2019-21 | Promuove **Eric Zemmour** | Show serale CNews |
| 2021 | Acquisisce Lagardere | Europe 1, Paris Match, JDD |
| 2022 | Zemmour candidato presidenziale | 7% primo turno |
| 2024 | Split per consolidare controllo | Voting rights multipli |

### CNews: Il Laboratorio Conservatore

| Aspetto | Dettaglio |
|---------|-----------|
| **Rebrand** | Da iTele (info) a CNews (opinione) |
| **Stile** | Talk show, dibattiti polarizzanti |
| **Linea** | Anti-immigrazione, valori tradizionali |
| **Critiche** | RSF: "non piu' canale news, canale opinione" |
| **Effetto** | Normalizzazione estrema destra |

### Cattolicesimo Conservatore

Bollore e' un **cattolico tradizionalista** che usa i media per promuovere:
- Valori familiari tradizionali
- Anti-aborto (film su C8)
- Programmazione religiosa (15 agosto)
- Critica secolarismo francese

> "Crociata mediatica" per riportare il cristianesimo politico al centro della politica francese.

### Pattern Comune Bollore-Murdoch

1. **Acquisizione canale esistente** → trasformazione editoriale
2. **Talk show polarizzanti** → audience e influenza
3. **Promozione candidato** → kingmaker
4. **Integrazione verticale** → content + distribution
5. **Controllo familiare** → decisioni rapide
