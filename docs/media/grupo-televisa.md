## Storia

### Timeline

| Anno | Evento |
|------|--------|
| **1930** | Azcárraga Vidaurreta crea prima radio messicana |
| **1950** | XHTV-TV - prima stazione TV |
| **1951** | Channel 2 (XEW-TV) |
| **1952** | XHGC-TV (Channel 5) |
| **1955** | Fusione in Telesistema Mexicano |
| **1963** | Prima trasmissione a colori (Channel 5) |
| **1972** | Morte Azcárraga Vidaurreta |
| **1973** | Fusione con TIM → **Televisa** |
| **1997** | Morte Azcárraga Milmo. Jean CEO a 29 anni |
| **2017** | Jean lascia CEO, resta Chairman |
| **2018** | De Angoitia e Gómez co-CEOs |
| **2022** | Merger con Univision → TelevisaUnivision |
| **2024** | Jean in leave per indagine DOJ FIFA |
