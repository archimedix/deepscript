## Storia

### Timeline

| Anno | Evento |
|------|--------|
| **1925** | Irineu Marinho fonda O Globo. Muore 21 giorni dopo |
| **1925** | Roberto Marinho (21 anni) assume direzione |
| **1944** | Lancio Rádio Globo |
| **1962** | Deal Time-Life (~€20M + supporto tecnico) |
| **1964** | Golpe militare in Brasile |
| **1965** | Lancio TV Globo (26 aprile) |
| **1968** | Termine contratto Time-Life (pressione opposizione) |
| **1984** | Censura Diretas Já (movimento pro-democrazia) |
| **1985** | Fine dittatura militare |
| **2003** | Morte Roberto Marinho. Roberto Irineu chairman |
| **2013** | Globo si scusa per supporto a dittatura 1964 |
| **2015** | Lancio Globoplay |
| **2016** | Ruolo in impeachment Dilma Rousseff |
| **2020** | Ristrutturazione: fusione divisioni in "Globo" |
| **2021** | João Roberto Marinho diventa chairman |
| **2024** | JV con MGM per sports betting |
| **2024** | 100° anniversario (1 dicembre) |

## Controversie Politiche

### Dittatura Militare (1964-1985)

| Aspetto | Dettaglio |
|---------|-----------|
| **Posizione** | Supporto attivo al regime |
| **Time-Life** | Deal 1962 violava costituzione (foreign investment ban) |
| **Censura interna** | Roberto Marinho ordinò di ignorare Diretas Já 1984 |
| **Testimonianza** | Boni (VP): "Mr. Roberto non voleva menzione di Diretas Já" |
| **Scuse** | 2013: Globo si scusa per supporto a golpe 1964 |

### Era PT (Lula/Dilma)

| Aspetto | Dettaglio |
|---------|-----------|
| **2006** | Accuse di bias contro Lula |
| **2005-2016** | Profitti da pubblicità governativa |
| **2016** | Ruolo nell'impeachment Dilma Rousseff |
| **Lula quote** | "Globo fu uno dei principali coordinatori del golpe" |

### Era Bolsonaro

| Aspetto | Dettaglio |
|---------|-----------|
| **Bolsonaro 2019** | "TV Globo nacque nel 1965, era un'azienda dittatoriale" |
| **Relazione** | Tensioni ma non rottura totale |
