## Struttura Proprietaria

```
FONDAZIONI (80.9% capitale, 0% voto)
├── Bertelsmann Stiftung (majority)
├── Reinhard Mohn Stiftung
├── BVG-Familienstiftung
└── BVG-Stiftung
        │
        │ Capitale
        ▼
BERTELSMANN SE & Co. KGaA
        │
        │ Voto (100%)
        ▼
BERTELSMANN VERWALTUNGSGESELLSCHAFT (BVG)
        │
        │ Controllata da
        ▼
FAMIGLIA MOHN (19.1% capitale + 100% voto via BVG)
```

### Il Modello "Fondazione"

| Aspetto | Dettaglio |
|---------|-----------|
| **Capitale** | 80.9% in fondazioni (tax-efficient, permanente) |
| **Voto** | 100% in BVG (famiglia) |
| **Risultato** | Controllo totale con capitale minimo |
| **Vantaggio** | No takeover possibile, no quotazione |

Modello simile a: IKEA (Stichting INGKA), Bosch (Robert Bosch Stiftung).
