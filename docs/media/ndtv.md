## L'Acquisizione Adani (2022)

### Meccanismo

| Aspetto | Dettaglio |
|---------|-----------|
| **Veicolo** | VCPL (Vishvapradhan Commercial Private Limited) |
| **Origine** | Loan 2009 Rs 403.85 crore con warrant conversion |
| **Shell company** | VCPL senza asset per 14 anni |
| **Diritto** | 99.99% shares RRPR (holding Roy) se esercitato |

### Deal Finale

| Fase | Dettaglio |
|------|-----------|
| **29.18%** | Via conversione warrant VCPL |
| **8%** | Open offer novembre 2022 |
| **27.26%** | Acquisto diretto da Roy (Rs 602 crore, Rs 342.65/share) |
| **Totale** | 64.71% |

### Reazione NDTV

> "L'acquisizione è stata eseguita senza input, conversazione o consenso dei fondatori NDTV."
