# Telegraph Media Group

> Il quotidiano conservatore britannico per eccellenza, dalla fondazione 1855 ai Barclay brothers al takeover DMGT 2025 - passando per Conrad Black e lo sceicco bloccato

## Executive Summary

| Campo | Valore |
|-------|--------|
| **Tipo** | Gruppo editoriale (quotidiani) |
| **Fondazione** | 1855 (Daily Telegraph) |
| **Sede** | London, UK |
| **Proprietario** | DMGT (deal £500M, novembre 2025, pending) |
| **Pubblicazioni** | Daily Telegraph, Sunday Telegraph |
| **Diffusione** | ~300,000 copie/giorno (print+digital) |
| **Status** | active |

---

## Proprieta - Storia

| Periodo | Proprietario | Note |
|---------|--------------|------|
| **1855-1928** | Famiglia Levy-Lawson (Lord Burnham) | Fondazione, penny press |
| **1928-1986** | Berry family (Viscounts Camrose) | William e Gomer Berry |
| **1986-2004** | [Conrad Black](../persons/conrad-black.md) / [Hollinger](../company/hollinger.md) | Acquisizione £43M, scandalo finanziario |
| **2004-2023** | [Barclay Brothers](../family/barclay.md) / [Press Holdings](../company/press-holdings.md) | Acquisto £665M, perdita controllo per debiti |
| **2023-2025** | Lloyds Banking Group (receivership) | AlixPartners come receiver |
| **2025-** | [DMGT](dmgt.md) / [Lord Rothermere](../persons/jonathan-rothermere.md) | Deal £500M, pending approval |

### La Saga 2023-2025

| Data | Evento |
|------|--------|
| **Giugno 2023** | Lloyds Banking Group seizes control per £1.2B debiti non pagati |
| **Novembre 2023** | RedBird IMI (Sheikh Mansour) offre £1.2B |
| **Marzo 2024** | UK government blocca foreign state ownership |
| **Aprile 2024** | RedBird IMI si ritira |
| **Maggio 2025** | RedBird Capital (senza UAE) rilancia a £500M |
| **Novembre 2025** | RedBird si ritira definitivamente |
| **Novembre 2025** | DMGT (Rothermere) deal £500M |

## Timeline Storica

| Anno | Evento |
|------|--------|
| **1855** | Fondazione Daily Telegraph and Courier da Col. Arthur Sleigh |
| **1855** | Joseph Moses Levy rileva il giornale, lancia penny press |
| **1861** | Primo quotidiano a vendere 1 milione di copie (Guerra Civile USA) |
| **1928** | Vendita ai Berry brothers (Camrose) |
| **1937** | Split Berry: William tiene Telegraph, Gomer tiene Sunday Times |
| **1961** | Fondazione Sunday Telegraph |
| **1986** | Conrad Black acquisisce controllo via Hollinger |
| **1994** | Lancio Electronic Telegraph (primo quotidiano UK online) |
| **2003** | Scandalo Hollinger, Black forzato a dimettersi |
| **2004** | Barclay brothers acquistano per £665M |
| **2006** | Trasferimento da Canary Wharf a Buckingham Palace Road |
| **2006** | Rinominato Telegraph Media Group |
| **2021** | Morte Sir David Barclay |
| **2023** | Lloyds seize control, Barclay family ousted |
| **2024** | UAE/RedBird IMI bloccato, Spectator venduto a Paul Marshall |
| **2025** | DMGT acquisisce per £500M |

## Digital Markets Act 2024

Il tentativo di acquisizione da parte degli UAE ha portato a legislazione storica:

| Aspetto | Dettaglio |
|---------|-----------|
| **Legge** | Digital Markets, Competition and Consumers Act 2024 |
| **Contenuto** | Vieta foreign state ownership di giornali UK |
| **Trigger** | Bid RedBird IMI / Sheikh Mansour |
| **Effetto** | Precedente per protezione press freedom |
