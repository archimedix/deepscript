## Glossario

| Termine | Definizione |
|---------|-------------|
| **Habatsu** | Sistema di fazioni interne al LDP |
| **Seiwakai** | Fazione conservatrice (ex-Abe/Fukuda); nazionalista |
| **Kochikai** | Fazione moderata (ex-Ikeda/Kishida); internazionalista |
| **1955 System** | Sistema politico dominato da LDP vs. socialisti (1955-1993) |
| **Koenkai** | Associazioni di sostegno locale dei parlamentari |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 15 novembre 1955 |
| **Fondatori** | Fusione Liberal Party (Yoshida) + Japan Democratic Party (Hatoyama) |
| **Contesto** | Unificazione conservatrice contro il blocco socialista |
| **Obiettivo** | Stabilita' politica, alleanza USA, crescita economica |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1955** | Fondazione da fusione Liberal Party + Japan Democratic Party |
| **1955-93** | "1955 System": dominio quasi incontrastato |
| **1960s** | Miracolo economico sotto Ikeda e Sato |
| **1972** | Tanaka Kakuei presidente; era del "money politics" |
| **1993** | Prima sconfitta: governo Hosokawa (coalizione non-LDP) |
| **1994** | Ritorno al potere in coalizione con socialisti |
| **1999** | Coalizione LDP-Komeito |
| **2001-06** | Era Koizumi: riforme strutturali, privatizzazione poste |
| **2006-07** | Primo mandato Abe (dimissioni per salute) |
| **2009-12** | All'opposizione: governo DPJ (Hatoyama, Kan, Noda) |
| **2012-20** | Era Abe: il PM piu' longevo, Abenomics, riarmo |
| **2020-21** | Suga PM dopo Abe |
| **2021-24** | Kishida PM; scandalo fondi neri fazioni |
| **2024** | Sconfitta elettorale: perde maggioranza alla Camera bassa |
| **2025** | Takaichi presidente; fine coalizione con Komeito; nuova coalizione con Japan Innovation Party |

### Il Sistema delle Fazioni (Habatsu)

Le fazioni sono il cuore del potere LDP. Controllano fondi, nomine ministeriali e la successione alla presidenza.

| Fazione | Orientamento | Leader storici |
|---------|-------------|----------------|
| **Seiwakai** | Nazionalista-conservatrice | Fukuda, Abe |
| **Kochikai** | Moderata-internazionalista | Ikeda, Miyazawa, Kishida |
| **Shikoukai** (ex-Aso) | Conservatrice moderata | Aso |
| **Heisei Kenkyukai** | Centrista | Takeshita, Obuchi |
