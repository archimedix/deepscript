## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|

| **Contesto** | Veicolo politico personale di Poroshenko, rinominato ad ogni fase della sua carriera |
| **Obiettivo dichiarato** | Integrazione euro-atlantica dell'Ucraina, pro-NATO, pro-UE |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2000** | Poroshenko fonda la fazione parlamentare "Solidarnist" |
| **2005** | Riorganizzazione come "NASTUP" (Alleanza Nazionale) |
| **2009** | Diventa "Unione Panucraiana Solidarnist" (BOS) |
| **2014** | Rinominato "Bloc Petro Poroshenko" per le parlamentari; vince 132 seggi su 423 |
| **2015** | Alleanza con UDAR di Vitali Klitschko |
| **2019** | Rebranding in "European Solidarity" dopo la sconfitta presidenziale contro Zelensky |
| **2019** | Elezioni parlamentari: 8.1%, 25 seggi (crollo da 132) |
| **2022** | Sostegno alla difesa nazionale durante l'invasione russa |

## Significato nel Sistema di Potere

### Pattern

| Pattern | Descrizione |
|---------|-------------|
| **Partito personale** | Veicolo politico costruito attorno a un singolo oligarca-politico |
| **Rebranding ciclico** | Cambia nome ad ogni fase (Solidarnist → NASTUP → BOS → BPP → ES) |
| **Oligarchia politica** | Poroshenko usa ricchezza personale (Roshen, 5 Kanal) per finanziare il partito |
| **Pro-occidentale** | Posizionamento costante pro-UE/NATO, leva per legittimita' internazionale |
| **Opposizione wartime** | Ruolo ambiguo: sostegno alla difesa ma critica all'accentramento di Zelensky |
