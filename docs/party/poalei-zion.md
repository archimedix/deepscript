# Poalei Zion (Workers of Zion)

> Partito sionista-socialista marxista: matrice del movimento laburista israeliano

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1906, Poltava (conferenza fondativa) |
| **Sede** | Multipla (Russia, Palestina, USA, Europa) |
| **Tipo** | Party |
| **Fondatore** | Ber Borochov |
| **Membri al picco** | ~25.000 (1907, solo Russia) |
| **Status** | Dissolto (1919 scissione; ramo destro → Ahdut HaAvoda → Mapai) |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Poalei Zion** | "Lavoratori di Sion" (ebraico); sionismo socialista marxista |
| **Ahdut HaAvoda** | "Unita' del Lavoro"; partito nato dalla scissione destra di Poalei Zion (1919) |
| **Borochov** | Ber Borochov: teorico marxista-sionista, autore di "Our Platform" |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origini** | Circoli operai ebraici marxisti-sionisti nell'Impero Russo, dopo il rifiuto del sionismo da parte del Bund (1901) |
| **Conferenza fondativa** | Poltava, 1906: "Jewish Workers' Social Democratic Party Po'alei Zion" |
| **Fondatore** | Ber Borochov: scrisse "Our Platform" (1906), sintesi marxismo + sionismo |
| **Tesi centrale** | La questione nazionale ebraica e' risolvibile solo con concentrazione territoriale in Eretz Israel; la lotta di classe richiede una base territoriale propria |
| **World Union** | 1907: unione mondiale dei partiti Poalei Zion |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **~1901** | Primi circoli Poalei Zion nell'Impero Russo (reazione al rifiuto sionista del Bund) |
| **1906** | Conferenza fondativa a Poltava; Borochov scrive "Our Platform" |
| **1907** | World Union of Poalei Zion; 25.000 membri in Russia |
| **1906-07** | Ben-Gurion e Ben-Zvi attivi nel ramo palestinese |
| **1915** | Golda Meir entra nel Young Poale Zion a Milwaukee |
| **1917** | Morte di Borochov a Kiev |
| **1919** | Scissione: Poalei Zion Right → Ahdut HaAvoda (Ben-Gurion); Poalei Zion Left prosegue |
| **1930** | Ahdut HaAvoda + Hapoel Hatzair = Mapai |
| **1946** | Poalei Zion Left + Ahdut HaAvoda Movement → Ahdut HaAvoda-Poale Zion |
| **1948** | Fusione con Hashomer Hatzair → Mapam |
| **1954** | Dissidenti Mapam ricostituiscono Ahdut HaAvoda-Poale Zion |
| **1968** | Confluenza in Israel Labour Party |

## Potere Esercitato

### Impatto Storico

| Aspetto | Dettaglio |
|---------|-----------|
| **Ideologia fondativa** | Il marxismo-sionismo di Borochov forni' la base teorica per il sionismo laburista che domino' Israele per 30 anni |
| **Vivaio di leader** | Ben-Gurion, Ben-Zvi, Golda Meir: tutti formati nei circoli Poalei Zion |
| **Ahdut HaAvoda** | Il ramo destro divenne il nucleo di Mapai (1930), partito dominante del pre-stato e dei primi decenni di Israele |
| **Histadrut** | I leader Poalei Zion fondarono la Histadrut (1920), proto-stato economico |
| **Pipeline** | Poalei Zion → Ahdut HaAvoda → Mapai → Israel Labour Party: una linea diretta di successione politica |
