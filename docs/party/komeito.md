# Komeito

> Partito centrista giapponese legato alla Soka Gakkai, junior partner del LDP per 26 anni (1999-2025), forza moderatrice su politica estera e difesa

## Executive Summary

| Metrica | Valore |

| **Sede** | Tokyo, Japan |
| **Tipo** | Party |

| **Status** | active |

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Komeito** | "Partito del Governo Pulito" (公明党) |
| **Soka Gakkai** | Organizzazione buddista laica, base elettorale del Komeito |
| **New Komeito** | Nome usato 1998-2014 dopo rifondazione |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 17 novembre 1964 |
| **Fondatore** | [Daisaku Ikeda](../persons/daisaku-ikeda.md), presidente Soka Gakkai |
| **Contesto** | Braccio politico della Soka Gakkai per promuovere "governo pulito" |
| **Obiettivo** | Pace, welfare, moderazione politica |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1930** | Fondazione Soka Gakkai (Tsunesaburo Makiguchi) |
| **1960** | Daisaku Ikeda diventa terzo presidente Soka Gakkai |
| **1964** | Fondazione Komeito come partito politico |
| **1970** | Separazione formale Soka Gakkai-Komeito dopo controversie |
| **1972** | Takeiri visita Zhou Enlai: ruolo nella normalizzazione Giappone-Cina |
| **1993-94** | Partecipa al governo di coalizione Hosokawa (anti-LDP) |
| **1998** | Rifondato come "New Komeito" |
| **1999** | Inizio coalizione con LDP |
| **2009-12** | All'opposizione con LDP (governo DPJ) |
| **2012** | Ritorno al governo come partner LDP |
| **2014** | Torna al nome "Komeito" |
| **2023** | Morte di Daisaku Ikeda (95 anni) |
| **2024** | Tetsuo Saito diventa presidente |
| **2025** | Fine coalizione con LDP dopo 26 anni |

## Soka Gakkai: la Base

| Aspetto | Dettaglio |
|---------|-----------|
| Membri stimati | ~8 milioni di famiglie in Giappone |
| Fondazione | 1930 (buddismo Nichiren) |
| Leader storico | [Daisaku Ikeda](../persons/daisaku-ikeda.md) (1960-2023) |
| Separazione formale | 1970, ma affiliazione di fatto continua |
| Voto organizzato | Blocco elettorale disciplinato, decisivo per candidati LDP |

---

## Ruolo nella Coalizione LDP (1999-2025)

| Aspetto | Dettaglio |
|---------|-----------|
| Moderazione difesa | Freno a revisione Art. 9 e riarmo |
| Politica estera | Pro-dialogo con Cina; pacifismo |
| Welfare | Spinta su politiche sociali e sussidi |
| Voto organizzato | ~7-8 milioni di voti Soka Gakkai per candidati LDP |
| Fine coalizione | 2025: sostituito da Japan Innovation Party |
