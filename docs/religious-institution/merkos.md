# Merkos L'Inyonei Chinuch

> Il braccio educativo di Chabad-Lubavitch: fondato nel 1943 dal 6o Rebbe, coordina 6.000 coppie di emissari in 111 paesi, controlla JLI, Chabad on Campus, Kehot, Chabad.org e l'intero apparato educativo del movimento.

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Merkos** | Abbreviazione di Merkos L'Inyonei Chinuch (ebraico: מרכז לענייני חינוך, "Centro per l'Educazione") |
| **Shliach/Shluchim** | Emissario/i: coppie rabbiniche inviate a fondare e gestire centri Chabad nel mondo |
| **Kinus Hashluchim** | Conferenza Internazionale degli Emissari (annuale) |
| **Merkos 302** | Dipartimento operativo (Suite 302, 788 Eastern Parkway); backbone logistico per 4.000+ centri |
| **Merkos Shlichus** | Programma "Roving Rabbis": studenti rabbinnici inviati in citta' remote (dal 1943) |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1943 |
| **Fondatore** | [Yosef Yitzchak Schneersohn](../persons/yosef-yitzchak-schneersohn.md) (6o Rebbe) |
| **Primo Direttore** | [Menachem Mendel Schneerson](../persons/menachem-schneerson.md) (futuro 7o Rebbe; nominato chairman 1943) |
| **Contesto** | Ricostruzione ebraica post-Shoah; necessita' di infrastruttura educativa negli USA |
| **Sede storica** | 770 Eastern Parkway, Crown Heights, Brooklyn (dal 1940) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1940** | Il 6o Rebbe si stabilisce al 770 Eastern Parkway |
| **1942** | Fondazione Kehot Publication Society |
| **1943** | Fondazione Merkos L'Inyonei Chinuch; primo Merkos Shlichus (10 citta', Upstate NY) |
| **1950** | Morte del 6o Rebbe |
| **1951** | Menachem Schneerson diventa 7o Rebbe e President di Merkos |
| **1979** | Riconoscimento legale di Merkos L'Inyonei Chinuch Italia (decreto Presidente Pertini) |
| **1990** | Nomina board allargato (Krinsky, Chodakov, Shemtov, Raskin e altri) |
| **1994** | Morte del 7o Rebbe; Krinsky assume la chairmanship |
| **1995-97** | Dispute interne; consolidamento sotto leadership Krinsky-Kotlarsky |
| **1998** | Fondazione Rohr Jewish Learning Institute (JLI) |
| **2004** | Creazione Merkos 302 (Mendy Kotlarsky) |
| **~2008** | Lancio CTeen (Chabad Teen Network) |
| **2020** | Sentenza tribunale NY: piena autorita' di Merkos/Aguch su 770, 784-788 Eastern Parkway |
| **Giu 2024** | Morte di Moshe Kotlarsky (74 anni, cancro pancreatico); figlio Mendy designato successore |

## Controversie

| Tema | Periodo | Dettaglio |
|------|---------|-----------|
| **Dispute proprieta' 770** | 2004-2020 | Contenzioso tra Merkos/Aguch e Congregation Lubavitch Inc. per il controllo del 770 Eastern Parkway; sentenza 2020 assegna piena autorita' a Merkos/Aguch |
