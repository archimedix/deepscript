# Kfar Chabad

> Sede centrale del movimento Chabad-Lubavitch in Israele; villaggio-istituzione fondato nel 1949 per assorbire i chassidim sopravvissuti

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1949 |
| **Sede** | Distretto Centrale, Israele (5 km a sud di Tel Aviv) |
| **Tipo** | ReligiousInstitution / Villaggio-istituzione |
| **Leadership** | Comitato (Agudah): Rabbi Menachem Lerer (chairman) |
| **Abitanti** | ~6.000 (100% Chabad chassidim) |
| **Shluchim prodotti** | ~1.500 emissari nel mondo |
| **Status** | active |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 21 Iyar 5709 (1949) |
| **Fondatore** | Rabbi Yosef Yitzchak Schneersohn (6° Rebbe) |
| **Contesto** | Assorbire chassidim sopravvissuti all'Olocausto e all'oppressione staliniana, in transito nei campi profughi europei |
| **Sito originale** | Area di Safaria, villaggio arabo abbandonato |
| **Primi abitanti** | 74 famiglie di immigrati dall'URSS |
| **Approvazione** | Governo israeliano (Levi Eshkol) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1949** | Fondazione con 74 famiglie di sopravvissuti |
| **1956** | Massacro di Kfar Chabad: attacco terroristico alla scuola agricola; 5 studenti e 1 operaio uccisi. Il Rebbe Schneerson invio' un telegramma: la comunita' rispose costruendo nuove istituzioni |
| **1963** | Yeshiva Tomchei Tmimim di Lod trasferita a Kfar Chabad |
| **1987** | Costruzione della replica di 770 Eastern Parkway |
| **2016** | Popolazione raggiunge ~6.000; eta' mediana 17 anni |
| **2024** | Elezioni comitato Agudah; nuovo chairman Rabbi Menachem Lerer |

### Rilevanza strategica

- **Seconda piu' grande concentrazione Chabad al mondo** (dopo Crown Heights, Brooklyn)
- **~1.500 shluchim** prodotti dal villaggio, operanti globalmente
- **Centro logistico** per distribuzione materiali rituali ed educativi alla rete globale
- **Modello villaggio-istituzione**: 100% Chabad, autogestione comunitaria, integrazione completa vita-religione-lavoro
