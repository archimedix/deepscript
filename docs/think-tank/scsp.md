# Special Competitive Studies Project (SCSP)

> Think tank di Eric Schmidt per continuare il lavoro NSCAI sulla competizione tecnologica USA-Cina

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | Ottobre 2021 |
| **Sede** | Washington DC, USA |
| **Tipo** | ThinkTank (Private Foundation) |
| **Leadership** | Eric Schmidt (Chair), Ylli Bajraktari (CEO) |
| **Parent** | Eric & Wendy Schmidt Fund for Strategic Innovation |
| **Status** | active |

## Critiche
