## Ideologia: Resource-Based Economy

### Principi fondamentali

| Principio | Descrizione |
|-----------|-------------|
| **Abolizione moneta** | Nessun denaro, credito, baratto o debito |
| **Risorse comuni** | Gestione scientifica delle risorse del pianeta |
| **Automazione** | Tecnologia per liberare l'umanita' dal lavoro ripetitivo |
| **Design scientifico** | Citta' progettate per efficienza e sostenibilita' |
| **Decisioni basate su dati** | Nessuna politica, solo metodo scientifico |

### Differenze da Technocracy Inc.

| Technocracy Inc. | Venus Project |
|------------------|---------------|
| Energy certificates | Nessuna valuta |
| Nord America | Visione globale |
| 1930s-40s | Post-scarsita' |
| Ingegneri al comando | Automazione + AI |
