## Glossario

| Termine | Definizione |
|---------|-------------|
| Biosecurity | Protezione da minacce biologiche (bioterrorismo, pandemie) |
| Biodefense | Difesa civile contro armi biologiche |
| Tabletop exercise | Simulazione strategica per testare preparazione |
| Gain-of-function | Ricerca che aumenta patogenicita'/trasmissibilita' virus |

## Esercitazioni Pandemiche

### Dark Winter (2001)

| Dato | Dettaglio |
|------|-----------|
| Data | Giugno 2001 |
| Scenario | Attacco bioterroristico con vaiolo |
| Partecipanti | Ex-senatori, funzionari governo |
| Lezione | USA impreparati a bioterrorismo |

### Clade X (2018)

| Dato | Dettaglio |
|------|-----------|
| Data | Maggio 2018 |
| Scenario | Pandemia da virus ingegnerizzato |
| Focus | Risposta governo federale |

### Event 201 (2019)

| Dato | Dettaglio |
|------|-----------|
| Data | 18 Ottobre 2019 |
| Luogo | New York |
| Co-organizzatori | [WEF](../forum/wef.md), [Gates Foundation](../foundation/gates-foundation.md) |
| Scenario | Coronavirus zoonotico (pipistrelli → maiali → umani) |
| Partecipanti | 130 leader business/governo/sanita' |
| Moderatore | Tom Inglesby |

**Conclusioni Event 201:**
- Grave gap nella preparazione pandemica
- Necessita' cooperazione pubblico-privato
- Importanza comunicazione e contrasto disinformazione
