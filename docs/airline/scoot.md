## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Annuncio** | Maggio 2011 |
| **Founding CEO** | Campbell Wilson |
| **Primo volo** | 4 giugno 2012 (Singapore-Sydney) |
| **Obiettivo** | LCC medium/long-haul per competere con AirAsia X |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2011** | SIA annuncia nuova LCC; Campbell Wilson nominato CEO |
| **2012** | Primo volo Singapore-Sydney con B777-200ER ex-SIA |
| **2015** | Transizione a Boeing 787 Dreamliner |
| **2016** | Campbell Wilson promosso a SVP SIA |
| **2017** | Merger con Tigerair (25 luglio); adotta codice TR |
| **2020** | Campbell Wilson ritorna come CEO (COVID crisis) |
| **2022** | Leslie Thng diventa CEO |
| **2024** | Thng promosso EVP SIA Group |
| **2025** | 70+ destinazioni; espansione flotta con E190-E2 |

## Modello di Business

### Posizionamento

| Aspetto | Scoot | Competitor |
|---------|-------|------------|
| Segmento | LCC medium/long-haul | AirAsia X, Jetstar |
| Widebody | Si (787) | Raro per LCC |
| Premium | ScootPlus | Economy-focused |
| Network | Asia-Pacific + Europa | Regionale |

### Differenziazione
- Unico LCC asiatico con flotta 787 significativa
- Integrazione con frequent flyer SIA (KrisFlyer)
- Accesso a slot premium Changi via parent
