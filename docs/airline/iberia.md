## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 28 giugno 1927 |
| **Fondatori** | Horacio Echevarrieta, Deutsche Luft Hansa |
| **Capitale iniziale** | 1.1 milioni di pesetas |
| **Prima rotta** | Madrid-Barcellona (14 dicembre 1927) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1927** | Fondazione con investimento tedesco-spagnolo |
| **1940s** | Nazionalizzazione sotto regime franchista |
| **1996** | Xabier de Irala avvia ristrutturazione |
| **2001** | Privatizzazione completata |
| **2011** | Fusione con British Airways in IAG |
| **2012** | Lancio Iberia Express (low-cost) |
| **2014-2020** | Luis Gallego risana la compagnia |
| **2024** | Record profitti post-COVID |

### Proprieta'

Dal 2011 Iberia e' subsidiaria al 100% di [IAG](../company/iag.md).

**Struttura IAG:**
- British Airways (55% alla fusione)
- Iberia (45% alla fusione)
- Vueling (acquisita 2012)
- Aer Lingus (acquisita 2015)
- LEVEL

## Trasformazione sotto Luis Gallego

| Anno | Risultato Operativo | Note |
|------|---------------------|------|
| **2012** | -€351M | Pre-ristrutturazione |
| **2013** | -€166M | Primo anno Gallego |
| **2014** | +€50M | Ritorno al profitto |
| **2015** | Positivo | Top airline puntualita' mondiale |

### Strategia di Risanamento

1. Taglio costi operativi
2. Accordi sindacali su salari
3. Lancio Iberia Express (low-cost corto raggio)
4. Focus su puntualita' (>90%)
5. Rafforzamento network America Latina
