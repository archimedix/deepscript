# Manhattan Project

> Programma militare segreto USA per lo sviluppo della bomba atomica (1942-1946): il progetto che inauguro' l'era nucleare e ridisegno' la geopolitica globale

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1942 (origini nel 1939 con la lettera Einstein-Szilard) |
| **Scioglimento** | 1946 (funzioni trasferite alla AEC) |
| **Sede** | Oak Ridge (TN), Los Alamos (NM), Hanford (WA) + 30 siti |
| **Tipo** | Defense |
| **Leadership** | Gen. Leslie Groves (militare), J.R. Oppenheimer (scientifico) |
| **Personale** | ~125.000 al picco |
| **Budget** | ~$2 miliardi (1945) — ~$30 miliardi attuali |
| **Status** | defunct |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origine** | Lettera Einstein-Szilard a Roosevelt (agosto 1939) |
| **Uranium Committee** | 1939, primo comitato di studio ($6.000 budget) |
| **OSRD / S-1** | 1941, Vannevar Bush riorganizza sotto Office of Scientific Research and Development |
| **Manhattan Engineer District** | Settembre 1942, Gen. Groves assume il comando |
| **Obiettivo** | Sviluppare armi atomiche prima della Germania nazista |
| **Successore** | Atomic Energy Commission (AEC), 1 gennaio 1947 |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1939** | Lettera Einstein-Szilard a Roosevelt; creazione Uranium Committee |
| **1941** | Creazione OSRD sotto Vannevar Bush; Pearl Harbor accelera il programma |
| **1942** | Gen. Groves nominato direttore; Oppenheimer scelto per Los Alamos; Fermi realizza la prima reazione nucleare controllata (Chicago Pile-1, 2 dicembre) |
| **1943** | Los Alamos operativo; accordi con UK (Quebec Agreement) e Canada |
| **1944** | Produzione plutonio a Hanford; arricchimento uranio a Oak Ridge |
| **1945** | Test Trinity (16 luglio); Hiroshima (6 agosto); Nagasaki (9 agosto); resa del Giappone |
| **1946** | Atomic Energy Act; trasferimento al controllo civile (AEC) |

## Critiche

| Tema | Dettaglio |
|------|-----------|
| Uso su civili | Hiroshima e Nagasaki: ~200.000 morti, dibattito sulla necessita' militare |
| Szilard Petition | 70 scienziati firmarono contro l'uso senza preavviso; ignorata |
| Segretezza | Compartimentazione estrema; molti lavoratori non sapevano cosa producevano |
| Spionaggio | Klaus Fuchs, Theodore Hall passarono segreti all'URSS |
| Salute lavoratori | Esposizione a radiazioni nei siti di produzione |
