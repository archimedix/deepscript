# Yuzhmash (Pivdenmash)

> Principale produttore missilistico e spaziale ucraino, ex fabbrica segreta di ICBM sovietici

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1944 |
| **Sede** | Dnipro, Ucraina |
| **Tipo** | Defense (aerospazio e difesa) |
| **Nome ufficiale** | State Enterprise Production Association Yuzhny Machine-Building Plant named after A.M. Makarov |
| **Leadership** | Sergii Voit, General Director |
| **Dipendenti** | ~10.000 (storico: fino a 50.000 in epoca sovietica) |
| **Proprieta'** | Impresa statale, dipende dalla State Space Agency of Ukraine |
| **Status** | active |

## Glossario

| Termine | Definizione |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1944, come stabilimento automobilistico |
| **Conversione militare** | 1951, trasformato in fabbrica missilistica segreta |
| **Fondatore bureau** | Mikhail Yangel (OKB-586, 1954) |
| **Nome attuale** | Intitolato a Oleksandr Makarov, direttore per 30 anni |
| **Obiettivo** | Progettazione e produzione di missili balistici e vettori spaziali |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1944** | Apertura come stabilimento automobilistico a Dnipropetrovsk |
| **1951** | Conversione in fabbrica missilistica segreta sovietica (Plant 586) |
| **1954** | Mikhail Yangel fonda OKB-586 (oggi Yuzhnoye Design Bureau) |
| **1966** | Rinominato Southern Machine-Building Plant (Yuzhmash) |
| **1961-1986** | Oleksandr Makarov direttore generale per 25 anni |
| **1986-1992** | [Leonid Kuchma](../persons/leonid-kuchma.md) direttore generale |
| **1991** | Indipendenza ucraina, inizio conversione civile |
| **1994** | Kuchma diventa Presidente dell'Ucraina |
| **2015** | Russia interrompe programma congiunto razzi Dnepr e booster Zenit |
| **2017** | Scandalo IISS: accuse (poi smentite) di trasferimento tecnologia missilistica alla Corea del Nord |
| **2022** | Attacco missilistico russo allo stabilimento durante l'invasione |

## Produzione

### ICBM sovietici

| Missile | Designazione NATO | Note |
|---------|-------------------|------|
| R-5M | SS-3 Shyster | Primo razzo sovietico con testata nucleare |
| R-12 Dvina | SS-4 Sandal | Dispiegato a Cuba (1962) |
| R-14 Chusovaya | SS-5 Skean | MRBM |
| R-16 | SS-7 Saddler | Primo ICBM sovietico ampiamente dispiegato |
| R-36 | SS-9 Scarp | ICBM pesante |
| R-36M | SS-18 Satan | ICBM piu' potente al mondo |

Capacita' produttiva in epoca sovietica: fino a 120 ICBM all'anno.

### Vettori spaziali

- **Zenit**: lanciatore medio-pesante
- **Dnepr**: conversione civile di ICBM R-36
- **Antares**: componenti strutturali per [Northrop Grumman](../defense/northrop-grumman.md) (programma rifornimento ISS)

## Controversie

| Tema | Dettaglio |
|------|-----------|
| **Scandalo Corea del Nord (2017)** | Rapporto IISS sostiene che motori missilistici nordcoreani provenissero da Yuzhmash. Indagine governativa ucraina conclude: nessun trasferimento. L'ipotesi riguardava reti criminali, non dirigenza o governo |
| **Crisi finanziaria post-sovietica** | Drastico calo delle commesse dopo il 1991, difficolta' nella riconversione civile |
| **Attacco russo 2022** | Putin conferma attacco missilistico allo stabilimento durante l'invasione |
