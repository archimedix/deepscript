## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 29 giugno 2019 |
| **Predecessori** | L3 Technologies (1997) + Harris Corporation (1895) |
| **Fusione** | All-stock merger da $35B |
| **Obiettivo** | Creare 6o prime contractor USA |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1895** | Harris Corp fondata come tipografia |
| **1997** | L-3 Communications spinoff da Lockheed Martin |
| **2018** | Annunciata fusione L3-Harris |
| **2019** | Completata fusione, nasce L3Harris |
| **2021** | Kubasik diventa CEO |
| **2022** | Annunciata acquisizione Aerojet Rocketdyne ($4.7B) |
| **2023** | Completata acquisizione Aerojet |
| **2024** | D.E. Shaw entra nel board (activist) |
