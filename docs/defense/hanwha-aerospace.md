## Gruppo Hanwha

Hanwha Aerospace e' parte del conglomerato Hanwha Group, settimo chaebol coreano per dimensioni.

### Struttura Difesa

| Entita' | Focus |
|---------|-------|
| **Hanwha Aerospace** | Motori, missili, spazio |
| **Hanwha Defense** | Veicoli corazzati (K9, K21) |
| **Hanwha Ocean** | Navi, sottomarini (ex DSME) |
| **Hanwha Systems** | Radar, elettronica difesa |
