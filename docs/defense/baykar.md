# Baykar Makina

> Leader mondiale nell'export di droni militari, controllata dalla famiglia Bayraktar e legata al presidente Erdogan

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1984 |
| **Sede** | Istanbul, Turkey |
| **Tipo** | Defense (Aerospace/Drones) |
| **Leadership** | Haluk Bayraktar (CEO), Selcuk Bayraktar (Chairman/CTO) |
| **Dipendenti** | ~5.000 |
| **Revenue** | $2 mld (2024, 90% export) |
| **Status** | active |

## Glossario

| Termine | Definizione |
|---------|-------------|
| UAV/UAS | Unmanned Aerial Vehicle/System - velivolo senza pilota |
| UCAV | Unmanned Combat Aerial Vehicle - drone armato |
| MALE | Medium Altitude Long Endurance - classe droni |
| TB2 | Bayraktar TB2, drone MALE piu' esportato al mondo |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1984 |
| **Fondatori** | Ozdemir Bayraktar |
| **Nome** | Portmanteau di "Bayraktar Kardeler" (Fratelli Bayraktar) |
| **Contesto** | Nasce come subfornitore CNC per automotive |
| **Obiettivo dichiarato** | Localizzazione industria turca, poi difesa |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1984** | Fondazione come Baykar Makina, produzione parti auto |
| **2000s** | Transizione verso UAV con ritorno di Selcuk da MIT |
| **2007** | Selcuk rientra in Turchia, sviluppo TB2 |
| **2014** | Primo volo Bayraktar TB2 |
| **2016** | Selcuk sposa Sumeyye Erdogan |
| **2019** | Export TB2 a Ucraina (12 unita', $69M) |
| **2020** | TB2 protagonista in Libia, Nagorno-Karabakh, Siria |
| **2021** | Morte Ozdemir Bayraktar (72 anni) |
| **2022** | TB2 simbolo resistenza ucraina; donazioni campagne crowdfunding |
| **2023** | Primo volo Bayraktar Kizilelma (jet unmanned) |
| **2024** | Acquisizione Piaggio Aerospace (Italia); export a 37 paesi |
| **2025** | Partnership Leonardo; stabilimento Ucraina operativo |

## Export e Clienti

### Statistiche

- **65%** del mercato globale export UAV e' turco
- **60%** dell'export turco e' Baykar
- **37 paesi** con contratti firmati (2024)
- **$1.8 mld** revenue export 2024 (90% del totale)

### Clienti NATO/UE

Albania, Croazia, Polonia, Romania, Ungheria

### Altri Clienti Principali

Ucraina, Azerbaijan, Etiopia, Libia, UAE, Arabia Saudita, Pakistan, Marocco, Qatar
