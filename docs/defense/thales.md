## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origini** | Compagnie Française Thomson-Houston (1893) |
| **Fusione** | Thomson-Brandt + CSF → Thomson-CSF (1968) |
| **Nazionalizzazione** | 1982 (governo Mitterrand) |
| **Privatizzazione** | 1987 (parziale), 1997 (completa) |
| **Rebranding** | Thomson-CSF → Thales (2000) |

### Timeline

| Anno | Evento |
|------|--------|
| **1893** | Fondazione CFTH (brevetti Thomson-Houston USA) |
| **1918** | Fondazione CSF (radio, radar) |
| **1968** | Fusione Thomson-Brandt + CSF → Thomson-CSF |
| **1982** | Nazionalizzazione governo Mitterrand |
| **1987** | Prima privatizzazione (Stato mantiene maggioranza) |
| **1996** | Tentativo vendita a Lagardère/Daewoo fallisce |
| **1997** | Privatizzazione con Alcatel e Dassault |
| **2000** | Rebranding Thomson-CSF → Thales |
| **2024** | Vendita divisione trasporti a Hitachi Rail ($2.5B) |
