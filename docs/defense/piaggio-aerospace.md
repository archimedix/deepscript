## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1884 |
| **Fondatore** | Rinaldo Piaggio |
| **Sede originale** | Sestri Ponente, Genova |
| **Contesto** | Allestimento navi e materiale rotabile ferroviario |
| **Transizione aeronautica** | 1915, acquisizione Officine Aeronautiche Francesco Oneto |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1884** | Fondazione Rinaldo Piaggio S.p.A. a Sestri Ponente, Genova |
| **1906** | Costruzione stabilimento Finale Marina |
| **1915** | Ingresso nel settore aeronautico |
| **1920s** | Assunzione ingegneri Giovanni Pegna e Giuseppe Gabrielli |
| **1986** | Primo volo P.180 Avanti |
| **1990** | Certificazione FAA per P.180 |
| **2006** | Mubadala (Abu Dhabi) acquisisce 35% |
| **2008** | Tata Group acquisisce 33% |
| **2013** | Primo volo P.1HH HammerHead; Mubadala e Tata iniettano €190M |
| **2014** | Mubadala diventa azionista di maggioranza |
| **2016** | Crash prototipo P.1HH al largo della Sicilia |
| **2018** | UAE cancella ordine HammerHead; receivership |
| **2021** | Italia cancella programma drone, ordina 6 P.180 Avanti Evo (€171M) |
| **2024** | Acquisizione da Baykar (27 dicembre) |
| **2025** | Closing definitivo (30 giugno); MoU Leonardo-Baykar per JV droni |

## L'acquisizione Baykar (2024-2025)

### Contesto
Dopo 6 anni di amministrazione straordinaria, il Ministero delle Imprese ha selezionato Baykar tra tre offerenti.

### Piani industriali Baykar
1. Rilancio P.180 Avanti Evo
2. Produzione in Italia di droni TB3 e Akinci
3. Centro europeo manutenzione motori e fusoliere
4. Accesso alla tecnologia HammerHead

### Partnership Leonardo-Baykar
- MoU firmato 6 giugno 2024 a Roma
- Joint venture per sistemi aerei senza pilota
- Baykar fornisce piattaforme, Leonardo sistemi missione e payload
