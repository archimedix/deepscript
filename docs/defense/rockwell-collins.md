## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1933 (incorporazione) |
| **Fondatore** | [Arthur A. Collins](../persons/arthur-collins.md) |
| **Origine** | Basement of Collins home, Cedar Rapids |
| **Prima fama** | Radio link con spedizione Byrd al Polo Sud |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1931** | Arthur Collins apre laboratorio nel basement |
| **1933** | Incorporazione Collins Radio Company |
| **1933** | Link radio con spedizione Byrd al Polo Sud |
| **1940s** | Contratti militari WWII |
| **1971** | Rockwell acquisisce controllo, rimuove Arthur Collins |
| **1973** | Acquisizione completa da Rockwell International |
| **2001** | IPO, spin-off da Rockwell International (NYSE: COL) |
| **2002** | Clay Jones diventa Chairman |
| **2013** | Acquisizione ARINC ($1.4 mld) |
| **2013** | Kelly Ortberg diventa CEO |
| **2017** | Acquisizione B/E Aerospace ($8.3 mld) |
| **2017** | UTC annuncia acquisizione ($30 mld) |
| **2018** | Merge con UTC Aerospace → Collins Aerospace |
