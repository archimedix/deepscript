# AvtoKrAZ

> Produttore ucraino di camion militari pesanti, nazionalizzato nel 2022

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1945 |
| **Sede** | Kremenchuk, Ucraina |
| **Tipo** | Defense (military trucks) |
| **Proprietario** | Kostiantyn Zhevago (fino al 2022), poi Stato ucraino |
| **Dipendenti** | ~3,000 |
| **Status** | active (nazionalizzata nov 2022) |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 31 agosto 1945, prima pietra posata nel 1946 |
| **Gruppo** | Finance&Credit, conglomerato finanziario-industriale di Kostiantyn Zhevago |
| **Produzione** | Camion pesanti fuoristrada (KrAZ-6322 e altri), veicoli speciali militari |
| **Rilevanza** | Fornitore chiave delle forze armate ucraine |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1945** | Fondazione dello stabilimento a Kremenchuk |
| **1946** | Posa della prima pietra |
| **Era sovietica** | Produzione camion pesanti per l'Armata Rossa |
| **Anni '90** | Privatizzazione, acquisizione da parte del gruppo Finance&Credit di Zhevago |
| **2022 nov** | Nazionalizzazione sotto legge marziale, produzione riorientata alla difesa |
| **2025** | Zhevago sanzionato da Zelensky per alto tradimento |
