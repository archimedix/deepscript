## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2018 |
| **Origine** | Fusione Rockwell Collins + UTC Aerospace Systems |
| **Contesto** | Consolidamento industria aerospace |
| **Obiettivo** | Creare terzo player mondiale dopo Boeing e Airbus |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1933** | Fondazione Collins Radio Company (Arthur Collins) |
| **1973** | Acquisizione da Rockwell International |
| **2001** | Spin-off come Rockwell Collins |
| **2013** | Kelly Ortberg diventa CEO Rockwell Collins |
| **2017** | UTC annuncia acquisizione Rockwell Collins per $30 mld |
| **2018** | Nasce Collins Aerospace dalla fusione |
| **2020** | Merger UTC-Raytheon crea Raytheon Technologies |
| **2023** | Rebrand in RTX Corporation |
| **2025** | Troy Brunk succede a Stephen Timm come President |

### Business Units

1. **Avionics** - Sistemi flight deck, display, comunicazioni
2. **Interiors** - Interni cabina, sedili, sistemi entertainment
3. **Mission Systems** - Soluzioni militari, ISR, C2
4. **Power & Controls** - Sistemi elettrici, idraulici, fuel
5. **Aerostructures** - Nacelle, thrust reversers, strutture
