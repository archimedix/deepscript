## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Dicembre 1990 |
| **Predecessori** | Aeritalia (1969) + Selenia |
| **Parent** | Finmeccanica/IRI |

| **Obiettivo** | Unificare capacita' aeronautiche ed elettroniche italiane |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1969** | Fondazione Aeritalia (fusione Fiat Aviazione + Aerfer) |
| **1989** | IRI avvia riorganizzazione; Aeritalia e Selenia sotto Finmeccanica |
| **1990** | Fusione Aeritalia + Selenia = **Alenia Aeronautica** |
| **1997** | JV con Lockheed Martin per C-27J (LMATTS) |
| **2002** | Alenia diventa societa' autonoma sotto Finmeccanica |
| **2007** | C-27J vince contratto US Joint Cargo Aircraft ($2.04B) |
| **2012** | Fusione con Aermacchi → **Alenia Aermacchi** |
| **2016** | Integrazione in Leonardo-Finmeccanica |
| **2017** | Leonardo S.p.A. - nome definitivo |

## Contesto Strategico

### Ruolo nel Sistema Italia

Alenia rappresento' il tentativo italiano di mantenere capacita' aerospaziali nazionali attraverso:
- **Partecipazione a consorzi europei** (Eurofighter, Tornado, ATR)
- **Partnership transatlantiche** (Lockheed Martin, Boeing)
- **Controllo statale** via Finmeccanica/IRI

### Fine Guerra Fredda

La fusione del 1990 avvenne nel contesto di:
- Riduzione budget difesa
- Necessita' di economie di scala
- Consolidamento industria europea
