## La Riforma del 2001

La crisi finanziaria del 2001 porto' a riforme strutturali della banca centrale, architettate da [Kemal Dervis](../persons/kemal-dervis.md):

| Aspetto | Prima | Dopo 2001 |
|---------|-------|-----------|
| **Obiettivo** | Multiplo | Stabilita' dei prezzi (primario) |
| **Indipendenza** | Limitata | Operativa (strumenti) |
| **Finanziamento governo** | Consentito | **Vietato** |
| **Decisioni** | Governatore | MPC (7 membri) |
| **Accountability** | Limitata | Report al Parlamento |

### Risultati

| Periodo | Inflazione media |
|---------|------------------|
| **1990s** | ~70% |
| **2003** | 12% |
| **2004-2017** | Single digit |
| **2022-2024** | Ritorno alta inflazione |
| **Mag 2024** | 75.5% (picco) |
| **Apr 2025** | 37.9% (in calo) |

## Significato nel Sistema di Potere

### Indipendenza vs Controllo Politico

La TCMB rappresenta la tensione tra:
1. **Ortodossia monetaria**: Indipendenza, inflation targeting, tassi alti
2. **Erdoganomics**: Tassi bassi = crescita, "lobby dei tassi"

| Fase | Caratteristica |
|------|----------------|
| **2001-2019** | Relativa indipendenza, disinflazione |
| **2019-2023** | Subordinazione politica, inflazione galoppante |
| **2023-** | Tentativo ritorno ortodossia (Simsek + Erkan/Karahan) |

### Revolving Door

| Pattern | Esempio |
|---------|---------|
| **FMI/WB → Governo → Academia** | Kemal Dervis |
| **TCMB → Think Tank** | Guven Sak (MPC → TEPAV) |
| **Fed NY → TCMB** | Fatih Karahan |
| **Wall Street → TCMB** | Hafize Gaye Erkan (First Republic) |
