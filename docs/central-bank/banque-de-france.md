## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 18 gennaio 1800 |
| **Fondatore** | Napoleone Bonaparte (Primo Console) |
| **Primo Governatore** | Emmanuel Crétet |
| **Contesto** | Stabilizzazione monetaria post-rivoluzionaria |
| **Obiettivo** | "Massa di granito" per stabilizzare la nazione |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1800** | Fondazione da Napoleone Bonaparte |
| **1803** | Monopolio emissione banconote a Parigi, primo franco germinale |
| **1805** | Crisi finanziaria, riforma governance |
| **1806** | Governatore e Vice-Governatori nominati dallo Stato |
| **1848** | Monopolio emissione esteso a tutta la Francia |
| **1936** | Portata sotto controllo governativo |
| **1945** | Nazionalizzazione |
| **1993** | Jean-Claude Trichet diventa Governatore |
| **1998** | Integrazione nell'Eurosistema (BCE) |
| **2003** | Christian Noyer Governatore |
| **2015** | François Villeroy de Galhau Governatore |
| **2017** | Co-fondazione NGFS (Network for Greening Financial System) |
| **2021** | Villeroy de Galhau rinominato per secondo mandato |
| **2025** | 225° anniversario |
