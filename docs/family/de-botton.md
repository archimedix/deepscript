# Famiglia de Botton (Spagna → Salonicco → Alessandria → Svizzera)

> Dinastia sefardita con 500 anni di storia: dal rabbinato di Salonicco alla finanza svizzera, passando per Alessandria e l'intelligence israeliana

## Dati della Dinastia

| Aspetto | Dettaglio |
|---------|-----------|
| **Origine** | Castiglia (Spagna), espulsi nel 1492 |
| **Capostipite documentato** | [Abraham de Boton](../persons/abraham-de-boton.md) (c. 1560-1605), rabbino capo a Salonicco |
| **Traiettoria** | Spagna → Salonicco → Istanbul/Alessandria → Europa |
| **Matriarca moderna** | Yolande Harmer (1913-1959) |
| **Costruttore patrimonio** | Gilbert de Botton (1935-2000) |
| **Settore** | Rabbinato → Intelligence → Finanza → Filantropia/Cultura |
| **Generazioni documentate** | ~15 (dal 1492 a oggi) |
| **Patrimonio stimato** | £234m (1999, vendita GAM a UBS) |
| **Status attuale** | Attiva (filantropia e cultura) |

---

## Significato Storico

La famiglia de Botton incarna una traiettoria tipica delle elite sefardite di Alessandria: da una comunita' cosmopolita sotto protezione reale egiziana, attraverso lo spionaggio per la nascita di Israele, alla costruzione di un impero finanziario svizzero legato ai Rothschild, fino alla trasformazione in dinastia filantropica e culturale.

> La transizione da intelligence a finanza a filantropia rispecchia il pattern delle famiglie che convertono il capitale sociale e informativo in capitale finanziario e poi in influenza culturale.
