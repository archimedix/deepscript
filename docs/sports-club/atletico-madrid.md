## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 26 Aprile 1903 |
| **Fondatori** | Studenti baschi a Madrid |
| **Nome originale** | Athletic Club Sucursal de Madrid |
| **Indipendenza** | 1912 (da Athletic Bilbao) |
| **Nome attuale** | Dal 1946 |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1903** | Fondazione come succursale dell'Athletic Bilbao |
| **1912** | Indipendenza dal club basco |
| **1923** | Inaugurazione primo Estadio Metropolitano |
| **1939** | Fusione con Club Aviación Nacional |
| **1946** | Diventa "Atlético de Madrid" |
| **1966** | Trasferimento al Vicente Calderón |
| **2011** | Diego Simeone diventa allenatore |
| **2014** | La Liga + finale Champions League |
| **2016** | Seconda finale Champions League |
| **2017** | Trasferimento al Wanda Metropolitano |
| **2018** | Quantum Pacific (Idan Ofer) diventa azionista 15% |
| **2020** | Quantum Pacific sale al 32% (acquisisce quota Dalian Wanda) |
| **2021** | 11° titolo La Liga |
| **2025 Nov** | Apollo Sports Capital acquisisce maggioranza (51-55%) per $2.55B |
