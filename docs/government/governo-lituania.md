# Governo della Lituania

> Repubblica parlamentare baltica, membro NATO e UE dal 2004, fronte anti-Russia in Europa orientale

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1990 (restaurazione indipendenza) |
| **Sede** | Vilnius |
| **Sistema** | Repubblica parlamentare semi-presidenziale |
| **Capo di Stato** | Presidente (Gitanas Nauseda, 2019-) |
| **Capo di Governo** | Primo Ministro (Inga Ruginiene, 2025-) |
| **PIL** | ~$75 mld (2024) |
| **Status** | active |

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica |
| **Forma di Governo** | Semi-presidenziale (presidente eletto direttamente, ma PM guida il governo) |
| **Legislativo** | Seimas (unicamerale, 141 seggi) |
| **Giudiziario** | Corte Costituzionale, Corte Suprema |
| **Partiti dominanti** | Homeland Union (centro-destra), Social Democrats (centro-sinistra), Dawn of Nemunas |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1990-1992** | Indipendenza | Rottura con URSS, resistenza non violenta, riconoscimento internazionale |
| **1993-2003** | Transizione | Alternanza ex-comunisti/conservatori, privatizzazioni, crisi russa 1998 |
| **2004** | Integrazione occidentale | Ingresso NATO e UE nello stesso anno |
| **2009-2019** | Consolidamento | Presidenza Grybauskaite, euro (2015), rafforzamento difesa |
| **2020-2024** | Fronte anti-autoritario | Crisi con Cina per Taiwan, sostegno Ucraina, rafforzamento NATO |
| **2025-** | Instabilita' interna | Caduta Paluckas, nuovo governo socialdemocratico |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1990** | 11 marzo: dichiarazione indipendenza (prima repubblica sovietica) |
| **1991** | Gennaio: attacco sovietico alla torre TV di Vilnius, 14 morti |
| **1991** | Riconoscimento internazionale dell'indipendenza |
| **2004** | Ingresso NATO (29 marzo) e UE (1 maggio) |
| **2009** | Grybauskaite prima presidente donna |
| **2015** | Adozione dell'euro |
| **2021** | Apertura ufficio rappresentanza Taiwan, crisi con Cina |
| **2022** | Forte sostegno militare all'Ucraina |
| **2023** | Vertice NATO a Vilnius |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Frontiera NATO** | Confine diretto con Russia (Kaliningrad) e Bielorussia, corridoio di Suwalki punto critico |
| **Fronte anti-Russia** | Tra i piu' duri sostenitori di sanzioni e supporto militare a Ucraina |
| **Taiwan gambit** | Unico paese UE ad aver aperto ufficio con nome "Taiwan", sfidando Pechino |
| **Ponte transatlantico** | Elite politica con forti legami USA (Atlantic Council, Stanford, George W. Bush Center) |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Dinastia Landsbergis** | Tre generazioni al governo: nonno (indipendenza), nipote (Min. Esteri) |
| **Revolving door UE** | Grybauskaite: Min. Finanze → Commissario UE → Presidente. Kubilius: PM → Commissario UE Difesa |
| **Small state, big voice** | 2.8M abitanti ma influenza sproporzionata nella politica estera UE grazie a posizioni nette |
| **Circuito think tank** | Post-mandato: leaders lituani assorbiti da Atlantic Council, ECFR, Stanford, GMF |
