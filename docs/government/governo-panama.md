# Governo di Panama

> Repubblica presidenziale, snodo tra Nord e Sud America, custode del Canale

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1903 (indipendenza dalla Colombia) |
| **Sede** | Panama City, Palacio de las Garzas |
| **Sistema** | Repubblica presidenziale |
| **Capo di Stato** | Jose' Raul Mulino (2024–) |
| **PIL** | ~$77 mld (2024) |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **PRD** | Partido Revolucionario Democratico — fondato da Omar Torrijos |
| **PA / Panameñismo** | Partido Panameñista (ex Arnulfista) — fondato da Arnulfo Arias |
| **ACP** | Autoridad del Canal de Panama |
| **Zona del Canale** | Area sotto sovranita' USA fino al 1999 |

---

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica |
| **Forma di Governo** | Presidenziale (mandato 5 anni, no rielezione consecutiva) |
| **Legislativo** | Asamblea Nacional (unicamerale, 71 deputati) |
| **Giudiziario** | Corte Suprema de Justicia |
| **Partiti dominanti** | PRD, Panameñismo, Cambio Democratico, Realizando Metas |
| **Esercito** | Abolito nel 1990 da Endara, sostituito da Servicio Nacional de Fronteras |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1903–1968** | Repubblica oligarchica | Indipendenza sponsorizzata da USA, Zona del Canale |
| **1968–1989** | Dittature militari | Torrijos (populista), poi Noriega (narco-militare) |
| **1989** | Invasione USA | Operation Just Cause, fine regime Noriega |
| **1989–1999** | Transizione democratica | Endara, Perez Balladares — ricostruzione istituzioni |
| **1999–2009** | Era del Canale | Handover (1999), espansione (2006), boom economico |
| **2009–oggi** | Democrazia consolidata | Alternanza PRD/PA/CD, scandali finanziari ricorrenti |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1903** | Indipendenza dalla Colombia (con appoggio USA) |
| **1914** | Apertura del Canale di Panama |
| **1968** | Golpe di Omar Torrijos |
| **1977** | Trattati Torrijos-Carter per restituzione Canale |
| **1981** | Morte di Torrijos in incidente aereo |
| **1989** | Invasione USA, caduta di Noriega |
| **1990** | Abolizione dell'esercito |
| **1999** | Handover del Canale dagli USA a Panama |
| **2006** | Referendum espansione Canale (78% si') |
| **2016** | Panama Papers — scandalo Mossack Fonseca |
| **2016** | Inaugurazione terzo set di chiuse del Canale |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Canale** | Choke point del commercio globale — 5% del commercio marittimo mondiale |
| **Hub finanziario** | Centro offshore, segreto bancario, societa' anonime |
| **USA** | Rapporto di dipendenza storica — invasione 1989, dollaro come valuta |
| **Intelligence** | Snodo operativo per CIA e Mossad (Harari) durante Guerra Fredda |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Oligarchia familiare** | Poche famiglie (Gonzalez-Revilla, Martinelli, Arias) controllano economia e politica |
| **Revolving door** | Imprenditori diventano presidenti e viceversa |
| **Hub offshore** | Panama Papers rivelano ruolo sistemico nel riciclaggio globale |
| **Dipendenza USA** | Dal Canale all'invasione al dollaro — rapporto asimmetrico strutturale |
| **Alternanza senza alternativa** | PRD e PA si alternano, stesse elite al comando |
