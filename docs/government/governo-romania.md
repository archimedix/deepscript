| Metrica | Valore |

## Glossario

| Termine | Definizione |
|---------|-------------|
| PSD | Partito Social Democratico (ex comunisti riformati) |
| PNL | Partito Nazionale Liberale (centro-destra pro-UE) |
| USR | Uniunea Salvati Romania (riformisti anti-corruzione) |
| UDMR | Unione Democratica Magiari Romania (minoranza ungherese) |
| AUR | Alleanza per l'Unione dei Romeni (nazionalisti) |

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica |
| **Forma di Governo** | Semipresidenziale |
| **Legislativo** | Parlamento bicamerale (Senato + Camera Deputati) |
| **Giudiziario** | Corte Costituzionale, Inalta Curte |
| **Partiti dominanti** | PSD, PNL, USR, UDMR, AUR |

### Poteri del Presidente
- Politica estera e difesa
- Nomina PM (con fiducia parlamentare)
- Scioglimento Parlamento (condizioni limitate)
- Rappresentanza internazionale

### Poteri del Primo Ministro
- Guida governo e politica interna
- Proposta ministri
- Politica economica

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1989-1996** | Transizione | Ion Iliescu, ex comunisti al potere |
| **1996-2000** | Prima alternanza | Constantinescu, tentativi riforme |
| **2000-2004** | Ritorno PSD | Nastase, corruzione endemica |
| **2004-2014** | Era Basescu | Ingresso UE/NATO, crisi 2008-09 |
| **2014-2024** | Era Iohannis | Stabilita' pro-UE, proteste anti-corruzione |
| **2024-2025** | Crisi elettorale | Annullamento elezioni, interferenze russe |
| **2025-** | Nuova era | Dan presidente, coalizione pro-UE |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1989** | Rivoluzione, caduta Ceausescu |
| **2004** | Ingresso NATO |
| **2007** | Ingresso UE |
| **2012** | Tentativo impeachment Basescu (fallito) |
| **2015** | Tragedia Colectiv, proteste anti-corruzione |
| **2017** | Proteste contro depenalizzazione corruzione |
| **2024** | Annullamento elezioni presidenziali (interferenze russe) |
| **2025** | Nicusor Dan eletto presidente |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Posizione** | Frontiera orientale NATO/UE |
| **Mar Nero** | Strategico per contenimento Russia |
| **Energia** | Corridoio alternativo gas (bypass Russia) |
| **Ucraina** | Paese confinante, supporto NATO |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Tecnocrati** | PM tecnici in momenti di crisi (Isarescu, Ciolos) |
| **Corruzione** | Problema endemico, proteste ricorrenti |
| **Alternanza** | PSD vs blocco anti-PSD |
| **Interferenze** | Russia attiva (crisi 2024) |
