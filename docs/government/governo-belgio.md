# Regno del Belgio

> Monarchia federale al crocevia d'Europa, sede di NATO e UE, laboratorio di governance multilingue

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1831 |
| **Sede** | Bruxelles (Palazzo della Nazione, Palazzo Reale) |
| **Sistema** | Monarchia costituzionale federale parlamentare |
| **Capo di Stato** | [Re Philippe](../persons/philippe-belgium.md) (dal 2013) |
| **Capo di Governo** | [Bart De Wever](../persons/bart-de-wever.md) (dal 2025) |
| **PIL** | ~580 miliardi EUR (2024) |
| **Popolazione** | ~11.6 milioni |
| **Status** | Attivo |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Monarchia costituzionale** | Il Re regna ma non governa; tutti gli atti richiedono controfirma ministeriale |
| **Federalismo belga** | Divisione in 3 regioni (Fiandre, Vallonia, Bruxelles) e 3 comunita' linguistiche |
| **Parita' linguistica** | Costituzione richiede 7 ministri fiamminghi e 7 francofoni |
| **Formateur** | Incaricato dal Re di formare il governo dopo le elezioni |
| **Arizona** | Coalizione De Wever 2025 (N-VA, CD&V, Vooruit, MR, Les Engages) |

---

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Monarchia federale |
| **Forma di Governo** | Parlamentare |
| **Legislativo** | Parlamento bicamerale (Camera + Senato) |
| **Giudiziario** | Corte Costituzionale, Corte di Cassazione |
| **Partiti dominanti** | N-VA, CD&V, Open VLD, MR, PS, PTB-PVDA |

### Struttura Federale

```
                GOVERNO FEDERALE
                      |
     +----------------+----------------+
     |                |                |
3 REGIONI        3 COMUNITA'      MONARCHIA
- Fiandre        - Fiamminga      - Re Philippe
- Vallonia       - Francese       - Casa S-C-G
- Bruxelles      - Germanofona
```

| Livello | Competenze |
|---------|------------|
| **Federale** | Difesa, finanze, giustizia, sicurezza sociale |
| **Regioni** | Economia, ambiente, trasporti, urbanistica |
| **Comunita'** | Cultura, istruzione, sanita', politiche linguistiche |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1831-1914** | Fondazione | Indipendenza, neutralita', industrializzazione |
| **1914-1945** | Guerre mondiali | Occupazioni, questione reale |
| **1945-1970** | Dopoguerra | Ricostruzione, questione linguistica |
| **1970-1993** | Federalizzazione | 6 riforme costituzionali |
| **1993-oggi** | Belgio federale | Instabilita' cronica, ruolo UE/NATO |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1831** | Indipendenza dai Paesi Bassi |
| **1885** | Congo Free State (proprieta' personale Leopold II) |
| **1908** | Congo ceduto allo Stato belga |
| **1914-18** | Occupazione tedesca, Albert I "Re Soldato" |
| **1940-44** | Occupazione nazista, controversia Leopold III |
| **1950** | "Questione Reale", abdicazione Leopold III |
| **1960** | Indipendenza Congo |
| **1970** | Prima riforma federale |
| **1993** | Costituzione federale completa |
| **2010-11** | 541 giorni senza governo (record mondiale) |
| **2025** | Primo PM nazionalista fiammingo (De Wever) |
