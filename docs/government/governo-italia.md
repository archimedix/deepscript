## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1946-1992** | Prima Repubblica | Egemonia DC, arco costituzionale, Guerra Fredda |
| **1992-1994** | Tangentopoli | Crollo sistema partitico, governi tecnici Amato-Ciampi |
| **1994-2011** | Seconda Repubblica | Bipolarismo, alternanza, Berlusconi dominante |
| **2011-2021** | Era Tecnocratica | Crisi economiche, governi tecnici (Monti, Draghi) |
| **2022-** | Terza Repubblica? | Meloni, stabilita' inedita, destra al governo |

## Significato nel Sistema di Potere

### Pattern dei Governi Tecnici

```
CRISI → GOVERNO TECNICO → PROFILO FORUM

1993: Tangentopoli ──► Ciampi ──► Aspen
2011: Crisi spread ──► Monti ──► Trilaterale chair, Goldman
2021: COVID/PNRR ──► Draghi ──► BCE, Goldman, Bilderberg
```

| Pattern | Descrizione |
|---------|-------------|
| **Goldman Sachs → Palazzo Chigi** | Draghi, Monti, Prodi: dalla finanza USA al governo |
| **McKinsey → Ministero → Azienda** | Colao, Passera: manager globali prestati alla politica |
| **Min. Economia → Chairman Banca** | Padoan, Saccomanni: revolving door MEF-UniCredit |
| **DIS → Infrastrutture Critiche** | Massolo, Ciocca: da intelligence a presidenze strategiche |
| **Aspen/Trilaterale → PM** | 83% dei PM recenti sono membri di forum internazionali |
