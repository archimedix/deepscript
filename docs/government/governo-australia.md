# Government of Australia

> Monarchia costituzionale federale, pilastro dell'alleanza anglo-americana nel Pacifico, snodo Five Eyes

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1901 (Federation) |
| **Sede** | Canberra, Parliament House |
| **Sistema** | Monarchia costituzionale federale, parlamentare |
| **Capo di Stato** | Re Carlo III (rappresentato dal Governor-General) |
| **Capo di Governo** | Anthony Albanese (PM, 2022-) |
| **PIL** | ~$1.7 trilioni USD |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Federation** | Unione delle 6 colonie britanniche nel 1901 |
| **Governor-General** | Rappresentante del monarca, poteri formali |
| **Dismissal** | Crisi 1975: Governor-General licenzio' PM Whitlam |
| **Five Eyes** | Alleanza intelligence: AUS, USA, GBR, CAN, NZL |
| **AUKUS** | Patto trilaterale AUS-GBR-USA (2021), sottomarini nucleari |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| Five Eyes | Membro fondatore dell'alleanza intelligence anglofona |
| AUKUS | Patto trilaterale con USA-UK per sottomarini nucleari e tech sharing |
| Indo-Pacifico | Pilastro della strategia USA di contenimento della Cina |
| Quad | Membro con USA, India, Giappone |
| Risorse | Maggior esportatore mondiale di minerale di ferro, carbone, LNG |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| Alternanza Labor-Coalition | Sistema bipartitico stabile dal 1945 |
| Dismissal 1975 | Unica rimozione di un PM da parte del Governor-General |
| Leadership spills | 2010-2018: instabilita' interna con 5 cambi di PM in 8 anni |
| Alleanza USA | Costante dal 1942 (ANZUS), rafforzata con AUKUS |
| Mining lobby | Potere delle compagnie minerarie sulla politica fiscale e climatica |
