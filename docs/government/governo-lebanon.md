## Glossario

| Termine | Definizione |
|---------|-------------|
| **Confessionalismo** | Sistema di divisione del potere tra comunita' religiose |
| **Troika** | I tre ruoli apicali: Presidente, PM, Speaker |
| **Patto Nazionale** | Accordo non scritto 1943 sulla divisione confessionale |
| **Accordo di Ta'if** | Riforme 1989 che riequilibrarono potere cristiani-musulmani |
| **March 14** | Coalizione anti-Siria/Hezbollah (dal 2005) |
| **March 8** | Coalizione pro-Siria/Hezbollah |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1943-58** | Prima Repubblica | Patto Nazionale, dominio notabili |
| **1958** | Crisi | Intervento USA, tensioni pan-arabe |
| **1958-75** | Chehabismo | Modernizzazione, stabilita' |
| **1975-90** | Guerra Civile | 15 anni, 120,000 morti |
| **1990-2005** | Pax Syriana | Occupazione siriana, Ta'if |
| **2005-** | Post-Siria | Rivoluzione Cedri, polarizzazione |
| **2019-** | Crisi multipla | Economica, COVID, esplosione porto, guerra 2024 |

## Significato nel Sistema di Potere

### Il Modello Confessionale

| Aspetto | Dettaglio |
|---------|-----------|
| **Origine** | Patto Nazionale 1943 |
| **Logica** | Evitare dominazione di una comunita' |
| **Effetto** | Paralisi decisionale, veto reciproco |
| **Riforma Ta'if** | 1989: parita' cristiani-musulmani |
| **Crisi** | Sistema sempre piu' disfunzionale |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Dinastie politiche** | Gemayel, Frangieh, Hariri, Jumblatt controllano partiti |
| **Generali-presidenti** | 5 su 14 presidenti erano comandanti esercito |
| **Patronato estero** | Siria (1976-2005), Saudi, Iran, USA |
| **Veto confessionale** | Ogni comunita' puo' bloccare decisioni |
| **Milizie-partiti** | Hezbollah, Lebanese Forces, Marada: da milizie a partiti |

### Sfide 2025

| Sfida | Status |
|-------|--------|
| **Disarmo Hezbollah** | Obiettivo dichiarato governo Aoun/Salam |
| **Crisi economica** | Default 2020, svalutazione 98% |
| **Ricostruzione** | Post-guerra 2024, fondi sauditi promessi |
| **FATF grey list** | Riforme necessarie per uscire |
