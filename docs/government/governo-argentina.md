| Termine | Definizione |
|---------|-------------|

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica federale (23 province + CABA) |
| **Forma di Governo** | Presidenziale (mandato 4 anni, rielezione 1 volta) |
| **Legislativo** | Congreso Nacional: Senado (72, 6 anni) + Cámara de Diputados (257, 4 anni) |
| **Giudiziario** | Corte Suprema de Justicia de la Nación |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1853-1930** | Repubblica oligarchica | Dominio élite agraria, Generación del 80 |
| **1930-1946** | Década infame | Golpe militari, frode elettorale |
| **1946-1955** | Peronismo classico | Industrializzazione, welfare, sindacalismo |
| **1955-1976** | Instabilita' | Alternanza civili-militari, proscrizione peronismo |
| **1976-1983** | Proceso | Dittatura, Guerra Sucia, 30.000 desaparecidos |
| **1983-2001** | Democrazia restaurata | Alfonsín, privatizzazioni Menem, crisi 2001 |
| **2003-2015** | Kirchnerismo | Neoperonismo, statalismo, politiche sociali |
| **2015-2023** | Pendolo | Macri (centro-destra), A. Fernández (centro-sinistra) |
| **2023-oggi** | Era Milei | Libertarismo, austerita', dollarizzazione |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1853** | Costituzione federale |
| **1946** | Elezione Perón |
| **1955** | Revolución Libertadora, esilio Perón |
| **1976** | Golpe militare, inizio Proceso |
| **1982** | Guerra delle Malvinas |

| **2005** | Ristrutturazione debito (Kirchner) |
| **2018** | Prestito FMI $57 miliardi (Macri) |
| **2023** | Elezione Milei, svolta libertaria |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| America Latina | Seconda economia del continente, membro G20, pendolo tra allineamento USA e autonomia |
| FMI | Maggior debitore storico: $57B (Macri 2018), rinegoziazione continua |
| Risorse | Vaca Muerta (shale gas), cereali, litio — attira investimenti globali |
| Geopolitica Milei | Svolta filo-USA e filo-Israele, distacco da BRICS, avvicinamento a Trump |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| Pendolo peronismo/anti-peronismo | Ciclo ricorrente tra populismo redistributivo e liberismo pro-mercato |
| Oligarchia italo-argentina | Techint (Rocca), Socma (Macri), Bridas (Bulgheroni) — famiglie di origine italiana dominano industria |
| Revolving door | Caputo: JP Morgan/Deutsche Bank → Min. Finanzas → BCRA; Bausili: Deutsche Bank → Sec. Finance → BCRA |
| Privatizzazione ciclica | Menem (anni '90), Milei (2024-) — concessioni a gruppi privati connessi |
| Debito come strumento | Ogni ciclo neoliberale genera debito FMI che vincola i governi successivi |
