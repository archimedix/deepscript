# Repubblica Popolare Cinese

> Stato a partito unico dove il Partito Comunista Cinese controlla ogni aspetto del potere politico, economico e militare

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1949 |
| **Sede** | Beijing, Zhongnanhai |
| **Sistema** | Stato a partito unico (one-party state) |
| **Capo di Stato** | Xi Jinping (Presidente) |
| **Capo di Governo** | Li Qiang (Premier) |
| **PIL** | $17.7 trillion (2024) |
| **Popolazione** | 1.4 miliardi |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **CCP/CPC** | Chinese Communist Party - Partito Comunista Cinese |
| **PSC** | Politburo Standing Committee - i 7 leader supremi |
| **NPC** | National People's Congress - parlamento (rubber stamp) |
| **CPPCC** | Chinese People's Political Consultative Conference - organo consultivo |
| **CCDI** | Central Commission for Discipline Inspection - anti-corruzione |
| **CMC** | Central Military Commission - comando forze armate |
| **State Council** | Consiglio di Stato - gabinetto esecutivo |
| **Paramount Leader** | Leader supremo de facto (titolo informale) |

---

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica popolare socialista |
| **Forma di Governo** | Stato a partito unico |
| **Legislativo** | National People's Congress (unicamerale, ~3000 delegati) |
| **Giudiziario** | Supreme People's Court (sotto controllo CCP) |
| **Partito unico** | Partito Comunista Cinese (98+ milioni membri) |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1949-1976** | Era Mao | Rivoluzione, Grande Balzo, Rivoluzione Culturale |
| **1978-1989** | Riforme Deng | Apertura economica, "socialismo con caratteristiche cinesi" |
| **1989-2002** | Consolidamento | Post-Tiananmen, crescita economica, WTO (2001) |
| **2002-2012** | Ascesa globale | "Peaceful rise", Olimpiadi 2008 |
| **2012-oggi** | Era Xi | Centralizzazione, Belt & Road, "China Dream" |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1949** | Fondazione PRC, Mao proclama Repubblica |
| **1958-62** | Grande Balzo in Avanti (carestia, 30-45M morti) |
| **1966-76** | Rivoluzione Culturale |
| **1976** | Morte Mao |
| **1978** | Deng avvia riforme economiche |
| **1989** | Massacro Tiananmen |
| **1997** | Morte Deng, ritorno Hong Kong |
| **2001** | Ingresso WTO |
| **2008** | Olimpiadi Beijing |
| **2012** | Xi Jinping diventa General Secretary |
| **2013** | Lancio Belt and Road Initiative |
| **2017** | "Xi Jinping Thought" nella costituzione |
| **2018** | Abolizione limiti mandato presidenziale |
| **2022** | Xi ottiene terzo mandato (senza precedenti) |
| **2020-23** | Zero-COVID policy |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Economia** | Seconda economia mondiale, "fabbrica del mondo" |
| **Militare** | PLA piu' grande esercito, nucleare |
| **Tecnologia** | Leader 5G, AI, semiconduttori (sfida USA) |
| **Finanza** | $3+ trillion riserve forex, creditori globale |
| **Soft power** | Confucius Institutes, Belt and Road |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Centralizzazione Xi** | Piu' potere concentrato dai tempi di Mao |
| **Partito sopra tutto** | CCP controlla stato, economia, societa' |
| **Techno-autoritarismo** | Sorveglianza di massa, social credit |
| **Capitalismo di stato** | SOE + "golden shares" in tech private |
| **Diplomazia del debito** | BRI come strumento geopolitico |
| **Wolf warrior diplomacy** | Diplomazia assertiva/aggressiva |

### Strumenti di Controllo

| Strumento | Funzione |
|-----------|----------|
| CCDI | Anti-corruzione (purga rivali) |
| United Front | Influenza diaspora e stranieri |
| Great Firewall | Censura internet |
| Social Credit | Controllo comportamento cittadini |
| Golden Shares | Controllo aziende tech private |
