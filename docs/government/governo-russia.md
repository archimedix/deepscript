# Governo della Federazione Russa

> Sistema semi-presidenziale dominato dal potere verticale di Putin, fusione di siloviki e oligarchi di stato

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1991 (dissoluzione URSS) |
| **Sede** | Mosca, Cremlino |
| **Sistema** | Repubblica federale semi-presidenziale |
| **Capo di Stato** | Vladimir Putin (Presidente) |
| **Capo di Governo** | Mikhail Mishustin (Primo Ministro) |
| **PIL** | ~$2.2 trilioni (2024) |
| **Status** | active |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Siloviki** | Funzionari con background nei servizi di sicurezza (FSB, KGB, GRU) |
| **Tandemocracy** | Periodo 2008-2012: Medvedev presidente, Putin primo ministro |
| **Vertical of Power** | Centralizzazione del potere sotto Putin |
| **Oligarchi di Stato** | Imprenditori il cui potere deriva dal controllo di aziende statali |

---

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica federale |
| **Forma di Governo** | Semi-presidenziale (de facto presidenziale) |
| **Legislativo** | Assemblea Federale (Duma di Stato + Consiglio Federazione) |
| **Giudiziario** | Corte Costituzionale, Corte Suprema |
| **Partito dominante** | Russia Unita (Единая Россия) |

### Struttura Esecutiva

```
                    PRESIDENTE
```

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1991-1999** | Era Yeltsin | Privatizzazioni selvagge, oligarchi, caos |
| **2000-2008** | Consolidamento Putin | Distruzione Yukos, verticale del potere |
| **2008-2012** | Tandemocracy | Medvedev presidente, Putin PM |
| **2012-2014** | Ritorno Putin | Proteste, giro di vite |
| **2014-2022** | Confronto Occidente | Crimea, sanzioni, isolamento |
| **2022-oggi** | Guerra Ucraina | Economia di guerra, mobilitazione |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1991** | Dissoluzione URSS, Yeltsin presidente |
| **1993** | Crisi costituzionale, bombardamento Parlamento |
| **1996** | Rielezione Yeltsin (supporto oligarchi) |
| **1998** | Default finanziario |
| **1999** | Putin nominato PM, poi presidente ad interim |
| **2003** | Arresto Khodorkovsky, distruzione Yukos |
| **2008** | Guerra Georgia, Medvedev presidente |
| **2011-12** | Proteste anti-Putin |
| **2014** | Annessione Crimea, sanzioni occidentali |
| **2020** | Riforma costituzionale (reset mandati Putin) |
| **2022** | Invasione Ucraina |
| **2024** | Putin rieletto per 5° mandato |

## Significato nel Sistema di Potere

### Struttura del Potere Reale

| Livello | Composizione |
|---------|--------------|
| **Cerchia intima** | Putin + 5-10 siloviki fidati |
| **Oligarchi di stato** | CEO Rosneft, Gazprom, banche |
| **Technocrati** | Mishustin, ministri economici |
| **Sicurezza** | FSB, SVR, GRU, Security Council |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Verticale del potere** | Tutti i flussi decisionali passano da Putin |
| **Sistema di clan** | Fazioni competono per favore presidenziale |
| **Oligarchi dipendenti** | Ricchezza subordinata a lealtà politica |
| **Rotazione controllata** | Nessun successore chiaro, instabilità programmata |

### Fazioni Principali

| Fazione | Leader | Base di potere |
|---------|--------|----------------|
| **Siloviki** | Patrushev, Bortnikov | FSB, sicurezza |
| **Rosneft** | Sechin | Energia, petrolio |
| **Gazprom** | Miller | Gas naturale |
| **Technocrati** | Mishustin | Amministrazione |
