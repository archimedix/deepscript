## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica |
| **Forma di Governo** | Presidenziale (dal 2018) |
| **Legislativo** | TBMM - 600 membri, unicamerale |
| **Giudiziario** | Corte Costituzionale, Consiglio di Stato |
| **Partito dominante** | AKP (dal 2002) |

### Transizione 2017-2018

| Anno | Evento |
|------|--------|
| **Apr 2017** | Referendum costituzionale (51.4% si) |
| **Giu 2018** | Elezioni presidenziali, abolizione PM |
| **2018** | Ultimo PM: Binali Yildirim |

Il sistema presidenziale ha concentrato il potere esecutivo nel Presidente, eliminando la figura del Primo Ministro e conferendo poteri di nomina diretta del gabinetto.

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1923-1938** | Fondazione Kemalista | Secolarismo, occidentalizzazione, partito unico |
| **1938-1950** | Inonu | Neutralita' WWII, transizione |
| **1950-1960** | Democrazia multipartitica | Bayar/Menderes, tensioni laici-religiosi |
| **1960-1980** | Instabilita' | 3 colpi di stato (1960, 1971, 1980) |
| **1980-2002** | Tutela militare | Kenan Evren, Ozal, coalizioni fragili |
| **2002-2018** | Era AKP | Erdogan PM, riforme UE, poi svolta autoritaria |
| **2018-oggi** | Presidenzialismo | Potere concentrato, polarizzazione |

### Colpi di Stato

| Anno | Tipo | Esito |
|------|------|-------|
| **1960** | Militare | Costituzione 1961 |
| **1971** | Memorandum | Governo tecnico |
| **1980** | Militare | Kenan Evren presidente |
| **1997** | "Postmoderno" | Caduta governo islamista Erbakan |
| **2016** | Tentato | Fallito, purghe massive |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Posizione** | Ponte Europa-Asia, controllo Bosforo |
| **NATO** | Secondo esercito per dimensione |
| **Energia** | Hub gas (TurkStream, TANAP) |
| **Migranti** | Cuscinetto UE (accordo 2016) |
| **Difesa** | Industria autonoma (droni Bayraktar) |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Stato profondo** | Tensione storica militari-civili |
| **Islam politico** | AKP ha rotto monopolio kemalista |
| **Oligarchi** | Grandi famiglie (Koc, Sabanci) bilanciano potere politico |
| **Presidenzialismo** | Concentrazione potere post-2018 |
| **Politica estera attiva** | Libia, Siria, Caucaso, Africa |

### Tensioni Strutturali

| Tensione | Manifestazione |
|----------|----------------|
| **Laici vs Islamisti** | Storica, ora sopita |
| **Governo vs Business** | TUSIAD vs politiche economiche |
| **Centralismo vs Curdi** | Conflitto irrisolto |
| **Occidente vs Russia** | Equilibrismo S-400/NATO |
