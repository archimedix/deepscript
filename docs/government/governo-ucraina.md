# Governo dell'Ucraina

> Repubblica semi-presidenziale post-sovietica, in guerra dal 2022

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1991 (indipendenza dall'URSS) |
| **Sede** | Kyiv |
| **Sistema** | Repubblica semi-presidenziale (parlamentare-presidenziale dal 2014) |
| **Capo di Stato** | Volodymyr Zelensky (Presidente, dal 2019) |
| **Capo di Governo** | Yulia Svyrydenko (Prima Ministra, dal 2025) |
| **PIL** | $190.7 mld (2024) |
| **Popolazione** | ~37.9M (2024, da ~44M pre-invasione) |
| **Spesa difesa** | 25% PIL (2025) |
| **Status** | active |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1991-94** | Fondazione | Indipendenza, rinuncia al nucleare, iperinflazione |
| **1994-2004** | Era Kuchma | Privatizzazioni selvagge, nascita sistema oligarchico, Cassette Scandal |
| **2004-05** | Rivoluzione Arancione | Proteste di massa contro frode elettorale, svolta occidentale |
| **2005-10** | Instabilita' | Conflitto Yushchenko-Tymoshenko, paralisi istituzionale |
| **2010-14** | Svolta filo-russa | Yanukovych, arresto Tymoshenko, rifiuto accordo UE |
| **2013-14** | [Euromaidan](../events/2014-euromaidan.md) | Rivoluzione, fuga Yanukovych, annessione Crimea, guerra Donbas |
| **2014-19** | Era Poroshenko | Riforme pro-UE, anti-corruzione, decentralizzazione |
| **2019-22** | Era Zelensky pre-guerra | De-oligarchizzazione, attore-presidente, cerchia Kvartal 95 |
| **2022-oggi** | Guerra totale | Invasione russa, legge marziale, accentramento potere, nazionalizzazioni |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| Posizione | Stato cuscinetto tra NATO/UE e Russia |
| Energia | Transito gas russo verso Europa (gasdotti), grano export mondiale |
| Nucleare | Ex terzo arsenale nucleare al mondo, ceduto 1994 |
| Guerra | Teatro del piu' grande conflitto europeo dal 1945 |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Sistema oligarchico** | Dall'indipendenza, oligarchi controllano media, industria, parlamento |
| **Revolving door** | Magnati diventano deputati, governatori, ministri (Poroshenko, Akhmetov, Zhevago) |
| **Cerchia presidenziale** | Ogni presidente porta la sua "famiglia" al potere (Kuchma-Pinchuk, Zelensky-Kvartal) |
| **Patronage estero** | Costante tensione tra influenza russa e occidentale |
| **De-oligarchizzazione** | Dal 2021 legge anti-oligarchi, nazionalizzazioni wartime 2022, arresti e sanzioni |
| **Accentramento wartime** | Legge marziale concentra potere nell'Ufficio Presidenziale (Yermak 2020-25) |
| **Partiti-veicolo** | SotP e ES sono entrambi partiti personali, non movimenti; la Rada e' braccio operativo del presidente |
| **Media → politica** | Kvartal 95 → SotP → Stato; 1+1 (Kolomoisky) ha lanciato Zelensky |
