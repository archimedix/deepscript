| Metrica | Valore |

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Bundeskanzler** | Cancelliere federale, capo del governo |
| **Bundespraesident** | Presidente federale, capo di stato (ruolo cerimoniale) |
| **Bundestag** | Camera bassa del parlamento (736 seggi) |
| **Bundesrat** | Camera alta, rappresentanti dei Laender |
| **Laender** | 16 stati federali |
| **Mitbestimmung** | Codeterminazione: dipendenti nei board aziendali |

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica federale |
| **Forma di Governo** | Parlamentare (Cancelliere eletto dal Bundestag) |
| **Legislativo** | Bundestag (camera bassa) + Bundesrat (camera alta) |
| **Giudiziario** | Bundesverfassungsgericht (Corte Costituzionale, Karlsruhe) |
| **Partiti dominanti** | CDU/CSU, SPD, Gruene, FDP, AfD |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Motore UE** | Piu' grande economia, 29% bilancio UE |
| **NATO** | 2a spesa militare NATO (dopo USA) post-Zeitenwende |
| **Ponte Est-Ovest** | Ostpolitik, dipendenza energia russa (fino 2022) |
| **Exportnation** | Modello export-led, surplus commerciale cronico |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Capitalismo renano** | Banche + industria + sindacati, stakeholder capitalism |
| **Mitbestimmung** | Dipendenti 50% supervisory board (aziende >2000 dip.) |
| **Federalismo** | Laender con poteri reali (VW Law: Lower Saxony) |
| **Atlantismo** | Atlantik-Brucke, MSC, NATO come pilastri |
| **Ordoliberalismo** | Bundesbank, austerita', regole fiscali |

### Crisi e Transizioni

| Anno | Crisi | Risposta |
|------|-------|----------|
| **1990** | Riunificazione | Kohl, costo EUR 2T |
| **2003** | "Malato d'Europa" | Agenda 2010, Hartz |
| **2008** | Crisi finanziaria | Salvataggio banche, austerita' UE |
| **2015** | Crisi migranti | Merkel "Wir schaffen das" |
| **2022** | Guerra Ucraina | Zeitenwende, fine gas russo |
