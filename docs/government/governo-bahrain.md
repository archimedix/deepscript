# Governo del Regno del Bahrain

> Monarchia costituzionale dove una famiglia (Al Khalifa) governa da 242 anni, ospita la Fifth Fleet USA, e ha normalizzato con Israele: il potere concentrato in un'isola strategica.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1783 (Al Khalifa), 1971 (indipendenza), 2002 (regno) |
| **Sede** | Riffa Palace, Manama |
| **Sistema** | Monarchia costituzionale |
| **Capo di Stato** | Re Hamad bin Isa Al Khalifa (1999-) |
| **Capo di Governo** | PM Salman bin Hamad Al Khalifa (2020-) |
| **PIL** | $44 miliardi (2024) |
| **Popolazione** | 1.5 milioni |
| **Status** | Active |

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Monarchia |
| **Forma di Governo** | Costituzionale (dal 2002), precedentemente assoluta |
| **Legislativo** | National Assembly bicamerale |
| **Shura Council** | 40 membri nominati dal Re |
| **Council of Representatives** | 40 membri eletti (ultime elezioni Nov 2022) |
| **Giudiziario** | Sistema misto civile/sharia |
| **Partiti** | Vietati; societa' politiche ammesse |

### Poteri del Re (Art. 33 Costituzione 2002)

- Nomina tutti i ministri
- Nomina i 40 membri Shura Council
- Puo' modificare la Costituzione
- Puo' sciogliere il parlamento
- Comanda le forze armate

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1783-1861** | Conquista | Al Khalifa prendono Bahrain dai Persiani |
| **1861-1971** | Protettorato UK | Trattati esclusivi con Gran Bretagna |
| **1971-1975** | Indipendenza democratica | Breve esperimento parlamentare |
| **1975-2002** | Monarchia assoluta | Parlamento sciolto, legge sicurezza |
| **2002-2011** | Riforma costituzionale | Nuova costituzione, bicameralismo |
| **2011-oggi** | Post-Primavera | Repressione, alleanza saudita rafforzata |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1783** | Ahmed Al-Fateh conquista Bahrain |
| **1861** | Primo trattato con UK |
| **1932** | Scoperta petrolio (primo nel Golfo) |
| **1948** | Arrivo US Navy |
| **1971** | Indipendenza da UK |
| **1973** | Prima costituzione, parlamento eletto |
| **1975** | Scioglimento parlamento |
| **1981** | Fondazione GCC |
| **1999** | Morte Isa, Hamad diventa Emir |
| **2002** | Nuova costituzione, Bahrain diventa Regno |
| **2011** | Primavera Araba repressa con truppe GCC |
| **2020** | Abraham Accords con Israele |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Posizione** | Stretto di Hormuz, 20% petrolio mondiale transita |
| **Fifth Fleet** | Unica base navale USA permanente nel Golfo |
| **Hub finanziario** | 103 banche, $160B+ asset, alternativa a Beirut post-1975 |
| **Normalizzazione** | Secondo paese arabo Abraham Accords (dopo UAE) |
| **Contenimento Iran** | Maggioranza shia ma governo sunnita pro-Saudi |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Micro-stato dipendente** | Sopravvivenza legata a USA/Saudi/UK |
| **Monarchia familiare** | 50% ministri storicamente Al Khalifa |
| **Hub strategico** | Compensa mancanza risorse con posizione |
| **Repressione selettiva** | Maggioranza shia esclusa dal potere |
| **Normalizzazione anticipata** | Segue UAE in 30 giorni su Israele |
