## Pattern di Potere

### Revolving Door Tech-Governo

```
GOJEK (fondatore Nadiem)
        │
        │ 2019
        v
GOVERNO INDONESIA
(Ministro Istruzione)
```

**Pattern**: Fondatori tech indonesiani entrano in politica. Nadiem Makarim e' l'esempio piu' chiaro.

### Cukong System (Era Suharto)

```
SUHARTO (potere politico)
        │
        │ licenze, protezione
        v
FAMIGLIE CINESI (Salim, Hartono, etc.)
        │
        │ finanziamenti, quote
        v
SUHARTO FAMILY + CRONIES
```

Il sistema *cukong* collegava businessmen sino-indonesiani al potere politico attraverso scambi di favori.

### Trasferimento di Ricchezza 1998

```
CRISI 1997-98
     │
     v
IBRA (nazionalizza)
     │
     v
SALIM perde BCA ────► HARTONO acquisisce BCA
```

La crisi del 1998 redistribui' asset dai vecchi *cukong* (Salim) a nuovi oligarchi (Hartono).

### Tecnocrati Internazionali

| Persona | Percorso |
|---------|----------|
| Sri Mulyani | Indonesia → FMI → World Bank → Indonesia |

Pattern classico: formazione internazionale, ruoli in istituzioni globali, ritorno come ministro "tecnico".
