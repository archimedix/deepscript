# Governo del Liechtenstein

> Monarchia costituzionale unica in Europa: il Principe detiene poteri reali (veto, scioglimento parlamento), micro-stato hub finanziario globale con PIL pro capite tra i piu' alti al mondo.

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Monarchia costituzionale (Costituzione 1921, riformata 2003) |
| **Forma di Governo** | Parlamentare con poteri monarchici reali |
| **Legislativo** | Landtag, 25 seggi, proporzionale, 2 circoscrizioni (Oberland/Unterland) |
| **Giudiziario** | Corte Suprema (Oberster Gerichtshof), giudici nominati dal Principe |
| **Partiti dominanti** | VU e FBP (sistema bipartitico de facto dal 1928) |
| **Democrazia diretta** | Referendum e iniziativa popolare; anche il popolo puo' chiedere abolizione della monarchia |
| **Peculiarita'** | Il Principe puo' bloccare leggi, sciogliere il Landtag, nominare giudici, dichiarare emergenza; immune da giurisdizione |

### Poteri del Principe (post-2003)

| Potere | Descrizione |
|--------|-------------|
| **Veto legislativo** | Puo' bloccare qualsiasi legge, senza possibilita' di override |
| **Scioglimento Landtag** | Puo' sciogliere il parlamento unilateralmente |
| **Nomina giudici** | Nomina tutti i giudici delle corti superiori |
| **Nomina PM** | Il PM deve avere la fiducia del Principe E del Landtag |
| **Stato di emergenza** | Poteri eccezionali senza limiti temporali definiti |
| **Sanzione leggi** | Ogni legge richiede la firma del Principe |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1699-1806** | Principato nel SRI | Acquisto territori, status imperiale |
| **1806-1866** | Sovranita' nella Confederazione del Reno/Germanica | Indipendenza formale |
| **1866-1918** | Alleanza con Austria-Ungheria | Unione doganale, protezione militare |
| **1918-1938** | Riorientamento verso la Svizzera | Fine legame con Austria, unione doganale con Svizzera (1923) |
| **1938-1984** | Modernizzazione | Franz Josef II primo principe residente, industrializzazione |
| **1984-2004** | Era Hans-Adam II | Riorganizzazione finanziaria, adesione ONU/EEA/WTO |
| **2004-oggi** | Reggenza Alois | Hub fintech (Blockchain Act 2020), diversificazione economica |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1719** | Carlo VI eleva Vaduz e Schellenberg a Principato |
| **1806** | Sovranita' piena dopo dissoluzione SRI |
| **1862** | Prima Costituzione |
| **1921** | Nuova Costituzione, forma attuale di governo |
| **1923** | Unione doganale e monetaria con la Svizzera (CHF) |
| **1938** | Franz Josef II: primo principe a risiedere in Liechtenstein |
| **1945** | Decreti Benes: confisca proprieta' Liechtenstein in Cecoslovacchia |
| **1984** | Suffragio femminile (ultimo paese europeo!) |
| **1990** | Adesione alle Nazioni Unite |
| **1991** | Adesione all'EFTA |
| **1995** | Adesione all'EEA (Spazio Economico Europeo) |
| **2003** | [Referendum costituzionale](../events/2003-liechtenstein-referendum.md): poteri ampliati al Principe (64.3%) |
| **2004** | Alois diventa reggente |
| **2008** | [Liechtenstein Tax Affair](../events/2008-liechtenstein-tax-affair.md): scandalo evasione fiscale |
| **2009** | Liechtenstein accetta standard OCSE di trasparenza fiscale |
| **2020** | Blockchain Act (TVTG): prima legislazione organica su blockchain/token |
| **2025** | Brigitte Haas, prima donna Primo Ministro |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Hub finanziario** | Terza piazza finanziaria tra i micro-stati europei, dopo Svizzera e Lussemburgo |
| **Neutralita'** | Non membro NATO ne' UE, ma nell'EEA e nello spazio Schengen |
| **Unione con Svizzera** | Moneta CHF, unione doganale, rappresentanza diplomatica svizzera in molti paesi |
| **Blockchain hub** | Blockchain Act 2020: prima legislazione organica su tokenizzazione |
| **PIL pro capite** | ~$232K, tra i piu' alti al mondo |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Sovranita'-Finanza** | Unico Stato europeo dove la famiglia regnante possiede la piu' grande banca del paese |
| **Micro-stato hub** | Dimensione ridotta = agilita' regolatoria, attrazione capitali |
| **Bipartitismo stabile** | VU e FBP alternano governo dal 1928, bassissima conflittualita' politica |
| **Democrazia + Monarchia forte** | Caso unico di poteri monarchici reali in una democrazia europea |
| **Libertarismo di stato** | Tasse basse, regolazione leggera, privacy finanziaria — modello teorizzato da Hans-Adam II |
| **Diplomazia di nicchia** | Ambasciatori principeschi (Nikolaus alla Santa Sede), soft power culturale (collezione d'arte) |
