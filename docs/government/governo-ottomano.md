# Impero Ottomano

> Impero multietnico 1299-1922, dalla Sublime Porta ai Giovani Turchi

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1299 |
| **Dissoluzione** | 1922 |
| **Sede** | Istanbul (Costantinopoli) |
| **Sistema** | Monarchia assoluta, poi monarchia costituzionale (1876, 1908) |
| **Ultimo Sultano** | Mehmed VI (1918-1922) |
| **Ultimo Gran Visir** | Ahmed Tevfik Pasha (1918-1922) |
| **Status** | defunct |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Sublime Porta** (Bab-i Ali) | Sede del Gran Visir e del governo, metonimia per il governo ottomano |
| **Gran Visir** | Capo del governo, primo ministro de facto |
| **Sultano** | Monarca assoluto, capo di Stato e Califfo (dal 1517) |
| **Vilayet** | Provincia dell'impero |
| **CUP** | Comitato Unione e Progresso, partito dei Giovani Turchi |
| **Tre Pascia** | Triumvirato CUP: Talaat, Enver, Cemal (1913-1918) |
| **Tanzimat** | Riforme di modernizzazione (1839-1876) |
| **Donme** | Cripto-ebrei sabbatiani convertiti all'Islam, influenti a Salonicco |

---

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Monarchia imperiale (Sultanato-Califfato) |
| **Forma di Governo** | Assoluta → Costituzionale (1876, sospesa; ripristinata 1908) |
| **Legislativo** | Parlamento ottomano (Meclis-i Mebusan) dal 1876 |
| **Giudiziario** | Tribunali della Sharia + tribunali civili (Nizamiye) |
| **Partiti dominanti** | CUP (1908-1918), Liberta' e Intesa (opposizione) |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1299-1453** | Espansione | Conquista dei Balcani, caduta di Costantinopoli |
| **1453-1566** | Apogeo | Solimano il Magnifico, massima estensione |
| **1566-1789** | Stagnazione | Declino militare, riforme sporadiche |
| **1789-1876** | Riforme | Tanzimat, modernizzazione, influenza europea |
| **1876-1908** | Autocrazia hamidiana | Abdul Hamid II, Costituzione sospesa |
| **1908-1913** | Rivoluzione | Giovani Turchi, restaurazione costituzionale |
| **1913-1918** | Dittatura CUP | Tre Pascia, WWI, genocidi |
| **1918-1922** | Dissoluzione | Occupazione, guerra d'indipendenza, abolizione sultanato |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1839** | Editto di Gulhane, inizio Tanzimat |
| **1876** | Prima Costituzione (Kanun-i Esasi) |
| **1889** | Fondazione societa' segreta proto-CUP |
| **1908** | Rivoluzione dei Giovani Turchi, ripristino Costituzione |
| **1909** | Deposizione Abdul Hamid II |
| **1913** | Colpo di Stato CUP (Raid sulla Sublime Porta) |
| **1914** | Entrata in WWI a fianco degli Imperi Centrali |
| **1915** | Inizio genocidio armeno |
| **1918** | Armistizio di Mudros, fuga dei Tre Pascia |
| **1920** | Trattato di Sevres, spartizione dell'impero |
| **1922** | Abolizione del Sultanato |

## Significato nel Sistema di Potere

### Salonicco come laboratorio

| Aspetto | Dettaglio |
|---------|-----------|
| **Convergenza** | Massoneria italiana, Donme sabbatiani, CUP si sovrappongono a Salonicco |
| **Rivoluzione 1908** | Organizzata nelle logge, non a Istanbul |
| **Modello** | Rete segreta che conquista lo Stato dall'interno |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Societa' segrete → Stato** | CUP nasce come societa' segreta, diventa partito-stato |
| **Minoranze nell'elite** | Donme, armeni, greci nelle finanze e nel commercio |
| **Triumvirato** | Tre Pascia come modello di potere oligarchico |
| **Genocidio come politica** | Ingegneria demografica come strumento statale |
