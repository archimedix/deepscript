## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Granducato (monarchia costituzionale) |

| **Partiti dominanti** | CSV, DP, LSAP |
| **Mandato parlamentare** | 5 anni |
| **Circoscrizioni** | 4 (proporzionale) |

### Ruolo del Granduca

| Aspetto | Dettaglio |
|---------|-----------|
| **Legislativo** | Promulga le leggi (dal 2008: automatico, non piu' assenso) |
| **Esecutivo** | Esercita con il Governo |
| **Internazionale** | Rappresenta il Granducato |
| **Neutralita'** | Deve rimanere politicamente neutrale |
| **Giuramento** | Davanti alla Chamber of Deputies |

### Il Pattern PM → Commissione

Il Lussemburgo (0.1% popolazione UE) ha prodotto 3 Presidenti della Commissione su 13 (23%):

| PM | Commissione | Fattori |
|----|-------------|---------|
| Gaston Thorn | 1981-1985 | Neutralita', multilinguismo |
| Jacques Santer | 1995-1999 | Network costruito in 11 anni PM |
| Jean-Claude Juncker | 2014-2019 | 18 anni PM, Eurogroup |

### Il Modello Lussemburghese

```
PICCOLO STATO + STABILITA' + HUB FINANZIARIO = LEVERAGE SPROPORZIONATO
       |              |              |
       v              v              v
   Neutralita'    Governi      Dipendenza
   percepita      longevi       UE/globale
       |              |              |
       +------+-------+------+------+
                      |

          3 PRESIDENTI COMMISSIONE UE
          Istituzioni UE a Kirchberg
          EUR 5.9T AUM in fondi
```

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **PM → Commissione** | 3 su 13 Presidenti vengono dal Lussemburgo |
| **Longevita'** | Juncker PM 18 anni, Werner Min. Fin. 26 anni |
| **Revolving door** | Frieden: Governo → BIL/Deutsche Bank → Governo |
| **Multilinguismo** | FR, DE, EN, LU tutti usati |
| **Neutralita' tattica** | Mediatori tra grandi potenze |
