# Governo di Singapore

> Stato-azienda: 65 anni di dominio monopartitico, controllo totale via SWF e GLCs

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1959 (autogoverno), 1965 (indipendenza) |
| **Sede** | The Istana, Singapore |
| **Sistema** | Repubblica parlamentare |
| **Capo di Stato** | Tharman Shanmugaratnam (Presidente, 2023-) |
| **Capo di Governo** | Lawrence Wong (PM, 2024-) |
| **PIL** | ~$400B USD |
| **Status** | active |

## Glossario

| Termine | Definizione |
|---------|-------------|
| **PAP** | People's Action Party - partito dominante dal 1959 |
| **GLC** | Government-Linked Company - azienda controllata dallo stato |
| **GIC** | Government Investment Corporation - SWF per riserve estere |
| **Temasek** | SWF per investimenti strategici |
| **4G** | Fourth Generation - attuale generazione di leader PAP |

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica |
| **Forma di Governo** | Parlamentare (Westminster modificato) |
| **Legislativo** | Parlamento unicamerale (93 seggi) |
| **Giudiziario** | Corte Suprema, basato su common law |
| **Partiti dominanti** | PAP (unico al potere dal 1959) |

### Caratteristiche Uniche

| Aspetto | Dettaglio |
|---------|-----------|
| **Democrazia dominata** | PAP mai perso elezioni generali |
| **Controllo media** | Stampa controllata dallo stato |
| **ISA** | Internal Security Act - detenzione senza processo |
| **Meritocrazia** | Sistema di selezione elite via PSC scholarships |

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1959-1965** | Autogoverno | Merger con Malesia, poi espulsione |
| **1965-1990** | Era LKY | Costruzione stato, industrializzazione |
| **1990-2004** | Transizione | Goh Chok Tong, stile piu' aperto |
| **2004-2024** | Dinastia Lee | LHL PM, Ho Ching Temasek |
| **2024-oggi** | Post-Lee | Lawrence Wong, ma LHL resta Chairman GIC |

### Pattern di Potere

| Pattern | Descrizione |
