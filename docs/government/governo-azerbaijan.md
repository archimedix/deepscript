# Governo dell'Azerbaijan

> Regime autoritario dinastico dominato dalla famiglia Aliyev dal 1969

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1991 (indipendenza da URSS) |
| **Sede** | Baku |
| **Sistema** | Repubblica semipresidenziale (de jure), autoritaria (de facto) |
| **Capo di Stato** | Ilham Aliyev, Presidente (2003-) |
| **Capo di Governo** | Ali Asadov, Primo Ministro (2019-) |
| **PIL** | ~$80 miliardi (2024) |
| **Status** | active |

---

## Sistema Politico

| Aspetto | Dettaglio |
|---------|-----------|
| **Forma di Stato** | Repubblica |
| **Forma di Governo** | Semipresidenziale (costituzionalmente), presidenziale-autoritaria (de facto) |
| **Legislativo** | Milli Majlis (Parlamento unicamerale, 125 seggi) |
| **Limiti mandato** | Aboliti con referendum 2009 |
| **Prima Vice Presidente** | Mehriban Aliyeva (moglie del Presidente, carica creata nel 2017) |

### Concentrazione del potere

Il potere e' concentrato nelle mani del Presidente Ilham Aliyev e della sua famiglia allargata. La carica di Prima Vice Presidente e' stata creata nel 2017 specificamente per Mehriban Aliyeva.

## Storia del Potere

### Fasi Storiche

| Periodo | Fase | Caratteristiche |
|---------|------|-----------------|
| **1918-1920** | Prima Repubblica | Breve indipendenza |
| **1920-1991** | Era Sovietica | RSS Azerbaigiana |
| **1969-1982** | Heydar Aliyev I | Primo Segretario PC Azerbaijan |
| **1991-1993** | Transizione | Instabilita', guerra Nagorno-Karabakh |
| **1993-2003** | Heydar Aliyev II | Consolidamento autoritario |
| **2003-** | Ilham Aliyev | Successione dinastica, petrolio boom |

### Eventi Chiave

| Anno | Evento |
|------|--------|
| **1991** | Indipendenza dall'URSS |
| **1993** | Heydar Aliyev torna al potere |
| **1994** | "Contratto del Secolo" con BP e oil majors |
| **2003** | Successione Heydar → Ilham |
| **2009** | Referendum abolisce limiti mandato presidenziale |
| **2017** | Mehriban Aliyeva nominata Prima Vice Presidente |
| **2020** | Guerra Nagorno-Karabakh, vittoria militare |
| **2023** | Operazione militare, ripresa completa Nagorno-Karabakh |
| **2024** | Elezioni presidenziali (Feb), Aliyev rieletto |

## Significato nel Sistema di Potere

### Ruolo Geopolitico

| Aspetto | Dettaglio |
|---------|-----------|
| **Energia** | Snodo chiave corridoio Sud del gas verso Europa (TAP/TANAP) |
| **Posizione** | Ponte tra Russia, Turchia, Iran, Asia Centrale |
| **Alleanze** | Partnership strategica con Turchia, relazioni complesse con Russia |
| **Investimenti** | SOFAZ investe globalmente (Italia, UK, real estate, infrastrutture) |

### Pattern di Potere

| Pattern | Descrizione |
|---------|-------------|
| **Successione dinastica** | Padre-figlio (Heydar→Ilham), moglie First VP |
| **Stato petrolifero** | Economia dipendente da oil & gas, redistribuzione selettiva |
| **Autoritarismo soft** | Facciata democratica, controllo reale concentrato |
| **Diplomazia energetica** | Leverage su Europa per forniture gas |
