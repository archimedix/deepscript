## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1996, Stamford, Connecticut |
| **Fondatore** | [Jay S. Walker](../persons/jay-walker.md) |
| **Modello originale** | "Name Your Own Price" — aste inverse per viaggi |
| **IPO** | 1999, primo giorno chiuse a $69 (offerta $16) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1996** | Jay Walker fonda Priceline.com |
| **1998** | Lancio del sito, modello Name Your Own Price |
| **1999** | IPO su NASDAQ, valutazione $9.8B al primo giorno |
| **2000** | Scoppio bolla dot-com, Walker lascia l'azienda |
| **2005** | Acquisizione di Booking.com (Paesi Bassi) — svolta strategica |
| **2007** | Acquisizione di Agoda.com (mercato asiatico) |
| **2010** | Acquisizione di TravelJigsaw (poi RentalCars.com) |
| **2013** | Acquisizione di KAYAK ($1.8B) |
| **2014** | Rinominata da priceline.com Inc. a The Priceline Group; acquisizione OpenTable ($2.6B) |
| **2017** | Glenn Fogel diventa CEO & President |
| **2018** | Rinominata Booking Holdings, ticker BKNG |
| **2021** | Acquisizione di Getaroom (~$1.2B) |
