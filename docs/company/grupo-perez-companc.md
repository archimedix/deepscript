## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1946 (come Petrolera Pérez Companc) |
| **Fondatore** | Famiglia Companc de Pérez Acuña, poi [Gregorio Pérez Companc](../persons/gregorio-perez-companc.md) |
| **Evoluzione** | Da petrolio e banca a conglomerato agribusiness-energia |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1946** | Fondazione Petrolera Pérez Companc |
| **1968** | Gregorio diventa direttore Banco Río de la Plata |
| **1990-94** | Espansione massiccia con privatizzazioni Menem |
| **1993** | Gregorio acquista controllo Banco Río dai fratelli |
| **1997** | Vendita Banco Río a Santander |
| **1998** | Acquisto 68% Molinos Río de la Plata da Bunge y Born ($380M) |
| **1998** | Creazione PC Holdings S.A. |
| **2002** | Vendita 60% Petrolera a Petrobras (~$1B) — pivot verso agribusiness |
| **2003** | Luis Pérez Companc presidente Molinos |
| **2015** | Luis presidente Pecom |
| **2016** | Spin-off Molinos Agro come unita' indipendente |
| **2024** | Riorganizzazione azionaria: Luis, Rosario e Pilar concentrano il controllo (~$550-650M) |
| **2024** | Morte di Gregorio Pérez Companc (14 giugno) |
| **2025** | Acquisto campo Manantiales Behr da YPF, iniezione $150M in Pecom |
