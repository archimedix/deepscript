# 7-Eleven

> Dal ghiaccio al convenience store globale: la catena texana diventata giapponese che ha inventato il modello del negozio aperto "dalle 7 alle 11" e oggi domina il retail mondiale con 85,000+ punti vendita in 20 paesi.

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1927 (come Southland Ice Company) |
| **Sede** | Irving, Texas, USA |
| **Tipo** | Convenience store chain |
| **Proprietario** | [Seven & I Holdings](seven-i-holdings.md) (100%) |
| **Negozi** | 85,000+ |
| **Paesi** | 20 |
| **#1 Paese** | Giappone (21,668) |
| **#2 Paese** | Thailandia (15,430) |
| **Status** | active |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1927 |
| **Luogo** | Dallas, Texas |
| **Fondatore** | Joe C. "Jodie" Thompson Sr. |
| **Idea originale** | John Jefferson Green (vendere latte/uova da ice dock) |
| **Nome originale** | Southland Ice Company |
| **Nome 1928-1946** | Tote'm Stores |
| **Nome dal 1946** | 7-Eleven (orari 7am-11pm) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1927** | Fondazione Southland Ice Company, Dallas |
| **1928** | Prima espansione, nome Tote'm Stores |
| **1946** | Rebranding in 7-Eleven (orari 7am-11pm) |
| **1961** | Inizio franchising |
| **1963** | Primi negozi 24 ore (Austin, Texas) |
| **1969** | Introduzione Slurpee |
| **1973** | Accordo franchise con Ito-Yokado (Giappone) |
| **1974** | Prima 7-Eleven in Giappone (Tokyo, Koto) |
| **1987** | LBO famiglia Thompson (difesa da raider) |
| **1988** | CP Group fonda CP All (licenza Thailandia) |
| **1989** | Prima 7-Eleven Thailandia (Patpong Road) |
| **1990** | Southland in bancarotta |
| **1991** | Ito-Yokado e Seven-Eleven Japan acquisiscono 70% |
| **1999** | Southland diventa 7-Eleven Inc. |
| **2005** | Seven-Eleven Japan acquisisce 100%, nasce Seven & I Holdings |
| **2021** | Acquisizione Speedway ($21B) |
| **2024** | Offerta Couche-Tard ($47B) respinta |

### L'Acquisizione Giapponese

| Fase | Anno | Evento |
|------|------|--------|
| Partnership | 1973 | Ito-Yokado ottiene licenza Giappone |
| Crisi | 1987 | LBO Thompson per difesa da raider |
| Bancarotta | 1990 | Default $1.8B debito |
| Salvataggio | 1991 | Ito-Yokado + SEJ iniettano $430M, prendono 70% |
| Controllo totale | 2005 | Tender offer, 7-Eleven Inc. diventa sussidiaria 100% |

> Il franchisee giapponese ha comprato il franchisor americano.

## Pattern Rilevanti

### "Franchisee Compra Franchisor"

Caso raro di inversione proprietaria:
1. 1973: Ito-Yokado prende licenza da Southland
2. 1991: Salva Southland dalla bancarotta
3. 2005: Acquisisce 100%
4. Oggi: La filiale giapponese controlla la casa madre americana

### Dominanza Asiatica

7-Eleven e' piu' forte in Asia che negli USA:
- Giappone: 21,668 negozi
- Thailandia: 15,430 negozi
- USA: ~13,000 negozi

### Integrazione CP Group

Per CP Group, 7-Eleven e' strategico:
- Retail pillar (con Makro, Lotus's)
- Distribuzione prodotti CP Foods
- 15,430 punti vendita = network logistico
