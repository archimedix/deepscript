# Adallom

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2012 |
| **Sede** | Menlo Park, California, USA (HQ); Tel Aviv, Israel (R&D) |
| **Tipo** | Company (Cybersecurity/CASB) |
| **Founders** | Assaf Rappaport (CEO), Ami Luttwak (CTO), Roy Reznik (VP Eng), Yinon Costica (VP Product) |

## Glossario

| Termine | Definizione |
|---------|-------------|
| **CASB** | Cloud Access Security Broker - software che monitora e controlla l'accesso aziendale alle applicazioni SaaS |
| **SaaS** | Software as a Service - applicazioni cloud-based in abbonamento |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2012, Tel Aviv |
| **Fondatori** | Assaf Rappaport, Ami Luttwak, Roy Reznik, Yinon Costica |
| **Background** | Tutti ex Unit 8200 e programma Talpiot IDF |
| **Obiettivo** | Proteggere l'uso aziendale di applicazioni SaaS |

### Tesi fondativa

I fondatori partirono dall'osservazione che le applicazioni SaaS sono intrinsecamente sicure, ma il modo in cui i dipendenti le utilizzano non lo e'. Adallom sviluppo' tecnologia per:
- Audit dell'attivita' utente
- Rilevamento anomalie comportamentali
- Protezione asset digitali in cloud

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2012** | Fondazione a Tel Aviv da 4 veterani Unit 8200 |
| **2013** | Serie A da Index Ventures e Sequoia Capital |
| **2014** | Espansione HQ a Menlo Park, California |
| **2015** | Acquisizione da Microsoft per $320M (luglio) |
| **2015** | Annuncio ufficiale Microsoft (8 settembre) |
| **2016** | Prodotto rinominato Microsoft Cloud App Security (MCAS) |
| **2021** | Rebrand a Microsoft Defender for Cloud Apps |

## Acquisizione Microsoft

| Aspetto | Dettaglio |
|---------|-----------|
| **Acquirente** | [Microsoft](microsoft.md) |
| **Valore** | $320M |
| **Data** | Luglio 2015 (annuncio ufficiale 8 settembre 2015) |
| **Return per founder** | ~$25M ciascuno (stimato) |

Post-acquisizione, i fondatori sono entrati in Microsoft:
- Assaf Rappaport: GM Cloud Security, Head Israel R&D
- Team: ha guidato Microsoft Azure Cloud Security Group

L'acquisizione e' stata descritta come la piu' grande mai pagata da Microsoft per un'azienda tech israeliana (al tempo).

## Prodotto e Legacy

### Evoluzione del prodotto

| Periodo | Nome | Piattaforma |
|---------|------|-------------|
| 2012-2015 | Adallom CASB | Standalone |
| 2016-2021 | Microsoft Cloud App Security (MCAS) | Azure |
| 2021- | Microsoft Defender for Cloud Apps | Azure/M365 |

### Impatto sul mercato

Adallom e' stata pioniera nel mercato CASB, diventato poi categoria standard nella cybersecurity enterprise. La tecnologia sviluppata e' ora parte integrante della security suite Microsoft.

## Pattern: Unit 8200 → Startup → Big Tech

Adallom rappresenta un pattern ricorrente nell'ecosistema tech israeliano:

1. **Formazione militare elite** (Talpiot/Unit 8200)
2. **Network duraturo** tra commilitoni
3. **Prima startup** con exit a Big Tech
4. **Seconda startup** con valutazione maggiore

I fondatori hanno replicato esattamente questo pattern con Wiz (2020), acquisita da Google/Alphabet per $32B nel 2025.
