## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1983 |
| **Fondatore** | Nicolas G. Hayek (consulente, poi proprietario) |
| **Contesto** | Crisi quarzo giapponese anni '70-'80, industria svizzera in declino |
| **Obiettivo dichiarato** | Salvare e consolidare industria orologiera svizzera |

### Origine: La Crisi del Quarzo

Negli anni '70 e '80, l'industria orologiera svizzera fu devastata dall'arrivo degli orologi al quarzo giapponesi (Seiko, Casio). I due maggiori gruppi svizzeri, ASUAG e SSIH, erano sull'orlo del fallimento.

Nicolas Hayek, consulente libanese-svizzero, fu incaricato dalle banche creditrici di studiare il problema. Invece di raccomandare la liquidazione, propose una fusione e una strategia di rilancio basata su:
- Innovazione (Swatch: orologio plastica economico)
- Integrazione verticale (controllo componentistica)
- Portfolio multi-segmento (dal lusso al mass market)

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1983** | Fusione ASUAG + SSIH → SMH (Société de Microélectronique et d'Horlogerie) |
| **1985** | Lancio Swatch: successo globale, 10M pezzi venduti primo anno |
| **1986** | Nicolas Hayek acquisisce quota di controllo |
| **1998** | SMH rinominata The Swatch Group |
| **1999** | Acquisizione Breguet (alta orologeria) |
| **2000** | Lancio Omega Speedmaster per nuovo millennio |
| **2003** | Nick Hayek Jr diventa CEO |
| **2010** | Morte Nicolas Hayek; Nayla Chairman |
| **2013** | Acquisizione Harry Winston ($1 miliardo) |
| **2022** | Lancio MoonSwatch (Omega x Swatch), fenomeno globale |
| **2024** | Marc Hayek entra nel Board |
| **2025** | Famiglia Hayek rafforza partecipazione |
