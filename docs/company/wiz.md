## Acquisizione Google/Alphabet

| Aspetto | Dettaglio |
|---------|-----------|
| **Acquirente** | [Alphabet Inc.](alphabet.md) / [Google Cloud](google.md) |
| **Prezzo** | $32 miliardi (all-cash) |
| **Annuncio** | 18 marzo 2025 |
| **Approvazione DOJ** | Novembre 2025 |
| **Chiusura prevista** | 2026 |
| **Record** | Piu' grande acquisizione nella storia di Alphabet |

### Contesto

Nel luglio 2024, Wiz aveva rifiutato un'offerta da $23B di Google, preferendo perseguire una IPO indipendente. Meno di un anno dopo, ha accettato $32B - un premio del 39%.

### Motivazioni strategiche

- Google Cloud rafforza offerta security per competere con AWS e Azure
- Wiz serve il 40% delle Fortune 100
- Piattaforma vendor-agnostic (funziona anche su AWS, Azure, Oracle)
- Integrazione con strategia "security-first AI" di Alphabet

---

## Prodotto

### Tecnologia

Wiz offre una piattaforma di cloud security **agentless** che fornisce visibilita' completa sull'infrastruttura cloud in minuti anziche' mesi.

### Clienti

- 40% delle Fortune 100
- Multi-cloud: AWS, Azure, Google Cloud, Oracle

### Differenziazione

A differenza dei competitor che richiedono l'installazione di agent su ogni workload, Wiz si connette via API e scansiona l'intera infrastruttura cloud senza deployment.
