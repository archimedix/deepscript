## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1997, come Siberian Aluminum |
| **Rinominata** | 2001, diventa Basic Element |
| **Fondatore** | [Oleg Deripaska](../persons/oleg-deripaska.md) |
| **Funzione** | Holding di gestione degli investimenti di Deripaska in Russia, CSI, Africa, Asia, Europa e America Latina |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1997** | Fondazione come Siberian Aluminum |
| **2000** | Merger asset alluminio con Millhouse Capital → nasce Rusal |
| **2001** | Rinominata Basic Element |
| **2002** | Creazione En+ Group (energia) |
| **2007** | Acquisizione quota 24% in [Strabag](strabag.md) (via Rasperia Trading, Cipro) |
| **2007** | Merger Rusal + SUAL + Glencore → UC Rusal |
| **2018** | Sanzionata da OFAC insieme a Deripaska |
