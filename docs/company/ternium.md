## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2005 |
| **Origini** | Consolidamento di Siderar (ARG), Sidor (VEN), Hylsa (MEX) |
| **Fondatore** | Techint Group / Famiglia Rocca |
| **Contesto** | Unificazione degli asset siderurgici piani del gruppo Techint in un'unica entità quotata |
| **Etimologia** | Dal latino *Ter* (tre) + *Eternium* (eterno): le tre aziende fondatrici |

### Componenti originarie

- **Siderar** (Argentina) — acciaieria integrata a San Nicolás, Buenos Aires, controllata da Techint dal 1992 (privatizzazione SOMISA)
- **Hylsa** (Messico) — pioniere della riduzione diretta del ferro (DRI), acquisita da Techint
- **Sidor** (Venezuela) — privatizzata nel 1997, rinazionalizzata da Chávez nel 2008

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1992** | Techint acquisisce Siderar (ex SOMISA) nella privatizzazione argentina |
| **1997** | Techint acquisisce Sidor (Venezuela) nella privatizzazione |
| **2005** | Costituzione Ternium S.A. in Lussemburgo, consolidamento Siderar + Hylsa + Sidor |
| **2006** | Quotazione NYSE (ticker TX) |
| **2008** | Venezuela nazionalizza Sidor; Ternium perde l'asset venezuelano |
| **2012** | Acquisizione quota in Usiminas (Brasile) |
| **2018** | Máximo Vedoya diventa CEO, Novegil passa a Vice-chairman |
| **2023** | Quota in Usiminas sale al 51,5% |

## Operazioni

Ternium produce acciaio piani (laminati a caldo, a freddo, zincati, preverniciati) per industria automotive, edilizia, elettrodomestici ed energia.

### Stabilimenti principali

| Paese | Stabilimento | Prodotto |
|-------|-------------|----------|
| Argentina | San Nicolás (ex SOMISA/Siderar) | Acciaio integrato |
| Argentina | Haedo, Ensenada, Florencio Varela | Laminati, zincati |
| Messico | Monterrey (ex Hylsa) | DRI, laminati |
| Messico | Pesquería (Nuevo León) | Nuovo impianto integrato |
| Colombia | | Laminati |
| Guatemala | | Prodotti lunghi |
