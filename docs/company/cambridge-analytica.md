## Operazioni e Metodo

### Data Harvesting

| Aspetto | Dettaglio |
|---------|-----------|
| **App** | "thisisyourdigitallife" (Aleksandr Kogan) |
| **Utenti diretti** | 270.000 |
| **Utenti harvested** | 87 milioni (via friends' data) |
| **Dati raccolti** | Likes, profili, network, messaggi |

### Profiling Psicografico

| Modello | Uso |
|---------|-----|
| OCEAN personality | Classificazione elettori per tratti psicologici |
| Microtargeting | Ads personalizzati per profilo psicologico |
| A/B testing | Ottimizzazione messaggi divisivi |
