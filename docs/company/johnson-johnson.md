## Vaccino COVID Janssen

### Timeline

| Data | Evento |
|------|--------|
| **Marzo 2020** | Partnership BARDA ($1B+ funding) |
| **Giugno 2020** | Trial fase 1 |
| **Febbraio 2021** | FDA EUA (Emergency Use Authorization) |
| **Aprile 2021** | Pausa per blood clots (TTS) |
| **Dicembre 2021** | CDC raccomanda Pfizer/Moderna over Janssen |
| **Maggio 2022** | FDA limita uso |
| **Maggio 2023** | Stock USA scaduto |
| **Agosto 2023** | Chiusura R&D vaccines/ID |

### Caratteristiche

| Metrica | Valore |
|---------|--------|
| **Tecnologia** | Viral vector (Ad26) |
| **Dosi** | Singola (vs 2 per mRNA) |
| **Storage** | Refrigerato (vs congelato mRNA) |
| **Efficacia** | 66% sintomatico, 85% severo |
| **Revenue totale** | ~$2.5B |

### Problemi

| Problema | Dettaglio |
|----------|-----------|
| **TTS** | Thrombosis with thrombocytopenia syndrome |
| **Preferenza CDC** | mRNA preferiti dal dicembre 2021 |
| **Chiusura** | R&D vaccines terminato 2023 |

## Revolving Door: Mark McClellan

### Timeline

| Periodo | Ruolo |
|---------|-------|
| 2002-2004 | FDA Commissioner |
| 2004-2006 | CMS Administrator |
| 2007-2013 | Brookings Institution |
| **2013-oggi** | Board J&J |

### Compensi

| Metrica | Valore |
|---------|--------|
| **Compenso annuale** | ~$300,000 |
| **Azioni J&J** | Migliaia di shares |
| **Durata board** | 10+ anni |

### Pattern: Tutti e 3 i Vaccini COVID

```
VACCINO COVID    →    EX-FDA COMMISSIONER NEL BOARD
    |
    +-- Pfizer         →    Scott Gottlieb (2017-2019)
    +-- J&J            →    Mark McClellan (2002-2004)
    +-- Moderna        →    Stephen Hahn (2019-2021)
```

**Implicazione**: Tutte e tre le aziende con vaccini COVID approvati da FDA hanno ex-Commissioner nei rispettivi board.

## Pattern Identificati

### 1. Revolving Door FDA Confermato

McClellan e' il secondo ex-FDA Commissioner in board pharma COVID (dopo Gottlieb/Pfizer). Pattern sistemico.

### 2. Exit Strategy Vaccini

J&J ha chiuso R&D vaccines/infectious disease dopo aver incassato $2.5B+ da COVID e funding BARDA. Pattern hit-and-run?

### 3. Liability Management

Texas Two-Step bankruptcy per isolare cause talco. Innovazione legale per limitare responsabilita'.
