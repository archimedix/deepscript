## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1865 (Fredrik Idestam, cartiera) |
| **Fusione** | 1967 (Nokia Ab + Finnish Rubber + Cable Factory) |
| **Trasformazione** | 1992 (focus telecomunicazioni sotto Ollila) |
| **Vendita mobile** | 2014 (a Microsoft per 5.4 mld EUR) |
| **Rinascita** | 2016 (acquisizione Alcatel-Lucent) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1865** | Fredrik Idestam fonda cartiera a Tampere |
| **1868** | Seconda cartiera vicino a Nokia (da cui il nome) |
| **1871** | Fondazione Nokia Ab con Leo Mechelin |
| **1967** | Fusione in Nokia Corporation |
| **1981** | Lancio NMT - primo network cellulare internazionale |
| **1992** | Jorma Ollila CEO, focus su telecomunicazioni |
| **1998** | Nokia primo produttore mondiale cellulari |
| **2007** | iPhone lanciato - inizio declino Nokia mobile |
| **2008** | Peak market share 38.6% |
| **2010** | Stephen Elop CEO (primo non-finlandese) |
| **2011** | Partnership Windows Phone con Microsoft |
| **2013** | Annuncio vendita divisione mobile a Microsoft |
| **2014** | Completata vendita a Microsoft (5.4 mld EUR) |
| **2016** | Acquisizione Alcatel-Lucent, ritorno brand consumer |
| **2020** | Pekka Lundmark CEO, Sari Baldauf Chairman |
| **2025** | Lundmark lascia, Justin Hotard nuovo CEO |

## Business Attuale

### Segmenti (2025)

| Segmento | Focus |
|----------|-------|
| **Network Infrastructure** | 5G RAN, optical, fixed networks |
| **Cloud & Network Services** | 5G Core, enterprise |
| **Nokia Technologies** | Licensing, brevetti |

### Posizionamento 5G

- 120+ accordi 5G SA core globali
- 35 deployment 5G SA (su 60 totali CSP)
- Acquisizione Infinera (febbraio 2025) per optical networks
- Partnership Lockheed Martin per 5G militare
