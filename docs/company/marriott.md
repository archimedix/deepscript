## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Maggio 1927 |
| **Fondatori** | J. Willard Marriott e Alice S. Marriott |
| **Prima attivita'** | Chiosco A&W Root Beer a 9 posti, Washington DC |
| **Obiettivo** | Catena hospitality familiare |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1927** | J. Willard e Alice Marriott aprono un chiosco root beer a Washington DC |
| **1957** | Primo hotel: Twin Bridges Motor Hotel, Arlington, Virginia |
| **1967** | Cambio nome in Marriott Corporation |
| **1985** | J.W. Marriott Jr. diventa CEO |
| **1993** | Scissione: Marriott International (gestione) e Host Marriott (immobili) |
| **2012** | Arne Sorenson diventa CEO: primo non-Marriott alla guida |
| **2015** | Annuncio acquisizione Starwood Hotels per $12.2B |
| **2016** | Chiusura acquisizione Starwood per $13.3B: nasce il piu' grande gruppo alberghiero al mondo |
| **2019** | Lancio programma fedelta' Marriott Bonvoy (147M+ membri) |
| **2021** | Morte di Arne Sorenson (62 anni, cancro pancreatico). Anthony Capuano diventa CEO |
| **2025** | 30+ brand, 9.700+ proprieta' in 143 paesi |

## Portfolio Brand (30+)

- **Luxury**: The Ritz-Carlton, St. Regis, W Hotels, EDITION, JW Marriott
- **Premium**: Marriott Hotels, Sheraton, Westin, Le Meridien, Autograph Collection
- **Select**: Courtyard, Fairfield, SpringHill Suites, Four Points, Aloft
- **Extended Stay**: Residence Inn, TownePlace Suites, Element
- **Timeshare**: Marriott Vacations Worldwide

### Acquisizione Starwood (2016)

L'acquisizione di Starwood Hotels per $13.3B ha aggiunto brand iconici: Sheraton, Westin, W Hotels, St. Regis, Le Meridien. L'offerta iniziale di $12.2B fu rialzata per contrastare Anbang Insurance Group (Cina) che tento' di fare una controfferta.
