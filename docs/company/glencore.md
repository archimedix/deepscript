## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1974, Zug, Svizzera |
| **Fondatore** | [Marc Rich](../persons/marc-rich.md) |
| **Nome originale** | Marc Rich + Co AG |
| **Rebranding** | 1994: Glencore (Global Energy Commodity Resources) |
| **Contesto** | Trader di petrolio sfida embargo Iran |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1974** | Marc Rich fonda Marc Rich + Co AG a Zug |
| **1983** | Marc Rich incriminato per evasione fiscale e violazione sanzioni Iran (caso piu' grande nella storia USA) |
| **1983** | Rich fugge in Svizzera, inserito nella lista FBI Most Wanted |
| **1994** | Management buyout ($600M): Glasenberg e altri comprano Rich dopo perdite zinc trading ($172M) |
| **1994** | Rinominata Glencore |
| **2001** | Bill Clinton grazia Marc Rich nel suo ultimo giorno di presidenza |
| **2002** | Ivan Glasenberg diventa CEO |
| **2011** | IPO Londra: $10B raccolti, valutazione $60B (piu' grande IPO non-privatizzazione europea) |
| **2012** | Annunciata fusione con Xstrata |
| **2013** | Completata fusione Glencore-Xstrata ($90B deal) |
| **2021** | Gary Nagle succede a Glasenberg come CEO |
| **2022** | Guilty plea USA: $1.1B per corruzione e manipolazione mercato |
| **2024** | Condanna Svizzera: CHF 2M + $150M per corruzione DRC |
