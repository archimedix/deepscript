1. **Pagamenti mobile** — P2P, POS commercianti, bollette, ricariche
2. **Rete indipendente** — Alternativa a Visa/Mastercard, commissioni inferiori
3. **Servizi finanziari** — Lancio investimenti previsto 2025
4. **Espansione europea** — Francia, Lussemburgo, altri mercati UE

### Il cluster Tiger su Satispay

La Series D di Satispay vede convergere due fondi con radice comune in [Tiger Management](../hedge-fund/tiger-management.md) di Julian Robertson:

```
Julian Robertson (Tiger Management)
    ├── Philippe Laffont → Coatue Management (Tiger Cub diretto)
    └── Chase Coleman → Tiger Global
                            └── Lee Fixel → Addition (Tiger Cub 2a gen.)
```

Entrambi investono nella stessa operazione, segnalando un pattern di "deal flow condiviso" dentro la rete Tiger.
