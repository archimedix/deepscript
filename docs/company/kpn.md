## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origine** | 1752 (Statenpost) |
| **PTT** | 1893 (fusione posta + telegrafo + telefono) |
| **Corporatizzazione** | 1 gennaio 1989 (da statale a azienda) |
| **Nome inventato da** | Wim Dik (primo CEO) |
| **Privatizzazione** | 1994-2006 |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1752** | Fondazione Statenpost |
| **1893** | Creazione PTT (posta + telecom) |
| **1989** | Corporatizzazione → **KPN** (Wim Dik CEO) |
| **1994** | IPO Amsterdam Stock Exchange (30% venduto) |
| **1995** | Altra vendita (25%), governo scende a 45% |
| **1998** | Split: KPN (telecom) + TPG (posta, poi TNT) |
| **2000-2001** | **Crisi debiti**, bolla dot-com |
| **2001** | Ad Scheepbouwer CEO - salva KPN |
| **2006** | Stato vende quota residua 7.8% - privatizzazione completata |
| **2013** | **Tentativo OPA Carlos Slim** (America Movil) - bloccato |
| **2019** | Joost Farwerck CEO |

## Il Tentativo OPA Carlos Slim (2013)

| Aspetto | Dettaglio |
|---------|-----------|
| **Offerente** | America Movil (Carlos Slim) |
| **Quota gia' posseduta** | ~30% |
| **Offerta** | €7.2 miliardi per restante 70% |
| **Reazione governo** | Warning ufficiale |
| **Difesa** | Stichting Preferente Aandelen B KPN |
| **Meccanismo** | Call option per acquisire ~50% azioni, "poison pill" |
| **Risultato** | OPA bloccata |

> **Pattern**: KPN ha usato una fondazione (Stichting) con opzione su azioni preferenziali per bloccare l'OPA ostile - meccanismo tipico olandese di difesa.

## Pattern Rilevanti

### 1. Privatizzazione Graduale
KPN rappresenta il modello olandese di privatizzazione: corporatizzazione (1989) → IPO parziale (1994) → riduzione progressiva → uscita totale (2006).

### 2. Difesa Anti-OPA Olandese
La "Stichting" con opzione su azioni preferenziali e' un meccanismo tipico olandese per bloccare acquisizioni ostili (usato anche da Philips, AkzoNobel).

### 3. Elite nel Board
La presenza di Halberstadt dimostra come i board delle grandi aziende olandesi siano interconnessi con i forum transnazionali.
