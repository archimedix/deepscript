## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Gennaio 2000 |
| **Fondatori** | Robin Li, Eric Xu |
| **Contesto** | Boom dot-com, assenza di search engine cinese |
| **Seed funding** | $1.2M da VC Silicon Valley |
| **Breakthrough** | Lancio motore di ricerca proprio (2001) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2000** | Fondazione a Beijing da Robin Li e Eric Xu |
| **2001** | Lancio motore di ricerca Baidu.com |
| **2004** | Prima profittabilita' |
| **2005** | IPO NASDAQ - azioni da $27 a $122 primo giorno |
| **2010** | Google esce dalla Cina, Baidu domina |
| **2013** | Acquisizione 91 Wireless ($1.9B) |
| **2014** | Lancio Baidu Translate |
| **2017** | Lancio Apollo (guida autonoma) |
| **2019** | Lancio Baidu Brain AI |
| **2021** | IPO secondaria Hong Kong |
| **2023** | Lancio ERNIE Bot (risposta a ChatGPT) |
| **2024** | Management rotation: nuovo CFO Junjie He |
| **Mar 2025** | Lancio ERNIE 4.5 e ERNIE X1 (reasoning) |
| **Jun 2025** | Open-source ERNIE 4.5 (10 varianti fino a 424B param) |
| **Nov 2025** | ERNIE 5.0 omni-modal, compete con GPT-5 |
