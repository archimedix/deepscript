## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 21 gennaio 2011 |
| **Fondatori** | Fusione British Airways + Iberia |
| **Contesto** | Consolidamento mercato aereo europeo post-crisi 2008 |
| **Quotazione** | London Stock Exchange, Bolsa de Madrid |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2011** | Fusione BA-Iberia, creazione IAG |
| **2012** | Acquisizione British Midland International (bmi) |
| **2013** | Acquisizione Vueling |
| **2015** | Acquisizione Aer Lingus; Qatar Airways entra con 10% |
| **2017** | Lancio LEVEL (low-cost long-haul) |
| **2020** | Qatar Airways sale al 25.1%; Luis Gallego diventa CEO |
| **2021** | Javier Ferrán diventa Chairman |
| **2025** | Ordine 76 aerei Boeing/Airbus |
