## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1898, Bermuda |
| **Fondatore** | Reginald Appleby (1865-1948) |
| **Contesto** | Bermuda come centro finanziario britannico offshore |
| **Obiettivo dichiarato** | Consulenza legale offshore per clienti internazionali |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1898** | Fondazione a Hamilton, Bermuda da Reginald Appleby |
| **1945** | Apertura ufficio Cayman Islands (il piu' antico studio legale delle Cayman) |
| **1979** | Prima espansione fuori Bermuda |
| **2004** | Fusione con Hunter & Hunter (Cayman) |
| **2006** | Fusione con Bailhache Labesse (Jersey) |
| **2017** | Hack dei server: 13.4M documenti sottratti → Paradise Papers |
| **2017** | ICIJ pubblica l'inchiesta Paradise Papers (5 novembre) |

## Paradise Papers (2017)

### Il Leak

| Aspetto | Dettaglio |
|---------|-----------|
| **Documenti** | 13.4 milioni |
| **Volume** | 1.4 terabyte |
| **Periodo coperto** | ~50 anni di attivita' |
| **Contenuto** | Accordi clienti, bilanci, trust deeds, email, documenti societari |
| **Ottenuti da** | Suddeutsche Zeitung (hack dei server Appleby) |
| **Pubblicati da** | ICIJ con NYT, BBC, Guardian e 95 media partner |
| **Data pubblicazione** | 5 novembre 2017 |

### Persone/Entita' Esposte (nel DB)

| Persona/Entita' | Connessione |
|------------------|-------------|
| [Wilbur Ross](../persons/wilbur-ross.md) | 50+ entita' offshore amministrate da Appleby; legami Navigator Holdings-Sibur |
| [Yuri Milner](../persons/yuri-milner.md) | Rivelati flussi VTB Bank-Gazprom → DST Global → Facebook/Twitter |
| [Stephen Bronfman](../persons/stephen-bronfman.md) | $60M offshore per Leo Kolber via Cayman |
| [Leo Kolber](../persons/leo-kolber.md) | Beneficiario fondi Bronfman offshore |
| [Gennady Timchenko](../persons/gennady-timchenko.md) | Comproprietario Sibur, cliente Navigator Holdings |

### Compliance Failures

ICIJ e OCCRP hanno documentato molteplici fallimenti di compliance da parte di Appleby, che operava come parte dell'"Offshore Magic Circle" fornendo servizi a clienti ad alto rischio, inclusi individui sotto indagine per frode e riciclaggio.
