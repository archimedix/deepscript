## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 30 ottobre 1941 |
| **Nome originale** | Societa' Nazionale Metanodotti |
| **Fondatori** | Ente Nazionale Metano (Agip), Regie Terme Salsomaggiore, SURGI |
| **Capitale iniziale** | 3 milioni di lire |
| **Contesto** | Guerra mondiale, necessita' energia alternativa al carbone |
| **Obiettivo** | Costruire rete nazionale trasporto metano |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1941** | Fondazione a San Donato Milanese |
| **1948** | Prima pipeline Salsomaggiore-Milano-Lodi completata (257 km) |
| **1949** | Agip acquisisce controllo totale |
| **1950** | Rete supera 700 km |
| **1952** | Rete supera 2,000 km |
| **1953** | ENI costituita, Snam diventa controllata |
| **1955** | Snam Montaggi fondata (diventa Saipem 1957) |
| **1956** | Snam Progetti fondata (Snamprogetti) |
| **1960** | Rete supera 4,600 km |
| **1961** | Gasdotti Centro-Sud Italia (Chieti-Roma, Basilicata-Puglia) |
| **2001** | Quotazione Borsa Italiana |
| **2009** | Acquisizione Italgas da ENI (€3.07 mld) |
| **2012** | Separazione da ENI, CDP RETI diventa azionista |
| **2014** | State Grid China entra in CDP RETI (35%, €2.1 mld) |
| **2016** | Spin-off Italgas (riquotata) |
| **2022** | FSRU Golar Tundra (Italis LNG) per crisi gas russo |
| **2023** | FSRU BW Singapore operativa |
| **2024** | Acquisizione storage Edison, capacita' sale a 18 bcm |

## Pattern: Da ENI a CDP

Snam rappresenta il modello di **privatizzazione controllata**:

1. **Fase ENI** (1949-2012): Controllata integrata nel gruppo petrolifero
2. **Separazione** (2012): Unbundling EU, ENI cede controllo
3. **CDP RETI** (2012-): Stato mantiene controllo via CDP
4. **State Grid** (2014-): Cina entra come partner finanziario

**Confronto con altre utility CDP**:

| Utility | Quota CDP RETI | Settore |
|---------|----------------|---------|
| **Snam** | 31.35% | Gas trasporto/storage |
| Terna | 29.85% | Elettricita' trasmissione |
| Italgas | 25.96% | Gas distribuzione |

## Crisi Gas Russo 2022

Sotto Venier, Snam ha guidato la risposta italiana alla crisi energetica:

| Azione | Dettaglio |
|--------|-----------|
| FSRU Golar Tundra | Acquisita e operativa in mesi (Piombino) |
| FSRU BW Singapore | Seconda unita' (Ravenna) |
| Diversificazione | Aumento forniture Nord Africa, Azerbaijan |
| TAP | Raddoppio capacita' da Azerbaijan |

Risultato: Italia ha ridotto dipendenza gas russo da ~40% a <10% in due anni.
