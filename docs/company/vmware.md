# VMware

> Pioniere della virtualizzazione x86, acquisito da Broadcom per $69B

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1998 |
| **Sede** | Palo Alto, California, USA |
| **Tipo** | Company (Software) |
| **Leadership** | Hock Tan (CEO Broadcom, parent company) |
| **Dipendenti** | ~38.000 (pre-acquisizione) |
| **Fatturato** | $13.4B (FY2023) |
| **Status** | acquired (Broadcom, Nov 2023) |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Virtualizzazione** | Tecnologia che permette di eseguire piu' sistemi operativi su un singolo hardware fisico |
| **Hypervisor** | Software che crea e gestisce macchine virtuali |
| **vSphere** | Piattaforma di virtualizzazione enterprise di VMware |
| **NSX** | Piattaforma di virtualizzazione di rete (software-defined networking) |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1998 |
| **Fondatori** | [Diane Greene](../persons/diane-greene.md), [Mendel Rosenblum](../persons/mendel-rosenblum.md), Edouard Bugnion, Scott Devine, Edward Wang |
| **Contesto** | Ricerca accademica a Stanford sulla virtualizzazione come strumento di simulazione |
| **Obiettivo dichiarato** | Portare la virtualizzazione x86 al mercato mainstream |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1998** | Fondazione a Palo Alto da team Stanford/Berkeley |
| **1999** | Lancio VMware Workstation, primo prodotto |
| **2001** | Ingresso nel mercato server enterprise |
| **2004** | Acquisizione da EMC Corporation per $625M |
| **2007** | IPO a $19.1B (maggiore IPO tech dell'anno) |
| **2008** | Diane Greene rimossa come CEO, sostituita da Paul Maritz |
| **2012** | Pat Gelsinger diventa CEO |
| **2012** | Acquisizione Nicira (SDN) |
| **2016** | Dell acquisisce EMC per $67B, ottiene 81% VMware |
| **2021** | Dell spin-off di VMware; Raghu Raghuram CEO |
| **2022** | Annuncio acquisizione Broadcom ($61B + $8B debito) |
| **2023** | Completata acquisizione Broadcom (Nov), Raghuram lascia |
| **2024** | Broadcom termina licenze perpetue, passa a subscription |

| **CEO** | Raghu Raghuram | 2021-2023 |

### Post-Acquisizione (2023)

Sotto Broadcom, VMware e' stata riorganizzata in 4 divisioni:
- VMware Cloud Foundation
- Tanzu (containers/Kubernetes)
- Software-Defined Edge
- Application Networking and Security

La divisione End-User Computing (EUC) venduta a KKR per $4B e rinominata Omnissa.

## Impatto Settore

VMware ha creato il mercato della virtualizzazione enterprise:
- 2008: 25% applicazioni Intel virtualizzate
- 2012: 60% applicazioni Intel virtualizzate
- Certificati VMware: da 25.000 (2008) a 125.000 (2012)

Fatturato triplicato sotto Gelsinger: da $4.6B a $12B.
