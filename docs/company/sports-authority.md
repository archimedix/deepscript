| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1987 |
| **Sede** | Englewood, Colorado, USA |
| **Tipo** | Retail (sporting goods) |
| **Peak** | 463 negozi, 45 stati |
| **Fallimento** | 2016 |
| **Dipendenti persi** | 14,000+ |
| **Status** | defunct |

## Cause del Fallimento

### Debito Insostenibile

| Fattore | Impatto |
|---------|---------|
| **Servizio debito** | Priorita' su investimenti |
| **Mancata modernizzazione** | No e-commerce competitivo |
| **Margini compressi** | Impossibilita' di competere sui prezzi |

### Competizione

| Competitor | Vantaggio |
|------------|-----------|
| **Dick's Sporting Goods** | Non gravato da debito LBO |
| **Amazon** | E-commerce dominante |
| **Walmart/Target** | Economie di scala |
| **Nike (direct)** | Brand-to-consumer |

## Pattern Rilevante

Sports Authority e' un **caso emblematico di LBO distruttivo**:

1. **Azienda sana** → acquisita con debito
2. **Debito sull'azienda** → non sugli acquirenti
3. **Cash flow** → servizio debito invece di investimenti
4. **Competizione** → impossibile rispondere
5. **Fallimento** → PE ha gia' estratto valore
6. **Lavoratori** → 14,000+ licenziati

Il pattern e' identico a Toys R Us, Payless ShoeSource, e decine di altri retailer vittime di LBO pre-crisi 2008.
