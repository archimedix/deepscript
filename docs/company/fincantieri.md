## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 29 dicembre 1959 |
| **Nome originale** | Societa' Finanziaria Cantieri Navali - Fincantieri |
| **Creatore** | IRI (Istituto per la Ricostruzione Industriale) |
| **Obiettivo** | Consolidare e modernizzare l'industria cantieristica italiana |

### Cantieri Storici Integrati

- Ansaldo (Genova)
- Cantieri Riuniti dell'Adriatico - CRDA (Trieste, Monfalcone)
- Navalmeccanica

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1959** | Fondazione come holding finanziaria IRI |
| **1984** | Trasformazione in societa' operativa (gestione diretta cantieri) |
| **1984** | Ingresso nel mercato crocieristico |
| **2002** | [Giuseppe Bono](../persons/giuseppe-bono.md) diventa CEO |
| **2013** | Acquisizione Vard (Norvegia) - raddoppio dimensioni |
| **2014** | **IPO Borsa Milano** - quotazione |
| **2016** | [Giampiero Massolo](../persons/giampiero-massolo.md) Chairman |
| **2022** | Bono si ritira dopo 20 anni, muore a 78 anni |
| **2022** | [Pierroberto Folgiero](../persons/pierroberto-folgiero.md) nuovo CEO |
| **2024** | [Biagio Mazzotta](../persons/biagio-mazzotta.md) nuovo Chairman |
| **2025** | Folgiero e Mazzotta confermati (mandato 2025-2027) |

## Attivita' e Settori

### Divisioni

| Divisione | Prodotti |
|-----------|----------|
| **Navi militari** | Portaerei, fregate, sottomarini, pattugliatori |
| **Navi da crociera** | Navi per Carnival, MSC, Princess, Viking |
| **Offshore** | Piattaforme, navi speciali (via Vard) |
| **Riparazioni** | Manutenzione flotte |

### Numeri

- 230+ anni di storia cantieristica
- 7,000+ navi costruite
- 4° gruppo mondiale (dopo acquisizione Vard 2013)
- Leader europeo

### Clienti Principali

**Militare:**
- Marina Militare Italiana
- US Navy (via Fincantieri Marine Group)
- Marine straniere (Qatar, UAE, Indonesia)

**Crociere:**
- Carnival Corporation
- MSC Crociere
- Royal Caribbean (parziale)
- Viking Ocean Cruises
