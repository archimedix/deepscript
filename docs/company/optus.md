## Storia

### Origini AUSSAT (1981-1992)

| Anno | Evento |
|------|--------|
| **1977** | Kerry Packer propone sistema satellitare |
| **1980** | Gruppo NSS dentro OTC per satellite nazionale |
| **1981** | Governo forma AUSSAT Pty Limited |
| **1985** | Lancio AUSSAT A1 e A2 (Hughes 376) |
| **1987** | Lancio AUSSAT A3, copertura SW Pacific |

**Uso:** Comunicazioni militari e civili, TV per comunita' remote outback.

### Privatizzazione (1991-1992)

| Aspetto | Dettaglio |
|---------|-----------|
| **Problema** | AUSSAT in perdita |
| **Soluzione** | Vendita + licenza telco |
| **Consorzio Optus** | Cable & Wireless (UK), BellSouth (USA), Mayne Nickless |
| **Licenza** | Gennaio 1991, secondo carrier generale |
| **Acquisizione AUSSAT** | 1992 |

**Obiettivo:** Competizione con Telecom Australia (ora Telstra).

### Consolidamento (1992-2001)

| Anno | Evento |
|------|--------|
| **1992** | Optus Communications Pty Limited |
| **1997** | Cable & Wireless acquisisce quota BellSouth (24.5%) per $750M |
| **1997** | Rebrand: Cable & Wireless Optus |

### Era Singtel (2001-oggi)

| Anno | Evento |
|------|--------|
| **2001** | Singtel acquisisce per A$16B (~S$11B) |
| **2001** | Rebrand: Singtel Optus Pty Limited |
| **2020** | Kelly Bayer Rosmarin CEO |
| **2022** | Data breach: 9.5M clienti esposti |
| **2023** | Outage nazionale → dimissioni CEO |
| **2023** | Michael Venter interim CEO |

## Crisi 2022-2023

### Data Breach (Settembre 2022)

| Aspetto | Dettaglio |
|---------|-----------|
| **Clienti colpiti** | 9.5 milioni (1/3 popolazione Australia) |
| **Dati esposti** | Nome, DOB, indirizzo, telefono, email, passaporto, patente |
| **Causa** | Errore umano API (contestato da Optus) |
| **Costo** | A$140M riservati |
| **Conseguenze legali** | Causa OAIC, fino a A$2.22M per violazione |

### Outage Nazionale (Novembre 2023)

| Aspetto | Dettaglio |
|---------|-----------|
| **Durata** | Ore |
| **Impatto** | Servizi emergenza, trasporti, business |
| **Conseguenza** | Dimissioni CEO Bayer Rosmarin |

### Risposta Regolamentare Australia

| Misura | Dettaglio |
|--------|-----------|
| **Nuove sanzioni** | Fino a A$50M per data breach |
| **Causa OAIC** | In corso, potenziale A$21B (9.5M x A$2.22M) |
