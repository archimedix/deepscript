## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1921 |
| **Luogo** | Song Sawat Road, Chinatown, Bangkok |
| **Fondatori** | Fratelli Chia Ek Chor e Chia Jin Hyang |
| **Origine** | Immigrati da Shantou (Swatow), Cina |
| **Nome originale** | Chia Tai Chung (negozio di semi) |
| **Business iniziale** | Import semi dalla Cina, export maiali/uova verso Hong Kong |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1921** | Fondazione Chia Tai Chung a Bangkok |
| **1954** | Prima sussidiaria: Charoen Pokphand Feedmill |
| **1964** | Dhanin Chearavanont (25 anni) guida la crescita |
| **Fine 1960s** | Leader thai nella produzione di mangimi |

| **1973** | Inizio export pollame in Giappone |

| **1989** | Prima 7-Eleven apre a Patpong Road |
| **1994** | Fondazione Lotus Supercenter |
| **1997** | Crisi asiatica: consolidamento in 3 linee |
| **1998** | Tesco acquisisce quota Lotus (diventa Tesco Lotus) |
| **2012** | Acquisizione 15.6% Ping An da HSBC ($9.4B) |
| **2013** | Riacquisto Siam Makro da SHV Holding |

| **2017** | Soopakij Chairman, Suphachai CEO |
| **2020** | Acquisizione Tesco Asia ($10.6B) |
| **2021** | Rebranding Tesco Lotus in Lotus's |

## Pattern Rilevanti

### Modello "First Mover in Cina"

CP ha costruito vantaggio competitivo essendo primo:
1. Primo investitore straniero (1979)
2. Prima azienda straniera in mangimi
3. Prima banca estera
4. Primo supermercato estero

### Integrazione Verticale

Il modello CP controlla l'intera filiera:
- Mangimi → Allevamento → Trasformazione → Distribuzione → Retail
- Ogni passaggio aggiunge margine

### Diversificazione Post-Crisi

Dopo la crisi asiatica 1997, consolidamento in tre pilastri:
- Food (core storico)
- Retail (7-Eleven, cash & carry)
- Telecom (True)
