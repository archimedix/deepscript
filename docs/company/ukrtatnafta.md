# Ukrtatnafta

> Operatore della raffineria di Kremenchuk, la piu' grande d'Ucraina, nazionalizzata nel 2022

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1994 (raffineria costruita dal 1961) |
| **Sede** | Kremenchuk, Ucraina |
| **Tipo** | Company (oil refining) |
| **Leadership** | Sergii Koretskyi, CEO (2022-2025) |
| **Capacita'** | 18.6M tonnellate/anno (368,500 bbl/d) |
| **Status** | active (sotto controllo statale dal 2022, raffineria danneggiata dalla guerra) |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1994, come joint venture ucraino-tatara per gestire la raffineria di Kremenchuk |
| **Raffineria** | Costruzione iniziata nel 1961 sotto pianificazione sovietica, operativa dalla fine degli anni '60 |
| **Azionariato pre-2022** | Naftogaz 43.05% (stato), Privat Group 56% (offshore), Tatneft/Tatarstan ~37% (storicamente, poi perso) |
| **Settore** | Raffinazione petrolio, produzione carburanti e petrolchimici |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1961** | Inizio costruzione della raffineria di Kremenchuk |
| **1994** | Costituzione di Ukrtatnafta come societa' per azioni |
| **2007** | Takeover ostile del Privat Group: acquisisce 56% tramite societa' offshore |
| **2007** | Tatneft (Russia) perde il controllo, avvia causa da $150M |
| **2018** | Inchieste sulla mancanza di trasparenza nella gestione Kolomoisky |
| **2022 apr** | Raffineria di Kremenchuk distrutta da attacco russo |
| **2022 nov** | Nazionalizzazione: quote Privat espropriate sotto legge marziale |
| **2022 nov** | Sergii Koretskyi nominato CEO (anche CEO di Ukrnafta) |
| **2025** | Corte Suprema USA ordina all'Ucraina il pagamento di $173M a Tatneft |
| **2025 mag** | Koretskyi promosso a Chairman di Naftogaz |
