# De Beers

> "A Diamond is Forever": il cartello che ha inventato il desiderio dei diamanti

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 1888 |
| **Sede** | London, UK (HQ) / Gaborone, Botswana |

## Glossario

| Termine | Definizione |
|---------|-------------|
| CSO | Central Selling Organisation - braccio vendite storico (1930-2000) |
| DTC | Diamond Trading Company - successore CSO |
| Sightholder | Acquirente autorizzato De Beers (30-40 aziende) |
| Sight | Evento di vendita programmato (10 all'anno) |
| Rough | Diamanti grezzi non tagliati |
| Debswana | JV 50-50 De Beers-Botswana per miniere |
| Forevermark | Brand De Beers per diamanti certificati |
| Lightbox | Brand De Beers per lab-grown diamonds |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1888 |
| **Fondatore** | [Cecil Rhodes](../persons/cecil-rhodes.md) |
| **Luogo** | Kimberley, Sudafrica |
| **Nome** | Dalla fattoria dei fratelli De Beer dove furono trovati diamanti |
| **Obiettivo** | Consolidare miniere diamanti sudafricane |

### Il Modello De Beers

```
MINIERE SUDAFRICANE
        │
        ▼
     DE BEERS
   (estrazione)
        │
        ▼
       CSO
 (controllo offerta)
        │
        ▼
      DTC
  (prezzi fissi)
        │
        ▼
  SIGHTHOLDERS
  (30-40 clienti)
        │
        ▼
"A DIAMOND IS FOREVER"
  (creazione domanda)
```

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1888** | Cecil Rhodes fonda De Beers Consolidated Mines |
| **1902** | Morte Rhodes |
| **1917** | Ernest Oppenheimer fonda Anglo American |
| **1927** | Oppenheimer diventa chairman De Beers |
| **1930** | Creazione Central Selling Organisation (CSO) |
| **1947** | Slogan "A Diamond is Forever" (N.W. Ayer) |
| **1957** | Harry Oppenheimer succede al padre |
| **1998** | Nicky Oppenheimer diventa chairman |
| **2000** | CSO diventa DTC, fine era stockpiling |
| **2001** | Accordo antitrust USA |
| **2012** | Oppenheimer vendono 40% ad Anglo American ($5.1B) |
| **2023** | Al Cook diventa CEO |
| **2023** | Nuovo accordo con Botswana |

## Il Monopolio dei Diamanti

### Controllo del Mercato (Picco)

| Periodo | Market Share | Meccanismo |
|---------|--------------|------------|
| 1888-1930 | 90%+ | Consolidamento miniere |
| 1930-1990 | 80-85% | CSO, stockpiling |
| 1990-2000 | 63% | Competizione Russia, Australia |
| 2018-oggi | 30-35% | Fine monopolio |

### Central Selling Organisation (CSO)

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondazione** | 1930 |
| **Funzione** | Controllo offerta mondiale |
| **Meccanismo** | Stockpiling, prezzi fissi, sightholders esclusivi |
| **Fine** | 2000 (rinominata DTC) |

### Diamond Trading Company (DTC)

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondazione** | 2000 (ex CSO) |
| **Sightholders** | 30-40 aziende autorizzate |
| **Sight** | 10 eventi vendita/anno |
| **Localita'** | Botswana, Namibia, Sudafrica |

## "A Diamond is Forever"

| Aspetto | Dettaglio |
|---------|-----------|
| **Anno** | 1947 |
| **Autrice** | Frances Gerety (N.W. Ayer) |
| **Riconoscimento** | Miglior slogan pubblicitario XX secolo (Ad Age, 2000) |
| **Impatto** | Trasformazione diamanti da commodity a simbolo amore eterno |

> Il genio di De Beers non fu vendere diamanti, ma vendere l'*idea* che i diamanti fossero necessari per l'amore.
