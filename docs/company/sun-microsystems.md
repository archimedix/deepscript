| Dato | Dettaglio |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1982** | Fondazione, seed da [Kleiner Perkins](../private-equity/kleiner-perkins.md) ($300K) |
| **1984** | McNealy diventa CEO, Khosla lascia per Kleiner Perkins |
| **1985** | Lancio NFS (Network File System) |
| **1986** | IPO, $1B revenue entro 1988 |
| **1987** | Lancio architettura SPARC |
| **1995** | Lancio Java ("write once, run anywhere") |
| **2001** | Peak revenue $18.3B, market cap $200B+ |
| **2006** | McNealy lascia, Jonathan Schwartz CEO |
| **2008** | Crisi finanziaria, lotta per sopravvivenza |
| **2010** | Acquisita da [Oracle](../company/oracle.md) per $7.4B |

## Tecnologie Chiave

### Innovazioni Fondamentali

| Tecnologia | Anno | Impatto |
|------------|------|---------|
| **NFS** | 1985 | Primo protocollo open per file sharing, ancora usato |
| **SPARC** | 1987 | Architettura RISC, standard enterprise per decenni |
| **Solaris** | 1992 | Unix enterprise, scalabilita' e affidabilita' |
| **Java** | 1995 | "Write once, run anywhere", rivoluzione software |
| **ZFS** | 2005 | File system avanzato, ancora in uso |

### Java: l'eredita' piu' duratura

Java emerge dai laboratori Sun nel 1995 con la promessa di portabilita' totale. [Eric Schmidt](../persons/eric-schmidt.md), allora CTO di Sun, ne fu un promotore entusiasta. Oggi Java rimane uno dei linguaggi piu' usati al mondo.

## Declino e Acquisizione

### Fattori del Declino

1. **Dot-com crash (2001)**: Da $200B market cap a frazioni
2. **Commoditization**: Linux su x86 vs Solaris su SPARC
3. **Cloud shift**: Mancata transizione al cloud computing
4. **Leadership**: McNealy resistente al cambiamento

### Acquisizione Oracle (2010)

| Aspetto | Dettaglio |
|---------|-----------|
| **Prezzo** | $7.4 miliardi |
| **Data** | 27 gennaio 2010 |
| **Delay** | EC preoccupata per MySQL |
| **Post-merger** | Esodo ingegneri chiave (Gosling, Cantrill) |

## Legacy

Sun Microsystems ha lasciato un'impronta indelebile:

- **Java**: Ancora tra i linguaggi piu' usati
- **NFS**: Standard de facto per network storage
- **Talent pipeline**: CEO di Google, Yahoo, Motorola
- **Open source**: Pioneer con NFS, OpenSolaris
- **SPARC/Solaris**: Standard enterprise per 20+ anni
