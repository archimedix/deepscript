## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1977 (come SDL) |
| **Fondatori** | Larry Ellison, Bob Miner, Ed Oates |
| **Prodotto core** | Database relazionale |
| **Evoluzione** | Database → ERP → Cloud → AI infrastructure |

### Timeline

| Anno | Evento |
|------|--------|
| **1977** | Fondazione come Software Development Laboratories |
| **1979** | Rinominata Relational Software Inc. |
| **1982** | Rinominata Oracle |
| **1986** | IPO |
| **2005** | Acquisizione PeopleSoft ($10.3B) |
| **2008** | Acquisizione BEA Systems ($8.5B) |
| **2010** | Acquisizione Sun Microsystems ($7.4B) |
| **2016** | Acquisizione NetSuite ($9.3B) |
| **2020** | HQ trasferito in Texas |
| **2022** | Acquisizione Cerner ($28B) - healthcare |
| **2025** | Partner Stargate ($500B AI infrastructure) |
