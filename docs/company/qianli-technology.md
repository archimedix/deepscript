## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1992, Chongqing |
| **Fondatore** | Yin Mingshan |
| **Capitale iniziale** | Officina riparazione moto con 9 dipendenti |
| **Contesto** | Boom manifatturiero Chongqing anni '90 |

### Il Fondatore: Yin Mingshan

| Dato | Dettaglio |
|------|-----------|
| **Nato** | 1938, Chongqing |
| **Background** | Figlio di proprietario terriero, 20 anni in campo di lavoro |
| **Pre-Lifan** | Editoria (anni '80), autodidatta inglese |
| **Fondazione Lifan** | 1992, a 54 anni |
| **Uscita** | 2017, dopo fallimento pivot EV |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1992** | Fondazione Lifan da Yin Mingshan, officina moto |
| **1997** | Rinominata Lifan Industry Group |
| **Late 1990s** | 2 milioni moto/anno, export in 160 paesi |
| **2003** | Acquisizione fabbrica auto Chongqing |
| **2006** | Prime auto Lifan in produzione |
| **2010** | IPO Shanghai Stock Exchange (601777.SH) |
| **2015** | Tentativo pivot EV, fallisce |
| **2017** | Yin Mingshan lascia l'azienda |
| **Ago 2020** | Richiesta fallimento, debiti RMB 17.86B ($2.6B) |
| **Nov 2020** | Geely entra come investitore ristrutturazione |
| **Gen 2021** | Rinominata Lifan Technology, Manjianghong Fund azionista |
| **2022** | Lancio brand Livan (EV con Geely) |
| **Lug 2024** | Yin Qi acquisisce 19.91% per RMB 2.43B |
| **Nov 2024** | Yin Qi nominato Chairman |
| **Feb 2025** | Rinominata Qianli Technology |
| **Mar 2025** | JV con Geely, Miletrip, Lotus per guida autonoma |
| **Set 2025** | Mercedes-Benz acquisisce 3% per RMB 1.34B |
| **Ott 2025** | Filing IPO Hong Kong (A+H dual listing) |
