## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 6 dicembre 1962 |
| **Contesto** | Nazionalizzazione industria elettrica italiana |
| **Nome originale** | Ente Nazionale per l'Energia Elettrica |
| **Obiettivo** | Elettricita' a basso costo per sviluppo nazionale |

### La Nazionalizzazione (1962)

Nel 1962, l'Italia nazionalizzo' circa 1.250 compagnie elettriche private, creando Enel come ente statale. Al momento della fondazione:
- Popolazione: 51 milioni
- 2 milioni di italiani senza elettricita'
- 300.000 km di linee elettriche

Prime aziende acquisite:
- SIP (Piemonte)
- Edison Volta (Lombardia)
- SADE (Veneto)
- SELT-Valdarno (Toscana)
- SRE (Lazio)
- SME (Campania)
- SGES (Sicilia)
- Carbosarda (Sardegna)

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1962** | Fondazione come ente statale (nazionalizzazione) |
| **1992** | Trasformazione in SpA (governo Amato) |
| **1999** | Privatizzazione e IPO (decreto Bersani), quotazione Milano e NYSE |
| **1999** | Scissione: Enel Produzione, Enel Distribuzione, Terna |
| **2002-2005** | Paolo Scaroni CEO |
| **2005-2014** | Fulvio Conti CEO |
| **2007** | Acquisizione Endesa (con Acciona, €42.5 mld) |
| **2008** | Fondazione Enel Green Power |
| **2009** | Buyout quota Acciona in Endesa (>92%) |
| **2014-2023** | Francesco Starace CEO (3 mandati, focus rinnovabili) |
| **2023** | Paolo Scaroni Chairman, Flavio Cattaneo CEO |

## Pattern: La Privatizzazione Italiana

Enel rappresenta il modello di privatizzazione "all'italiana":

| Caratteristica | Dettaglio |
|----------------|-----------|
| **Controllo statale** | Ministero Economia mantiene 23.6% |
| **Golden power** | Governo mantiene poteri speciali |
| **Management** | Nomine concordate con governo |
| **IPO** | Maggiore della storia finanziaria (1999) |
