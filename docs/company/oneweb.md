# OneWeb

> Costellazione satellitare LEO per connettività globale, ora parte di Eutelsat

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2012 |
| **Sede** | Londra, Regno Unito |
| **Tipo** | Company (satellite telecom) |
| **Leadership** | Ora sotto Eutelsat Group |
| **Satelliti** | 648+ in LEO (~1.200 km) |
| **Valutazione merger** | $3.4 miliardi (2022) |
| **Status** | merged (Eutelsat 2023) |

---

## Glossario

| Termine | Definizione |
|---------|-------------|
| LEO | Low Earth Orbit, orbita bassa (1.200 km vs 36.000 km GEO) |
| Costellazione | Rete di satelliti coordinati per copertura globale |
| Backhaul | Connessione tra rete locale e backbone internet |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2012 (come WorldVu) |
| **Fondatore** | [Greg Wyler](../persons/greg-wyler.md) |
| **Contesto** | Dopo O3b (MEO), Wyler punta a LEO per latenza minima |
| **Obiettivo** | Connettività broadband globale per aree remote |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2012** | Fondazione come WorldVu da Greg Wyler |
| **2015** | Raccolta $500M, rinominata OneWeb |
| **2016** | [SoftBank](../company/softbank.md) investe $1B |
| **2017** | Wyler lascia la compagnia |
| **2019** | Lancio primi 6 satelliti (febbraio) |
| **2020** | Bancarotta Chapter 11 (marzo) |
| **2020** | Salvataggio: [UK Government](../government/governo-uk.md) $500M + [Bharti Global](../company/bharti-global.md) |
| **2021** | [Eutelsat](../company/eutelsat.md) entra come azionista |
| **2022** | Annuncio merger con Eutelsat ($3.4B) |
| **2023** | Completamento merger, diventa Eutelsat OneWeb |
| **2025** | Eutelsat unifica brand, OneWeb resta nome costellazione |

## Analisi: La Corsa allo Spazio per la Connettività

OneWeb rappresenta un caso emblematico di:

1. **Tech vision vs execution**: Wyler ha anticipato il mercato ma non ha retto la competizione con SpaceX
2. **Geopolitica spaziale**: UK ha investito $500M per mantenere asset strategico fuori da mani cinesi/russe
3. **Consolidamento**: Il merger con Eutelsat crea il secondo operatore multi-orbit dopo SpaceX
4. **India connection**: Bharti/Mittal porta connessione con mercato emergente cruciale
