## Fondazione e Storia

### Le Origini: Due Giganti del Mare

| Compagnia | Fondazione | Sede | Fondatore |
|-----------|------------|------|-----------|
| **HAPAG** | 1847 | Hamburg | Adolph Godeffroy |
| **Norddeutscher Lloyd** | 1857 | Bremen | H.H. Meier, E. Crusemann |

### Albert Ballin: Il Genio Ebreo di HAPAG

| Aspetto | Dettaglio |
|---------|-----------|
| **Nascita** | 1857, Hamburg (famiglia ebrea danese) |
| **Ruolo** | Direttore Generale HAPAG (1899-1918) |
| **Innovazioni** | Crociere di piacere, emigrazione di massa |
| **1913** | HAPAG = piu' grande compagnia navale del mondo (175 navi, 25,000 dipendenti) |
| **Morte** | 9 novembre 1918 (suicidio, giorno abdicazione Kaiser) |
| **Era nazista** | Nome cancellato (nave "Albert Ballin" → "Hansa") |
| **1947** | Riabilitazione: Ballindamm ad Amburgo |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1847** | Fondazione HAPAG (Hamburg-Amerikanische Packetfahrt-Actien-Gesellschaft) |
| **1857** | Fondazione Norddeutscher Lloyd (Bremen) |
| **1899** | Albert Ballin diventa DG di HAPAG |
| **1913** | HAPAG piu' grande shipping company del mondo |
| **1918** | Suicidio Ballin, perdita flotta (WWI reparations) |
| **1935** | Nazisti cancellano nome Ballin |
| **1945** | Perdita quasi totale flotta (WWII) |
| **1970** | **Fusione HAPAG + NDL = Hapag-Lloyd** |
| **1998** | Acquisizione da TUI AG |
| **2005** | Acquisizione CP Ships |
| **2008** | Albert Ballin Consortium (Hamburg + privati) compra da TUI |
| **2009** | TUI vende maggioranza a investitori Hamburg |
| **2014** | **Merger con CSAV** (Luksic entra) |
| **2017** | **Merger con UASC** (Qatar, Saudi entrano) |
| **2022** | Profitto record EUR 17B |

## Pattern Emergenti

### 1. Controllo Familiare Resiliente

Come BMW (Quandt) e VW (Porsche-Piech), Hapag-Lloyd e' controllata da un piccolo gruppo:
- Un miliardario tedesco (Kuhne)
- Una famiglia sudamericana (Luksic)
- Una citta'-stato (Hamburg)

### 2. Capitale del Golfo

Qatar (QIA) e' presente sia in Hapag-Lloyd (12.3%) che in Volkswagen (17%), Deutsche Bank (~6%), Siemens (4%).

### 3. Eredita' Storica Controversa

Sia HAPAG (Ballin) che Kuhne+Nagel hanno storie legate all'era nazista:
- Ballin: ebreo, suicida, nome cancellato
- Kuhne+Nagel: trasporto beni confiscati
