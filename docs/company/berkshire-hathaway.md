## Storia

### Timeline

| Anno | Evento |
|------|--------|
| **1839** | Fondata Valley Falls Company (tessile, Rhode Island) |
| **1955** | Fusione → Berkshire Hathaway Inc. |
| **1962** | Warren Buffett inizia ad acquistare azioni |
| **1965** | Buffett acquisisce controllo (industria tessile in declino) |
| **1967** | Prima acquisizione assicurativa (National Indemnity) |
| **1970** | Buffett diventa Chairman & CEO |
| **1978** | Charlie Munger Vice Chairman |
| **1985** | Chiusura definitiva operazioni tessili |
| **1996** | Azioni classe B (democratizzazione) |
| **2008** | Crisi: investe $5B in Goldman Sachs |
| **2010** | Acquisizione BNSF ($26.5B) |
| **2016** | Acquisizione Precision Castparts ($37.2B) |
| **2018** | Abel e Jain Vice Chairman |
| **2021** | Abel designato successore |
| **2023** | Morte Charlie Munger |
| **2024** | Market cap supera $1 trilione |
| **2025** | Buffett annuncia pensionamento, Abel CEO dal 2026 |

### L'Errore Tessile

> Buffett considera l'acquisto di Berkshire Hathaway il suo peggior investimento - un'azienda tessile morente che ha drenato capitale per 20 anni prima di chiudere.

La lezione: usare il "float" assicurativo per investire in aziende migliori.

## Filosofia Buffett

### Principi di Investimento

1. **Margine di sicurezza**: comprare sotto valore intrinseco
2. **Cerchio di competenza**: solo ciò che si capisce
3. **Moat economico**: vantaggi competitivi duraturi
4. **Holding period**: "Il nostro preferito è per sempre"

### Citazioni

> "Rule No. 1: Never lose money. Rule No. 2: Never forget rule No. 1."

> "Be fearful when others are greedy and greedy when others are fearful."

> "Price is what you pay. Value is what you get."
