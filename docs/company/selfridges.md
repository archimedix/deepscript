## Fondazione e Storia

| Dato | Dettaglio |

| **Luogo** | 400 Oxford Street, London |

### Harry Gordon Selfridge

| Aspetto | Dettaglio |
|---------|-----------|
| Nascita | 11 gennaio 1858, Ripon, Wisconsin, USA |
| Morte | 8 maggio 1947, Putney, London |
| Background | 25 anni a Marshall Field, Chicago |
| Intuizione | Londra mancava di department store all'americana |
| Fine | Estromesso dal board 1941, mori' in poverta' |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1858** | Nascita Harry Gordon Selfridge in Wisconsin |
| **1879** | Selfridge entra in Marshall Field, Chicago |
| **1906** | Selfridge visita Londra, identifica opportunita' |
| **1909** | Apertura Selfridges, Oxford Street (15 marzo) |
| **1910** | Supporto pubblico alle suffragette |
| **1919** | Acquisizione Brown Thomas (Irlanda) |
| **1940s** | Vendita negozi provinciali a John Lewis |
| **1941** | Selfridge estromesso dal board |
| **1947** | Morte di Harry Gordon Selfridge |
| **1951** | Acquisizione da Lewis's (Liverpool) |
| **1965** | Acquisizione da Sears Group (Charles Clore) |
| **1998** | Apertura Trafford Centre, Manchester |
| **2002** | Apertura Exchange Square, Manchester |
| **2003** | Galen Weston Sr acquisisce per £598M |
| **2003** | Apertura Bullring, Birmingham |
| **2011** | Acquisizione De Bijenkorf (Paesi Bassi) |
| **2015** | Acquisizione Arnotts (Irlanda) |
| **2021** | Accordo vendita a Central/Signa (~£4B) |
| **2022** | Completamento acquisizione (23 agosto) |
| **2023** | Signa Holding fallisce (novembre) |
| **2024** | PIF acquisisce quota Signa (40%) |

## Innovazioni Storiche

### Harry Gordon Selfridge: Pioniere del Retail

| Innovazione | Impatto |
|-------------|---------|
| "Shopping as entertainment" | Trasformo' acquisti in esperienza |
| Ristoranti in-store | Donne potevano pranzare senza accompagnatore |
| Supporto suffragette (1910) | Rifiuto' querela per vetrine rotte |
| "Only X Shopping Days Until Christmas" | Inizio' la tradizione a Marshall Field |
| Numero telefonico "Gerrard 1" | Marketing innovativo |
