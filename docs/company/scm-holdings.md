## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2000, Donetsk |
| **Fondatore** | Rinat Akhmetov |
| **Contesto** | Consolidamento degli asset acquisiti durante le privatizzazioni ucraine degli anni '90 |
| **Obiettivo** | Holding per gestire il portafoglio industriale di Akhmetov |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2000** | Fondazione di SCM a Donetsk |
| **2000** | Oleg Popov entra come Deputy CEO |
| **2006** | Popov diventa CEO |
| **2006** | Creazione di UMG (private equity arm) |
| **2007** | Fusione asset mining con Smart-Holding sotto Metinvest B.V. |
| **2011** | Acquisizione Ukrtelecom (via EPIC/ESU, completata 2013) |
| **2014** | Crisi Donbas: sede spostata a Kyiv, asset in zona di conflitto |
| **2018** | Ristrutturazione corporate: SCM Holdings Limited (Cipro) con Popov Director |
| **2022** | Invasione russa: Azovstal e Illich Steel distrutti a Mariupol |
| **2022** | Cessione Media Group Ukraine allo stato |
| **2022** | Causa CEDU contro Russia per danni di guerra |
| **2023** | Arbitrato investimenti bilaterali Ucraina-Russia |
| **2025** | Danni stimati $17-20 miliardi, di cui $11 mld solo per Azovstal e Illich |

## Impatto Invasione Russa 2022

| Aspetto | Dettaglio |
|---------|-----------|
| **Distruzione Mariupol** | Azovstal e Illich Steel completamente distrutti durante l'assedio |
| **Danni stimati** | $17-20 miliardi totali; $11 mld solo Azovstal + Illich |
| **Crimea (2014)** | DTEK Krymenergo espropriata; arbitrato vinto per $270M |
| **CEDU** | Causa contro Russia depositata 2022 |
| **Arbitrato BIT** | Claim sotto trattato bilaterale Ucraina-Russia |
| **Aiuti umanitari** | $350M+ tramite SCM, Akhmetov Foundation e Shakhtar |
| **Cessione media** | Media Group Ukraine trasferita allo stato per evitare status oligarca |
