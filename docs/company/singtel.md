## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origini** | 1879 - primi telefoni a Singapore |
| **Singapore Telephone Board** | 1955 - governo coloniale |
| **Telecoms** | 1974 - fusione STB + TAS |
| **Incorporazione SingTel** | Marzo 1992 |
| **IPO** | Ottobre 1993, SGX |
| **Fine monopolio** | 2000 |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1879** | Prime linee telefoniche a Singapore |
| **1955** | Singapore Telephone Board (governo coloniale) |
| **1974** | Fusione in Telecoms |
| **1992** | Incorporazione Singapore Telecommunications Pte Ltd |
| **1993** | IPO - Temasek mantiene 89% |
| **1995** | Lee Hsien Yang diventa CEO |
| **1997** | Compensazione $1.5B per fine monopolio anticipata |
| **2000** | Fine monopolio domestico |
| **2001** | Acquisizione Optus (Australia) per A$16B |
| **2007** | Lee Hsien Yang lascia, Chua Sock Koong CEO |
| **2021** | Yuen Kuan Moon CEO |
