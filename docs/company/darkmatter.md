## Glossario

| Termine | Definizione |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2014 |
| **Fondatore** | Faisal Al Bannai (figlio di un generale della Dubai Police) |
| **Contesto** | UAE voleva capacita' cyber offensive autonome |
| **Obiettivo dichiarato** | Cybersecurity difensiva per il governo e il settore privato |
| **Obiettivo reale** | Sorveglianza di dissidenti, giornalisti, governi stranieri |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2014** | Fondazione da parte di Faisal Al Bannai |
| **2016** | Subentra a CyberPoint come contractor per Project Raven |
| **2016-2019** | Baier, Adams, Gericke sviluppano exploit Karma/Karma 2 per iOS |
| **2018** | Fatturato ~$400M, 80% da governo UAE |
| **2018** | Peng Xiao lascia DarkMatter, fonda G42 (spin-off da subsidiary In-Q) |
| **2019** | Faisal Al Bannai nominato CEO di EDGE Group (conglomerato difesa UAE) |
| **2019** | Attivita' cyber offensive iniziano trasferimento a Digital14/EDGE |
| **2021** | Tre ex NSA (Baier, Adams, Gericke) firmano DPA con DOJ USA |
| **2021** | Attivita' residue trasferite a Digital14 (secure comms Katim) |
