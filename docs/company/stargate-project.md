# Stargate Project

> Il piu' grande investimento in infrastruttura AI della storia - $500B per dominare l'era dell'intelligenza artificiale

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | Gennaio 2025 |
| **Sede** | Delaware, USA |
| **Tipo** | Joint Venture |
| **Chairman** | Masayoshi Son (SoftBank) |
| **Investimento totale** | $500B (entro 2029) |
| **Investimento immediato** | $100B |
| **Capacita' target** | 10 GW |
| **Status** | active |

---

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Annuncio** | 21 gennaio 2025, Casa Bianca |
| **Annunciato da** | Presidente Donald Trump |
| **Fondatori** | SoftBank, OpenAI, Oracle, MGX |
| **Obiettivo** | Costruire infrastruttura AI dominante in USA |

### Timeline

| Data | Evento |
|------|--------|
| **21 Gen 2025** | Annuncio alla Casa Bianca con Trump |
| **Apr 2025** | SoftBank annuncia $10B finanziamento da Mizuho |
| **Mag 2025** | JPMorgan presta $2.3B per data center Abilene |
| **Mag 2025** | Annunciato UAE Stargate (Nvidia, Cisco, OpenAI) |
| **Lug 2025** | Tensioni OpenAI-SoftBank su location |
| **Ago 2025** | Bloomberg: ritardi, nessun fondo raccolto per $500B |
| **Set 2025** | Annunciati 5 nuovi siti (Texas, NM, Ohio, Midwest) |
| **Ott 2025** | Stargate Argentina annunciato ($25B) |

## Finanziamento

### Capitale Annunciato

| Fonte | Importo | Status |

### Sfide

| Issue | Dettaglio |
|-------|-----------|
| **Funding gap** | Ago 2025: nessun fondo raccolto per $500B |
| **Market uncertainty** | Trade policy, AI hardware valuations |
| **Internal tensions** | Disaccordi OpenAI-SoftBank su location |
| **Oracle friction** | Deal $30B OpenAI-Oracle separato da Stargate |
