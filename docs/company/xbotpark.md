## Il Modello di Incubazione

### Differenze vs Silicon Valley

| Aspetto | Silicon Valley | XbotPark |
|---------|----------------|----------|
| **Focus** | Software, SaaS | Hardware, robotics |
| **Time to prototype** | 6-12 mesi | 1-2 mesi |
| **Costo sviluppo** | 1x | 0.2-0.25x |
| **Supply chain** | Global, frammentata | Locale, 1-2h da Shenzhen |
| **Educazione** | Separata | Integrata (learn by doing) |

### Pipeline

```
Studenti/Founders
       │
       ▼
  Selezione XbotPark
       │
       ▼
┌──────────────────┐
│  INCUBAZIONE     │
│  - Mentorship    │
│  - Seed funding  │
│  - Office space  │
│  - Supply chain  │
└────────┬─────────┘
         │
         ▼
   Prototipo (1-2 mesi)
         │
         ▼
┌──────────────────┐
│  SCALING         │
│  - Series A      │
│  - Manufacturing │
│  - Distribution  │
└────────┬─────────┘
         │
         ▼
  Unicorn / IPO / Exit
```

### Pattern: HKUST → XbotPark → Unicorn

```
HKUST (Hong Kong)
       │
       ├── Li Zexiang (Professor 1992-)
       │         │
       │         ├── Lab 3126 (50+ startup)
       │         │         │
       │         │         └── DJI ($15B)
       │         │
       │         └── XbotPark (2014)
       │                   │
       │                   ├── Narwal ($1B+)
       │                   ├── HAI Robotics ($1B+)
       │                   ├── EcoFlow ($1B+)
       │                   ├── CiDi ($1B+)
       │                   └── (altri)
       │
       └── Gao Bingqiang (Dean → Angel)
                 │
                 └── Early investor DJI, Googol, XbotPark
```

**Lezione**: XbotPark rappresenta la scalabilita' del modello Li Zexiang:
1. **Lab universitario** (HKUST) → prova del concept
2. **Incubatore dedicato** (XbotPark) → pipeline industrializzata
3. **Network governativo** (Dongguan) → infrastrutture e policy
4. **Supply chain locale** (Greater Bay Area) → speed to market
5. **Capitale paziente** (Clear Water Bay VC) → focus long-term
