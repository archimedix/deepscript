## Storia

### Pre-privatizzazione

Thames Water era parte del sistema pubblico di gestione idrica UK.

### Privatizzazione (1989)

| Aspetto | Dettaglio |
|---------|-----------|
| **Anno** | 1989 |
| **Contesto** | Privatizzazione settore idrico UK (Thatcher) |
| **Debito iniziale** | £0 (venduta senza debito) |
| **Nome** | Thames Water plc |

### RWE AG (2001-2006)

| Aspetto | Dettaglio |
|---------|-----------|
| **Acquirente** | RWE AG (Germania) |
| **Anno** | 2001 |
| **Dividendi a RWE** | £1B+ in 5.5 anni |
| **Critiche** | Leakage record, £270M dividendi nel 2006 |

### Macquarie (2006-2017)

| Aspetto | Dettaglio |
|---------|-----------|
| **Acquirente** | Kemble Water (consorzio Macquarie) |
| **Anno** | 2006 |
| **Prezzo** | £2.3B |
| **Consorzio** | 14 investitori internazionali |
| **Fondi principali** | MEIF, MEIF II |

### Timeline

| Anno | Evento |
|------|--------|
| **1989** | Privatizzazione, debito £0 |
| **2001** | Acquisita da RWE AG |
| **2006** | Venduta a Macquarie consortium (£2.3B) |
| **2006** | Debito: £3.2B |
| **2017** | Macquarie esce, debito: £10.5B |
| **2017** | Dividendi pagati da Macquarie: £2.8B |
| **2024** | Kemble Water default su £190M |
| **2024** | Richiesta aumento bollette 40% rifiutata |
| **Dic 2024** | Multa Ofwat £18.2M per dividendi illegali |
| **Feb 2025** | Prestito £3B approvato da High Court |
| **Mar 2025** | KKR nominato bidder preferito |
| **Giu 2025** | KKR si ritira |
| **Lug 2025** | Perdita £1.65B, debito £16.8B |
| **Ago 2025** | DEFRA prepara SAR (nazionalizzazione) |

## L'Era Macquarie (2006-2017)

### Il Modello di Estrazione

| Metrica | 2006 | 2017 | Variazione |
|---------|------|------|------------|
| **Debito** | £3.2B | £10.5B | **+228%** |
| **Dividendi pagati** | - | £2.8B | - |
| **RAB** | £6.5B | £12.9B | +98% |

### Come Hanno Fatto

| Meccanismo | Dettaglio |
|------------|-----------|
| **Whole Business Securitisation** | Ricavi futuri usati come garanzia prestiti |
| **Cayman Islands** | Prestiti via subsidiary offshore |
| **Dividend recapitalization** | Anno 1: dividendi £656M vs profitti £241M |
| **Transfer pricing** | Costi acquisizione trasferiti a Thames |

### Il Primo Anno

| Metrica | Valore |
|---------|--------|
| **Profitti** | £241M |
| **Dividendi** | £656M |
| **Rapporto** | Dividendi = 272% dei profitti |

### Risposta Macquarie

> "Durante la nostra gestione Thames Water ha investito oltre £11 miliardi, circa £1 miliardo all'anno, più del doppio degli investimenti nei 5 anni pre-privatizzazione."

**Critica**: Gran parte degli "investimenti" erano finanziati con debito, non con capitale proprio.

## Pattern: Privatizzazione UK

Thames Water rappresenta il **fallimento del modello privatizzazione UK**:

1. **Monopolio naturale** venduto a privati
2. **Debito zero** alla vendita
3. **WBS** permette leverage estremo
4. **Dividendi** pagati con debito
5. **Regolatore debole** (Ofwat)
6. **Exit PE** con profitto, debito resta
7. **Crisi** → contribuenti pagano
