## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Gennaio 1994 (incorporated 2 marzo 1995) |
| **Fondatori** | Jerry Yang, David Filo |
| **Contesto** | Progetto PhD a Stanford, directory web |
| **Nome originale** | "Jerry and David's Guide to the World Wide Web" |
| **Obiettivo dichiarato** | Organizzare il web in una directory navigabile |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1994** | Yang e Filo creano directory web a Stanford |
| **1995** | Incorporazione; Sequoia Capital investe $1M per 25% (Michael Moritz); Tim Koogle primo CEO |
| **1996** | IPO Nasdaq ($33.8M); JV Yahoo Japan con SoftBank |
| **1999** | Acquisizione GeoCities ($4B+) e Broadcast.com ($5.7B) |
| **2001** | Terry Semel CEO (ex-Warner Bros.) |
| **2005** | Acquista 40% Alibaba per $1B; acquisisce Flickr |
| **2007** | Jerry Yang torna come CEO |
| **2008** | Rifiuta offerta Microsoft $44.6B |
| **2009** | Carol Bartz CEO |
| **2011** | Bartz licenziata per telefono; Daniel Loeb (Third Point) entra come attivista |
| **2012** | Scott Thompson CEO (dimesso in 4 mesi per resume fraud); Marissa Mayer CEO |
| **2013** | Acquisisce Tumblr ($1.1B); data breach 3 miliardi di account |
| **2017** | Verizon acquisisce core business per $4.48B; asset Alibaba → Altaba |
| **2019** | Altaba vende quota Alibaba (~$46B) e si dissolve |
| **2021** | Apollo Global acquisisce 90% per $5B; Jim Lanzone CEO |

## Critiche e Controversie

| Tema | Dettaglio |
|------|-----------|
| Caso Shi Tao (2004-2005) | Yahoo China consegno' dati utente al governo cinese; giornalista condannato a 10 anni per "segreti di stato" |
| Data breach 2013 | 3 miliardi di account compromessi, il piu' grande nella storia, rivelato solo nel 2016 |
| Data breach 2014 | 500+ milioni di account, rivelato 2016 |
| Rifiuto Microsoft (2008) | Offerta $44.6B rifiutata; core business venduto per $4.48B nel 2017 |
| Resume fraud CEO (2012) | Scott Thompson falsifico' laurea in CS, dimesso dopo 4 mesi |
| Distruzione valore acquisizioni | GeoCities ($4B), Broadcast.com ($5.7B), Tumblr ($1.1B) — tutte fallimentari |
