## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Match.com** | 1993, Gary Kremen e Peng T. Ong, San Francisco |
| **IAC acquisisce Match.com** | 1999 |
| **Match Group** | 2009, IAC raggruppa i brand dating sotto Match Group |
| **Modello** | Acquisizioni seriali per dominare il mercato dating online |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1993** | Gary Kremen fonda Match.com (Electric Classifieds Inc.) |
| **1995** | Lancio sito beta, feature su Wired |
| **1999** | IAC acquisisce Match.com |
| **2009** | IAC crea formalmente Match Group come holding dating |
| **2011** | Acquisizione OkCupid ($50M) |
| **2012** | Tinder nasce in Hatch Labs (incubatore IAC) |
| **2015** | Acquisizione Plenty of Fish; IPO Match Group su NASDAQ |
| **2017** | Investimento in Hinge |
| **2018** | Acquisizione completa Hinge |
| **2019** | Acquisizione Hyperconnect (espansione Asia) |
| **2020** | Spin-off completo da IAC; Match Group diventa indipendente |
| **2022** | Settlement $441M con fondatori Tinder |
| **2025** | Spencer Rascoff (co-fondatore Zillow) diventa CEO |

### Rete Barton-Rascoff-Diller

Spencer Rascoff come CEO Match Group chiude il cerchio della rete Diller:
- Rich Barton fonda Expedia (per Microsoft, sotto IAC) → poi co-fonda Zillow con Rascoff
- Rascoff co-fonda anche Hotwire (brand Expedia)
- Rascoff ora guida Match Group, spin-off IAC
- Stessa rete di persone ruota tra travel, real estate, dating
