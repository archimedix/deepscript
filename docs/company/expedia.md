## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1996, come divisione di Microsoft |
| **Fondatore** | Rich Barton (in Microsoft) |
| **IPO** | Novembre 1999 (Microsoft deteneva 86,4%) |
| **Acquisizione IAC** | 2001-2003, IAC (Barry Diller) acquisisce il controllo |
| **Spin-off** | 2005, IAC cede Expedia come entita' separata; Diller resta chairman |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1996** | Fondata come divisione Microsoft |
| **1999** | IPO su NASDAQ |
| **2001** | IAC (USA Networks) acquisisce quota di controllo |
| **2003** | IAC completa l'acquisizione |
| **2005** | Spin-off da IAC; Barry Diller chairman, Dara Khosrowshahi CEO |
| **2012** | Acquisizione Trivago (quota di maggioranza) |
| **2015** | Acquisizione Orbitz Worldwide e Travelocity |
| **2015** | Acquisizione HomeAway (poi rinominata Vrbo) |
| **2017** | Khosrowshahi lascia per diventare CEO Uber |
| **2019** | Acquisizione Liberty Expedia Holdings |
| **2020** | Peter Kern diventa CEO |
| **2023** | Lancio One Key (programma fedelta' unificato) |
| **2024** | Ariane Gorin diventa CEO |

## Azionariato e Controllo

| Azionista | Quota | Note |
|-----------|-------|------|
| Barry Diller | ~29% voting power | Controllo via azioni classe B |
| Vanguard Group | ~12% | Istituzionale |
| BlackRock | ~7,6% | Istituzionale |

**Struttura di controllo**: Diller mantiene il controllo tramite azioni a voto multiplo ereditate dalla struttura IAC. Nonostante una quota economica ridotta, detiene circa il 29% del potere di voto.

## Connessioni PowerLink

### Rete Barry Diller

Barry Diller controlla sia IAC che Expedia Group, creando un ecosistema interconnesso:
- **IAC/InterActiveCorp**: holding madre, ha generato Expedia via spin-off
- Chelsea Clinton siede nei board di entrambe (IAC dal 2011, Expedia dal 2017)
- Alexander von Furstenberg (figlio di Diane von Furstenberg, moglie di Diller) nel board Expedia
