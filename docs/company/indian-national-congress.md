## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 28 dicembre 1885, Bombay |
| **Fondatori** | A.O. Hume, Dadabhai Naoroji, Dinshaw Wacha |
| **Primo Presidente** | W.C. Bonnerjee |
| **Contesto** | Piattaforma per dialogo civile tra indiani istruiti sotto dominio britannico |
| **Obiettivo iniziale** | Riforme graduali, autogoverno |
| **Evoluzione** | Da movimento moderato a lotta per indipendenza completa |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1885** | Fondazione a Bombay con 72 delegati |
| **1906** | Dadabhai Naoroji presidente per la terza volta |
| **1917** | Annie Besant prima donna presidente |
| **1920** | Gandhi assume leadership movimento non-cooperazione |
| **1929** | Nehru presidente; dichiarazione Purna Swaraj (indipendenza totale) |
| **1942** | Movimento Quit India |
| **1947** | Indipendenza; Nehru primo PM |
| **1969** | Scissione: Congress (R) di Indira vs Congress (O) |
| **1977** | Prima sconfitta elettorale (Emergency) |
| **1984** | Assassinio Indira; Rajiv PM piu' giovane |
| **1991** | Assassinio Rajiv; riforme economiche sotto Rao |
| **1998** | Sonia Gandhi presidente |
| **2004** | Vittoria UPA; Manmohan Singh PM |
| **2014** | Sconfitta storica contro BJP di Modi |
| **2022** | Kharge primo presidente non-Gandhi in 24 anni |
| **2024** | Recupero a 99 seggi, opposizione ufficiale |

## Dinastia Nehru-Gandhi

```
Motilal Nehru (1861-1931)
    |-- INC President 1919-20, 1928-29
    |
    v
Jawaharlal Nehru (1889-1964)
    |-- PM 1947-1964
    |
    v
Indira Gandhi nee Nehru (1917-1984)
    |-- PM 1966-77, 1980-84
    |-- sposata Feroze Gandhi (non parente del Mahatma)
    |
    +-- Rajiv Gandhi (1944-1991)
        |-- PM 1984-1989
        |-- sposato Sonia Maino (ITA)
        |
        +-- Rahul Gandhi (1970)
        |   |-- INC President 2017-2019
        |
        +-- Priyanka Gandhi (1972)
            |-- AICC General Secretary
```

**Nota**: I Gandhi della dinastia Nehru non sono imparentati con Mahatma Gandhi.

## Ideologia e Posizioni

| Tema | Posizione |
|------|-----------|
| **Economia** | Economia mista, welfare state, poi liberalizzazione (1991) |
| **Secolarismo** | Laicita' dello stato, pluralismo religioso |
| **Politica estera** | Non-allineamento (Cold War), poi pragmatismo |
| **Riforme sociali** | Affirmative action, diritti delle donne |
