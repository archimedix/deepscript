## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Agosto 2008 |
| **Fondatori** | Brian Chesky, Joe Gebbia, Nathan Blecharczyk |
| **Contesto** | Chesky e Gebbia non riuscivano a pagare l'affitto a SF; affittano materassi ad aria durante una conferenza di design |
| **Obiettivo dichiarato** | Creare un mercato globale per esperienze di ospitalita' peer-to-peer |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2007** | Chesky si trasferisce da Gebbia a SF, idea "AirBed & Breakfast" |
| **2008** | Fondazione, primo sito web (Ruby on Rails, Blecharczyk) |
| **2009** | Y Combinator (Winter 2009): $20K seed; Sequoia investe $585K |
| **2011** | 1 milione di notti prenotate; espansione internazionale |
| **2014** | Valutazione $10B, round Series D |
| **2016** | Lancio "Experiences"; fondatori aderiscono al Giving Pledge |
| **2017** | Blecharczyk diventa CSO; Gebbia chairman Airbnb China |
| **2019** | Angela Ahrendts entra nel board |
| **2020** | COVID: crollo e ripresa; IPO dicembre (NASDAQ: ABNB), apertura a $146 (+113%) |
| **2022** | Gebbia lascia ruolo operativo; 7M+ annunci in 220+ paesi |
| **2025** | Gebbia entra in DOGE come Chief Design Officer USA; Super PAC "Affordable New York" ($5M) |

## Potere e Impatto

### Disruption dell'Ospitalita'

| Aspetto | Dettaglio |
|---------|-----------|
| Scala | 7M+ annunci, 220+ paesi, 5M+ host |
| Modello | Economia della condivisione: trasforma proprietari in micro-imprenditori |
| Impatto housing | Accusata di contribuire alla crisi abitativa nelle citta' turistiche |
| Revenue | ~$10B+ annui |

### Potere Politico

| Aspetto | Dettaglio |
|---------|-----------|
| Lobbying USA | Super PAC "Affordable New York" ($5M, 2025) |
| Lobbying EU | Home-sharing clubs come grassroots lobby in citta' sotto pressione regolatoria |
| Revolving door | Gebbia da Airbnb a DOGE/governo USA (2025) |
| Obama connection | Chesky donatore $100M alla Obama Foundation |
| Battaglie regolatorie | NYC, Barcellona, Firenze, Parigi — scontri con amministrazioni locali |

### Italia

| Aspetto | Dettaglio |
|---------|-----------|
| Regolamentazione | Nuove norme sugli affitti brevi in preparazione |
| Overtourism | Proteste anti-turismo a Firenze, Venezia, Roma (2025) |
| Impatto fiscale | Dibattito su tassazione degli host e cedolare secca |

|------|-----------|
