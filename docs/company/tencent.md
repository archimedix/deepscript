## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Novembre 1998 |
| **Fondatori** | Ma Huateng (Pony Ma), Zhang Zhidong, Xu Chenye, Chen Yidan, Zeng Liqing |
| **Contesto** | Boom internet cinese, necessita' di IM locale |
| **Obiettivo dichiarato** | Servizi internet per consumatori cinesi |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1998** | Fondazione a Shenzhen da 5 co-fondatori |
| **1999** | Lancio OICQ (poi QQ), instant messenger |
| **2000** | Investimento IDC/PCCW ($2.2M per 40%) |
| **2001** | Naspers acquista 46.5% ($32M) - investimento cruciale |
| **2004** | IPO Hong Kong Stock Exchange (HKEx: 0700) |
| **2008** | Entra nell'Hang Seng Index |
| **2011** | Lancio WeChat (Weixin) |
| **2012** | Investimento 40% in Epic Games |
| **2014** | Acquisisce 15% JD.com |
| **2015** | Acquisisce 100% Riot Games (League of Legends) |
| **2017** | Prima azienda asiatica a superare $500B market cap |
| **2018** | Supera $500B market cap |
| **2021** | Si avvicina a $1T market cap |
| **2025** | Inserita nella lista DoD "Chinese Military Companies" |

## Settori di Business

1. **Gaming** - Maggiore azienda gaming al mondo per revenue
2. **Social Media** - WeChat (1B+ utenti), QQ
3. **Fintech** - WeChat Pay, Tencent FiT
4. **Cloud** - Tencent Cloud (terzo in Cina)
5. **Media & Entertainment** - Tencent Video, Tencent Music
6. **AI** - Sviluppo modelli proprietari
