## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1 gennaio 1925 |
| **Origine** | Fusione dipartimento ingegneria Western Electric + AT&T R&D |
| **Primo presidente** | Frank B. Jewett |
| **Obiettivo** | Ricerca fondamentale e sviluppo tecnologico per il sistema telefonico |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1925** | Fondazione Bell Telephone Laboratories |
| **1937** | Primo Nobel: Clinton Davisson (diffrazione elettroni) |
| **1947** | Invenzione del transistor (Shockley, Bardeen, Brattain) |
| **1948** | Claude Shannon pubblica teoria dell'informazione |
| **1954** | Prima cella solare pratica |
| **1956** | Nobel per il transistor |
| **1958** | Primo laser |
| **1962** | Telstar 1, primo satellite comunicazioni |
| **1969** | Unix e linguaggio C |
| **1978** | Nobel: Penzias e Wilson (radiazione cosmica di fondo) |
| **1984** | AT&T divestiture; Bell Labs resta con AT&T |
| **1996** | Spin-off in Lucent Technologies |
| **2006** | Lucent acquisita da Alcatel |
| **2016** | Nokia acquisisce Alcatel-Lucent, diventa Nokia Bell Labs |

## Modello Bell Labs

### Filosofia di Ricerca

Bell Labs e' stato il prototipo del laboratorio di ricerca industriale moderno:

1. **Ricerca fondamentale** - Liberta' di esplorare senza obiettivi commerciali immediati
2. **Interdisciplinarita'** - Fisici, chimici, ingegneri, matematici insieme
3. **Prossimita' fisica** - Edifici progettati per incontri casuali
4. **Tenure di fatto** - Stabilita' lavorativa per ricercatori
5. **Monopolio benevolo** - AT&T poteva investire a lungo termine

### Declino

| Fattore | Impatto |
|---------|---------|
| **1984 divestiture** | Fine monopolio AT&T, pressione sui costi |
| **1996 spin-off** | Lucent deve generare profitti trimestrali |
| **Dot-com bust** | Crollo investimenti telecom |
| **Acquisizioni** | Perdita identita' culturale |

## Eredita'

Bell Labs ha creato le fondamenta tecnologiche dell'era digitale:

- **Transistor** → Microprocessori → Smartphone
- **Teoria informazione** → Internet → AI
- **Unix/C** → Linux → Cloud computing
- **Laser** → Fibra ottica → Telecomunicazioni globali
- **CCD** → Fotografia digitale → Imaging medico
