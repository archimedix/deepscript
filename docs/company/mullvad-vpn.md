## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Marzo 2009 |
| **Fondatori** | Fredrik Strömberg, Daniel Berntsson |
| **Contesto** | Protesta contro la FRA law svedese sulla sorveglianza di massa |
| **Obiettivo dichiarato** | Rendere la sorveglianza di massa e la censura impraticabili |
| **Parent company** | Amagicom AB (100% Strömberg e Berntsson) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2008** | Inizio sviluppo come risposta alla FRA law svedese |
| **2009** | Lancio Mullvad VPN AB |
| **2019** | Partnership con Mozilla per Mozilla VPN (server WireGuard) |
| **2020** | Client nativi iOS e Android con WireGuard |
| **2023 apr** | Polizia svedese (NOA) perquisisce la sede su mandato tedesco; nessun dato trovato |
| **2023** | Lancio Mullvad Browser in collaborazione con Tor Project |
| **2023** | Migrazione a server diskless RAM-only |

## Prodotti e Servizi

| Prodotto | Descrizione |
|----------|-------------|
| **Mullvad VPN** | VPN no-log, protocolli WireGuard e OpenVPN |
| **Mullvad Browser** | Browser privacy-first, co-sviluppato con Tor Project |
| **Mullvad DNS** | DNS criptato (DNS over HTTPS) |
| **Mullvad LETA** | Motore di ricerca privacy (proxy Google) |

### Infrastruttura Tecnica

- Server diskless RAM-only (dal 2023)
- Protocolli: WireGuard, OpenVPN
- Pagamenti anonimi: cash, crypto (BTC, BCH, Monero)
- Sistema account numerico (no email/password richiesti)
