## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2007, San Francisco |
| **Fondatori** | [Drew Houston](../persons/drew-houston.md), [Arash Ferdowsi](../persons/arash-ferdowsi.md) |
| **Contesto** | Houston frustrato dal portare USB drive e email di file a se stesso |
| **Acceleratore** | [Y Combinator](../private-equity/y-combinator.md) W07 batch |
| **Obiettivo** | Sincronizzazione file seamless tra dispositivi |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2007** | Drew Houston e Arash Ferdowsi fondano Dropbox, entrano in Y Combinator W07 |
| **2007** | Sequoia Capital investe nel seed round |
| **2008** | Accel Partners entra come investor |
| **2011** | Series B $250M guidata da Index Ventures, valutazione $4B |
| **2014** | Series C valutazione $10B |
| **2018** | IPO NASDAQ a $21/share, primo giorno chiude a $28.48 (+36%) |
| **2025** | Warren Jenson (ex-Amazon, Electronic Arts) entra nel board |

## Pattern di Potere

### "YC Mafia"

Dropbox rappresenta il caso emblematico del modello Y Combinator:
- **MIT founders** (Houston, Ferdowsi)
- **YC accelerator** → credibilita' + network
- **Sequoia lead** → capital + governance
- **NASDAQ IPO** → liquidita' per tutti

### Silicon Valley Ecosystem

```
MIT → Y Combinator → Sequoia → NASDAQ
 │         │            │         │
 └─────────┴────────────┴─────────┘
           Drew Houston
```

### Forum Presence

A differenza di altri fondatori Big Tech (Zuckerberg, Bezos), Drew Houston ha:
- **WEF Davos:** Partecipante dal 2013
- **Bilderberg:** Non documentato
- **CFR:** Non documentato

Pattern simile a altri fondatori "builder" (non "forum elite").
