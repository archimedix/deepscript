## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2019 |
| **Fondatori** | Aidan Gomez, Ivan Zhang, Nick Frosst |
| **Contesto** | Tre giovani canadesi, uno co-autore del paper Transformer, creano alternativa enterprise a OpenAI |
| **Obiettivo dichiarato** | Portare AI enterprise-grade con focus su privacy, sicurezza e deployment on-premise |

## Prodotti

| Prodotto | Descrizione |
|----------|-------------|
| **Command A** | Flagship model 2025 |
| **Command R+** | Enterprise RAG-optimized (2024) |
| **Command R** | Mid-tier, 128K context |
| **Embed** | Text embeddings per search |
| **Rerank** | Semantic reranking |
| **Coral** | Enterprise chat interface |

### Focus Enterprise

A differenza di OpenAI (consumer) e Anthropic (safety), Cohere si concentra esclusivamente su enterprise:
- Deployment on-premise e cloud privato
- No training su dati clienti
- Multilingual (23+ lingue)
- RAG-first design
