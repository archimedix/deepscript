# Africa Israel Investments

> Conglomerato israeliano fondato nel 1934: dal mandato britannico all'era Leviev, un impero di real estate, costruzioni e infrastrutture tra Israele, Russia e USA, crollato con la crisi del 2008.

## Executive Summary

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1934, durante il Mandato Britannico di Palestina |
| **Fondatori** | Investitori ebrei sudafricani per sviluppo economico dello Yishuv |
| **Espansione post-1948** | Ampliamento attivita' commerciali e finanziarie dopo la fondazione di Israele |
| **TASE** | Tra le prime aziende quotate alla Borsa di Tel Aviv (anni '70) |
| **Bank Leumi** | Controllo acquisito dalla banca negli anni '70 |
| **Era Leviev** | Novembre 1996: Lev Leviev acquista il controllo da Bank Leumi |

### Timeline

| Anno | Evento |
|------|--------|
| **1934** | Fondazione da investitori ebrei sudafricani durante il Mandato Britannico |
| **Anni '70** | Controllo acquisito da Bank Leumi; quotazione TASE |
| **1996-97** | Lev Leviev acquista il controllo da Bank Leumi |
| **2001** | Fondazione AFI Development Plc per real estate in Russia (Mosca) |
| **2002** | Fondazione AFI USA per progetti residenziali esclusivi (NYC, Miami, LA, Phoenix) |
| **2007** | Acquisto The Apthorp (NYC), edificio residenziale storico |
| **2008** | Crisi finanziaria globale: crollo valori immobiliari in USA, Russia, Est Europa |
| **2010** | Default su obbligazioni; ristrutturazione 7.4 miliardi di shekel di debito |
| **2016** | Secondo accordo di ristrutturazione del debito |
| **2018** | Leviev perde il controllo durante la ristrutturazione |
| **2020** | Delisting dalla TASE dopo 47 anni; transizione a societa' privata |

## Crisi 2008 e Ristrutturazione

| Aspetto | Dettaglio |
|---------|-----------|
| **Causa** | Esposizione massiva al real estate in USA, Russia e Est Europa con alta leva |
| **Default** | Serie di obbligazioni in default |
| **Debito ristrutturato** | 7.4 miliardi di shekel (2010) |
| **Seconda ristrutturazione** | 2016 |
| **Conseguenza** | Leviev perde il controllo; delisting TASE (2020) |
