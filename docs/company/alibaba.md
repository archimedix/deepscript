## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 28 giugno 1999 |
| **Fondatori** | Jack Ma + 17 co-fondatori (inclusi Joe Tsai, Eddie Wu) |
| **Contesto** | Boom internet, necessita' B2B per PMI cinesi |
| **Obiettivo dichiarato** | "Make it easy to do business anywhere" |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1999** | Fondazione a Hangzhou (appartamento di Jack Ma) |
| **2000** | SoftBank investe $20M (Masayoshi Son) |
| **2003** | Lancio Taobao (C2C marketplace) |
| **2004** | Lancio Alipay (pagamenti digitali) |
| **2005** | Yahoo! acquista 40% per $1B |
| **2008** | Lancio Tmall (B2C) |
| **2009** | Alibaba Cloud (Aliyun) |
| **2013** | Fondazione Cainiao (logistica) |
| **2014** | IPO NYSE ($25B) - piu' grande IPO della storia |
| **2019** | Jack Ma lascia ruolo di Chairman |
| **2020** | IPO Ant Group bloccata da regolatori |
| **2021** | Multa antitrust $2.8B da governo cinese |
| **2023** | Joe Tsai Chairman, Eddie Wu CEO |
| **2024** | SoftBank disinveste completamente |
| **2025** | Piano investimenti AI/Cloud RMB 380B in 3 anni |
