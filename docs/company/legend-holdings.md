## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1 novembre 1984 |
| **Fondatori** | Liu Chuanzhi + 10 ingegneri CAS |
| **Capitale iniziale** | RMB 200,000 |
| **Origine** | Institute of Computing Technology, CAS |
| **Location originale** | Stanza di 20mq a Beijing |

Liu Chuanzhi, ingegnere frustrato della Chinese Academy of Sciences, fondo' Legend con 10 colleghi in una stanza di 20 metri quadri. L'azienda sviluppo' il primo sistema di caratteri cinesi per computer (1985), poi divenne produttore di PC (1990).

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1984** | Fondazione Legend Group (11 ingegneri CAS) |
| **1985** | Sviluppo sistema caratteri cinesi per PC |
| **1990** | Inizio produzione PC |
| **1994** | IPO Hong Kong Stock Exchange |
| **1997** | #1 PC maker in Cina |
| **2001** | Creazione Legend Holdings come holding diversificata |
| **2003** | Fondazione Hony Capital (PE) |
| **2004** | Rebranding Legend → Lenovo |
| **2005** | Acquisizione IBM PC Division (ThinkPad) |
| **2012** | Liu Chuanzhi lascia chairmanship Legend Holdings |
| **2013** | Lenovo diventa #1 PC mondiale |
| **2015** | IPO Legend Holdings (HKEX) |
| **2018** | Acquisizione 89.9% BIL (Lussemburgo) |
| **2019** | Liu Chuanzhi si ritira definitivamente |
| **2020** | Ning Min diventa Chairman |
| **2024** | Partnership strategica con PIF/Alat (Saudi) |

## Pattern: Da Laboratorio a Conglomerato

Legend Holdings rappresenta il modello cinese di trasformazione:

1. **Origine accademica** (CAS) → legittimita' statale
2. **Campione nazionale** (Lenovo) → scala globale
3. **Diversificazione** (PE, VC, banking) → resilienza
4. **Acquisizioni occidentali** (IBM, BIL) → know-how globale

### Confronto con Modelli Simili

| Gruppo | Origine | Core Business | Diversificazione |
|--------|---------|---------------|------------------|
| Legend Holdings | CAS | PC/IT | PE, VC, Banking, Agri |
| Fosun | Privato | Pharma | Insurance, Tourism |
| HNA | Privato | Aviation | Hotels, Finance |
