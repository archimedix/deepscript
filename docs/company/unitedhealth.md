## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1974 (come Charter Med Inc) |
| **Fondatore** | Richard Taylor Burke |
| **Sede originale** | Minnesota |
| **Rinominata** | United HealthCare (1977) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1974** | Richard Burke fonda Charter Med Inc |
| **1977** | Rinominata United HealthCare, inizia acquisizioni HMO regionali |
| **1984** | IPO alla borsa |
| **1988** | William McGuire entra come EVP (acquisizione Peak Health Plan) |
| **1991** | McGuire diventa CEO e Chairman |
| **1998** | Acquisizione MetraHealth |
| **2006** | Scandalo stock options, McGuire dimesso, Stephen Hemsley CEO |
| **2011** | Creazione divisione Optum |
| **2017** | Hemsley passa a Executive Chairman, David Wichmann CEO |
| **2021** | Andrew Witty CEO |
| **2023** | Acquisizione LHC Group ($5.4B), accordo Amedisys ($3.3B) |
| **Dec 2024** | Brian Thompson (CEO UnitedHealthcare) assassinato a NYC |
| **May 2025** | Stephen Hemsley ritorna CEO |

## Assassinio Brian Thompson (2024)

| Dato | Dettaglio |
|------|-----------|
| **Vittima** | Brian Thompson, CEO UnitedHealthcare |
| **Data** | 4 dicembre 2024 |
| **Luogo** | New York Hilton Midtown |
| **Contesto** | Investor conference |
| **Bossoli** | Incisi "delay, deny, depose" |
| **Sospettato** | Luigi Mangione (26 anni) |
| **Arresto** | 9 dicembre 2024, Altoona PA |
| **Arma** | Pistola stampata 3D |
| **Accuse** | Omicidio di primo grado, terrorismo |

**Reazione pubblica:** L'assassinio ha scatenato dibattito sul sistema sanitario USA. Mangione e' diventato figura simbolica per critici delle assicurazioni sanitarie.
