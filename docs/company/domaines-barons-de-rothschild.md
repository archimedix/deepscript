## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Acquisizione originale** | 1868, Baron James de Rothschild acquista Château Lafite all'asta pubblica a Parigi |
| **Ristrutturazione** | Post-1981, dopo la nazionalizzazione bancaria sotto Mitterrand |
| **Ramo familiare** | Éric de Rothschild (figlio di Alain, nipote di Robert) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1868** | Baron James de Rothschild acquista Château Lafite Rothschild |
| **1962** | Acquisizione Château Duhart-Milon |
| **1970** | Acquisizione Château Paradis Casseuil |
| **1974** | [Éric de Rothschild](../persons/eric-de-rothschild.md) assume la guida |
| **1984** | Acquisizione Château Rieussec (Sauternes, 1er Grand Cru Classé) |
| **1988** | Prima espansione internazionale: Viña Los Vascos (Cile) |
| **1990** | Acquisizione Château L'Evangile (Pomerol) |
| **1999** | Acquisizione Domaine d'Aussières (Languedoc); JV Bodegas Caro (Argentina) |
| **2005** | JV Champagne Barons de Rothschild (con altri rami familiari) |
| **2011** | Impianto vigneti in Cina (Penglai, Shandong) |
| **2018** | [Saskia de Rothschild](../persons/saskia-de-rothschild.md) succede al padre come Chairwoman |
| **2019** | Prima release di Domaine de Long Dai (Cina) |
| **2021** | Saskia diventa anche CEO e Director General |
| **2024 gen** | Acquisizione Domaine William Fevre (Chablis, 72 ha, ex Artemis/Pinault) |
| **2024** | Certificazione B Corp (score 95.5); prima vendemmia interamente biologica a Lafite |
