## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1983 |
| **Costituzione** | Fusione di 4 raffinerie statali |
| **Privatizzazione** | 2005-2006 |
| **Acquirente** | Koc Holding/Shell consortium |
| **Prezzo** | $4.14 miliardi (51%) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1983** | Fondazione Tupras (fusione raffinerie statali) |
| **1991** | Prima IPO, 2.5% venduto al pubblico |
| **2005** | Free float raggiunge 49%; gara privatizzazione |
| **Set 2005** | Koc/Shell vincono con offerta $4.14 mld per 51% |
| **Gen 2006** | Firma accordo trasferimento azioni |
| **2006** | Completamento privatizzazione |
| **~2008** | Shell esce; Koc Holding controlla 100% di Enerji Yatirimlari |
| **2008** | Omer Koc diventa Chairman |
| **2019** | Investimenti post-privatizzazione superano $6.5 mld |
| **2024** | Utile netto 18.32 mld lira ($505M) |
| **2025** | Stop import petrolio russo dopo sanzioni; accordo SAF con Turkish Airlines |

### Proprieta'

```
KOC HOLDING (quotata BIST)
     |
     v
ENERJI YATIRIMLARI A.S. (100%)
     |
     v
TUPRAS (51%)
     |
     +---> 49% free float (BIST)
```

**Nota:** Enerji Yatirimlari fu creata nel 2005 come JV Koc/Shell (80/10/7/3). Dopo l'uscita di Shell, e' 100% Koc.

## Russia e Sanzioni

### Import Petrolio Russo 2022-2024

| Anno | Import (bpd) | % Import Turchia |
|------|--------------|------------------|
| **2023** | 178,000 | ~60% |
| **2024** | 225,000 | 65% |

Tupras divenne uno dei maggiori importatori di greggio russo dopo l'invasione dell'Ucraina (2022). Nel 2025, dopo nuove sanzioni, ha interrotto gli acquisti.
