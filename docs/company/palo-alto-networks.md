## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2005 |
| **Fondatore** | Nir Zuk |
| **Contesto** | Zuk lascia Juniper Networks (che aveva acquisito la sua NetScreen) per creare un nuovo paradigma di firewall |
| **Obiettivo dichiarato** | Rivoluzionare la sicurezza di rete con firewall "next-generation" capaci di identificare applicazioni e minacce |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2005** | Nir Zuk fonda Palo Alto Networks con 25 colleghi ex-NetScreen |
| **2007** | Rilascio primo firewall next-generation al mondo |
| **2012** | IPO su NYSE (PANW) |
| **2018** | Nikesh Arora diventa Chairman & CEO |
| **2019-2024** | Strategia di acquisizioni per piattaforma cybersecurity integrata |
| **2025 Feb** | Nomina board: Helle Thorning-Schmidt e Ralph Hamers |
| **2025 Jul** | Annuncio acquisizione CyberArk per $25B |
| **2025 Aug** | Nir Zuk annuncia ritiro dopo 20 anni; Lee Klarich diventa CTO |
| **2025 Nov** | Mark Goodburn entra nel board |

## Prodotti e Focus

1. **Next-Generation Firewall** - Prodotto fondante, identificazione applicazioni
2. **Cortex** - Piattaforma AI-driven per security operations
3. **Prisma** - Cloud security (SASE, Cloud Workload Protection)
4. **Unit 42** - Threat intelligence e incident response

Strategia "platformization": consolidare strumenti di sicurezza in piattaforma unica AI-powered.
