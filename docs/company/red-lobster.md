# Red Lobster

| **Fondazione** | 1968 |

| **Tipo** | Ristorazione (casual dining) |
| **Peak** | 700+ ristoranti |
| **Bankruptcy** | Maggio 2024 |
| **Ristoranti chiusi** | 100+ |
| **Status** | crisis |

| Termine | Definizione |
|---------|-------------|
| **Sale-leaseback** | Vendita immobili + affitto degli stessi |
| **Triple-net lease** | Affitto dove il tenant paga tutto (tasse, assicurazione, manutenzione) |

## L'Era Golden Gate (2014-2020)

### Il Deal

| Aspetto | Dettaglio |
|---------|-----------|
| **Anno** | 2014 |
| **Venditore** | Darden Restaurants |
| **Acquirente** | Golden Gate Capital |
| **Prezzo** | $2.1B |

### La Sale-Leaseback

| Aspetto | Dettaglio |
|---------|-----------|
| **Immobili venduti** | Maggioranza delle proprieta' |
| **Ricavo** | $1.5B |
| **Conseguenza** | Red Lobster diventa affittuario |
| **Tipo lease** | Triple-net |
| **Affitto annuo (2023)** | $200M (~10% ricavi) |

**Meccanismo**: Golden Gate recupera $1.5B dei $2.1B investiti vendendo gli immobili. Red Lobster deve pagare affitto permanente.

### Quote Josh Olshansky (Golden Gate)

> "Achieved strong returns for our investors."

## L'Era Thai Union (2020-2024)

### L'Acquisizione

| Aspetto | Dettaglio |
|---------|-----------|
| **2016** | Thai Union acquista 25% ($575M) |
| **2020** | Thai Union acquista resto da Golden Gate |
| **Conflitto** | Thai Union e' fornitore di gamberi |

### Il Conflitto di Interessi

| Problema | Dettaglio |
|----------|-----------|
| **Thai Union** | Uno dei maggiori fornitori shrimp |
| **Incentivo** | Aumentare acquisti/prezzi shrimp |
| **Azione** | Eliminati 2 fornitori shrimp, Thai Union esclusivo |
| **Conseguenza** | "Burdensome supply obligations" |

### L'Endless Shrimp Disaster

| Aspetto | Dettaglio |
|---------|-----------|
| **Promozione** | Ultimate Endless Shrimp $20 |
| **Lancio** | Maggio 2023 (permanente) |
| **Perdite** | $11M in meno di un anno |
| **Decisione** | CEO Paul Kenny (nominato da Thai Union) |
| **Opposizione** | "Significant pushback" dal management |

### Write-Down Thai Union

| Metrica | Valore |
|---------|--------|
| **Investimento** | $575M+ |
| **Write-down** | $535M |
| **Perdita** | Quasi totale |

## Pattern: Sale-Leaseback

Red Lobster e' il secondo caso documentato di Golden Gate Capital:

| Azienda | Anno | Sale-leaseback | Conseguenza |
|---------|------|----------------|-------------|
| **Payless** | 2012 | Dividend recap $700M | Bankruptcy 2017, 2019 |
| **Red Lobster** | 2014 | $1.5B | Bankruptcy 2024 |

### Golden Gate Pattern

1. **Acquisizione** con LBO
2. **Sale-leaseback** degli immobili
3. **Recupero** maggior parte investimento
4. **Exit** con "strong returns"
5. **Azienda** paga affitto permanente
6. **Bankruptcy** inevitabile
