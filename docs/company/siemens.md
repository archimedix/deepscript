## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 12 Ottobre 1847, Berlino |
| **Fondatori** | Werner von Siemens, Johann Georg Halske |
| **Nome originale** | Telegraphen Bau-Anstalt von Siemens & Halske |
| **Prima milestone** | 1848: Prima linea telegrafica europea (500km Berlino-Francoforte) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1847** | Fondazione a Berlino |
| **1848** | Prima linea telegrafica europea (Berlino-Francoforte) |
| **1866** | Fusione: Siemens & Halske + Siemens-Schuckert + Siemens-Reiniger |
| **1897** | Quotazione in borsa |
| **2014** | Spin-off Osram (illuminazione) |
| **2017** | Jim Hagemann Snabe diventa Chairman |
| **2020** | Spin-off Siemens Energy |
| **2021** | Roland Busch diventa CEO |
| **2024** | Acquisizione divisione IDT di ebm-papst |
| **2024** | Nuovo centro R&D £100M in UK (Wiltshire) |
