## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 22 maggio 1930 |
| **Fondatore** | [Battista "Pinin" Farina](../persons/battista-pininfarina.md) |
| **Contesto** | Battista lascia la carrozzeria del fratello (Stabilimenti Farina) per fondare la propria |
| **Obiettivo dichiarato** | Design automobilistico d'eccellenza |
| **Nome originale** | Carrozzeria Pinin Farina |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1930** | Fondazione a Torino come "Società anonima Carrozzeria Pinin Farina" |
| **1930s** | Carrozzerie per Lancia, Alfa Romeo, Fiat, Cadillac, Rolls-Royce |
| **1951** | Inizio partnership con Ferrari (212 Inter) |
| **1961** | Battista autorizzato a cambiare cognome in "Pininfarina"; Sergio eredita la guida |
| **1966** | Morte di Battista; Sergio diventa Chairman |
| **2006** | Picco occupazionale: 2.768 dipendenti |
| **2008** | Andrea muore in incidente; Paolo diventa Chairman |
| **2011** | Ferrari crea Centro Stile interno, inizio fine partnership |
| **2012** | Sergio Pininfarina muore; fine produzione auto in serie |
| **2015** | Mahindra Group acquisisce 76.06% per €168M |
| **2017** | F12berlinetta: ultimo modello Ferrari disegnato da Pininfarina |
| **2018** | Lancio Automobili Pininfarina (EV) |
| **2019** | Presentazione Battista hypercar elettrica |
| **2024** | Paolo Pininfarina muore (aprile) |

## Divisioni

Design e ingegneria per clienti esterni (automotive, industrial design, architettura)

### Pininfarina Extra

## Partnership Ferrari (1951-2017)

La più lunga e iconica collaborazione nella storia dell'auto.

| Periodo | Modelli Chiave |
|---------|----------------|
| 1951 | 212 Inter (prima collaborazione) |
| 1956 | 250 GT Coupé |
| 1968 | 365 GTB/4 Daytona |
| 1984 | Testarossa |
| 1987 | F40 |
| 2012 | F12berlinetta (ultimo) |

**~200 modelli Ferrari** disegnati in 66 anni di partnership.

Fine partnership: Ferrari crea Centro Stile interno (2011); LaFerrari (2013) primo modello interamente in-house.
