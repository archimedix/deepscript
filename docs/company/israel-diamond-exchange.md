## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 8 ottobre 1937 (Israel Diamond Club) |
| **Fondatori** | Diamantieri immigrati da Anversa, inclusi Asher Anshel Daskal e Zvi Rosenberg |
| **Contesto** | WWII chiuse Anversa/Amsterdam; De Beers cercava centri alternativi |
| **Obiettivo** | Centro trading diamanti per industria israeliana |

### Preistoria

- **1933**: Aharon Ben Zvi Moritz ottiene abolizione dazi doganali su diamanti grezzi
- **1936**: Via libera per importazione rough diamonds
- **1937**: Prima fabbrica taglio diamanti aperta a Petach Tikva

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1937** | Fondazione Israel Diamond Club, Tel Aviv |
| **1940** | De Beers inizia forniture durante WWII (Anversa occupata) |
| **1945** | Fine guerra, De Beers taglia forniture per ricostruire Anversa |
| **1947** | Fondazione Eretz Israel Diamond Exchange (societa' per azioni) |
| **1948** | Merger tre organizzazioni in struttura unificata |
| **1950** | De Beers riprende forniture regolari a Israele |
| **1961** | Accordo governo-CSO: fornitura diretta garantita, controllo produzione melee |
| **1968** | Trasferimento nel nuovo complesso a Ramat Gan |
| **1990s** | Indebolimento monopolio De Beers, apertura mercati Russia/Africa |
| **2006** | Morte Moshe Schnitzer, figura storica |
| **2021** | Boaz Moldawsky eletto presidente |
| **2024** | Crisi export (-35% polished), Nissim Zuaretz nuovo presidente |

## Relazione con De Beers

```
1940s: WWII - De Beers cerca alternative ad Anversa
         ↓
     ISRAEL DIAMOND INDUSTRY
         ↓
1945: Fine guerra - De Beers taglia forniture
         ↓
1950: Ripresa forniture regolari
         ↓
1961: Accordo storico CSO-Israele
      → Fornitura diretta garantita
      → Controllo produzione melee globale
         ↓
1990s: Fine monopolio De Beers
       → Mercati alternativi (Russia, Africa)
       → Maggiore indipendenza IDE
```

Per decenni, l'IDE fu **cliente privilegiato** del sistema De Beers/CSO. L'accordo 1961 diede a Israele una posizione dominante nella produzione di diamanti piccoli (melee), ancora oggi un punto di forza.
