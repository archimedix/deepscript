## Glossario

| Termine | Definizione |
|---------|-------------|
| Class A (GOOGL) | Azioni con 1 voto ciascuna, pubbliche |
| Class B | Azioni con 10 voti, detenute solo dai fondatori |
| Class C (GOOG) | Azioni senza diritto di voto, pubbliche |
| Dual-class structure | Struttura che separa proprieta' economica da controllo |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione Google** | 4 settembre 1998 |
| **Fondatori** | Larry Page, Sergey Brin (Stanford PhD students) |
| **Ristrutturazione** | 2 ottobre 2015 - creazione Alphabet |
| **Obiettivo Alphabet** | Separare core business (Google) da "moonshots" |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1996** | Page e Brin iniziano progetto "BackRub" a Stanford |
| **1998** | Google Inc. fondata, primo investimento $100K (Andy Bechtolsheim) |
| **1999** | Sequoia Capital e Kleiner Perkins investono $25M |
| **2001** | Eric Schmidt diventa CEO |
| **2004** | IPO Google ($1.67B raccolti, $85/azione) |
| **2006** | Acquisizione YouTube ($1.65B) |
| **2007** | Lancio Android |
| **2008** | Lancio Chrome browser |
| **2011** | Larry Page ritorna CEO Google |
| **2015** | Creazione Alphabet, Sundar Pichai CEO Google |
| **2017** | Sundar Pichai entra nel board Alphabet |
| **2018** | John Hennessy diventa Chairman |
| **2019** | Page e Brin lasciano ruoli esecutivi, Pichai CEO Alphabet |
| **2022** | Sergey Brin ritorna operativo per rispondere a ChatGPT |
| **2023** | Lancio Bard (poi Gemini) AI |
| **2024** | DOJ ruling: Google monopolio illegale in search; Brin: "uscito dal ritiro per l'AI" |

## Controllo Azionario: La Dual-Class Structure

### Fondatori Controllano Nonostante Minoranza Economica

| Azionista | Quota Economica | Quota Voti | Note |
|-----------|-----------------|------------|------|
| Larry Page | ~3.5% | ~26% | Class B shares |
| Sergey Brin | ~3.2% | ~25% | Class B shares |
| **Totale Fondatori** | **~6.7%** | **~51%** | **Controllo assoluto** |

I fondatori controllano 51.4% dei voti con solo 6.7% delle azioni grazie alle Class B shares (10 voti ciascuna).

## Potere e Influenza

### Dominio Digitale

| Metrica | Quota | Note |
|---------|-------|------|
| Search market share | ~90% | Monopolio de facto |
| Mobile OS (Android) | ~72% | 3B+ dispositivi attivi |
| Browser (Chrome) | ~65% | |
| Video (YouTube) | ~80% | 2B+ utenti mensili |

### Dati e Sorveglianza

Google possiede probabilmente il piu' grande dataset sulla popolazione mondiale:
- Search queries (intent)
- Gmail (comunicazioni)
- Maps/Waze (location)
- Android (tutto)
- YouTube (interessi)
- Chrome (browsing)

### AI Leadership

- **DeepMind**: AlphaGo, AlphaFold (Nobel 2024)
- **Google AI**: Gemini, TPU chips
- Competizione diretta con OpenAI/Microsoft
