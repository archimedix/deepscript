## Il Merger True-DTAC (2023)

### I Numeri del Deal

| Aspetto | Dettaglio |
|---------|-----------|
| Valore | $8.6 miliardi (enterprise value combinato) |
| Tipo | Amalgamation (fusione paritaria) |
| Annuncio | 22 novembre 2021 |
| Approvazione NBTC | 20 ottobre 2022 (3-2, "riconoscimento") |
| Completamento | 1 marzo 2023 |
| Nome risultante | True Corporation |
| Quotazione | SET: TRUE (3 marzo 2023) |

### Pre-Merger: Due Operatori

| Operatore | Abbonati | Posizione | Proprietario |
|-----------|----------|-----------|--------------|
| True Corporation | 32.2 milioni | #2 | CP Group |
| DTAC | 19.6 milioni | #3 | Telenor |
| **Combinato** | ~49 milioni | #1 | CP + Telenor |

### Significato

- Maggiore fusione telecom nel Sud-Est asiatico
- Maggiore fusione tra societa' quotate thailandesi (per capitalizzazione)
- Mercato mobile ridotto a duopolio (True + AIS)

## Mercato Telecom Thailandese

### Struttura Post-Merger (Duopolio)

| Operatore | Abbonati | Controllo |
|-----------|----------|-----------|
| **True Corporation** | ~49M | CP Group + Telenor |
| **AIS (Advanced Info Service)** | ~46M | Intouch Holdings (Temasek) |

### Critiche al Merger

| Critica | Dettaglio |
|---------|-----------|
| Duopolio | Riduzione da 3 a 2 operatori |
| Prezzi | Mancata riduzione promessa |
| MVNO | Collasso operatori virtuali |
| Regolatore | NBTC accusato di inerzia |
| Consumer protection | Condizioni non applicate |

> Critici parlano di "1,000 giorni di sonno regolamentare" dopo il merger.

## Pattern Rilevanti

### Modello "Convergenza Telecom"

True ha costruito un conglomerato convergente:
1. Mobile (TrueMove + DTAC)
2. Broadband (TrueOnline)
3. Pay-TV (TrueVisions)
4. Tech hub (True Digital Park)

### Consolidamento Mercato

Il merger True-DTAC segue un pattern globale:
- Riduzione operatori per scala
- Duopolio come equilibrio stabile
- Regolatore passivo (NBTC)

### CP Group come "Kingmaker"

CP Group applica lo stesso modello in telecom:
- Ingresso in mercato frammentato
- Acquisizioni successive
- Mega-merger per dominanza
