## Glossario

| Termine | Definizione |
|---------|-------------|
| **Pegasus** | Spyware zero-click capace di sorveglianza totale su smartphone (iOS/Android) |
| **Zero-click** | Exploit che non richiede interazione dell'utente per infettare il dispositivo |
| **Entity List** | Blacklist commerciale USA del Dept. of Commerce |
| **Spyware** | Software di sorveglianza che raccoglie dati da un dispositivo senza consenso |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2010 |
| **Fondatori** | Niv Karmi (ex Mossad), Shalev Hulio, Omri Lavie |
| **Nome** | NSO = Niv, Shalev, Omri |
| **Contesto** | Hulio e Lavie avevano fondato CommuniTake (remote access mobile); un'agenzia di intelligence europea mostro' interesse per uno strumento di accesso senza autorizzazione; Karmi fu portato per i contatti Mossad |
| **Obiettivo dichiarato** | Fornire strumenti di cyber-intelligence a governi per combattere terrorismo e criminalita' |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2010** | Fondazione a Herzliya da Niv Karmi, Shalev Hulio, Omri Lavie |

| **2021** | Investitori di Novalpina strappano il controllo, affidato a Berkeley Research Group |
| **2022** | Shalev Hulio si dimette da CEO |
| **2022** | Polizia israeliana usa Pegasus su cittadini israeliani senza mandato |
| **2024** | Dic: giudice USA dichiara NSO responsabile nella causa WhatsApp |
| **2025** | Mag: giuria assegna $168M danni a Meta (poi ridotti a ~$4M) |
| **2025** | Ott: gruppo di investitori USA guidati da Robert Simonds acquista il controllo |
| **2025** | Nov: David Friedman (ex ambasciatore Trump in Israele) nominato Executive Chairman |
| **2025** | Omri Lavie lascia l'azienda |
| **2025** | Dic: CEO Yaron Shohat si dimette |

## Prodotto: Pegasus

| Aspetto | Dettaglio |
|---------|-----------|
| **Tipo** | Spyware zero-click per smartphone |
| **Capacita'** | Accesso completo: messaggi, email, foto, microfono, camera, GPS, password |
| **Vettori** | Exploit zero-click via iMessage, WhatsApp, altri; non richiede interazione utente |
| **Classificazione** | Classificato come arma da Israele; export richiede approvazione Min. Difesa |
| **Clienti** | Governi di 40+ paesi (intelligence e forze dell'ordine) |
| **Target stimati** | 50.000+ numeri di telefono (lista Progetto Pegasus 2021) |

### Clienti confermati (da cause legali e indagini)

|-------|----------|

| **Arabia Saudita** | Confermato in tribunale; connessione caso Khashoggi |
| **Uzbekistan** | Confermato da avvocato NSO in tribunale (2025) |
| **UAE** | Citizen Lab: Ahmed Mansoor target (2016); sorveglianza Qatari |
| **Bahrain** | Citizen Lab: attivisti target |
| **Marocco** | Citizen Lab + Progetto Pegasus |
| **India** | Progetto Pegasus 2021 |
| **Ungheria** | Progetto Pegasus: giornalisti target |
| **Polonia** | Citizen Lab: opposizione politica target |
| **Spagna** | Indagine CatalanGate: politici e avvocati catalani |

## Critiche

| Tema | Dettaglio |
|------|-----------|
| **Progetto Pegasus (2021)** | Consorzio di 17 testate (Forbidden Stories + Amnesty) rivela abuso su scala globale: giornalisti, attivisti, politici, avvocati tra i target |
| **Caso Khashoggi** | Pegasus usato su persone vicine a Jamal Khashoggi prima del suo assassinio (2018) |
| **Blacklist USA** | Inserita nella Entity List dal Dept. of Commerce (Nov 2021) per attivita' contrarie alla sicurezza nazionale USA |
| **Causa WhatsApp/Meta** | 1.400 utenti hackerati in 2 settimane (2019); NSO dichiarata responsabile (2024); $168M danni poi ridotti a ~$4M |
| **Causa Apple** | Apple fa causa per hacking iPhone tramite exploit FORCEDENTRY (2021) |
| **Sorveglianza domestica ISR** | Polizia israeliana usa Pegasus su cittadini israeliani senza mandato (2022) |
| **CatalanGate** | Politici e avvocati catalani target; Hulio rischia accuse penali in Spagna |
| **Causa Khashoggi v. NSO** | Eredi Khashoggi portano causa al 4th Circuit USA (2025) |
| **Transizione USA** | Acquisizione da investitori USA e nomina di Friedman (ex Trump) vista come tentativo di riabilitazione politica |

## Pattern di Potere

NSO Group rappresenta il caso emblematico di:
- **Privatizzazione intelligence**: capacita' state-level vendute come servizio commerciale
- **Classificazione come arma**: export controllato dal Min. Difesa israeliano = leva geopolitica
- **Pipeline Unit 8200 â†’ startup**: talento militare israeliano convertito in prodotto commerciale
- **Revolving door USA-ISR**: Friedman (ex ambasciatore Trump) diventa chairman; tentativo di delisting dalla blacklist
- **Accountability gap**: clienti governativi abusano dello strumento, NSO nega responsabilita'
