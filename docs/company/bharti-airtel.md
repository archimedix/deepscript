## Fondazione e Storia

### Origini

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondatore** | Sunil Bharti Mittal |
| **Background** | Da telefoni push-button (1984) a mobile |
| **Licenza mobile** | Delhi 1992, approvata 1994 |
| **Lancio servizi** | Delhi 1995, brand AirTel 1997 |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1986** | Bharti Telecom Limited, accordo con Siemens (telefoni) |
| **1992** | Licenza mobile Delhi |
| **1995** | Lancio servizi cellulari Delhi |
| **1997** | Brand "AirTel" |
| **2000** | Singtel investe $400M (prima major estera in India) |
| **2002** | IPO su BSE e NSE |
| **2003** | Rebrand unico sotto Airtel |
| **2006** | Bharti Tele-Ventures → Bharti Airtel |
| **2009** | Espansione Sri Lanka |
| **2010** | Acquisizione Zain Africa ($10.7B, 15 paesi) |
| **2013** | Acquisizione Warid Congo |
| **2024** | 550M+ clienti, #2 mondiale |
