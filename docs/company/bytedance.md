## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2012 |
| **Fondatori** | Zhang Yiming + Liang Rubo (compagni di universita' Nankai) |
| **Location originale** | Beijing |
| **Primo prodotto** | Toutiao (news aggregator con AI) |
| **Primo investitore** | Susquehanna International Group (2012) |

Zhang Yiming, ingegnere software di Fujian, fondo' ByteDance con la visione di usare l'AI per personalizzare i contenuti. La maggior parte dei VC rifiuto' l'idea finche' Susquehanna non investi' nel 2012.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2012** | Fondazione ByteDance; lancio Toutiao |
| **2014** | Toutiao raggiunge 13M utenti giornalieri; Sequoia investe $100M |
| **2016** | Lancio Douyin (TikTok cinese) |
| **2017** | Lancio TikTok internazionale |
| **2018** | Acquisizione Musical.ly ($800M), merge con TikTok |
| **2018** | Valutazione $75B, supera Uber come startup piu' preziosa |
| **2020** | Trump ordina divestiture TikTok USA |
| **2021** | Zhang Yiming lascia ruolo CEO a Liang Rubo |
| **2021** | Shou Chew diventa CEO TikTok |
| **2021** | Golden share 1% al governo cinese |
| **2024** | Revenue $155B, supera Alibaba e Tencent |
| **Apr 2024** | Biden firma legge divestiture TikTok |
| **Gen 2025** | TikTok e app ByteDance bannate in USA |
| **2025** | Valutazione $300-414B |

## Pattern: Il Paradosso ByteDance

ByteDance rappresenta un caso unico:

1. **Proprieta' maggioritaria USA** (60%) ma percepita come "cinese"
2. **Fondatore emigrato** a Singapore dal crackdown
3. **Golden share** statale (1%) con diritto veto
4. **Prima vittima** della guerra tech USA-Cina

### Confronto con Altri Tech Cinesi

| Aspetto | ByteDance | Tencent | Alibaba |
|---------|-----------|---------|---------|
| **Fondatore** | Emigrato (Singapore) | In Cina | Rientrato (2025) |
| **Golden share** | 1% | Si | Si |
| **Ban USA** | TikTok bannato | No | No |
