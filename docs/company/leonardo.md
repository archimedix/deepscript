## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1948 |
| **Nome originale** | Societa' Finanziaria Meccanica (Finmeccanica) |
| **Fondatore** | IRI (subholding meccanica) |
| **Rinominazione** | Leonardo SpA (1 gennaio 2017) |

### Le Origini IRI (1948-1992)

Finmeccanica fu creata nel 1948 dall'IRI per gestire le partecipazioni statali nell'industria meccanica e cantieristica. L'industria italiana della difesa era frammentata:

| Ente | Aziende controllate |
|------|---------------------|
| IRI/Finmeccanica | Cantieri navali, meccanica |
| EFIM | Agusta (elicotteri), Oto Melara (difesa), Officine Galileo |
| STET | Selenia, Elsag, SGS Thomson |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1948** | Fondazione Finmeccanica come subholding IRI |
| **1989** | Riorganizzazione: aziende elettroniche STET → Finmeccanica |
| **1989** | Fusione Aeritalia-Selenia → Alenia |
| **1992** | Liquidazione EFIM: Agusta, Oto Melara, Galileo → Finmeccanica |
| **1993** | Privatizzazione parziale, quotazione Borsa Milano |
| **2000** | Joint venture Agusta + Westland (GKN) → AgustaWestland |
| **2002** | Liquidazione IRI, controllo passa a MEF |
| **2004** | Acquisizione quota GKN in AgustaWestland |
| **2011** | Scandalo Guarguaglini, dimissioni |
| **2013** | Arresto CEO Orsi (caso elicotteri India) |
| **2016** | Fusione divisioni: AgustaWestland, Alenia, Selex ES, Oto Melara, WASS |
| **2017** | Rinominazione: Finmeccanica → Leonardo |
| **2023** | Nuova leadership: Pontecorvo Chairman, Cingolani CEO |

### Pattern: Nomine Controverse

Leonardo ha una storia di CEO con vicende giudiziarie:
- **Orsi**: arrestato 2013 per corruzione India (poi assolto)
- **Moretti**: condannato per incidente ferroviario Viareggio 2009
- **Profumo**: condannato per frode MPS (confermato in carica)
