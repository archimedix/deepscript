## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 27 luglio 1927, Torino |
| **Fondatore** | Senatore Giovanni Agnelli (1866-1945) |
| **Obiettivo originale** | Riunire le partecipazioni Agnelli sotto un'unica società |
| **Nome originale** | Istituto Finanziario Industriale (IFI) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1927** | Fondazione IFI a Torino dal Sen. Giovanni Agnelli |
| **1945** | Morte Giovanni Agnelli; partecipazioni ai discendenti (35% a Gianni) |
| **1957** | IFI acquisisce ICLI (poi IFIL dal 1966) |
| **1969** | Partnership con Enzo Ferrari |
| **1984** | Gianni Agnelli crea Giovanni Agnelli S.r.l. per consolidare controllo |
| **1991** | Acquisizione Exor Group (Lussemburgo) |
| **2003** | Ristrutturazione sotto Umberto Agnelli; attività operative trasferite a IFIL |
| **2007** | John Elkann diventa Chairman IFI |
| **2009** | Fusione IFI + IFIL → **Exor S.p.A.** |
| **2014** | Fusione Fiat + Chrysler → FCA |
| **2016** | Trasferimento sede in Olanda; loyalty voting structure |
| **2021** | Fusione FCA + PSA → Stellantis |
| **2022** | Vendita PartnerRe a Covéa (€9.3B) |
| **2025** | Vendita parziale Ferrari (€3B); vendita Iveco a Tata/Leonardo |

### Loyalty Voting Structure

Il framework olandese permette azioni speciali (Class A) che concedono diritti di voto aggiuntivi:
- 5 voti dopo 5 anni di possesso
- 10 voti dopo 10 anni di possesso

Questo consolida il controllo di Giovanni Agnelli B.V. (56.9% equity → 86% voto).
