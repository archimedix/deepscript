## Glossario

| Termine | Definizione |
|---------|-------------|
| **Orb** | Dispositivo sferico che scansiona l'iride per creare World ID |
| **World ID** | Identita' digitale unica basata su biometria |
| **WLD** | Token cryptocurrency del progetto |
| **Tools for Humanity** | Societa' che sviluppa Worldcoin |
| **Proof of Personhood** | Verifica che l'utente sia umano, non bot/AI |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 2019 |
| **Fondatori** | Sam Altman, Alex Blania, Max Novendstern |
| **Entita' legale** | Tools for Humanity |
| **Obiettivo dichiarato** | Creare sistema ID globale per distinguere umani da AI |

## Controversie e Ban

### Paesi con Restrizioni

| Paese | Azione | Data |
|-------|--------|------|
| **Kenya** | Ban, ordine cancellazione dati | 2023, confermato mag. 2025 |
| **Spagna** | Ban temporaneo AEPD | mar. 2024 |
| **Hong Kong** | Cessazione operazioni | mag. 2024 |
| **Germania** | Indagine, ordine cancellazione | dic. 2024 |
| **Corea del Sud** | Restrizioni | set. 2024 |
| **Brasile** | Ban totale, multa $8,800/giorno | gen. 2025 |
| **Indonesia** | Sospensione operazioni | mag. 2025 |
| **Filippine** | Cease and desist | ott. 2025 |
| **Thailandia** | Ordine cancellazione 1.2M record | nov. 2025 |

### Critiche Principali

| Tema | Dettaglio |
|------|-----------|
| **Consenso sotto coercizione** | Offerta $55 per scansione in paesi poveri |
| **Minori** | Raccolta dati da minorenni senza consenso |
| **Trasferimento dati** | Dati Kenya trasferiti a server Germania senza autorizzazione |
| **Revoca consenso** | Impossibilita' di ritirare il consenso |
| **Centralizzazione** | Contraddizione con ethos crypto decentralizzato |

### Caso Kenya (Dettaglio)

| Data | Evento |
|------|--------|
| **2022** | Worldcoin offre ~$55 per scansione iride |
| **2023** | 350,000+ keniani scansionati |
| **2023** | Governo sospende operazioni |
| **Mag 2025** | Alta Corte dichiara operazioni illegali |
| **Nov 2025** | ODPC supervisiona cancellazione dati in Germania |

## Connessione con OpenAI/AI

### La Tesi di Altman

> "It's really hard to both identify unique people and do that in a privacy-preserving way."
> — Sam Altman, 2025

| Aspetto | Dettaglio |
|---------|-----------|
| **Problema** | AI genera contenuti indistinguibili da umani |
| **Soluzione proposta** | Proof of Personhood via biometria |
| **Critica** | Crea dipendenza da sistema centralizzato |
| **Conflitto interessi** | Altman crea problema (AI) e soluzione (Worldcoin) |
