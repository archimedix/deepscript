## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1993 (da rete sovietica delle comunicazioni) |
| **Nome originale** | Ukrelectrozvyazok (fino al 1994) |
| **Corporatizzazione** | 2000, trasformata in OJSC |
| **Privatizzazione** | 2011, venduta a EPIC/ESU per $1.3 miliardi |
| **Controllo SCM** | Dal 2013, parte dell'impero Akhmetov |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1991** | Rete telecomunicazioni passa al Ministero delle Comunicazioni dopo dissoluzione URSS |
| **1993** | Creazione di Ukrtelecom |
| **1994** | Cambio nome da Ukrelectrozvyazok a Ukrtelecom |
| **2000** | Corporatizzazione in OJSC |
| **2011** | Privatizzazione: EPIC/ESU acquista 92.79% per $1.3 miliardi |
| **2011** | Beneficiari occulti: Valeriy Khoroshkovsky e Dmytro Firtash dietro EPIC |
| **2013** | SCM (Akhmetov) acquista ESU/Ukrtelecom per ~$1 miliardo |
| **2017** | Fondo Proprieta' Statale contesta mancati investimenti ($450M promessi) |
| **2019** | Corte Suprema respinge ricorso statale, Akhmetov mantiene proprieta' |
| **2022** | Cyberattacco russo (marzo): connettivita' cala al 13% dei livelli pre-guerra |
| **2022** | Dipendenti sabotano equipaggiamento nei territori occupati per impedirne il controllo russo |

## La Privatizzazione Controversa (2011)

La privatizzazione di Ukrtelecom e' uno dei casi piu' opachi della storia ucraina.

| Aspetto | Dettaglio |
|---------|-----------|
| **Acquirente ufficiale** | EPIC Services Ukraine (ESU), sussidiaria della austriaca EPIC |
| **Prezzo** | $1.3 miliardi (92.79% delle azioni) |
| **Beneficiario reale** | Valeriy Khoroshkovsky (proprietario occulto di EPIC/Raga Establishment) |
| **Altro partecipante** | Dmytro Firtash, partecipante segreto |
| **Obbligo investimenti** | $450M in 5 anni per sviluppo rete |
| **Investimenti reali** | Mai realizzati |
| **Passaggio a SCM** | 2013, Akhmetov acquista ESU per ~$1 miliardo |
| **Contenzioso** | 2017-2019, Fondo Proprieta' Statale chiede annullamento; Corte Suprema rigetta |
