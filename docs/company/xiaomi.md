## Struttura Proprietaria

```
┌─────────────────────────────────────────┐
│     Lei Jun (Fondatore)                 │
│     24% shares → 65% voting power       │
│     (Class A super-voting shares)       │
└───────────────┬─────────────────────────┘
                │
┌───────────────┴─────────────────────────┐
│         Xiaomi Corporation              │
│      (quotata Hong Kong 1810.HK)        │
└─────────────────────────────────────────┘
        │                    │
   ┌────┴────┐          ┌────┴────┐
   │ Lin Bin │          │ Public  │
   │   9%    │          │  ~67%   │
   │ shares  │          │ shares  │
   └─────────┘          └─────────┘
```

### Dual-Class Structure

| Classe | Voti per azione | Detentori |
|--------|-----------------|-----------|
| Class A | 10 voti | Lei Jun, insiders |
| Class B | 1 voto | Public investors |

Questa struttura garantisce a Lei Jun il controllo assoluto (65% voting) pur possedendo solo 24% del capitale.

## Pattern Imprenditoriale Lei Jun

Lei Jun rappresenta un pattern distintivo nell'ecosistema tech cinese:

1. **Corporate → Entrepreneurship**: Carriera corporate (Kingsoft) → Angel investing → Fondazione startup
2. **Multi-platform**: Controlla simultaneamente azienda tech (Xiaomi), software (Kingsoft), VC (Shunwei)
3. **Political access**: NPC Deputy garantisce accesso diretto al potere politico
4. **Founder control**: Dual-class shares per mantenere controllo nonostante diluizione

Questo pattern è simile a quello di altri tech founder cinesi (Jack Ma, Pony Ma) ma con maggiore continuità di controllo.
