## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Giugno 2012 |
| **Fondatori** | [Brian Armstrong](../persons/brian-armstrong.md), [Fred Ehrsam](../persons/fred-ehrsam.md) |
| **Contesto** | Armstrong, ex ingegnere Airbnb, e Ehrsam, ex trader Goldman Sachs, si incontrano via Reddit sulla visione di rendere Bitcoin accessibile |
| **Obiettivo dichiarato** | Creare un sistema finanziario aperto al mondo, iniziando dal rendere facile comprare e vendere Bitcoin |
| **Incubatore** | [Y Combinator](../private-equity/y-combinator.md) Summer 2012, seed $150K |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **2012** | Fondazione; Y Combinator Summer 2012; lancio servizio buy/sell BTC via bonifico |
| **2013** | Serie A da [Andreessen Horowitz](../private-equity/andreessen-horowitz.md); 1M utenti |
| **2015** | Lancio di Coinbase Exchange (poi GDAX, poi Coinbase Pro) per trading avanzato |
| **2017** | Fred Ehrsam lascia il ruolo di President (resta Board Director); boom crypto |
| **2018** | Acquisizione Earn.com → [Balaji Srinivasan](../persons/balaji-srinivasan.md) diventa primo CTO; Emilie Choi entra come VP Corp Dev |
| **2019** | Balaji lascia il ruolo di CTO |
| **2021** | Direct listing su NASDAQ (14 aprile), ticker COIN; valutazione ~$86B al debutto |
| **2022** | Crypto winter; licenziamenti (~18% workforce); passaggio a remote-first |
| **2023** | SEC fa causa per vendita di securities non registrate; lancio Base (L2 Ethereum) |
| **2025** | Revenue $7.18B (+9.4% YoY); raddoppio volume trading; inclusione nell'S&P 500 |

### Rete crypto-potere

Coinbase e' il nodo centrale della rete crypto-finanza tradizionale:
- **a16z**: investitore principale + Andreessen nel board + Balaji ex GP a16z poi CTO Coinbase
- **Goldman Sachs**: Fred Ehrsam ex trader GS
- **BlackRock/Vanguard**: top institutional holders, ponte con finanza tradizionale
- **Governo USA**: sotto scrutinio SEC dal 2023, ma ruolo chiave nella regolamentazione crypto
