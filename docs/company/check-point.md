## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1993, Ramat Gan, Israel |
| **Fondatori** | Gil Shwed, Marius Nacht, Shlomo Kramer |
| **Contesto** | Shwed sviluppo' l'idea di stateful inspection durante il servizio in Unit 8200 |
| **Seed Funding** | $250,000 da BRM Group |

### Connessione IDF

I tre fondatori hanno background militare israeliano:
- **Gil Shwed**: Unit 8200, dove sviluppo' la tecnologia firewall
- **Marius Nacht**: 8 anni nell'Air Force, programma Talpiot
- **Shlomo Kramer**: Unit 8200, collega di Shwed

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1993** | Fondazione a Ramat Gan |
| **1994** | OEM agreement con Sun Microsystems; Nir Zuk tra i primi dipendenti |
| **1995** | Distribuzione HP; apertura ufficio USA (Redwood City) |
| **1996** | IPO NASDAQ ($67M); leader mondiale firewall (40% market share) |
| **1998** | Partnership con Nokia |
| **1999** | Shlomo Kramer lascia (fondera' Imperva); Nir Zuk lascia (fondera' Palo Alto Networks) |
| **2004** | Acquisizione Zone Labs/ZoneAlarm ($205M) |
| **2006** | Acquisizione PointSec ($586M) |
| **2009** | Acquisizione Nokia Security business |
| **2018** | Acquisizione Dome9 (cloud security) |
| **2020** | Marius Nacht lascia la chairmanship |
| **2024** | Gil Shwed diventa Executive Chairman; Nadav Zafrir nuovo CEO |
| **2025** | Partnership con Wiz; acquisizione Lakera (AI security) |

## Pattern Unit 8200 → Cybersecurity Industry

Check Point rappresenta il modello paradigmatico del pipeline israeliano:

```
Unit 8200 (training) → Check Point (incubation) → Spinoff ecosystem
```

Secondo uno studio 2018, l'80% dei founder cybersecurity israeliani ha esperienza in IDF intelligence.
