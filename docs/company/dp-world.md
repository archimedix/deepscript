## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origini** | Port Rashid (1972), Jebel Ali Port (1979) |
| **Dubai Ports International** | Fondato 1999, gestione porti internazionali |
| **Fondazione DP World** | 2005, fusione Dubai Ports Authority + Dubai Ports International |
| **Obiettivo** | Operatore portuale e logistico globale, braccio commerciale dell'emirato |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1972** | Apertura Port Rashid, Dubai |
| **1979** | Apertura Jebel Ali Port, piu' grande porto artificiale al mondo |
| **1999** | Creazione Dubai Ports International |
| **2005** | Fusione DPA + DPI = **DP World** |
| **2006** | Acquisizione **P&O** (Peninsular & Oriental Steam Navigation) per **$6.9B**; diventa 3o operatore portuale mondiale |
| **2006** | Controversia USA: opposizione bipartisan a gestione porti americani da parte di entita' emiratina; DP World cede i porti USA |
| **2007** | IPO parziale su Nasdaq Dubai; Sultan Ahmed bin Sulayem diventa Chairman |
| **2016** | Bin Sulayem assume anche ruolo CEO |
| **2019** | Delisting da Nasdaq Dubai; torna interamente sotto controllo Dubai |
| **2022** | Scandalo P&O Ferries: licenziamento immediato di 800 marinai britannici |
| **2026 Feb** | Bin Sulayem si dimette dopo rivelazioni Epstein files DOJ; Essa Kazim Chairman, Yuvraj Narayan CEO |
