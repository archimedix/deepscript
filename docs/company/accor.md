## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1967 (SIEH), Lille |
| **Fondatori** | [Paul Dubrule](../persons/paul-dubrule.md), [Gerard Pelisson](../persons/gerard-pelisson.md) |
| **Contesto** | Ispirazione dal modello Holiday Inn americano |
| **Obiettivo** | Portare l'hospitality moderna in Francia |

### I Fondatori

Paul Dubrule (1934-) e Gerard Pelisson (1932-2023) si incontrarono negli USA negli anni '60, entrambi lavorando nel settore informatico. Ispirati dal modello Holiday Inn, fondarono SIEH nel 1967 aprendo il primo Novotel vicino a Lille.

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1967** | Fondazione SIEH; primo Novotel a Lille |
| **1974** | Lancio brand Ibis |
| **1975** | Acquisizione Mercure |
| **1980** | Acquisizione Sofitel (43 hotel) |
| **1983** | Fusione con Jacques Borel International → nasce Accor; quotazione |
| **1997** | Dubrule e Pelisson lasciano la gestione operativa |
| **2005** | Colony Capital acquisisce €1 mld di azioni |
| **2006** | [Serge Weinberg](../persons/serge-weinberg.md) Chairman |
| **2013** | [Sebastien Bazin](../persons/sebastien-bazin.md) Chairman & CEO |
| **2015** | Ingresso Jin Jiang (Cina) come azionista |
| **2016** | Acquisizione FRHI (Fairmont, Raffles, Swissotel) |
| **2017** | Spin-off AccorInvest (immobiliare) |
| **2024** | Bazin rinnovato per altri 3 anni |

## Pattern: Convergenza Globale nell'Hospitality

Accor rappresenta un caso emblematico di convergenza di capitali:

1. **Origini imprenditoriali**: Due francesi ispirati dall'America (1967)
2. **Private equity americano**: Colony Capital (2005-2013)
3. **Transizione manageriale**: Da PE owner a CEO (Bazin)
4. **Capitali del Golfo**: QIA + Kingdom Holding (~13%)
5. **Capitale cinese**: Jin Jiang (~10%)
6. **Asset manager globali**: BlackRock
7. **Connessione politica**: Sarkozy intermediario per Qatar

Il settore hospitality diventa arena di soft power: chi controlla gli hotel di lusso (Raffles, Fairmont) controlla luoghi di incontro delle elite.
