## Sanzioni USA

### Entity List (2019)

| Data | Azione |
|------|--------|
| 9 ottobre 2019 | Aggiunta a Bureau of Industry and Security Entity List |
| Motivazione | Uso tecnologia per violazioni diritti umani contro Uiguri in Xinjiang |
| Effetto | Divieto export tecnologia USA a Megvii |

### Investment Ban (2021)

| Data | Azione |
|------|--------|
| Dicembre 2021 | Treasury vieta investimenti USA in Megvii |
| Motivazione | Complicita' in persecuzione Uiguri |
| Effetto | Fondi pensione/asset manager USA non possono detenere Megvii |
