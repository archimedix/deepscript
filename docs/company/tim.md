## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Origine** | SIP (Societa' Italiana per l'Esercizio Telefonico) 1964 |
| **Fusione** | 1994 - SIP, Italcable, Telespazio, Iritel → Telecom Italia |
| **Privatizzazione** | 27 ottobre 1997, Governo Prodi |
| **Ricavi IPO** | 26.000 miliardi lire (~€14 mld) per 35.26% |
| **Rebrand TIM** | 2016 |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1964** | Fondazione SIP |
| **1994** | Fusione → Telecom Italia |
| **1997** | Privatizzazione, quotazione Borsa |
| **1999** | Scalata Olivetti-Colaninno (LBO €30 mld) |
| **2001** | Pirelli-Benetton acquisiscono Olivetti |
| **2007** | Telefonica entra (10%), uscita Pirelli-Benetton |
| **2015** | Vivendi inizia scalata, arriva al 24% |
| **2018** | Elliott vs Vivendi, cambio governance |
| **2021** | KKR offre acquisizione totale |
| **2024** | Vendita NetCo a KKR (€22 mld), scorporo rete |
| **2025** | Poste acquisisce 27.32% da Vivendi |

## Lo Scorporo della Rete (2024)

Operazione storica: primo grande operatore europeo a separarsi dalla rete fissa.

### Vendita NetCo a KKR

| Aspetto | Dettaglio |
|---------|-----------|
| **Data closing** | 1 luglio 2024 |
| **Acquirente** | [KKR](../private-equity/kkr.md) via Optics BidCo |
| **Veicolo** | [FiberCop](./fibercop.md) |
| **Valutazione** | €18.8 mld (fino a €22 mld con earn-out) |
| **Riduzione debito TIM** | €14.2 mld |
| **Dipendenti trasferiti** | 20,000 |

### Azionariato FiberCop

## Pattern: La Privatizzazione Fallita

TIM rappresenta il caso emblematico di privatizzazione distruttiva:

| Metrica | 1997 | 2025 |
|---------|------|------|
| Fatturato | €23.2 mld | €15.8 mld (-32%) |
| Dipendenti | 120,345 | 52,333 (-56%) |
| Debito netto | €0 | €23.3 mld |
| Ranking Italia | #4 | #17 |
| Patrimonio immobiliare | €10 mld | Ceduto |

### Il Ciclo delle Scalate

```
1997: Privatizzazione (nocciolo duro fragile 6.62%)
        │
        ▼
1999: Olivetti-Colaninno (LBO €30 mld, debito su TIM)
        │
        ▼
2001: Pirelli-Benetton (acquisiscono Olivetti)
        │
        ▼
2007: Telefonica (10%), exit Pirelli-Benetton
        │
        ▼
2015: Vivendi (scalata al 24%)
        │
        ▼
2018: Elliott vs Vivendi (governance war)
        │
        ▼
2024: KKR (compra rete €22 mld)
        │
        ▼
2025: Poste (27.32%, rinazionalizzazione de facto)
```
