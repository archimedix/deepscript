## Fondazione e Storia Oscura

### Eugène Schueller: Il Fondatore

| Dato | Dettaglio |
|------|-----------|
| **Fondatore** | Eugène Schueller (1881-1957) |
| **Anno** | 1909 |
| **Prodotto originale** | Tintura per capelli "Oréale" (1907) |
| **Nome originale** | Société Française de Teintures Inoffensives pour Cheveux |

### La Cagoule e il Collaborazionismo

Schueller fu uno dei principali finanziatori di **La Cagoule** (1936-), organizzazione fascista francese:

| Aspetto | Dettaglio |
|---------|-----------|
| **Fondazione La Cagoule** | 1936 |
| **Finanziatore** | Schueller fornì fondi e uffici L'Oréal |
| **Attività** | Omicidi antifascisti, attentati, tentato colpo di stato 1937 |
| **Durante l'occupazione** | Collaborazione con nazisti |
| **Crescita patrimonio** | 10x tra 1940-43 |
| **Vendite L'Oréal** | 4x tra 1940-44 |
| **Lista Knochen** | Schueller tra i 45 "collaboratori volontari" |

### Il Processo 1946

| Fatto | Dettaglio |
|-------|-----------|
| **Accusa** | Collaborazionismo |
| **Difensori** | François Mitterrand, André Bettencourt |
| **Esito** | Tutte le accuse ritirate |
| **Giustificazione** | Protezione dipendenti ebrei, finanziamento Resistenza |

> Ironia storica: L'azienda fondata da un collaborazionista nazista sarà ereditata dai nipoti di un rabbino ucciso ad Auschwitz (Robert Meyers, nonno di Jean-Victor).

### Pattern Power
