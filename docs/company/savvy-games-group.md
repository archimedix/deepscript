# Savvy Games Group

> Holding di gaming ed esports del fondo sovrano saudita PIF - veicolo per dominare l'industria videoludica globale entro il 2030

## Executive Summary

| Metrica | Valore |
|---------|--------|
| **Fondazione** | 2021 |
| **Sede** | Riyadh, Saudi Arabia |
| **Tipo** | Gaming/Esports holding company |
| **Proprietario** | PIF Saudi Arabia (100%) |
| **Chairman** | Mohammed bin Salman |
| **Vice Chairman** | Prince Faisal bin Bandar |
| **CEO** | Brian Ward |
| **Capitale allocato** | $38B (SR142B) |
| **Speso ad oggi** | ~$10B+ |
| **Dipendenti** | ~3,500 |
| **Status** | active |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | Novembre 2021 |
| **Lancio pubblico** | Gennaio 2022 |
| **Fondatore** | PIF Saudi Arabia |
| **Obiettivo** | Posizionare Saudi Arabia come hub globale del gaming entro 2030 |

### Timeline

| Anno | Evento |
|------|--------|
| **Nov 2021** | Incorporazione |
| **Gen 2022** | Lancio pubblico, acquisizione ESL/FACEIT ($1.5B) |
| **Mag 2022** | Stake 5% in Nintendo |
| **Mag 2022** | Investimento $1B in Embracer Group (8%) |
| **Set 2022** | Annuncio piano $37.8B |
| **Feb 2023** | Investimento $235M in VSPO (esports cinese) |
| **Apr 2023** | Annuncio acquisizione Scopely |
| **Lug 2023** | Completata acquisizione Scopely ($4.9B) |
| **Lug 2024** | Esports World Cup Riyadh ($60M+ prize pool) |
| **2024** | Stake Nintendo ridotto da 8.58% a 7.54% |
| **Mar 2025** | Acquisizione Niantic gaming division ($3.5B) |
| **2025** | Consolidamento asset gaming PIF sotto Savvy |
