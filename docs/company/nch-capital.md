# NCH Capital Inc

> Il veicolo di investimento della famiglia Rohr: $3B+ AUM, pioniere degli investimenti occidentali nell'Europa orientale post-sovietica, 5o maggior proprietario terriero in Ucraina tramite Agroprosperis.

| Metrica | Valore |

| **Fondazione** | 1993 |
| **Sede** | New York, USA |
| **Tipo** | Private Equity / Agribusiness |
| **Leadership** | George Rohr (President), Moris Tabacinic (Principal) |
| **AUM** | $3B+ |
| **Uffici** | New York + 10+ sedi in Europa orientale e America Latina |
| **Status** | active |

## Glossario

| Termine | Definizione |
|---------|-------------|
| **Agroprosperis** | Braccio agribusiness di NCH in Ucraina; tra i maggiori produttori ed esportatori di prodotti agricoli nella regione del Mar Nero |
| **AgroTerra** | Braccio agribusiness di NCH in Russia; tra le top 20 aziende agricole russe |
| **Land bank** | Superficie totale di terreni agricoli controllati |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1993 |
| **Fondatori** | [George Rohr](../persons/george-rohr.md) e [Moris Tabacinic](../persons/moris-tabacinic.md) (cognati) |
| **Contesto** | Crollo dell'URSS; privatizzazioni post-sovietiche; opportunita' in mercati capital-starved |
| **Inizio investimenti** | 1991 (Rohr e Tabacinic iniziano a investire nella regione, 2 anni prima della fondazione formale) |
| **Famiglia** | Impresa familiare Rohr: George (figlio di Sami) e Moris (marito di Lillian, figlia di Sami) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1991** | Rohr e Tabacinic iniziano a investire in Europa orientale |
| **1993** | Fondazione formale di NCH Capital Inc, New York |
| **Anni '90** | Tra i primi investitori occidentali nelle privatizzazioni post-sovietiche |
| **~2006** | Inizio trasformazione terreni agricoli in Ucraina e regione Mar Nero |
| **2007** | Fondazione Agroprosperis (braccio agribusiness Ucraina) |
| **2010** | Espansione in America Latina (ufficio Rio de Janeiro, Brasile) |
| **2015** | Fondazione Agroprosperis Bank; acquisizione Astra Bank (Ucraina) |
| **Apr 2024** | Putin ordina gestione temporanea statale di AgroTerra (Russia) |
| **Dic 2024** | Procura russa chiede sequestro beni NCH per 73 miliardi di rubli (~$937M) |
| **Gen 2025** | Tribunale di Mosca respinge la richiesta di sequestro (durante Davos) |
