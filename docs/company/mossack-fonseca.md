## Glossario

| Termine | Definizione |
|---------|-------------|
| Shell company | Societa' di comodo senza attivita' operativa reale, usata per occultare beneficiari |
| Bearer shares | Azioni al portatore: chi detiene fisicamente il certificato e' il proprietario, anonimato totale |

## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1977 (studio Mossack); 1986 (fusione Mossack Fonseca) |
| **Fondatori** | Jurgen Mossack (tedesco-panamense) e Ramon Fonseca Mora (panamense) |
| **Contesto** | Panama come hub offshore: segretezza societaria, nessun registro beneficiari, bassa tassazione |
| **Obiettivo dichiarato** | Servizi legali per commercio internazionale, trust, consulenza investitori |
| **Obiettivo reale** | Creazione industriale di shell companies per evasione fiscale e occultamento patrimoni |

### Background Fondatori

| Fondatore | Background |
|-----------|------------|
| **Jurgen Mossack** | Nato a Furth (1948). Padre Erhard fu Rottenfuhrer nella Waffen-SS, emigro' a Panama, offri' servizi alla CIA per spiare attivita' comuniste a Cuba. Jurgen studio' legge a Panama, lavoro' a Londra, poi torno' a fondare lo studio |
| **Ramon Fonseca Mora** | Nato nel 1952. Rhodes Scholar a Oxford. Romanziere rispettato. Consigliere del presidente Varela (2014-16), dimesso dopo il leak. Morto di polmonite l'8 maggio 2024 durante il processo |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1977** | Jurgen Mossack apre studio legale a Panama City |
| **1986** | Fusione con Ramon Fonseca Mora, nasce Mossack Fonseca |
| **1987** | Apertura ufficio British Virgin Islands |
| **Anni '90** | Crescita: ~10.000 societa' incorporate all'anno |
| **1994** | Accordo esclusivo con Niue per registrazione offshore |
| **2001** | MF paga $1.6M del budget annuo di Niue ($2M). USA vietano trasferimenti a Niue |
| **2003** | Collasso accordo Niue, clienti spostati a Samoa |
| **2006** | Emerge coinvolgimento nello scandalo tangenti Siemens |
| **2009** | Picco: 80.000+ societa' attive |
| **2010** | Accordi "special treatment" con HSBC e Credit Suisse |
| **2014** | Whistleblower vende primi dati a autorita' fiscali tedesche |
| **2015** | "John Doe" contatta Sueddeutsche Zeitung con 2.6 TB di documenti |
| **3 apr 2016** | Pubblicazione Panama Papers: 11.5 milioni di documenti |
| **Feb 2017** | Mossack e Fonseca arrestati a Panama |
| **Apr 2017** | Rilasciati su cauzione di $500.000 ciascuno |
| **14 mar 2018** | Annuncio chiusura: "danno irreparabile alla reputazione" |
| **Ott 2020** | Procura di Colonia emette mandati d'arresto internazionali |
| **Apr 2024** | Inizio processo a Panama (28 imputati) |
| **8 mag 2024** | Morte di Ramon Fonseca Mora (polmonite, 71 anni) |
| **28 giu 2024** | Tribunale panamense assolve tutti i 28 imputati |

## Critiche

| Tema | Dettaglio |
|------|-----------|
| Evasione fiscale industriale | 300.000 shell companies create per occultare patrimoni |
| Compliance inesistente | Audit Seychelles: 6 leggi AML violate, nessun monitoraggio PEP |
| Accordi con banche | "DD light" e "special treatment" = bypass sistematico dei controlli |
| Clienti criminali | Narcotrafficanti, mafiosi, Ponzi schemers, soggetti sanzionati USA |

## Conseguenze Legali

| Giurisdizione | Esito |
|---------------|-------|
