## Storia e Privatizzazione

### Timeline

| Anno | Evento |
|------|--------|
| **1924** | Fondazione come monopolio statale |
| **1997** | Privatizzazione completa (governo Aznar) |
| **1996-2000** | Juan Villalonga Chairman |
| **2000-2016** | Cesar Alierta Chairman |
| **2016-2025** | Jose Maria Alvarez-Pallete CEO |
| **2023** | STC acquisisce 9.9% |
| **2024** | SEPI acquisisce 10% (ritorno Stato) |
| **2025** | Marc Murtra sostituisce Alvarez-Pallete |

### La Privatizzazione del 1997

| Dato | Dettaglio |
|------|-----------|
| **Quota venduta** | 20.9% residuo statale |
| **Valore** | $4.4 miliardi |
| **Record** | Piu' grande operazione borsa spagnola |
| **Domanda** | 5.4x l'offerta |
| **Governo** | Aznar (PP) |

### Il Pattern Villalonga-Aznar

> Juan Villalonga, amico d'infanzia di Aznar dal Colegio del Pilar, fu nominato presidente pochi mesi prima della privatizzazione.

| Figura | Ruolo | Connessione Aznar |
|--------|-------|-------------------|
| Juan Villalonga | Chairman 1996-2000 | Compagno di scuola |
| Cesar Alierta | Chairman 2000-2016 | Ex-Tabacalera (privatizzata PP) |

Villalonga dimesso nel 2000 dopo accuse (poi archiviate) di insider trading.

## Pattern: La Privatizzazione Spagnola

Telefonica esemplifica il pattern delle **privatizzazioni strategiche spagnole**:

1. **Monopolio statale** → privatizzazione sotto Aznar (PP)
2. **Nomine clientelari**: amici/alleati del governo
3. **Revolving door**: ministri → board aziendali
4. **Ritorno dello Stato**: quando interessi stranieri (Saudi) minacciano controllo

```
1924-1997          1997-2023           2023-oggi
MONOPOLIO    →    PRIVATIZZAZIONE  →   RITORNO STATO
 STATALE          (PP/Aznar)           (vs Saudi)
                       │
                       │
              Villalonga, Alierta
              (connessioni PP)
```
