## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 1909 |
| **Fondatori** | Sindaco Ernesto Nathan, Assessore Giovanni Montemartini |
| **Nome originale** | AEM - Azienda Elettrica Municipale di Roma |
| **Contesto** | Elettrificazione quartiere Prati, referendum popolare |
| **Quotazione** | 16 luglio 1999 (Borsa di Milano) |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1909** | Fondazione AEM per elettrificazione Roma |
| **1912** | Inaugurazione centrale elettrica Via Ostiense (Montemartini) |
| **1926** | AEM diventa AEG (Azienda Elettrica del Governatorato) |
| **1937** | AEG diventa AGEA (aggiunta gestione acquedotti) |
| **1945** | AGEA diventa ACEA (Azienda Comunale Elettricita' e Acqua) |
| **1989** | ACEA = Azienda Comunale dell'Energia e dell'Ambiente |
| **1992** | Trasformazione in Agenzia Speciale |
| **1998** | Trasformazione in SpA |
| **1999** | Quotazione Borsa di Milano |
| **2014** | Alberto Irace AD (nomina Marino) |
| **2017** | Stefano Donnarumma AD (nomina Raggi) |
| **2022** | Fabrizio Palermo AD |
| **2023** | Barbara Marinali Presidente |

## Il Pattern "Utility Comunale → Board Elite"

ACEA rappresenta il modello italiano di **utility pubblica con governance mista**:

### Caratteristiche

| Aspetto | Dettaglio |
|---------|-----------|
| **Controllo** | Pubblico (Comune Roma 51%) |
| **Gestione** | Manageriale (CEO da CDP, Fincantieri) |
| **Soci privati** | Multinazionali (Suez) + famiglie italiane (Caltagirone) |
| **Board** | Mix tecnici + politologi (Tocci) |

### Fabrizio Palermo: il Pattern CDP → Utility

```
MORGAN STANLEY (Londra)
         │
         ▼
    McKINSEY
         │
         ▼
    FINCANTIERI (CFO, Vice DG)
         │
         ▼
      CDP (CEO 2018-21)
         │
         ▼
     ACEA (CEO 2022-)
         │
         ▼
   GENERALI (Board 2025)
```

Palermo incarna il **manager di Stato**: da banche d'affari a aziende pubbliche strategiche.

## Osservazioni

1. **Controllo pubblico**: Roma Capitale mantiene 51% dal 1999
2. **Soci strategici**: Suez (expertise internazionale) + Caltagirone (radicamento italiano)
3. **Revolving door PA → utility**: Marinali 25 anni nella PA prima di presiedere ACEA
4. **Pattern CDP → utility**: Palermo da CEO CDP a CEO ACEA
5. **Board politologi**: Tocci (IAI, advisor UE) nel board 2020-23
6. **Nomine politiche**: Ogni sindaco nomina i propri vertici (Marino→Irace, Raggi→Donnarumma, Gualtieri→Palermo)
