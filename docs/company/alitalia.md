## Fondazione e Storia

| Dato | Dettaglio |
|------|-----------|
| **Fondazione** | 16 settembre 1946 |
| **Nome originale** | Alitalia - Aerolinee Italiane Internazionali |
| **Fondatori** | Governo italiano, BEA (British European Airways) |
| **Capitale iniziale** | 900 milioni lire + 1.5 miliardi |
| **Primo volo** | 5 maggio 1947, Torino-Roma-Catania |
| **Primo aereo** | Fiat G.12 Alcione |

### Timeline Storica

| Anno | Evento |
|------|--------|
| **1946** | Fondazione Alitalia-Aerolinee Italiane Internazionali |
| **1947** | Primo volo (5 maggio), primo volo internazionale Roma-Oslo |
| **1957** | Fusione con LAI → Alitalia-Linee Aeree Italiane |
| **1960** | Flotta jet, crescita olimpiadi Roma |
| **1996** | Fine controllo IRI, inizio privatizzazione |
| **1998** | Unico anno di profitto nella storia |
| **1998** | Aiuti statali €1.5 miliardi (Prodi) |
| **2002** | Aumento capitale €1.4 miliardi (Berlusconi) |
| **2008** | Fallimento, vendita a CAI per €1.05 miliardi |
| **2009** | Air France-KLM acquista 25% (€323M) |
| **2014** | Etihad acquista 49% |
| **2017** | Amministrazione straordinaria, fine supporto Etihad |
| **2021** | Ultimo volo (15 ottobre), nasce ITA Airways |
| **2025** | Lufthansa acquisisce 41% ITA Airways |

## Lezioni del Caso Alitalia

1. **Nazionalismo economico**: protezione del "vettore di bandiera" a costi insostenibili
2. **Moral hazard**: aspettativa di salvataggio statale ha disincentivato gestione efficiente
3. **Interferenza politica**: decisioni aziendali subordinate a logiche elettorali
4. **Ritardo strutturale**: low-cost (Ryanair, easyJet) hanno eroso mercato domestico
